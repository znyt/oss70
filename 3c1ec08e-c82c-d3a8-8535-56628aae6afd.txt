　　此时此刻特等区。
<br/>　　陆勇带着残余部下推到了这里。
<br/>　　实际上他自己都不知道他为什么会来这里，外围之墙都挡不住，何况这特等区不到十米的墙。
<br/>　　每个人都在等死，他们只是听到了许多丧尸的怒吼，他们知道，丧尸和怪物群冲了进来。
<br/>　　这群丧尸要想来到特等区，必须要过河，这条河，足足有五六米的深度，足够淹死许多的丧尸和怪物。
<br/>　　而此时此刻。
<br/>　　码头上。
<br/>　　几百个宪兵队已经开始交起火来了，强大的火力让大量的丧尸犹如割麦子一样的倒下，数千只丧尸死在了他们枪口之下，形成了尸体墙壁，阻挡了后面丧尸和怪物的前进。
<br/>　　哒哒哒哒哒。
<br/>　　无数的子弹犹如暴雨，每个宪兵都疯狂的扣动扳机，这是他们此时此刻唯一能做的事情了。
<br/>　　也就在这个时候。船来了。
<br/>　　顿时几百个宪兵眼睛一亮，全都是收起枪甚至有的人枪械都不要了，就是朝战船冲去。
<br/>　　数百人，没一个人殿后，因为殿后就是死，他们这些人谁都不想死，船就是他们最后的希望，只要上了船他们就有的救。
<br/>　　于是乎数百人全部朝船涌去，而那些原本被他们强大火力压制住的丧尸顿时犹如潮水一样涌来。
<br/>　　一些变异狗变异猫变异老鼠更是速度最快直接朝他们扑了过去，两条腿的人怎么可能快的过四条腿地上爬的变异动物。
<br/>　　顿时其中跑得慢的一群人直接被扑倒在地。
<br/>　　而其余人见到这一幕更是吓得魂魄都没了，一个个此时此刻就恨爹妈少生了两条腿。
<br/>　　而开船的船夫看到这么多丧尸和怪物已经在码头上了，也知道，驱逐之墙已经被攻破了，吓得二话不说掉头就是把船给开走了，他也不是傻子，他也是人，这群宪兵怕死，他也怕死，他知道，特等区才是最后的希望。
<br/>　　“你特么去哪里！”
<br/>　　“草！我们还没上船呢！”
<br/>　　剩下的宪兵都傻了眼了，然而下一刻，丧尸和怪物就冲了过来，把他们一个个扑倒在地。
<br/>　　“你特么给老子回来！老子给你钱！给你黄金！阿阿阿阿阿……”
<br/>　　“救我啊谁来救救我啊。”
<br/>　　“滚开！你们这群丧尸！！”
<br/>　　顿时一道道凄惨的令人起鸡皮疙瘩的惨叫声不断的响起。一个宪兵傻傻的瘫坐在地上，看着一个丧尸在啃他的同僚。
<br/>　　“难道，这就是报应吗？来得这么快？？”
<br/>　　轰
<br/>　　下一刻一个丧尸朝他扑来。
<br/>　　大量的鲜血汇聚进入了这个原本还算很清澈的小河之中，把小河变成了血红色，在河面上还有丧尸和怪物扑腾着，大量被枪杀的平民尸体漂浮在河面之上，随波逐流。
<br/>　　林峰来到自己小区，直接杀了几头变异猫。
<br/>　　然后冲到了自己家。
<br/>　　林峰打开门的一刹那一把刀对准了他。
<br/>　　“别过来。”
<br/>　　然后就是一道尖叫。
<br/>　　是保姆王莉。
<br/>　　“是我。”
<br/>　　林峰一把抓住对方的手，王莉这才回神来，才看到进屋内的是林峰而不是丧尸。
<br/>　　她刚才太紧张了，都忘记了林峰是开门进来的，丧尸和怪物可不会开门。他们若是闻到了活人的气息，那么会破门而入，所谓的防盗门，根本没有任何意义。
<br/>　　“林峰，你回来了。”
<br/>　　林峰点了点头，看了眼他和他的孩子，“你们没事吧。”
<br/>　　“我们没事，外面到底怎么了。乱哄哄的。”王莉害怕的脸色都白了。
<br/>　　“基地被破了，外面都是丧尸，不过你们放心，你们躲在这里还是很安全的，我刚才一路过来，我们小区这里没看到多少丧尸，更不谈楼道里了。就是你们等下拿报纸把窗户都贴上去，窗帘都拉起来，还有一些马桶下水道洗手池的下水口，全部用东西堵起来，免得有变异鸟变异虫子之类的撞碎玻璃，或者顺着排污口进入屋内。”
<br/>　　“嗯。”
<br/>　　王莉点了点头。
<br/>　　“苏云呢。
<br/>　　林峰看了眼屋内问道。
<br/>　　“啊，苏云小姐没回来啊，她这几天都没有回来过。”
<br/>　　王莉摇了摇头道。
<br/>　　没回来？
<br/>　　林峰面色一变，难道苏云老师还在白云镇？
<br/>　　林峰暗道糟糕，他以为苏云回到屋里，搞半天，对方居然还在白云镇！！！
<br/>　　林峰面色难看到了极点，他老龙口被袭击了，那么白云镇肯定也一样。
<br/>　　若是刚才他从老龙口回来第一时间就杀到白云镇，也许还有些希望，但是现在都耽搁这么久了。
<br/>　　“希望苏云老师无碍吧。”
<br/>　　林峰深吸口气，他没有冲动的前往白云镇。
<br/>　　从现在去白云镇。已经根本不及了。说难听点，若是苏云老师很脆弱，没有办法保住自己，那么她现在已经死了。
<br/>　　当机立断，林峰咬着牙按耐住了心中的冲动。
<br/>　　他决定先找到陆勇，看陆勇有没有战死，陆勇不死自然是最乐观的局面了，陆勇在才可以稳定下军心，曙光基地才有救，然后把战门的人给他干了。
<br/>　　这一次，林峰身边的人，陈勾李广外加一个苏云，现在一个个都生不见人死不见尸。这一切都是他们所赐。
<br/>　　这个仇不得不报。
<br/>　　“对了，吴三那个家伙。”
<br/>　　林峰立即跑到新开的店去看吴三，吴三这小子一心一意就想着开店做生意，吃喝拉撒都在店里面。这几天一直忙着店内生意。
<br/>　　林峰来到这里时，这个店已经关了，林峰看样子是从里面锁上来的，看来吴三这小子已经躲起来了。
<br/>　　林峰也不再墨迹，直接朝特等区奔去，在他看来，陆勇若是活着，外围墙壁守不住，那么也只能退到内围墙壁再踞守。
<br/>　　林峰来到船码头，看着码头上一地的鲜血和那些飘在河里面的丧尸和人类尸体，心中已经是充满了悲愤。
<br/>　　这么多人，因为战门，而无缘无故的死去，他们每个人都代表了一个家庭，他们没有死在外面，而是死在了他们自以为的天堂。
<br/>　　林峰拿着战刀，要想进入特等区必须要过这条河，这条河有个五六米宽，小河一条，林峰深吸口气，直接是一口扎进了这条河水之中，顺着这条河，林峰游了足足一刻钟，才是抵达了特等区墙角之下。
