　　“到齐了啊。”钟馗用力拍了拍地板，一个童子模样的少年恭恭敬敬地进来，钟馗让他去请大帝。
<br/>　　“咱们先聊着啊，”钟馗把酒葫芦递出来，“谁喝酒？”
<br/>　　没人理他。
<br/>　　钟馗抓了抓胡须，说道：“你们大家都听说了吧，青衣死了，在轩辕山，元神寂灭。那个，师尊找我们过来，就是讨论这件事吧。”
<br/>　　赵公明端着架子说道：“有什么好讨论的，这叛徒，死不足惜，死在轩辕山，正好省了我们动手。”
<br/>　　钟馗翻着白眼道：“哪有这么简单，要是这么简单，还找我们来讨论什么。”
<br/>　　赵公明哼了一声：“当初我得知他所做的那些，是大帝安排的时候，我就说过，不靠谱，他一点也不靠谱。”
<br/>　　赵公明好歹是客串财神，经常去人间转悠，为了与时俱进，时代性的用语也学了一些。
<br/>　　“你得了吧，你也就会事后诸葛亮。”钟馗不屑理他，暗自叹了口气，转过脸去看崔钰，道：“老崔，青衣之死，你怎么看？”
<br/>　　崔钰平静地反问了一句：“他死了吗？”
<br/>　　“你……你不知道？”
<br/>　　“没见到他的精魄，你怎么知道他死了？”
<br/>　　“这还用见到吗？”钟馗捋着胡子，喃喃道：“是有点奇怪，他既然死了，精魄为何不来报道，却是不知去了什么地方。”
<br/>　　赵公明道：“管这个作甚，来了也是要到轮回司报道，百世畜生道轮回，少不了他的！”
<br/>　　钟馗冷着脸道：“如何处罚他，可不是你说了算吧。”
<br/>　　“你说了算？”
<br/>　　几个超级大佬，抬杠起来跟普通人一样。
<br/>　　“咱们说了都不算，转轮在这呢，让转轮说，对了转轮，地藏菩萨那边怎么说？”
<br/>　　“菩萨并无关照。”转轮王是佛门那边的人，跟他们几个多少有点距离，说话没那么随便。
<br/>　　赵公明插嘴道：“我回头就找大帝请旨，带兵去平了风之谷这帮逆贼！”
<br/>　　一直没开口的阎罗王突然转头瞥了他一眼，道：“不该是去平了北斗观吗？”
<br/>　　“北斗观？”赵公明一愣，“找他们做什么？”
<br/>　　阎罗王端着架子，冷冷说道：“青衣，是我大帝门生，纵然反出阴司，也是我等头号要犯，北斗真人有什么资格杀他！再说，他身上担着太多干系，要拿他来问，他如今死了，很多事便死无对证，如何查起？不找北斗真人，你说找谁？”
<br/>　　赵公明嘿嘿笑道：“凡事抬不过一个理字，道风那是自己擅闯轩辕山，被人诛杀，怪得了谁？”
<br/>　　阎罗王瞥他一眼：“说理？你可知星月奴那厮，从人间掳掠了多少生灵，去了不知道什么地方，致使我那许多案子查无对证，却当如何？再说，她掳掠生灵，便同邪修一般行径，阴司不该管了？”
<br/>　　崔钰道：“不错。”
<br/>　　“这根本是两码事！何况，你说的那些事，你怎知是星月奴所为？”赵公明不服。
<br/>　　“无非是没证据，懒得跟你辩。”
<br/>　　“我也不跟你辩！”赵公明赌气转到一边去，过了一会，又转过来，问阎罗王：“老五，你说的可是当真，那些事，真的是星月奴所为？”
<br/>　　转轮王抢着答道：“不错，这是阴阳司查证的，徐师爷一直盯着，虽无证据，但那个圣灵会，八成就是法术公会掩人耳目的手段。”
<br/>　　赵公明听了，一拳打在身下的蒲团上，怒道：“这也太欺负人了不是，难道他们真想统治人间？”
<br/>　　“狼子野心。”转轮王道。
<br/>　　“一码归一码，道风死有余辜，这个没得辩，但这法术公会，可也真是过分，仗着轩辕山的后台，在人间一直搞事，阴司真的就不管啦？拱手把人间让给他们？”
<br/>　　崔钰道：“有太阴山在，阴司而今危机重重，法术公会也是抓住这点，方才肆无忌惮。”
<br/>　　“我说，咱们就这么忍了？”赵公明逐个看过去，几个大佬都没什么表情，不知道心里在想什么。
<br/>　　钟馗道：“待会记得请旨，若是请不下来，我自己去人间，会会星月奴！”
<br/>　　崔钰道：“即日起，若有轩辕山鬼魂灵魄来投，一律判入畜生道！”
<br/>　　阎罗王道：“好，若是他们抗辩，到我那先打一百杀魂棍！”
<br/>　　正说话着，之前那个童子进来了，手里碰着一张紫色灵符。几个人一看，都站起来，微微躬身，面带肃穆。崔钰离得近，走过去把灵符接了，转过身就走了。
<br/>　　这也是阴司的一种传统，如果是传达什么不那么机密的事，一般都是几个鬼使出面，照本宣读。但太多机密的事，便用这紫符写上，派人送到对方手中。
<br/>　　收到的人可以自己到暗处，把紫符点了，字会浮在空中，顷刻消失。这么说，是为了避免朗读旨意被什么灵物听到，谨慎起见，关键是这种法术只有大帝才会，做不了假。
<br/>　　崔钰凌空一抖，紫符燃烧，放出一道道紫色的光，在空中组成了几行汉子，大伙一起都围上来看。
<br/>　　旨意就写了两件事：
<br/>　　一是要将道风的精魄找到，带回阴司，二是定星月奴为通缉要犯，捉拿她到阴司受审。
<br/>　　不一会工夫，字迹就凭空消失了。
<br/>　　几个大佬互相望去，钟馗道：“还是老师高吧，早就想好了，行嘞，老赵，这抓捕星月奴的责任，非你莫属了。”
<br/>　　酆都大帝一向只传达旨意，至于让谁去干，自然不可能说得太细，再说这几个都是阴司大佬，这种事当然是自己分配，然后各自再去安排人马，指定战略。
<br/>　　赵玄坛脸色有点不自然，道：“且不说如何抓的住她，就算抓得住，轩辕山那边岂能罢休？”
<br/>　　钟馗斜眼笑道：“既然大帝有旨，出事了自然不用你来扛着，你怕什么？”
<br/>　　赵公明道：“你不怕，你去啊。”
<br/>　　“一起吧，你出兵，我出人。”
<br/>　　赵公明这是同意的。
<br/>　　两人商量了一下，觉得这件事得慢慢来，而且得一步到位，要干就直接抓到星月奴，否则千万不能打草惊蛇，毕竟星月奴的背景太深，本人实力也极强。
