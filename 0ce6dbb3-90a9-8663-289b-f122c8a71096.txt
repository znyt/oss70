　　不过在四大尊者之中，开道和明法悟道都比较迟，实力也是四人中稍差的，尽管如此，两位尊者一起动手，再加上地狱犼，以三对一，饶是道风修为登峰造极，也是不可能顶得住。
<br/>　　道风却不慌乱，面对三人的围攻，轻轻一笑，说道：“今日却是没有工夫陪你们了！”
<br/>　　说完，借着三清鬼符摇摇欲坠但还勉强支撑的工夫，挥舞双袖，一时间无数的灵体鬼魂从他两只袖子里飞了出来，鬼哭狼嚎，铺天盖地，朝着两位尊者和地狱犼扑了过去。
<br/>　　两位尊者一下子就怔住了。
<br/>　　面对扑到面前来的一批鬼魂，开道尊者刚要动手灭了，明法尊者喝了一声：“不可！我佛慈悲，这些都是被道风拘禁的鬼魂，岂可乱杀！”
<br/>　　当下取出了乾坤袋，作法之后，凭空产生了一股罡风，裹挟着这些鬼魂，陆续吸了进去。
<br/>　　地狱犼那边，却是没他们这慈悲心肠，两只手一起挥舞，将面前敢于扑过来的鬼魂一概拍碎，一时间精魄漫天飞舞，将视线都阻挡了。
<br/>　　开道和明法看见这一幕，也是知道地狱犼的心性，知道阻拦不了，也只好由他了。
<br/>　　等到这些鬼魂死的死，被抓的抓，总算是被清理干净，这时候，场面之中，已经早就没有了道风的影子……
<br/>　　“嗷……”
<br/>　　地狱犼仰天怒吼，声音震动了整个阴司。
<br/>　　道风已经飞出了枉死城，他走的是南门，南门的话，不用在经过那些府衙的所在，那里头藏龙卧虎，钟馗，黑白无常牛头马面这些都在那边，还有各大司主，都不是好惹的，单个打的话，叶少阳其实谁都不怕的，不过他现在最重要的是逃出阴司，如果逗留下去，他很清楚，就算是十个他，最后也得乖乖伏诛。
<br/>　　出了枉死城，道风也是长出了一口气，刚要飞得远一点，前方原本雾蒙蒙的天空中，突然现出了一抹金光，逐渐绽放开来，形成了一方七品莲台，上面盘膝坐着一个巨大的身躯，是一尊菩萨，一只手拿着念珠，另一只手捏成兰花指，面对道风，一脸的威和悲悯的神情。
<br/>　　道风一见到这尊菩萨，心中立刻一惊，拱手道：“见过地藏菩萨。”
<br/>　　在他面前的这座巨大的神祇，就是地藏菩萨的显影。
<br/>　　佛国在阴司的特派员，法诀寺的老大，他一心超度地府亡魂，很少参与具体的事情，但是就算是酆都大帝和三王一判，在阴司都要卖他的面子。
<br/>　　地藏菩萨金口开启，道：“孽徒，这一路来，杀人放火不易。”
<br/>　　道风笑道：“菩萨错了，我非佛门弟子，这一声孽徒，还是当不起的。”
<br/>　　“一日修佛，便为我徒。”
<br/>　　“我修的只是佛法，并非佛缘。”道风沉吟了一下，道：“菩萨何以拦我去路？”
<br/>　　地藏菩萨双手合十，闭上眼睛，沉吟不语。
<br/>　　道风腾空而起，飞到了跟地藏菩萨的脑袋平行的高度，调整了一下角度，试图从地藏菩萨脑袋边上绕过去，结果地藏菩萨的形象一动，堵在了道风的必经之路上，道风试了几次，都是这样，只好停下来，道：“菩萨不让我走么？”
<br/>　　“放下屠刀，立地成佛。”
<br/>　　道风笑道：“我杀人太多，早就放不下了。菩萨，你再拦我，莫怪我亵渎神威了。”
<br/>　　他早就看出，这根本不是真的地藏菩萨，或者说，不是地藏菩萨的本尊，地藏菩萨的本尊，一向都是在地狱地中，离阴司有很长一段距离，地藏菩萨虽然神通广大，也不能违背天地规则，在这么短的时间内赶过来，此刻出现在自己面前的，只是地藏菩萨的一缕分身显影而已，能够同步传达本尊的声音和行动，但是不具有任何法力。
<br/>　　但是，他毕竟是菩萨，是阴司几位大佬之一，破了他的显影分身，也只莫大的罪过，因此，道风一直没有这么做，现在为了逃走，也不得不发出警告。
<br/>　　地藏菩萨双手合十，闭上眼睛，一句话也没说，意思就是你要是敢这么做，你就去做。
<br/>　　在整个阴司，都不可能找的出敢这么亵渎地藏菩萨的人，不过，道风毕竟不是一般人，他咬了咬牙，手拿打神鞭，朝地藏菩萨飞了过去。
<br/>　　“你若毁我分身，你便是魔。”
<br/>　　“纵然是魔，这一棍我也要砸下去！得罪！”
<br/>　　打神鞭重重地击打在了地藏菩萨的脑袋上，没有遇到任何阻碍，金身缓缓碎裂，然后坍塌……道风头也不回地从中间飞了过去。
<br/>　　砸毁地藏菩萨的金身，不会有什么实质性的害处，不过，道风之后，从此之后，地藏菩萨不会再保佑他，或许……道风猛然想到，地藏菩萨早就算出他会这么做，故意摆出自己的分身让自己砸碎，好断绝跟自己的一切关系。
<br/>　　他是道佛双修，很多人不知道，只有他自己知道，在失踪十年的岁月里，他曾经问道于佛国，修炼过佛门的法术，并且与地藏菩萨有过一段渊源，如今，自己砸碎了地藏菩萨的金身，已经回不了头了……
<br/>　　回不了头，那就越走越远吧。
<br/>　　道风一路疾行，总算离开了阴司的势力范围，来到了荫水河边，望着浑浊的河水，怔怔出神。
<br/>　　徐福从后面走了出来，身后跟着建文帝。
<br/>　　道风就知道他会在安全的地方来找自己，并不意外，仍旧望着河水发呆。
<br/>　　徐福笑道：“道风子，你实力又有精进，离斩三尸证道却是不远了。”
<br/>　　道风冷冷道：“你跑的可真快，若不是我牺牲了血海万魔幡中的阴魂，只怕是逃不出来了。”
<br/>　　徐福道：“我知道你有办法的，事已成，在此谢过。”
<br/>　　“白起何在？”
<br/>　　徐福拿出了一面铜镜，道风转头看去，从铜镜上看到了一道人影，正是曾经不可一世的杀神白起，盘膝而坐。
<br/>　　这是白起的元神，因为无魂无魄，不能见风，只能通过媒介来栖身。
<br/>　　白起睁着眼睛，目光锁定了道风，冷冷端详了一会，哼了一声道：“是你！”
