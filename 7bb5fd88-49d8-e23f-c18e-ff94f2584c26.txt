　　尤天韩四人的速度极快，几乎是三分之一个眨眼的时间，便各自飞逃到了千里之外。
<br/>　　但是，仅仅是飞出了千里距离，他们四人便感受到了强烈的吞噬之力突然降临，瞬间就束缚住了他们的身形。
<br/>　　除了尤天韩能勉强稳住身形之外，其他三人飞速向后倒飞而回。
<br/>　　“怎么回事？”
<br/>　　“怎么会这样？”
<br/>　　“苏莫，你想怎么样？”
<br/>　　几人奋力大吼，个个满脸的惊恐之色，他们依旧没有看到苏莫的身影，但是通过吞噬之力的来源，他们已经知道了苏莫的藏身之地。
<br/>　　这三名青年，修为从武帝境六重到八重不等，面对强大的吞噬之力，全身气血和玄力完全混乱，连战魂都要自行离体了，完全没有任何的反抗之力。
<br/>　　他们惊恐不已，苏莫突然出现，突然对他们出手，让他们几乎吓破了胆。
<br/>　　轰隆！
<br/>　　就在三人惊恐的大吼之间，虚空中突然出现了一个巨大的旋涡，漆黑如墨，森然恐怖。
<br/>　　“不！”
<br/>　　三人只来得及大吼一声，便像被长鲸吸取的海水一般，被漆黑的旋涡一口吞掉了。
<br/>　　而后，漩涡消失不见，但吞噬之力并没有减弱多少。
<br/>　　“这……！”尤天韩见到这一幕，脸色惨白如纸，心中亦是惊恐不已。
<br/>　　以前他不怕苏莫，但自从经过了真圣界一役，他被苏莫一招击成重伤之后，现在说不怕那是假的。
<br/>　　啊！
<br/>　　尤天韩怒吼，拼命的挣扎，虽然能勉强继续飞行，但速度简直可以忽略不计。
<br/>　　“尤天韩，你不是要感谢我吗？”苏莫的声音在尤天韩的前方响起，声音淡漠无比，仿佛不含丝毫的感情。
<br/>　　此刻，他正站在尤天韩身前千丈之外，冷漠的注视着对方。
<br/>　　“苏莫，我们无冤无仇，你想要怎么样？”尤天韩停止了挣扎，心念急转，紧张的问道。
<br/>　　“呵呵，无冤无仇？”苏莫轻笑一声，不置可否，他依旧没有现身。
<br/>　　“不错，就算有点仇恨，上次在真圣界你将我打成重伤，也算是报仇了吧！”尤天韩说道。
<br/>　　“嗯，的确是报仇了！”苏莫点头说道。
<br/>　　“那既然你已经报仇了，就放我离开吧！”尤天韩心中忐忑的说道。
<br/>　　苏莫闻言轻笑一声，戏虐的道：“不好意思，我就是看你不爽，你可以死了！”
<br/>　　言罢，苏莫就要出手抹杀对方。
<br/>　　“杀！”
<br/>　　就在此时，尤天韩却是突然暴起，率先出手了，其手中瞬间出现了一柄战刀，狠狠的向苏莫所在的方向劈了过去。
<br/>　　他知道苏莫不可能放过他，早就有所准备，通过声音他也知道了苏莫藏身在何地，瞬间便出手了。
<br/>　　咻！
<br/>　　耀眼的刀气冲天而起，威势惊天，撕碎长空，裹挟毁灭般的杀机，向苏莫袭杀而去。
<br/>　　这一刀，算是尤天韩的拼命一击了，威力强大，刀势惊人，贯穿天地。
<br/>　　但是，这一击的威力距离他的巅峰一击，差距还是不小。
<br/>　　因为在吞噬之力的笼罩之下，尤天韩能发挥出的战力，只有五六成的样子。
<br/>　　他的战魂更是在体内瑟瑟发抖，战魂之力大大的被削弱了，能发挥出五六的战力，亦是他的极限了。
<br/>　　“螳臂当车！”苏莫不屑一笑，他武帝境六重之时，不用吞噬战魂，都能一招将对方重创，更别说现在已经提升到武帝境八重境界了。
<br/>　　嘭！
<br/>　　脚下一踏，空间炸裂，苏莫的身形显露而出，如同离弦之箭，迎着强大的刀气，直接向尤天韩冲了过去。
<br/>　　他没有直接出手，因为不能将对方的肉身破坏了，否则没法炼成分身了。
<br/>　　苏莫身上光芒闪耀，金刚之体和琉璃宝体等血脉力量催动了起来，他整个人都仿佛化为了一柄金属宝剑。
<br/>　　“碎！”
<br/>　　手臂一挥，袭来的刀气瞬间崩碎，苏莫的身形穿过溃散的气浪，直接冲到了尤天韩的面前。
<br/>　　轰隆隆！
<br/>　　苏莫大手一探，化为了一只巨大的玄力手掌，直接碾碎虚空，向尤天韩抓了过去。
<br/>　　“什么？”尤天韩见此心中顿时大骇，因为苏莫乍一现身，身上的气息便清清楚楚的展露了出来，武帝境八重的修为。
<br/>　　苏莫居然达到了武帝境八重的境界，这让尤天韩的心瞬间沉到了谷底，感受到了死亡的临近。
<br/>　　“给我滚！”
<br/>　　大手即将临身，电光火石之间，尤天韩无法逃避，只能怒吼一声，挥舞手中长刀，奋力一斩。
<br/>　　轰！
<br/>　　战刀斩到了大手之上，不仅没有斩碎大手，还遭到了巨大的反震之力。
<br/>　　噗！
<br/>　　尤天韩全身剧震，口喷鲜血狂喷，连手中的战刀都被崩飞了出去。
<br/>　　而且，由于受到了吞噬之力的束缚，他的身体没有被震退，强大的反震之力全部作用在了他的身上。
<br/>　　啊！
<br/>　　尤天韩口中发出一声惨叫，凄厉无比，但是下一刻，惨叫声便戛然而止。
<br/>　　因为，他的身体已经被苏莫的大手擒拿住了。
<br/>　　“死吧！”苏莫一把抓住了尤天韩，冷喝一声，手中玄力猛吐，直接冲进了对方的识海之中，将对方的战魂逼了出来。
<br/>　　嗡嗡~~
<br/>　　两声刀鸣震荡虚空，只见一金一银两种长刀战魂，从尤天韩的体内浮现而出，锋芒毕露。
<br/>　　两种战魂乍一出现，便想要逃离此地，但是它们根本没有任何逃走的机会。
<br/>　　巨大的黑色漩涡出现，瞬间便将两种战魂吞噬一空。
<br/>　　“搞定！“苏莫微微一笑，随即意念一动，便将尤天韩的肉身收进了玉镯空间之中。
<br/>　　而后，他目光如电，谨慎的扫视四方，发现四周并没有什么人来到此地，便立刻急速离去。
<br/>　　现在，肉身已经有了，除了尤天韩之外，那三人中有一名武帝境八重修为的人，也可以炼制成分身。
<br/>　　武帝境八重的修为，虽然有些低了，实力比较弱，但和他的修为正好相仿，完全可以冒充他，前往皇祖圣朝。
<br/>　　苏莫继续催动了隐身战魂，隐藏身形，飞行了两个时辰之后，来到了一片荒芜的山脉之中。
<br/>　　而后，他身形降落，直接钻进了荒山之中不见了踪迹。
