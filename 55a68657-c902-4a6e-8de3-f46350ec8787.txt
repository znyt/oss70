　　苏莫离开了金銮大殿之后，便径直离开了皇宫。
<br/>　　走在皇城宽广的街道上，苏莫一步数百丈，仿佛似一阵狂风刮过。
<br/>　　不多时，他便离开了皇城，来到了千里之外，青俞岛上的另一座城池。
<br/>　　随后，苏莫仿佛没事人一般，在城中闲逛了起来。
<br/>　　不过，他虽然远离了皇宫，但他的灵识却是一直牢牢的监视着皇宫中的屈覃国师。
<br/>　　他总感觉屈覃有些不对劲，但到底哪里不对劲，又完全说不上来。
<br/>　　当然，也可能是他多想了。
<br/>　　不过，为了追查黄泉魔宗的下落，苏莫还是要验证一番。
<br/>　　屈覃若真的是黄泉魔宗之人，想必此刻必然坐不住了，就算不知道他的身份，也能猜出来他是苍穹神宫的人。
<br/>　　如此的话，他离开了皇城，对方必定会有所动作，说不定会逃离皇宫。
<br/>　　若是对方逃跑，那就说明对方心中有鬼，必然就是黄泉魔宗之人。
<br/>　　苏莫在城中闲逛，不时的询问一名名武者，关于青俞帝国屈覃国师的情况。
<br/>　　屈覃作为青俞帝国的国师，整个青俞岛上的人，基本上都认识他。
<br/>　　随着询问，苏莫得到的消息，基本上都和屈覃所说相差无几。
<br/>　　看来屈覃并未说谎，对方的确在青俞帝国做了四十余年的国师，很可能的确和黄泉魔宗没有关系。
<br/>　　“嗯？”
<br/>　　正在闲逛，突然苏莫神情一动，随即无奈的摇了摇头。
<br/>　　他发现，居然有人在暗中监视他，而且还不止一人。
<br/>　　苏莫灵识一探，监视他的有三人，正是青俞皇室的三名强者。
<br/>　　不过，这三人只有武皇境一重的修为，即便再小心谨慎，也逃不过他的感知。
<br/>　　对于青俞皇室的打算，苏莫心中非常的清楚，他杀了汪晟，青俞皇室之人肯定要知道他的动向，好向北海冰宫交代。
<br/>　　无奈的摇了摇头，苏莫没有理会，毕竟人是他杀的，若是北海冰宫真的将怒火洒在青俞皇室的头上，倒是他害了别人。
<br/>　　闲逛了一个多时辰，苏莫了解了一些屈覃的情况之后，索性找了一家客栈住了下来。
<br/>　　客栈房间中，苏莫盘膝而坐，一边闭目修炼，一边用灵识监视着屈覃的情况。
<br/>　　从他离开青俞皇宫之后，屈覃一直在金銮大殿中没有离开，皇帝青谷和青雅公主，亦是没有离开。
<br/>　　不过，皇室的强者却是离开了三人。
<br/>　　而此刻，这三人正在他居住的客栈附近，监视着他的动向。
<br/>　　时间缓缓流逝，苏莫静静的修炼，约莫又过了两个时辰之后，他猛然睁开了双眼。
<br/>　　“原来如此！”
<br/>　　苏莫的嘴角，陡然勾起了一丝冷笑，他终于知道屈覃哪里不对了！
<br/>　　是镇定！
<br/>　　正常情况下，若是屈覃真的只是青俞帝国的国师，遇到他这般询问调查，即便实力不如他，不敢反抗，也会恼怒吧！
<br/>　　最起码，对方也会询问他为何调查自己吧！
<br/>　　可之前的情况，对方一直都很镇定，他所有的问题，对方都老老实实的回答了。
<br/>　　这一切，都好像是计划好的一般，对方没有任何慌乱之处。
<br/>　　这镇定，就是破绽！
<br/>　　“果然是黄泉魔宗之人！”
<br/>　　苏莫眸中冷光一闪，随即长身而起，一步跨出，直接冲碎了房间的木窗，飞出了客栈。
<br/>　　而后，苏莫身形如电，急速向青俞皇宫飞去。
<br/>　　咻！
<br/>　　苏莫的速度极快，整个人如同一道剑光，片刻间便飞进了青俞皇宫，直接冲进了金銮大殿之中。
<br/>　　皇宫中的那些守卫别说阻拦了，甚至都看不见苏莫的身影，只感觉到一阵风突然刮过。
<br/>　　金銮大殿中，皇帝青谷和一众皇室的强者，本来正在静等北海冰宫之人的到来。
<br/>　　他们已经时刻监视着苏莫的动向，知道苏莫栖身之地，只要北海冰宫的强者来临，他们告知实际情况，让苏莫来承受北海冰宫的怒火便可。
<br/>　　但就在他们焦急的等待之时，空间一阵波动，苏莫的身形瞬间出现在了大殿中。
<br/>　　“小……小友？”青谷见到突然到来的苏莫，顿时神情一怔。
<br/>　　其他人亦是一脸愕然，不明白苏莫为何又突然回来了！
<br/>　　“过来！”
<br/>　　身形落在大殿中，苏莫二话没说，直接向屈覃国师出手了，大手凌空一抓。
<br/>　　“什么？”众人见此顿时心中一惊，不明白苏莫发什么疯，为何突然对屈国师发难？
<br/>　　屈覃亦是神情微怔，眸中闪过一丝慌乱。
<br/>　　不过，苏莫出手，岂容他人反抗，虚幻的玄力大手瞬息而至，一把便将屈覃擒拿。
<br/>　　屈覃不过是武皇境二重的修为，和青谷相当，哪里能抵挡苏莫的擒拿，就算躲避都来不急。
<br/>　　唰！
<br/>　　一把将屈覃擒回身边，苏莫面色冰寒，眸光直视对方的双眼。
<br/>　　“告诉我，黄泉魔宗之人都在哪里？我饶你不死！”苏莫冷冷的问道。
<br/>　　屈覃闻言面色微变，他正欲开口，异变陡生。
<br/>　　就在此时，蓦地，一阵浩浩荡荡的皇者威压，突然从天而降，覆盖了整座皇宫，霎时之间，无边的寒气如同一阵狂风刮过。
<br/>　　整个皇宫中的温度，瞬间降到了冰点，空间仿佛都被冰冻住了。
<br/>　　“终于来了！”
<br/>　　“北海冰宫的强者来了！”
<br/>　　“好强大！”
<br/>　　一众皇室的强者惊呼出声，个个震惊的站了起来，皇帝青谷、青雅公主等人，面上均是充满了凝重。
<br/>　　轰！
<br/>　　不待众人有所动作，金銮大殿的屋顶突然发出一声爆响，而后直接被掀飞了出去。
<br/>　　随即，众人便见到天穹之上，伫立着十数道身影。
<br/>　　这十数道身影有男有女，除了其中一人之外，其他人尽皆是武皇强者。
<br/>　　为首的一人，是一位花甲老者，此人身穿灰色长袍，身形略显臃肿，脸盘宽大，下颌还留着一缕短须。
<br/>　　而此刻，这位老者的脸庞上，充满了戾气，眸中更是杀机森然。
<br/>　　其身上逸散的气息，冰寒刺骨，让的其周身的空间都被冰封了
<br/>　　而那位唯一不是武皇的强者，是一名黄衣青年，正是之前汪晟的同伴，后来逃走之人。
<br/>　　“三长老，就是他！就是他杀了汪师兄！”金銮殿的屋顶被掀飞，黄衣青年一眼便看到了殿中的苏莫，顿时大声疾呼了起来。
<br/>　　“小畜生，你是自己自缢，还是要老夫出手？”
<br/>　　听闻黄衣青年之言，花甲老者的目光瞬间落在了苏莫身上，厉声暴喝。
