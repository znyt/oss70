　　第307章
<br/>　　清点货物，露宿扎营所需要的装备，队伍人数控制在五十人左右，每人身上携带十天干粮。程大雷在做队伍出发前的最后准备。
<br/>　　秦蛮发现一件事，当樊梨花出现后，大当家像变个人似的，惜字如金，言简意赅，这对平常嘴有些碎的大当家来说可是很罕见的事情。相比之下，樊梨花却活跃得紧，面带春风，谈笑风声。
<br/>　　大家的眼睛是雪亮的，这件事不仅秦蛮发现了，如云中龙、乔鹤、游九楼也不是瞎子。至于究竟为何，诸人也没有去细想。
<br/>　　队伍清点完毕，一切准备就绪，程大雷大手一挥：“出发。”
<br/>　　接下来的路崎岖难走，离了落玉寨一切都是山路。山路蜿蜒曲折，只能用一匹马紧贴着山壁前进，稍有闪失，就会坠崖丧生。中间还有通过一段狭窄的索桥，云桥是用两根铁索拉着，中间铺上木板，下面就是万丈深渊，人走上去都是心惊胆战，何况是马匹。
<br/>　　程大雷终于明白为何这条路没人走了，自然，如果轻易就可以走通，今天也轮不到程大雷出马了。不得不说，因为这条索道，程大雷把这次的任务难度从困难模式调到地狱模式。
<br/>　　事情没有自己想的那么简单。
<br/>　　最后是福德勒想出个主意，把货物卸下来，由人背过去，然后再用黑布蒙住马匹的眼睛，由人牵着通过。
<br/>　　福德勒在山寨一直跟着李行哉混，堪称形影不离。这次是因为要和戎族做生意，所以特意将他带在身边，程大雷不是很喜欢和福德勒接触，这个人寡言少语，好像又很怕自己。但这次多亏将他带上，想出这样一个主意，不然的话，单单是因为眼前这条索桥，大家就要打道回府。
<br/>　　事情进行下去，是个很慢的过程，足足用了一天时间，才把人和货都送到索桥对面。等完成这一切，天已经黑下来，夜里走上路，几乎相当于送死。
<br/>　　大家在山间找了个平地扎营，将马匹围在靠山壁一侧，人守在外面。大家用凉水冲开干粮裹腹，程大雷已经决定今晚亲自守夜，既要防备山间的猛兽，也要防备夜里有人睡晕了头跌入悬崖。
<br/>　　这不是什么好差事，程大雷觉得，这条路根本无法撑起日后的贸易，想要支撑日后源源不断的货物来往，必须开辟一条新的商路。
<br/>　　夜里，围着篝火，程大雷把秦蛮、云中龙、乔鹤、游九楼几个交到身边，问他们关外的事情，当然，还包括樊梨花。
<br/>　　“也不是没有别的路啦。”樊梨花道：“上次西颜部的事情记得么，他们把守着一道山谷，路要比这里好走许多倍。”
<br/>　　樊梨花说完没人吭声，大家目光都看着程大雷，等着他说话。程大雷表情有些不自然，轻咳一声道：“为什么我们不从他们那里过？”
<br/>　　“因为那里有西颜部守着，西颜部兵强马壮，又残忍好杀，那条路走不通的。”樊梨花很自然的回答道。
<br/>　　程大雷心里寻思着，今天是第一次，本就是一个不停试错的过程。若有下次，还是应当走西颜部那条路。
<br/>　　其实，除开西颜部还有一条路，就是莫鸣米把守的琴川关，但朝廷不许和戎族做生意，这条路自然是走不通的。除非，程大雷能够取莫鸣米而代之，程大雷不是没有这个打算，但现在还真没有和莫鸣米硬碰硬的资格。
<br/>　　除非能开出三级山寨。
<br/>　　可开三级山寨需要钱，程大雷现在做的，就是挣钱的事情。
<br/>　　夜来，露水渐重，程大雷又问了一些周围的情况，诸人一一散去。
<br/>　　云中龙三人聚在帐篷内，也没有睡，他们被绑在程大雷这条船上，至于前途如何，其实都看不清楚，难免有几分迷茫。
<br/>　　“你说他真能走通这条路么？”
<br/>　　“路这么难走，这还是小事，关键是过了山头，如何和戎族接触。上次莫老大的事情，现在戎族看到帝国的商队，就抢就杀……”
<br/>　　“我倒是觉得，他或许真能做到。”云中龙缓缓道：“想想最近的事情，刚开始谁以为他能在这里站稳脚跟，现在他不仅站稳了，而且还能让莫老大低头。”
<br/>　　乔鹤和游九楼听着，缓缓点点头。
<br/>　　“如果我们跟着莫老大，虽然不至于饿死，但无非就是这样的日子。可这个人若真有本事，日后我们的前程如何，谁也说不好。我可听说，南边已经有人举起了反旗……”
<br/>　　现在三人都惶惶无主，云中龙这话倒是成了诸人的定心丸，等二人睡熟后，云中龙悄悄离开帐篷。
<br/>　　程大雷捧着大斧头，坐在悬崖上，云中龙悄悄过来。
<br/>　　“程当家，话都给他们说了。”
<br/>　　“嗯，你也回去歇着吧，等这件事成了，我少不了你的好处。”
<br/>　　“是，是，以后我唯程当家马首是瞻。”
<br/>　　乔鹤和游九楼都不知道，云中龙的脚步早站到程大雷这边了，一荣俱荣，一损俱损。而程大雷心里也明白，只要自己能让他们吃饱饭，忠心是很容易获得的。
<br/>　　程大雷今日亲自守夜，今夜能否平安过去，他心里也有些打鼓。
<br/>　　忽然听得身后有脚步声，程大雷还没来得及回头，就察觉一个温软的身子坐在自己旁边。
<br/>　　程大雷的心脏开始打鼓，草原女儿，敢爱敢恨，这句话浅白理解就是，爱你是面对面看着还想你，恨你时巴不得杀你全家。
<br/>　　现在程大雷脚下就是万丈悬崖，樊梨花稍微伸伸手，他就得跌死在悬崖下。
<br/>　　没理由两次坠崖不死呀。
<br/>　　“嗨，樊家姑娘还没睡呐？”程大雷不得不开口，他可担心樊梨花会推他坠崖。
<br/>　　“睡不着，过来看看你。”
<br/>　　樊梨花随口说了一句，程大雷感觉她的手搁在自己背心，一刹那白毛汗都冒了出来。
<br/>　　樊梨花表情自然，手里拿着个小包袱，道：“我给你做了双靴子，你看看喜不喜欢？”
<br/>　　程大雷看着面前的靴子，针脚杂乱，线头四外冒。
<br/>　　“喜欢，喜欢，怎么能不喜欢呢。”程大雷哆哆嗦嗦：“怎么敢不喜欢。”
<br/>(本章完)
