夏夕绾话音落下，夏妍妍的瞳仁倏然收缩放大，里面露出了不安和惊恐。

<br/>        她的右眼皮一直在跳，心里有不好的预感，果然，这一切是计，她中计了，她妈还被当场捉获了。
<br/>
<br/>        夏妍妍阖了阖唇，发现自己已经说不出话来了，很久她才找回自己的声音，“那…那又如何，我什么都不知道，这一切都是我妈做的，我毫不知情，我甚至还不知道她做了些什么。”
<br/>
<br/>        夏夕绾看着她强装镇定的样子，“夏妍妍，你嘴巴还挺严的，不过你妈已经招了，她说这一切都是你主使的。”
<br/>
<br/>        “不可能！”夏妍妍迅速摇头，“我妈不可能这样说的，我知道了，夏夕绾，你在骗我！”

<br/>        夏夕绾伸手，一把拽住了夏妍妍的衣领，让她看着草坪上那些警察，“我骗你，这些警察还能骗你？夏妍妍，你妈都招了，再加上我手上这份监控视频，完全可以给你定罪了，你的下半辈子就在监狱里度过吧！”
<br/>
<br/>        不！
<br/>
<br/>        她不要！
<br/>
<br/>        她不要进监狱！
<br/>
<br/>        夏妍妍想将夏夕绾给推开，“我不信你说的，一个字都不信，让他们将我带走啊，我什么都不会说，我要找我的律师！”
<br/>
<br/>        夏夕绾冷笑，她拽着夏妍妍的衣领一路将她往别墅里面扯，动作近乎野蛮粗暴，直接将夏妍妍往上扯。
<br/>
<br/>        夏妍妍挣扎，但是夏夕绾不知道哪来的力气，她根本挣扎不了，她被跌跌撞撞的扯了一路，站稳身体的时候已经到了楼梯上面。
<br/>
<br/>        “夏妍妍，还记得这个位置吗，十一年前你将我约到了这里，将我弄晕之后亲手将爷爷推下了楼梯，你好好看看，爷爷就躺在那里，倒在一片血泊了，他在看着你啊，爷爷在看着你！”
<br/>
<br/>        夏妍妍往下看，这里的楼梯很高，她站在这里的时候就有一种随时会摔下去的恐惧感。
<br/>
<br/>        记忆回到了十一年前，当时她伸手将老爷子推了下去，老爷子拉了她一把，她差点就跟着一起摔了下去。
<br/>
<br/>        后来老爷子倒在了血泊里，当时他还睁着眼，那浑浊的双目就那样冰冷而狠厉的盯着她，将她盯得浑身发毛。
<br/>
<br/>        当时她也才十几岁大，还是一个孩子，直接瘫在了地上，恐惧的大口大口呼吸。
<br/>
<br/>        “放开我！夏夕绾，你快点放开我！”夏妍妍崩溃的尖叫，她好像又看到了老爷子那双眼睛，那双眼睛犹如一条毒蛇般悄然盘上了她的脖子，让她骨头生寒。
<br/>
<br/>        夏夕绾用力的钳制住夏妍妍，将她看着这个楼梯，甚至她还用力一推，将夏妍妍半个身体都推了出去，“夏妍妍，这么多年你都生活在这里，每天都会在这个楼梯上走来走去，难道你就没有做过噩梦吗，爷爷回来找你了！”
