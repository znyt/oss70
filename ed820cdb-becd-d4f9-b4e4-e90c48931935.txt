　　第214章
<br/>　　人群一一从考场走出，太监恭恭敬敬捧着盒子离开，里面封着各自的考卷。这些将会被送到宫中，据说今夜将由明帝亲自阅卷。
<br/>　　这一方面说明明帝也有勤政的时候，另一方面也说明他无人可用。
<br/>　　“喂，你答得怎样？”
<br/>　　“太难了，太难了，我根本不知道该从何处回答。”
<br/>　　“是呐，是呐，可答的太多，反而最难破题。”
<br/>　　周围人议论着刚才的题目，大多摇头叹息，程大雷鄙夷的看着这一幕：最讨厌你们这些虚伪的渣渣，头摇得最狠的，往往也是成绩最好的，自己绝对不能成为他们这样的人。
<br/>　　“牛校尉。”百里胜突然出现在程大雷面前，道：“不知牛校尉答得怎样？”
<br/>　　程大雷想到刚才心中的鄙夷，立刻昂起头道：“很好。”
<br/>　　“呃……那牛校尉觉得刚才的题目？”
<br/>　　“很简单啊，一点难度都没有。”
<br/>　　“呃……牛校尉还真是，还真是……”
<br/>　　还真是特么的不好让人接话啊，本来百里胜想要表达一下长官对下官的关怀，类似于答得不好没什么，以后你就是我的小弟，我让你送死的时候可别犹豫。
<br/>　　可没想到程大雷的回答这么耿直，看来他不仅不懂官场规矩，脑中的自我认知也差一些。
<br/>　　“不知牛校尉是从何处破题？”
<br/>　　程大雷挺直腰杆，缓缓道出三个字：“军之魂。”
<br/>　　这里的谈话已经吸引起很多人注意，身在官场，无处不是眼睛，你随地吐痰都会被所有人知道。
<br/>　　大家对程大雷的回答不屑一顾，哼，看这个人的样子，怕是认字都够呛吧，还真是不知道天高地厚。
<br/>　　“不知牛校尉估计自己可以获得怎样名次？”宋天函问。
<br/>　　“满分。”程大雷顿了顿：“喔，你问名次呐，第一吧。”
<br/>　　呃……
<br/>　　所有人自动和程大雷隔开一定距离，此人有毒，距离他太近会被拉低智商的。
<br/>　　到晚上的时候，关于程大雷今日的言行已经传开，无他，对于他这种充满槽点的人物，大家都是不吐不快。
<br/>　　“哈哈哈。”
<br/>　　将军府内，传来洪亮的笑声，一个身穿便服的老者拍案大笑。
<br/>　　帝国有很多将军，就连程大雷也曾经被封为游骑将军。而当大家只提及将军，不加任何前缀后缀时，往往只代表一个名字——大武天策大将尉迟离。
<br/>　　“却是没想到这牛三斤倒是替我出了口气。”
<br/>　　“义父你没有看到，当时宋天函的脸都绿了。他后来还说这次考核，他必是第一。”百里胜。
<br/>　　“呃……”就连尉迟离也顿了顿：“该不太可能吧，这若是结果出来，我们还不成为相府的笑柄。”
<br/>　　“义父不用担忧，他现在还不是我们的人。”
<br/>　　“嗯，和这个人还是不要走得太近，该用要用，却要和他保持距离。”
<br/>　　一般人常常会被尉迟离的外表欺骗，以为他是个心肠耿直的武夫，可能掌一国军权，又怎会是个耿直汉子。
<br/>　　“这次武举，我和相府那家伙互为主考官，但事情还是要你们下面的人做。关乎帝国军权的事情，决不能让相府的人再插一杠子，所以最后武状元必须是我们的人。”
<br/>　　“义父，已经有人选了？”
<br/>　　尉迟离口中道出三个字：“柳轻名。”
<br/>　　百里胜点点头：“原也该是他。”
<br/>　　尉迟离递给百里胜一张名单：“这是我们的人，你看一遍，把名单烧掉，这些人必须在今次武科举中有一席之地，最后的状元必须是柳轻名。”
<br/>　　同一时间，相府内，一个留着两撇胡须的老者，把一份名单交给宋天函。
<br/>　　“这上面都是我们的人啦，你知道该怎么办，而这次的状元，必须是他。”
<br/>　　宋天函看着老者手指的位置，那是一个人的名字——屈九江。
<br/>　　“下官明白。”
<br/>　　“这次的事情怕是不好做，那武夫也会想法子的，你要小心，我们的对手并不弱。”
<br/>　　程大雷晚上住的地方是百里胜安排的，是一个临街的院落。过程中，百里胜还不忘展现自己的人格魅力。
<br/>　　“牛校尉在长安还没有住的地方吧，呵呵，长安物贵居大不易，我恰好有个宅子闲着……”
<br/>　　程大雷不知要不要告诉对方，其实我在长安有十几套宅子闲置着呢。从前生那样变态的市场过来，所以程大雷对房产有某种偏执就可以理解了。于是他在长安这大半年，很是给自己添置了一些房产，对于程大雷来说，反正价格也不贵。
<br/>　　不过如今他是『朝廷命官』，于是也只能悉听尊便。
<br/>　　咚咚咚！
<br/>　　门外响起敲门的声音，程大雷打开院门，女扮男装的李婉儿溜了进来。
<br/>　　“李董，你真是翘家有瘾呐，也不怕被人发现。”
<br/>　　“我这次来是重要的事情，还记得我给你的任务么。”
<br/>　　“当然，时刻铭记于心。”程大雷道：“不就是把你看不上的人淘汰掉，扶持你相中的人上位，怎样你有人选了？”
<br/>　　“当然，我喜欢谁我不知道，我不喜欢谁我已经很清楚了。”李婉儿认真道：“有两个人你一定要帮我淘汰掉。”
<br/>　　“那两个？”
<br/>　　“柳轻名，屈九江。”李婉儿随口道。
<br/>　　“保证完成任务。”程大雷昂首挺胸，然后又补充一句：“不会有什么难度吧？”
<br/>　　“能有什么难度，一个娘娘腔，一个死疙瘩，一点实力都没有的，说不定你不做手脚，他们就被淘汰掉了。”
<br/>　　“呵呵，那就发挥不出我的作用了。”程大雷笑着。
<br/>　　武举在即，以将军府和相府为代表的两大集团为了瓜分蛋糕，已经站上擂台，准备刺刀见血，拳拳带肉。
<br/>　　而他们并不知道，这场战斗并不是他们以为的一对一单挑，不知何时，一个小人物也站上了擂台，举目四望，信心百倍。
<br/>　　关键是战上擂台的程大雷也并不明白自己的对手是什么段位，他还以为这是一场单机收割游戏。
<br/>　　“喂，我让你帮忙查我六哥的消息，你查到没有？”李婉儿。
<br/>　　“我连长安都出不了，去哪里查呀。”程大雷随口道了一句：“大概死了吧。”
<br/>　　
<br/>(本章完)
