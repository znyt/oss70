        诺筱颖不禁怀疑，自己到底是不是这个老妈亲生的？
<br/>
<br/>        殊不知，诺筱颖的这种怀疑还真是怀疑对了。
<br/>
<br/>        吃过晚饭后，两家人坐在茶几前，面对面地谈话。
<br/>
<br/>        而话题的中心点，就是她诺筱颖和童梦萌。
<br/>
<br/>        表妹童梦萌是个高材生，离本科大学毕业还有三年，也急需学费。
<br/>
<br/>        童爸和童妈原来经营着一家糖果厂，后来不知道是什么原因，两个星期前突然破产了，而且还负债累累。
<br/>
<br/>        所以综上所述，童家如今基本上是一穷二白。
<br/>
<br/>        诺筱颖也是现在才知道，小姨妈和小姨父在家里已经住了一个多星期了。
<br/>
<br/>        小姨妈和小姨父曾经有钱，诺母还算很客气地待见，如今没钱了还负债累累，诺母便十分嫌弃起来。
<br/>
<br/>        这人多口杂，两家人才相处一个多星期，矛盾便接肘而来。
<br/>
<br/>        更何况，她诺筱颖的老妈还是个“尖酸刻薄”的人。其实，这么说她老妈真不对，但是她老妈确实是那性子。
<br/>
<br/>        也就在这个时候，诺母对大家说出了一个“惊雷”般的秘密。
<br/>
<br/>        而这个秘密，犹如狗血般泼在了诺筱颖的脸上。
<br/>
<br/>        “梦萌才是我的亲生女儿！”诺母一脸严肃，十分认真地宣布。
<br/>
<br/>        十八年前，两个亲姐妹同年同月同日同时在同一个产房内生产。
<br/>
<br/>        早就得知彼此宝宝都是女娃的诺母，花钱买通了护士，趁着自己的妹妹因为生产疲惫而昏睡过去时，将自己亲妹妹的女儿和自己的亲生女儿给换了。
<br/>
<br/>        所以，她诺筱颖本是凤凰，就这么狗血地成为了麻雀。
<br/>
<br/>        成为“麻雀”也就算了，她努力工作，为自己打拼，却没想到中途还惨遭诺母嫌弃。
<br/>
<br/>        为了要换女儿这事，两家父母都是彻夜难眠。
<br/>
<br/>        最终，换女儿这事双方父母终于全部点头了。
<br/>
<br/>        诺筱颖苦笑，她终于明白，从小到大，诺母为什么只疼表妹梦萌了。原来，梦萌才是诺母亲生的，而她不是，所以不得宠。
<br/>
<br/>        再看看如今落魄的小姨妈和小姨父，虽是自己的亲生父母，但诺筱颖并不嫌弃他俩。
<br/>
<br/>        只是，当诺筱颖刚想改口叫一声爸妈，谁知他们俩的脸却冷了。
<br/>
<br/>        童爸童妈的脸不仅仅只是冷了，而且还憔悴了。
<br/>
<br/>        甚至，诺筱颖还隐约听到这对夫妻心碎的声音。
<br/>
<br/>        诺筱颖看得出，他们并不想把表妹梦萌跟她换过来。
<br/>
<br/>        梦萌毕竟是高材生，大学一本，前途无量，她诺筱颖，只不过是一个三流大学毕业的大专生而已。
<br/>
<br/>        既然女儿已经换回来了，诺母便开始对童家下逐客令了。
<br/>
<br/>        表妹梦萌虽然开口替他们求情过，但是诺母认为，童家住这里，开销太大，更何况，童家还欠了高利贷。
<br/>
<br/>        哪天债主要是找上门来，他们梁家岂不是也跟着遭殃。
<br/>
<br/>        表妹梦萌面对曾经疼爱她的养父养母被自己的亲生母亲逐出门外也显得很无奈，她还在念大学，压根就没钱抚养他们，只能眼睁睁地看着生母将童爸和童妈赶出门外。
<br/>
<br/>        诺筱颖实在是看不下去了，不忍心童爸童妈，也就是自己的亲生父母露宿街头，索性也跟着他们一起离开梁家。
<br/>
<br/>        她跟童爸童妈离开梁家的时候，曾经的“老妈”只对她冷冷地说了一句：“筱颖，别怪老妈对你太狠心，这都是生活所迫。”
<br/>
<br/>        诺筱颖听着，对“老妈”的话，不屑地笑了笑。
<br/>
<br/>        被喊回老家，结果换了一对爸妈。
<br/>
<br/>        诺筱颖替童爸童妈整理着行李箱，希望他们跟她一起去滨江城，但是被他们拒绝了。
<br/>
<br/>        这里毕竟是他们的根，扎的太深，而舍不得离开。
<br/>
<br/>        但是，如果不离开这里，童爸童妈又负债累累，诺筱颖也不知道自己该怎么办。
<br/>
<br/>        看着这“一家三口”像是被逼得走投无路，诺司峻还是有些心疼诺筱颖，他们走之前，他主动出来送他们。
<br/>
<br/>        “哥……”
<br/>
<br/>        “筱颖，不如，你和你那个男朋友，早点结婚吧！”诺司峻接着说道。
<br/>
<br/>        诺筱颖怔了怔，有些不知所以然地看着诺司峻。
<br/>
<br/>        诺司峻接着说道：“小姨和小姨父欠了三十万的外债，你倘若回滨江城，丢下他们不管，他们肯定想死的心都有。所以，你干脆和你那个男朋友早点结婚。有礼金，你们就有……”
<br/>
<br/>        “哥，我……其实没男朋友。”诺筱颖一脸窘迫地回答，那次只是一个误会而已。
<br/>
<br/>        “你没男朋友！”诺司峻顿时错愕了。
<br/>
<br/>        诺筱颖重重地点了点头。
<br/>
<br/>        “那……哎……算了！等我回滨江城了，我就给你物色相亲对象吧！反正，我在婚恋网站当主编。”诺司峻接着说道。
<br/>
<br/>        而童爸童妈对此事，却不吭声。
<br/>
<br/>        “好，谢谢哥哥。”诺筱颖苦笑，心里虽然不接受，但也不想枉费哥哥的一片好心。
<br/>
<br/>        跟哥哥诺司峻告别后，诺筱颖在镇上的宾馆里开了一间房让童爸童妈暂时住下，明天她再去找房子长租。
<br/>
<br/>        这是诺筱颖第一次有种被逼得走投无路的感觉，她的银行卡上仅仅只有五千块钱了。
<br/>
<br/>        她没想到，自己的人生会这么悲催。更令她感到悲催的事情，又接踵而来。
<br/>
<br/>        那个放高利贷的男人，带着三四个手下，竟然找到宾馆来了。
<br/>
<br/>        诺筱颖只是出去给童爸童妈买晚饭，两手提着盒饭，走到房门口。
<br/>
<br/>        当她看到一群流氓打扮的男人围着童爸童妈，逼着他俩当众下跪，顿时她的心都痛了。
<br/>
<br/>        “还不起钱，那就剁了手指！你们夫妻俩，是想剁哪只？”为首的男人耀武淫威。
<br/>
<br/>        诺筱颖放下手里的盒饭袋子，直接走进去，喊道：“你们都给我住手！”
<br/>
<br/>        “你是谁？”为首的男人回头看向诺筱颖，不耐烦地问。
<br/>
<br/>        童妈为了保护诺筱颖，立马抢声说道：“她是我侄女，只是过来看看我们而已。你别……”
<br/>
<br/>        但童妈话还未说完，为首的那个男人便已经色眯眯地朝诺筱颖走了过去。
<br/>
<br/>        诺筱颖害怕地咽了咽口水，脚步不听使唤地后退。
<br/>
<br/>        男人逼近诺筱颖后，忽然问道：“你是不是诺筱颖啊？”
