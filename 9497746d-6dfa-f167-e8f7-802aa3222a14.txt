        &lt;h3&gt;第三卷第二百八十五章  一切正常运转&lt;/h3&gt;
<br/>
<br/>        宁雅告别了凌辰踏上了飞往北京的归途。
<br/>
<br/>        陈子昂已经将耗材推向市场的日期提前了。他必须返回北京。明天约了相关部门的人员，商谈后期的合作情况、一些商务要求，还有官方的宣传问题。
<br/>
<br/>        她的装修工程已经进入尾声，晾上两个月，人员到齐后就可以就可以启动她自己的事业。
<br/>
<br/>        这一次到杭州，她失去了，也获得了，人生安能两全？上帝为你关上一扇门也会为打开另外的一扇窗。
<br/>
<br/>        宁家放弃了她，她却收获了爱情，凌辰对她的心她已经感觉到了，后期一起共同面对，无论前路如何，他们会携手前进的。
<br/>
<br/>        给陈子昂发了邮件，陈子昂就回复了“辛苦了”三个字，一切都在不言中。
<br/>
<br/>        也许，陈子昂早就料到了结局？
<br/>
<br/>        见到王律师，宁雅问王律师：“签合同的时候可顺利？”
<br/>
<br/>        “签合同的时候比较顺利，都是商务签的，我没有见到你妹妹。”王律师全程见到了宁玥和宁雅的来来往往。
<br/>
<br/>        宁雅不语，宁玥下午没有去签合同，那肯定是和她父亲在一起，有可能是陪着凌思永？
<br/>
<br/>        她不知道宁玥爱凌辰爱到什么地步，但是宁玥那种做作让她感觉心寒，从小到大，宁家好的资源都会是宁玥的，只要是宁雅看上的，宁玥都会要，而兰乔也会告诉她，你是老大，你得让着弟弟妹妹，后来宁雅才知道她并非兰乔所生。
<br/>
<br/>        她努力的学习，努力的成长，她以为她足够优秀，他们会对她另眼相待，谁知道，最终她都融入不到宁家。
<br/>
<br/>        她庆幸阴差阳错中进了陈子昂的公司，认识了陈子昂，让她感觉靠别人不如靠自己，乘着自己年轻有能力还可以重新过另外的一种生活，她已经从那种悲观低落的情绪里走出来了，经过这一次，宁雅相信，以后的她也会是无坚不摧的。
<br/>
<br/>        下了飞机，打开手机就收到了凌辰的微信：“雅儿，相信你的凌辰一定会将事情处理好，目前凌老头能够和我站在同一条线上。”
<br/>
<br/>        “雅儿，你到北京了吗，刚下飞机吧，我的世界与你同步，我的心情与你同步，我与你一起神游到了北京。”
<br/>
<br/>        宁雅看着凌辰的信息不由笑出了声，这个人是越来越没脸没皮了。
<br/>
<br/>        她准备给凌辰发个信息，想了想还是给凌辰打了个电话，告诉凌辰她已经平安抵达北京。
<br/>
<br/>        “凌辰，我刚下飞机，已经安全抵达北京。”
<br/>
<br/>        “哈，我就知道你安全抵达了，我一直都在关注着航班信息。”凌辰第一时间接到宁雅的电话，别提心里有多开心了。
<br/>
<br/>        “你没事吧？”宁雅还是想知道凌辰在杭州的情况。
<br/>
<br/>        “老爷子现在在我身边，让我和他一起回家，他说宁玥的心思有些深沉，怕姐姐宁雅也会是那样，他得为他的儿子找个好媳妇。”凌辰毫不犹豫的出卖了他的父亲。
<br/>
<br/>        凌思永听得只翻白眼，有这样坑老爹的吗？
<br/>
<br/>        “确实应该给他儿子找个好媳妇儿，好好管一管那个不成器的儿子，别在暖昧不清，胡搞成精。”宁雅的心情莫名其妙的好。
<br/>
<br/>        “哪有暖昧不清，哪有胡搞成精？我就和你一个人暖昧，和你一个人.....”
<br/>
<br/>        “你这臭小子，你说什么呢？你在和谁说话？”凌思永听见凌辰的话敲了凌辰的脑壳一下。
<br/>
<br/>        “喂，你打我干嘛？我在和我媳妇儿说话呢，你还要不要儿媳妇了？”凌辰被父亲的那一敲打断了话，在电话那头冲着父亲喊。
<br/>
<br/>        宁雅笑着说：“凌辰，再见，我准备回家了。”
<br/>
<br/>        人生的路上有你陪着一起吵吵闹闹说说笑笑也很好。
<br/>
<br/>        .........
<br/>
<br/>        宁雅一回到北京就投入到紧张的工作中，耗材推向市场的日期也越来越近，她找到皇甫双双和李凌菲还有倪小美开了一个小型的会议，陈子昂这几天在西城，是不会回北京的，而且她似乎是比平时更忙，回复邮件也就是几个字，多的话一句也懒得说。
<br/>
<br/>        宁雅不知道陈子昂这几天在干些什么，她也不知道陈子昂去甸城的目的，但是她知道陈子昂的父亲生病了，陈子昂请了专家回去给她的父亲做手术，这几天陈子昂一定是在陪着她的父亲，也不知道她父亲的手术做的怎么样了。
<br/>
<br/>        宁雅将大家召集到办公室对大家说：“陈总这几天家里有事情，规定耗材推向市场的日期也越来越近，我想听听你们的建议和意见。”
<br/>
<br/>        这里面的几个人，除了他和李凌菲到公司晚一点，倪小美和皇甫双双都是老员工，在公司说话也具有一定的分量。
<br/>
<br/>        “按照正常的步骤走肯定来不及了，我和你一起跑跑商务，分工合作，在官方宣传上我的方案已经出了，就等你回来和你商量。”皇甫双双开口。
<br/>
<br/>        “我和凌菲已经将要合作的几个友商已经谈妥，他们都同意在上市的那一天支持我们，合同我们也已经做好了，这几个都是国外的单子。”倪小美沉稳的开口。
<br/>
<br/>        “全国各分部我也已经确认，静等花开。”李凌菲永远一副不急不缓的语气。
<br/>
<br/>        “那这么说都已经万事俱备只欠东风了？”宁雅开心的问。
<br/>
<br/>        “是的！”
<br/>
<br/>        “明天我再去有关部门洽谈一下日期，看看还有什么要补充的？凌菲，你有没有统计最近的出货量？”宁雅问李凌菲。
<br/>
<br/>        “出货量在财务科就能看到，如果你想要了我看看就是。”李凌菲淡淡的说。
<br/>
<br/>        “不用，我等一会看下。”宁雅也是忙糊涂了，她们几个中层是可以看到财务的信息的。
<br/>
<br/>        “双双，你等一会和钱喵喵沟通一下，看看他们生产的硒鼓库存量够不够，另外让苟江陵和郝总他们公司对接一下，在全店联动的前三天货物必须得到达各分部。我等会去找一下绾绾，看看我们的系统是不是已经很完善了。”陈子昂不在公司，江俞轩也不在公司，做为办公室主任的宁雅自然而然的担负起了公司的责任，而公司的各个中层也在有条不紊的工作着。
<br/>
<br/>        就像是陈子昂说的，这个公司是一个整体，离开谁都能运转，只不过是转得快慢而已，大家都只字不提江俞轩离职的事情，他们在一起共事了很多年，都彼此了解，江俞轩是江家大少的身份大家都知道了，迟早江俞轩都会要离开的，虽然大家的心里有不舍和惋惜，但是不会影响到他们的工作，他们都是公司的一部分，也许后期他们也会离开，生老病死，来来去去都是人之常情。
<br/>
<br/>        虽然江俞轩离开了，但不妨碍他们还是好朋友，好同事，他们在一起同甘共苦的那几年是每个人人生中最重要的财富，也是无法磨灭的印记。
