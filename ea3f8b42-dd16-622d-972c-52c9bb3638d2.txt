　　东海龙王、北海女皇、战空等十几名愿意一搏的强者，只是十数日后，便再次来到了苍穹神宫，各自带领了几名强者。
<br/>　　而他们所在的宗门或家族，有的直接解散了，有的开始低调行事。
<br/>　　总之，所有人都做好了拼死一搏的准备，也做好了陨落的准备。
<br/>　　苏莫每日都在修炼，即便没有足够的资源，他靠吞噬武魂吸纳灵气，速度也是别的武者的数十倍。
<br/>　　时光匆匆，一个月又一个月，半年时间转瞬即过，苏莫的修为达到了武皇境八重的巅峰，再也难有寸进了。
<br/>　　没有海量的资源，他根本无法冲破武皇境九重的晋阶瓶颈。
<br/>　　期间，苏莫数次动用吞噬武魂，帮助司空炎和青雅等人，将几人的修为全部提升到了武皇境九重，超过了他自己的修为境界。
<br/>　　苍穹神宫中资源虽然不多，但司空炎等人的修炼，耗费的资源极少，这点资源苍穹神宫还是不缺的。
<br/>　　对于司空炎几人的实力，苏莫还算比较满意，其九人的战力相当强大，任何一人，都能秒杀一般的准帝。
<br/>　　因为几人获得的九宫传承，尽皆都是大帝传承，强大无比，战力远不是北海女皇、战空等准帝可以比拟。
<br/>　　而司空炎几人，也多次帮助苏莫提升五行属性之力，让他的实力再增，五门帝级上品功法的境界，也提升到了第六层的境界。
<br/>　　帝级上品功法，同样是分为九层境界，但苏莫提升到第六层境界之后，便也无法再提升了。
<br/>　　因为这需要沉淀，足够的积累底蕴才行，他提升的太快了，一味的强行灌输属性之力，效果已经很低了。
<br/>　　五门功法提升到第六层境界，让得苏莫的属性玄力威力更加的强大。
<br/>　　目前，连他自己都不清楚，自己的实力达到了什么地步。
<br/>　　因为他早已无敌，也没有了参考的对象。
<br/>　　……
<br/>　　这一日，苍穹神宫不远处的山脉中，一处千丈山峰之上。
<br/>　　苏莫盘膝坐在山巅，如同一块亘古长存的岩石，纹丝不动。
<br/>　　而在他的头顶上空，二十八柄本命灵剑，组成一个巨大的磨盘。
<br/>　　磨盘急速旋转，爆发出万道剑光，照耀天地，无边的锋锐之气，即便相隔万里，都能清晰的感觉得到。
<br/>　　唰！
<br/>　　蓦地，剑光爆闪，二十八柄本命灵剑齐齐一动，瞬间向数百里外的一座山峰猛然一斩。
<br/>　　霎时之间，二十八道剑气形成一股绝强的剑气风暴，以毁灭一切之势，击在了那座高山上。
<br/>　　轰隆隆！
<br/>　　一阵爆响之后，漫天烟尘四起，高达千丈的山峰，在这一击之下，彻底灰飞烟灭。
<br/>　　苏莫见此暗暗点头，这一击名为‘星宿剑暴’，算是二十八星宿剑阵中。一种颇为厉害的攻击手段。
<br/>　　他刚才并未全力催动剑阵，只是随意一试，但剑阵的威力还是让他非常满意的。
<br/>　　这大星宿剑阵，他已经修炼的颇为纯属，即便没有到出神入化的地步，也已经算是炉火纯青了。
<br/>　　本命灵剑经过数年的温养，其本身威力，比刚炼制之时，早已不可同日而与。
<br/>　　咻咻咻！
<br/>　　苏莫心神再动，二十八柄本命灵剑瞬间连成一线，强大的气机彼此相连，爆射长空，如同一道幻影。
<br/>　　咻！
<br/>　　刺耳的音爆声不绝与耳，灵剑只是半个眨眼的时间，便返回了苏莫身前。
<br/>　　而此山脉中数百座山脉，已经被灵剑全部洞穿，速度快的不可思议。
<br/>　　轰隆隆！
<br/>　　就在此刻，异变陡起，只听南方天际传来阵阵沉闷的声响，仿佛有无数的炸雷，在虚空深处爆炸。
<br/>　　天地昏暗了下来，日月无光，狂风骤起，呜咽阵阵，如同鬼哭狼嚎。
<br/>　　“嗯？”苏莫见此立刻眸光一凝，他的感知力极为强大，他清晰的感觉到，南方的虚空都在震颤，好像受到了强烈的攻击一般。
<br/>　　嗖嗖嗖！！！
<br/>　　就在此刻，破空声响起，数十道身形急速而至，飞到了苏莫的身后。
<br/>　　来人正是苍穹神宫的一众强者，九宫之主，帝释、海明江、钱副阁主、东海龙王等一众准帝强者。
<br/>　　“宫主，试炼者要来了！”海明江颤声说道，其眼眸中露出深深的担忧。
<br/>　　其他人亦是如此，大劫近在眼前了，没有人心中不担心。
<br/>　　虽然他们都做好了失败的准备，但还是心中忐忑。
<br/>　　现在，他们只能寄希望于苏莫了，他们也不清楚苏莫的实力，如今达到了什么地步？
<br/>　　但是，毫无疑问，苏莫是他们的主心骨，是他们的希望。
<br/>　　“来了吗？”苏莫闻言，顿时眸中精光爆射，如同一柄犀利的剑光，将他身前的空间洞穿数十丈。
<br/>　　“宫主，这是外界在打开通向苍穹世界的空间通道，在南洲地域，最多半个时辰，试炼者便会降临！”原太玄宗宗主究楚北说道。
<br/>　　“回宫！”
<br/>　　苏莫点了点头，吩咐一声，便立刻飞身返回苍穹神宫。
<br/>　　回到苍穹神宫之后，苏莫便立刻行动了起来，将数以百万计的苍穹神宫弟子和长老全部聚集了起来。
<br/>　　而后，所有武皇境九重修为以下的人，全部被他收进了虚界神图之中。
<br/>　　这些弟子和长老，修为不足，实力低下，不宜留在外界。
<br/>　　很快，整个苍穹神宫中，便变的空空荡荡了起来。
<br/>　　剩下的人，除了苏莫和司空炎等九宫之主外，不足一百人。
<br/>　　但这一百人，却是苍穹神宫目前的巅峰力量。
<br/>　　山门处，白玉广场。
<br/>　　“各位前辈！再见了！”
<br/>　　苏莫望着身前的九名守护傀儡人，眸中露出深深的不舍。
<br/>　　他不可能在苍穹神宫中等待试炼者杀来，准备带领众强者离开苍穹神宫，伺机而动。
<br/>　　因为他无论是成功还是失败，都不可能回来了，成功则离开苍穹世界，失败则生死道消。
<br/>　　所以，苍穹神宫为避免毁灭，要再次遁入虚空中。
<br/>　　而九名傀儡人，是无法离开苍穹神宫的，只能和苏莫分开了。
<br/>　　“宫主，珍重！”
<br/>　　“宫主，若你能离开，日后为苍穹世界报仇雪恨！为苍穹神宫报仇雪恨！”
<br/>　　“宫主，希望我们还有再见面的一日。”
<br/>　　“……”
<br/>　　九名守护傀儡纷纷开口，个个眸中露出不舍之色，他们不知道苏莫能否成功，但也无法给予任何帮助。
<br/>　　“我会的，若是能逃得一命，我会成为巫族的噩梦！”苏莫郑重点头，随即，他将圣子令牌交给了金一。
<br/>　　无论他成功与否，苍穹神宫都可以再选传人，日后再次出世。
<br/>　　而后，司空炎等九人，也分别和各自所属的傀儡人依依道别。
<br/>　　最后，苏莫等人，亲眼看着苍穹神宫再次开启了虚空挪移大阵，消失在了苍穹世界。
<br/>　　“走，去南洲！”随即，苏莫率领众强者，向南洲方向而去。
