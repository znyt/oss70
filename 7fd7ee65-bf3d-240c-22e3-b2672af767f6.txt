        童筱颖看着傅夜沉，眨巴了一下眼睛。
<br/>
<br/>        他刚刚有听？！
<br/>
<br/>        她还以为……
<br/>
<br/>        童筱颖发现，傅夜沉和傅昼景一样，从未嫌弃过她的农村出身，这点令她很欣慰。
<br/>
<br/>        其实，他们傅家的人，家大业大，不管是冷爷爷，还是那位冷夫人，都从未排斥过她的出身。
<br/>
<br/>        冷爷爷表面上看起来很淡漠昼景，但实际他作为长辈，还是很关心昼景。
<br/>
<br/>        而昼景……
<br/>
<br/>        他似乎对冷爷爷有什么芥蒂一样。
<br/>
<br/>        童筱颖也说不清那种感觉。
<br/>
<br/>        傅夜沉一直垂着眸子，不敢多看童筱颖一眼，他怕自己多看了她一眼，就忍不住想把她从弟弟阿景的手中夺回来。
<br/>
<br/>        隐忍、隐忍、再隐忍……
<br/>
<br/>        直到苏漫雪从楼上下来，打破了两人间那和谐的气氛。
<br/>
<br/>        如果不是刘婶进屋来打扫房间，苏漫雪还真就怀疑他傅夜沉是不是把她给忘记了。
<br/>
<br/>        “夜沉，你也真是的，自己起来了也不叫醒我。害得我一睁开眼睛，发现你不在我身边，让我心里空空的。”苏漫雪趿着拖鞋，走过来一把从傅夜沉身后环住了他的脖子，并偏着脑袋嘟起红唇，在他耳边娇嗔道。
<br/>
<br/>        她就是要当着童筱颖的面与他亲昵，这样他就不好推开她了。
<br/>
<br/>        童筱颖看了苏漫雪一眼，然后低头，默默地喝着自己碗里的汤。心里想的却是，她是不是也应该像苏漫雪那样，在昼景面前多撒撒娇，增进一下两个人之间的感情？
<br/>
<br/>        面对苏漫雪故意跟自己亲热，傅夜沉面无表情地站起身来，并对苏漫雪的话充耳不闻。
<br/>
<br/>        “我先去车上等你，待会一起去公司。”傅夜沉淡淡地丢下这句话，踱步离开。
<br/>
<br/>        苏漫雪见傅夜沉出了大门后，转脸间就对童筱颖发起火来：“童筱颖，你是觉得我家夜沉的钱不是钱吗？觉得我家夜沉赚钱不辛苦是吗？这一大清早的，你竟然给我做这种山珍海味！你不知道早餐要吃清淡点，才对身体好吗？你不知道早餐应该多吃膳食纤维的食物，才对身体有益吗？你这一大清早就是鲍鱼、海参、鱼翅的，敢情你是把我家当你家了，钱可以随便挥霍了是不是？”
<br/>
<br/>        童筱颖被苏漫雪说得一脸懵，完全没有任何反驳的余地。
<br/>
<br/>        苏漫雪说得一点都没错，这里是她苏漫雪的家，她怎么会因为傅夜沉是昼景的大哥，而她苏漫雪又是她曾经的闺蜜，并且自己又和刘婶关系好，跟那个连医生也是自来熟，就把这儿当自己家一样住着悠闲自在了呢？
<br/>
<br/>        以前，她总是一个人，自从昼景负债忙着工作后，她其实一直觉得很孤独。
<br/>
<br/>        突然间，身边一下子多了这么多人，同住一个屋檐底下，不再是一个人吃饭、一个人看电视、一个人做家务。
<br/>
<br/>        这里热闹又有家的温馨，让她几乎忘却了自己的身份。
<br/>
<br/>        苏漫雪的话，真的就像一巴掌一样，打醒了童筱颖。
<br/>
<br/>        “对不起。”童筱颖垂下眼帘，诚恳并深沉地向苏漫雪道歉。
<br/>
<br/>        苏漫雪双手抱臂，不禁得意地扬了扬眉。
<br/>
<br/>        果然，她要在童筱颖面前建立起自己的威信来。
<br/>
<br/>        她可是傅家的大少奶奶啊！
<br/>
<br/>        哪怕她只是个冒牌货，但在大家眼里，她就是个货真价实的傅大少爷的未婚妻！！
<br/>
<br/>        她办不了傅夜沉，但是能制服她童筱颖也不错。
<br/>
<br/>        从今以后，傅夜沉加之在她身上的痛苦，她便一一在她童筱颖的身上讨回来。
<br/>
<br/>        “这次就不跟你计较了，我警告你啊！童筱颖！这种事情，下不为例！真是的，一大清早起来，就要为这种琐事操心。做人，都不知道自己寄人篱下，还想反客为主了。”苏漫雪白了童筱颖一眼，然后甩着手，冲着楼上吆喝了一声，“刘婶，赶紧下来做早餐，我肚子都饿了。”
<br/>
<br/>        童筱颖抿了抿唇，将碗筷收拾了一下，端去了厨房，才去楼上提起自己的包包，闷闷不乐地出了门。
<br/>
<br/>        是她的心思上疏忽了，一时间，竟然忘了自己的本分。
<br/>
<br/>        童筱颖走出大门，低着头下台阶。
<br/>
<br/>        直到她把脚下的台阶走完，下意识地抬头，才看到傅夜沉的豪车就停在自己面前，并且司机还站在后座旁，拉开了车门等候着她上车。
<br/>
<br/>        童筱颖立即识趣地钻进车内，向已经坐在车内等候多时的傅夜沉，抱歉地说了声：“大哥，对不起，让你久等了。”
<br/>
<br/>        傅夜沉只是轻瞥了一眼她所坐的位置与自己的距离，冷淡地说道：“我是魔鬼吗？中间这么宽敞你不坐，非得挨着车门坐。”
<br/>
<br/>        童筱颖愣了一下，只觉有些尴尬，又不得不挪动了身子往傅夜沉那边坐过去点。
<br/>
<br/>        车门关上后，司机小跑着回到了驾驶座上发动了车子。
<br/>
<br/>        因为昨晚为了做好那盅佛跳墙，童筱颖并未睡好，又加之司机开车很平稳，她不知不觉中双手抱着包包，昏昏欲睡。
<br/>
<br/>        傅夜沉本来在看平板上的财经杂志，无意间见童筱颖睡着了，便将身子挪了挪，挨到她身边坐着。
<br/>
<br/>        过了一会儿，童筱颖沉睡了过去，身体的支点开始往车窗那边倾斜。
<br/>
<br/>        傅夜沉担心她会被她自己给吓醒，连忙伸手扶了一下她的双肩。
<br/>
<br/>        不知不觉中，童筱颖的身子渐渐地往傅夜沉这边倒，最后，脑袋很自然地靠在了傅夜沉的手臂上。
<br/>
<br/>        为了让这女人靠得更舒适，傅夜沉轻轻地调整了一下自己的坐姿，直接让她的脑袋枕着他的肩头而睡。
<br/>
<br/>        他只要稍稍偏头，鼻息间，就有她头顶传来的发香。
<br/>
<br/>        这样的味道，还是和第一次时，他吻上她，那种令他流连忘返的、淡淡的柠檬香气……
<br/>
<br/>        豪车已经抵达了傅氏集团大厦前坪，司机刚停稳车，下意识地转身看向傅夜沉，等待他下达命令。
<br/>
<br/>        傅夜沉见童筱颖仍旧未醒过来，于是抬起另一只手伸出食指在半空中画了个圈。
<br/>
<br/>        司机会意地点了点头，又重新发动车子，然后开着车，围着傅氏集团大厦转圈。
<br/>
<br/>        这样纹丝不动地坐着，对傅夜沉来说并不是一什么件难事。
<br/>
<br/>        曾经，他可以和自己的战友，握着狙击枪，在丛林里纹丝不动地趴好几个小时。
<br/>
<br/>        哪怕有毒蛇或是毒虫从自己身上脸上爬过，他也可以淡定地一动不动，只为等对方敌人上钩，将他们一举歼灭。
<br/>
<br/>        不知不觉中，被筱颖靠着的这边手臂，渐渐泛起麻痹，本来应当是很难受，可是傅夜沉却只觉得手臂上像是只有几只蚂蚁在那里爬着，一种异样的酥痒，让他想去挠挠又怕吵醒她而强忍着继续一动不动。
<br/>
<br/>        童筱颖在梦里犹自舒展了眉心，嘴角微微上扬，圆润的唇珠在车窗透进来隐约的光线里，泛着蜜一样的润泽。
<br/>
<br/>        傅夜沉不敢再看她那秀色可餐的唇，怕自己把持不住，不得不转过脸去瞧着车窗外。
<br/>
<br/>        围绕着傅氏集团大厦的是一条宽敞的柏油路，柏油路的路缘石旁，种植了一棵棵高大的松柏。
<br/>
<br/>        这些松柏，还是当年，他小时候让爷爷派人种的。
<br/>
<br/>        因为，军人如松，他说他长大了以后要像父亲一样当一名保家卫国的军人。
<br/>
<br/>        爷爷允许了，如今，他长大了，这些松柏也高耸入云、绿意盎然。
<br/>
<br/>        此时此刻，有筱颖的陪伴，仿佛天长地久，时间定格，他亦是不想动弹，情愿就这样永远地坐下去一样。
<br/>
<br/>        现在，与她在一起时的点点滴滴，都是值得让他去珍惜。
<br/>
<br/>        而傅夜沉的私宅里，连华生一早起来就在询问他的“佛跳墙”，他想吃这道菜可是想了很久了。
