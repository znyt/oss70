　　长乐如此大手大脚的花钱，自然是有人看不惯的，可是不等他们上书弹劾，那位年轻的皇帝陛下又开始出妖蛾子了。
<br/>　　“皇宫开放日？把皇宫对外开放让百姓参观？陛下，这，这万万不可啊。”房玄龄在听到这个消息的时候几乎不敢相信自己的耳朵，看着脸露得意之色的李承乾，脑袋要多大就有多大，暗恨自己怎么就不跟着老皇帝去旅游，如果那样就不用在这里受这份罪了。
<br/>　　李承乾并没有在意老房的话，负手立于大殿之上的他，看着下面不知所措的一群大臣，一本正经的说道：“为什么不可？亲民嘛，这马上就要除夕了，朕要与民同乐。”
<br/>　　“陛下，太上皇也曾与民同乐，不过那也是在外城，而且太上皇也是在宫城之上，您，您这让百姓入皇宫，这，这不合律法啊。”李孝恭试图阻止李承乾，并且抬出老李。
<br/>　　但现在地球人已经无法再阻止李承乾这位新皇帝了，短暂的沉默之后，李承乾再次开口说道：“律法还不是人定的，朕决定了，以后为了表示亲民，每旬都要有一天皇宫对外开放，百姓可以自由出入皇宫。”
<br/>　　完了，彻底完了，这娃爱折腾的毛病这一定是又犯了，以前老皇帝在位的时候，还有人能压制他，可如今李二已经带着老婆满世界旅游去了，而且还连这娃的舅舅也一起带走了，整个大唐也就他一个人说了算，还有谁能阻止他？
<br/>　　一群持反对意见的臣子纷纷看向正在打盹的程咬金和数房梁的老孔，结果这俩人就像什么也没有听到一样，依旧是该干啥干啥，对众人的目光视而不见。
<br/>　　最后长孙冲倒霉蛋儿不知被身后哪个老人渣踹了一脚，踉踉跄跄的站到了大殿的中间。
<br/>　　李承乾在上面自然看到了这一切，不过心情不错的他很不地道的装作没有看到长孙冲被踹出来的一幕，反而一本正经的问道：“咦？表兄可是有什么话要讲？”
<br/>　　我有个屁话要讲！长孙冲回头愤怒的看了一眼，结果入眼所见个个都是眼观鼻，鼻观心，没有任何一人露出特殊的表情。
<br/>　　无奈，真是太无奈的，这帮老人渣为什么不早点退休呢，长孙冲心中无数草泥马在横冲直撞，最后把心一横：丫的，出卖老子，那老子就破罐子破摔，看你们怎么办！
<br/>　　想到这里，长孙冲梗着脖子，一脸真诚的说道：“臣以为陛下所言甚是，臣举双手赞成。”
<br/>　　而随着长孙冲话音刚落，立刻有老货跳出来：“长孙冲，你敢蛊惑圣上？！”
<br/>　　但长孙冲也不是什么善茬，闻言立刻还击：“陛下高瞻远瞩，深谋远略，岂是某能蛊惑的？”
<br/>　　“明知君上有错而不谏，这便是欺君！”
<br/>　　“是非对错又非你一家之言，陛下只是要亲民，这怎么就错了？莫非亲民也有错？”
<br/>　　“胡说八道，亲民之举千千万万，何必非要开放皇宫。”
<br/>　　“千千万万？那你怎么不举个例子，为何这么多年不见你谏言？知而不报你又该当何罪！”
<br/>　　一言不合就扣帽子，这样的行为李承乾已经不知道见过多少次，自然不会当真，不过任由这帮家伙吵下去也不是个事儿。
<br/>　　所以在事情还没有演变到不可收拾的时候，李承乾用专门打造出来的小木锤在桌子上‘哐哐’凿了几下：“都安静，吵什么！当朕的朝堂是菜市场么！”
<br/>　　“陛下，非是臣等要吵，实在是皇宫不能开放啊，若是因此而发生什么意外……”刚刚与长孙冲吵的正欢的乃中书令褚遂良，这老头儿字写的极好，也有学问，不过就是有些古板，很多时候不知道变通。
<br/>　　李承乾知道他的脾性是什么样的，索性不等他说完便打断他道：“会发生什么意外呢？刺杀？不至于吧？如果朕沦落到会有人刺杀的程度，这大唐怕是也要毁了吧？”
<br/>　　“陛下，臣的意思并非如此，臣……”。
<br/>　　不等褚遂良说完，李承乾再次打断了他道：“你们是什么意思朕很清楚，现在朕身负重任，身系天下，是吧？可是你们这样把朕围起来，朕怎么事知道你们天天给朕说的那些东西是真是假？你们是想要变相的软禁朕么？”
<br/>　　“臣等不敢！”一看李承乾似乎真的急了，满朝文武立刻集体请罪。
<br/>　　“朕量你们也不敢，不过这件事情你们不用劝朕了，朕意已决，皇宫每旬最后一日对民间开放，人数限为五百！退朝！”讲不通道理那就硬来，反正也不是什么原则上的问题，李承乾相信惹不出什么大乱子。
<br/>　　“陛下！”眼看着李承乾玩起混不吝，一群老人渣也没有什么好办法，叫了两声见他头也不回的走了，这才有些不甘的闭上了嘴。
<br/>　　“不行，这事儿不能就这么定了，怎么也要跟陛下好好说说，让陛下收回成命才是。”倔驴这种生物任何时候都不会缺，比如褚遂良便是如此，虽然李承乾已经离开了，但这老家伙还是不依不饶，喋喋不休的唠叨，同时大步向殿外走去，看样子好像真的要去找李承乾理论。
<br/>　　其他几个老家伙见他如此，一个个大叫一声等等，也都纷纷跟上。
<br/>　　不过这些老头子并没有见到李承乾，一路追到老李以前在两仪殿的书房时才被人告知，皇帝陛下溜腿儿去了，至于去了哪里，则谁也不知道。
<br/>　　这下老头儿们没招了，只能唉声叹气的垂头离开。
<br/>　　而在他们走了之后，李承乾这才鬼鬼祟祟的从书房的屏风后面走出来，长长的出了一口气，骂了句‘老不死的’之后，苦笑着摇了摇头。
<br/>　　“陛下，您何必如此呢，这皇宫本就是高贵机密的地方，如何能让百姓进来。”小白见到李慕云无奈的样子，忍不住问道。
<br/>　　因为在她看来李承乾这样做完全就是自找麻烦，那些古板的老头儿如果能答应才是怪事。
