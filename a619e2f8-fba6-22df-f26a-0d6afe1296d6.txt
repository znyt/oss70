　　这些离开之人，大多都是上一届参加的武者。
<br/>　　如今留下的人，许多都是第一次参加，虽然屠雄凶名显赫，但是他们也未尝没有想要尝试一番的心。
<br/>　　毕竟这考核，却是混战，双拳难敌四手，也许真的可以趁乱，得到机会。
<br/>　　屠雄眼神扫视了一圈，看着还有许多人留下，言语冰冷的开口道。
<br/>　　“看来你们一个个，都是在自己找死，那可就不要怪我了。”
<br/>　　这话中的意思，直接便是威胁，毫不掩饰的威胁。
<br/>　　在场的武者，许多都是大家族出身的武者，一个个心高气傲到极点，哪里可以忍受如此威胁。
<br/>　　“哼，倒时候谁死谁活，还不知道呢。”
<br/>　　一尊武者，也是在场武者中的佼佼者，来自一大家族青家的青厉，冷哼一声开口道。
<br/>　　这青厉的实力，也足以达到巅峰圣者的地步，他也未必怕了那屠雄。
<br/>　　听到屠雄的威胁，直接冷声反击。
<br/>　　“恩？”
<br/>　　听到青厉的话，屠雄眼睛转了过去，看着青厉，眼神中尽是杀意。
<br/>　　屠雄面露狰狞的说道。
<br/>　　“等到台上，我必杀你！”
<br/>　　“那就来吧。”
<br/>　　青厉也不甘示弱，反击道。
<br/>　　“准备就绪了吗？没有人认输了，就全部上台吧。”
<br/>　　北邙学院的弟子，高声说道，便是在擂台上打开了一个缺口，在场所有武者，便是一下子涌入其中。
<br/>　　这擂台上，布下了无数禁制，以防止力量波及出去，同时也是防止武者擅自离开。
<br/>　　这布下禁制的人，实力只怕是极为强大，楚风眠观察了一下，便是清楚，就算是他，也不能简单将这禁制打破。
<br/>　　这一切，都在显露着北邙学院的强大底蕴。
<br/>　　擂台极大，容纳百人分开站立，倒是轻轻松松，每个人都是选好了，位置。
<br/>　　不少武者都是选择站在擂台边缘，角落，以防止被人偷袭。
<br/>　　毕竟混战，在场的任何一人，互相之间，都是敌人。
<br/>　　楚风眠也是选择了一个角落，站在原地，他的脸色倒是极为平静。
<br/>　　擂台上的所有人中，只有一人，才是胜者，剩下的人，便都是垫脚石，被踩落下去，甚至直接陨落，都有可能。
<br/>　　“开始！”
<br/>　　随着那北邙学院弟子一声惊喝。
<br/>　　擂台上的所有武者，都是一股灵力，猛然爆发出来。
<br/>　　所有人，都是将力量全然爆发出来，却是没有任何一人主动出手。
<br/>　　在这样的混战之中，谁先贸然出手，谁便是就有可能遭到围攻。
<br/>　　“一群废物！”
<br/>　　屠雄站在台上，看着没有人出手，眼神中露出几分不屑，他的双眼，猛然看向了那青历。
<br/>　　一双眼睛中，尽是杀意。
<br/>　　“小子，我就先杀了你！让那些废物看看！敢跟我为敌的下场！”
<br/>　　屠雄身上的灵力，完全爆发出来，这一股力量，已经不比寻常巅峰圣者要弱了。
<br/>　　屠雄脚下一动，顿时便是直接冲到了那青历的面前，一身灵力爆发而出，整个身躯，都变的庞大无比，足有寻常武者三四倍之高。
<br/>　　“死！”
<br/>　　他一拳砸出，整个擂台上，都是微微有些晃动。
<br/>　　面对屠雄的一拳，那青历也不甘示弱，猛然拔剑，一剑对着屠雄斩了过去。
<br/>　　这一剑，已经蕴含了几分剑意，斩出一刻，空间都在嘶嘶作响。
<br/>　　这二人，便是在场所有人中实力最为强大的两位，看到这屠雄跟青历交手，在场无数众人便是安静的围观这。
<br/>　　许蚌相争渔翁得利，所有人都想要当最后的得利之人。
<br/>　　“轰！”
<br/>　　一拳一剑，这两大强横无比的灵力，顿时交锋。
<br/>　　这交锋，没有任何技巧可言，便都是真正的力量碰撞。
<br/>　　“小子，就凭你这几分实力，也敢在我面前放肆？”
<br/>　　两股力量相对，那屠雄的脸上，尽是冷笑。
<br/>　　下一刻，他打出的一拳，却是将青历的剑锋，直接打破。
<br/>　　这一拳，不止是将青厉的剑锋打破，更是将他逼退了数步之远，令青厉的口中，直接一口鲜血便是喷了出来。
<br/>　　“不可能！我的实力明明已经达到了巅峰圣者的地步，怎么会败！”
<br/>　　青厉看着屠雄，眼神中尽是不可置信。
<br/>　　屠雄的实力，看起来也就是在巅峰圣者的实力，青厉的实力，同样如此。
<br/>　　可如今，双方交手，青厉却是败的如此之惨，完全没有还手之力。
<br/>　　青厉的眼神中，露出几分惧怕之色。
<br/>　　屠雄的实力，已经是远远超出了他的意料之中，根本不是他可以抗衡的。
<br/>　　青厉的心中，都有着几分后悔，早知屠雄的实力如此强大，他肯定不会出言挑衅屠雄。
<br/>　　可是现在，后悔也晚了。
<br/>　　“我认输……”
<br/>　　青厉口中的话，还未说到一半。
<br/>　　屠雄的身躯，已经动了，他脚下一动，瞬间便是冲到了青厉的面前去。
<br/>　　猛然一拳，对着青厉胸口，砸了过去。
<br/>　　“砰！”
<br/>　　无数鲜血飞溅出去，这青厉的胸口，直接被砸碎，五脏六腑，尽数破碎。
<br/>　　青厉就连认输都没有来得及喊出来，便是直接被屠雄灭杀。

<br/>　　楚风眠看着屠雄的举动，心中暗自想到。
<br/>　　这屠雄出手，便是没有给青厉任何的机会，直接便是将青厉灭杀。
<br/>　　在这擂台上，虽然可以选择投降，但是在投降之前，也可以被轰杀。
<br/>　　屠雄的手段，便是丝毫没有给青厉投降的机会，直接雷厉风行，将青厉瞬间灭杀。
<br/>　　“该死！”
<br/>　　“上，杀了他！”
<br/>　　“不杀了他，我们谁都没有可能赢！”
<br/>　　几尊身影，突然动了。
<br/>　　这几道身影，四面八方之间，便是向着屠雄猛然打了过去。
<br/>　　刀，枪，剑，矛，无数灵器，纷纷出现，便是向着屠雄，直接轰杀了过去。
<br/>　　就连青厉，都不是屠雄的一招之敌，这屠雄已经是擂台上最大的威胁。
<br/>　　这其他武者，便是一起联手之下，同时出手，无数武技，俨然已经是向着屠雄轰杀了过去。
<br/>　　参加考核的弟子，谁没有一颗想要胜利的心，如今他们便是准备围攻屠雄，将这个最大的威胁铲除。
