　　走进石门中，四人眼前的场景立刻变换，来到了一片广阔的虚空之中，虚无中屹立着一根根粗大的灰色石柱。
<br/>　　这些石柱极长，仿若连天接地，放眼望去，数之不尽的石柱排列在虚空中，整整齐齐，每一根石柱间的距离都一般无二。
<br/>　　“这就是迷宫，是一个强大无比的幻阵，若是没有方法，一辈子都飞不出这片虚空，大家跟紧我！”廖云剑朗声说道。
<br/>　　随即，廖云剑脚踏虚空，在一根根石柱间穿梭了起来。
<br/>　　苏莫三人紧随其后，不敢有丝毫的懈怠。
<br/>　　廖云剑带领着三人在石柱群中绕来绕去，看似杂乱无章，实则是掌握了某种规律。
<br/>　　绕了约莫一刻钟后，四人眼前的场景变换，出现了一处通道之中，身后正是那阎魔地宫的大门。
<br/>　　咔擦！
<br/>　　苏莫脚下传来了清脆的响声，低头一看，居然是一具骸骨，被他一脚踏碎了。
<br/>　　抬眼看去，此处通道中居然有很多骸骨，粗略一扫不下十数具。
<br/>　　“这里怎么如此多的骸骨？”聂乾惊讶的问道。
<br/>　　“这些曾经都是学府的弟子，被困死在了此地！我当初也被困了整整一年！”廖云剑面带唏嘘的说道。
<br/>　　他当初和两位同门一起进来，最后只有他一人活着逃了出去。
<br/>　　“我们继续走！”廖云剑招呼一声，便继续向前走去。
<br/>　　苏莫走在最后，他在几具骸骨的身上发现了储物戒，全部一一收取了。
<br/>　　廖云剑瞥见苏莫的行为，暗暗摇了摇头，眸中露出一丝轻蔑。
<br/>　　几人在通道中前进了片刻，便再次来到了一座略小的石门之前，然后停了下来。
<br/>　　“这便是第二关，你们两人准备好了吗？”廖云剑瞥了一眼苏莫和聂乾，沉声问道。
<br/>　　苏莫和聂乾均是点了点头。
<br/>　　聂乾面色凝重，但苏莫却是心中隐隐有些兴奋，凶灵或许是很好的养料。
<br/>　　“那我们走吧！”
<br/>　　廖云剑的语气中，也是隐含兴奋之色，随即四人一步踏出，瞬间便进入了石门中。
<br/>　　唰！
<br/>　　四人眼前的场景变换，再次来到了一片虚无的空间之中。
<br/>　　这片空间辽阔无比，一眼望不到尽头。
<br/>　　在极远之处，一座黑色的宫殿悬浮在虚空之中，仿若亘古长存。
<br/>　　黑色的宫殿魔气冲天，仿若滚滚狼烟，让得虚空都扭曲了起来。
<br/>　　而在宫殿的四周，有着数之不尽的虚幻灵体，在盘旋飞舞，张牙舞爪，发出各种尖锐刺耳的声音。
<br/>　　这些灵体形状各异，有的呈人形，有的呈各种野兽的形状。
<br/>　　桀桀桀！
<br/>　　嘎嘎嘎！
<br/>　　这些虚幻的灵体，发现了突然出现的苏莫四人，顿时发出一阵怪笑，而后有一小半都向苏莫等人扑了过来。
<br/>　　“聂兄、苏莫，靠你们两人了，只有你们将这些凶灵吸引住，我和韩姑娘才能进入宫殿！”廖云剑急声说道。
<br/>　　聂乾闻言点了点头，立刻便出手了，一拳向那些飞来的凶灵轰去。
<br/>　　轰隆隆！
<br/>　　一声爆响，拳茫轰进了凶灵群中，顿时发生猛烈的爆炸。
<br/>　　聂乾的拳茫强大无比，一击便将大量的凶灵被轰飞了出去，有些弱小的凶灵，顿时四分五裂，被全家轰成了渣。
<br/>　　苏莫同样出手了，一掌拍出，浩大的掌印击向凶灵，立刻便将几只凶灵震退。
<br/>　　聂乾和苏莫的出手，顿时激怒了此地所有的凶灵，那黑色的宫殿周围，足足有上千只凶灵，向几人飞扑了过来，铺天盖地。
<br/>　　嘎嘎嘎嘎！！
<br/>　　一阵阴森的怪笑声，从凶灵群中响起，让人毛骨悚然。
<br/>　　“聂兄，交给你们两人了！”
<br/>　　廖云剑吩咐一声，随即向韩盈示意，便向一旁闪掠而去，速度如风似电。
<br/>　　他们要趁着苏莫和聂乾吸引住众凶灵，然后乘机进入宫殿中。
<br/>　　铺天盖地的凶灵，疯狂的向苏莫和聂乾扑了上来。
<br/>　　聂乾不敢怠慢，立刻疯狂的攻击了起来，铁拳不断的轰出，漫天的拳影浩浩荡荡，击向八方。
<br/>　　这些凶灵虽然没有什么攻击手段，但着实不弱，张牙舞爪间，便将所有拳影撕成了碎片。
<br/>　　聂乾见此，面色微变，立刻一边攻击一边后退，凶灵太多，让他感觉头皮发麻。
<br/>　　桀桀桀！！
<br/>　　嘎嘎嘎！！
<br/>　　刺耳的怪笑声冲击人的耳膜，聂乾的攻击已经激怒了所有的凶灵，铺天盖地的凶灵很快便包围了上来。
<br/>　　而此刻，因为凶灵都被聂乾和苏莫吸引，廖云剑带着韩盈，一路畅通无阻，已经接近了远处的那座宫殿。
<br/>　　虽然宫殿周围还有少量的凶灵，但根本挡不住廖云剑。
<br/>　　“聂兄、苏莫，你们要撑住啊！”廖云剑朗喝一声，声音中带着难以言喻的笑意，很快他便带着韩盈，冲进了宫殿中。
<br/>　　轰轰轰！！
<br/>　　聂乾不停的攻击飞扑而来的凶灵，爆响连连，不过凶灵太多，他虽然能勉强轰杀几只，但根本抵挡不住众多凶灵的围杀，只能不停的后退。
<br/>　　“苏兄，你怎么不出手？”后退之间，聂乾发现苏莫一直跟在他身边，但却并不出手，这让他顿时有些恼怒了起来。
<br/>　　“出手也没用，这些凶灵根本杀不死！”苏莫面色阴沉的说道。
<br/>　　“什么？”聂乾闻言一惊，随即眼眸扫视四方，顿时双眼一瞪。
<br/>　　因为他发现，之前被他轰杀的一些凶灵，身体被轰碎之后，居然又再次聚拢在了一起。
<br/>　　这些凶灵，居然无法轰杀！
<br/>　　聂乾顿时惊恐了起来，凶灵无法被杀死，那他们遭到这么多凶灵的围杀，岂不是凶多吉少了！
<br/>　　就算他们能抵挡片刻，但也挡不住多久的时间。
<br/>　　“廖兄居然没提前告诉我们！”聂乾脸色有些发白，心中急思如何才能冲破众多凶灵的围杀，而逃出去。
<br/>　　苏莫闻言翻了翻白眼，真为这个聂乾的智商感到着急。
<br/>　　“他只是把我们当炮灰而已！”苏莫淡淡的说道。
<br/>　　“什么？”聂乾闻言，顿时脸色一沉，他并不傻，经苏莫一提醒，立刻便明白了。
<br/>　　“这个混蛋！”聂乾大怒，口中怒喝一声，随即急忙向苏莫问道：“那我们该怎么办？”
<br/>　　一边询问，聂乾也没有停止出手，他双拳齐出，轰出漫天的拳影抵挡凶灵。
<br/>　　“交给我吧！”苏莫轻笑一声，随即他背后虚影升腾，一个硕大的黑色漩涡升腾而起。
<br/>　　这一刻，苏莫直接释放出了吞噬武魂。
