夏夕绾推开办公室的门，抬眸向里面看去，办公室里面有几个国外来的医学教授，都是神经科的顶尖人物，他们拿着陆寒霆的病历在那里激烈争论些什么，吵得面红耳赤，最后都摇了摇头。
<br/>
<br/>        夏夕绾的目光落在了窗户那里，现在窗户拉开了，有一道高大沉稳的身躯伫立在那里，正在抽烟。
<br/>
<br/>        这些人的吵架并不能影响到他，他指尖染着香烟，听到开门声，他侧过身，那双深邃如墨染的狭眸看了过来，落在了她的身上。
<br/>
<br/>        两秒后他拔腿走到了办公桌边，单手抄裤兜里微微弯腰将指尖的半截香烟掐灭在了烟灰缸里，他弯腰时，商务马甲上的那枚胸针流苏在空中荡漾出潋滟瑰丽的弧线。
<br/>
<br/>        夏夕绾已经在财经杂志上见过陆司爵了，但是现在见到本人，她依然觉得心悸，这个惊艳了一个商业时代的传奇男人比她想象的更加优雅倨贵，她也见过现在帝都的几大公子，陆寒霆，陆子羡，徐少南…都是风云人物，但是跟陆司爵这个长辈相比总差了一些什么。
<br/>
<br/>        夏夕绾知道了，他们差的是陆司爵身上最纯正的贵族气息，还有一个男人浸淫商场几十年，位居高位的帝王气场。
<br/>
<br/>        刚才他投来的一眼平淡的不含任何情绪，但是夏夕绾就觉得他好像看到了她的灵魂深处，已经将她看穿。
<br/>
<br/>        这时“叩叩”的敲门声响起了，宋明站在门边看着那几个外国教授，用一口流利的英文跟他们交流了几句，那几个外国教授就离开了。
<br/>
<br/>        宋明又看了里面的陆司爵一眼，十分恭敬的一眼，然后关上门。
<br/>
<br/>        夏夕绾跟宋明才打交道的，不过宋明给她的第一印象就是精明而倨傲，很有才干，但是宋明看陆司爵的眼神是十分恭敬的，还带着对上司的无比忠诚和崇拜，能让下属这样死心塌地的，肯定是陆司爵这个上司的个人魅力了。
<br/>
<br/>        办公室里就剩下了两个人，陆司爵坐在了办公椅上，用目光指了一下对面的椅子，然后开腔道，“夏小姐，请坐。”
<br/>
<br/>        他低沉的嗓音像珍藏版的法国红酒，十分磁性好听。
<br/>
<br/>        明明是很平常的语气，但是莫名透着一股深沉的威严，让人臣服。
<br/>
<br/>        夏夕绾走过去，坐在了对面。
<br/>
<br/>        陆司爵抬眸看过来，面色始终很平淡，不透喜怒，“冒昧请夏小姐过来还请见谅，不过我们需要谈一谈阿霆。”
<br/>
<br/>        夏夕绾澄亮的翦瞳看着他，也没有丝毫的怯场，“陆伯父想谈什么？”
<br/>
<br/>        陆司爵拿出了一个信封，推给她。
<br/>
<br/>        夏夕绾打开了信封，里面全是照片。
<br/>
<br/>        她拿起一张，这张照片是她从医院里出去后失明了，站在大街上打电话给他，希望他来接自己。
<br/>
<br/>        当时他在电话里冷漠薄情，说不来，她自己过马路的时候感觉到了他，向他伸出手问，陆先生，是你吗？
<br/>
<br/>        夏夕绾看着这张照片，整座海城霓虹灯璀璨，她和他站在马路中央，她向他伸出手，他痛苦而缱绻的望着她，这一幕被定格在了照片里，一座城市因为他们而安静下来。
