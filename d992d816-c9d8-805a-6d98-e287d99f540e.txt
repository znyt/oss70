        &lt;h3&gt;第四十九章  郝景文福建分公司成立&lt;/h3&gt;
<br/>
<br/>        因了冷三叔的原因，郝景文在家乡租房子都有些为难，只好用家里的房子，人员上面也更是为难，原本别的地方直接就是别人加盟，反倒到了家门口会吃闭门羹。
<br/>
<br/>        郝家其实是在业界是数一数二的大佬，只是冷三叔一直不服气，本来两家一直关系不错，后来郝爸爸研制了新的产品，将冷家拉了一大截，所以冷三叔心里很是不舒服，然后郝景文又重新开始做新的事情，冷三叔怎么会帮助郝家？
<br/>
<br/>        平时维持着表面的关系，现在就更不想维持了。
<br/>
<br/>        当然，郝爸爸是心怀宽广的人，也就没有想到这一层，让郝景文去当然会碰壁，不但会碰壁，而且冷三叔逢人就说郝景文不务正业，好好的茶叶生意不做，去搞什么运输，运输是一般人能搞的吗？
<br/>
<br/>        而且那些个东西来来去去的，也不知道是些什么东西，脏兮兮，放家里乱七八糟的，把家里都弄得晦气了，大家听冷三叔如此一说，便也不想租房子给郝景文了。
<br/>
<br/>        人说好事不出门，坏事传千里，郝景文明明做的是一件好事情，被有心人一利用，郝景文便遭到了抵触。
<br/>
<br/>        各路亲戚朋友也来郝家对郝爸爸说：“景文这孩子读书回来，好好的不帮着家里干事情，搞什么物流？”
<br/>
<br/>        “孩子想干什么就干什么，年轻人你不给他机会，他永远不知道自己能干什么。”
<br/>
<br/>        “那干个别的总比搞物流好吧，听说是从外面运一些乱七八糟的东西，然后给别人送，丢人现眼的。”
<br/>
<br/>        “怎么叫乱七八糟的东西呢？你的茶叶卖不卖？”
<br/>
<br/>        “卖啊。”
<br/>
<br/>        “卖茶叶要不要托运？你的茶叶是乱七八糟的东西吗？”
<br/>
<br/>        “怎么可能，我们的茶叶可是金贵的东西啊。”
<br/>
<br/>        “文仔搞的这个事情就是帮大家托运茶叶的，而且运费便宜，速度更快，你说是不是乱七八糟的东西？况且别人给你们邮寄的东西，也会直接给你们送到家里，简便快捷，这样的事情不好吗？”
<br/>
<br/>        “你这样一说，还真的好事啊，对大家也很好啊。”
<br/>
<br/>        “文仔在别的城市已经开了很多家，别人都很看好这个事情，都是追着找文仔，到了我们这里反而不行了，你说奇不奇怪？”郝爸爸就问问大家奇怪不奇怪，别的也不说。
<br/>
<br/>        “明天我找文仔了解了解情况，文仔今天不在家吗？”
<br/>
<br/>        “出去招人去了。”
<br/>
<br/>        “要招什么样的人？”亲戚问。
<br/>
<br/>        “文仔说要思想品德好的，身体健康，能骑摩托车就行。”
<br/>
<br/>        “我们几个家里的孩子都可以骑摩托车，而且知根知底的，孩子们也没做过什么坏事。”
<br/>
<br/>        “这个我说了不算，你们明天去问文仔吧。”
<br/>
<br/>        郝景文首先确定了地点，原本想着寻找可以入股、加盟的，但是，看样子，都没有人会愿意，不行了就自己招人，先期把点建立起来，再经过培训，找一个信得过的店长，郝景文在街道支了一张桌子开始招人。
<br/>
<br/>        一整天郝景文就招了三个人。
<br/>
<br/>        晚上回到家，父亲问郝景文：“今天什么情况？”
<br/>
<br/>        “招到了三个人，不是很理想。”
<br/>
<br/>        “今天来了几个亲戚，说他们家里的孩子也是可以的，你明天看看情况。”
<br/>
<br/>        “爸，我害怕亲戚之间不好管理。”
<br/>
<br/>        “你看情况吧，条件先讲好。能用就用，不能用就重新招人。”
<br/>
<br/>        “好的。”
<br/>
<br/>        第二天来了几个族里的年轻人，大家都是认识的。
<br/>
<br/>        “文哥，你单独出来创业不叫我们？”
<br/>
<br/>        “这个不是怕大家看不上这个事情吗？”
<br/>
<br/>        “你都没有给我们说，人家说打虎需得亲兄弟，上阵还需父子兵，我们可是你的兄弟啊。”
<br/>
<br/>        “是啊，文哥，我可是都听说了，冷家说你不务正业，我们就想看看你到底务的是那门子业，我们也想和你一同不务正业。”
<br/>
<br/>        郝景文真的是从内心感到温暖和激动，是啊，他们说得很对，打虎需得亲兄弟，上阵还需父子兵！前几天的不快一扫而空。
<br/>
<br/>        “对了，文哥，听我爸爸说，你的这个可以是入股，你就当我们是入股吧。需要多少钱，我们平均分担。”
<br/>
<br/>        “我们目前有了运输的车辆，现在只是缺人手，地方也有了，我自家的院子腾出来了。如果大家愿意陪着我一起折腾，那我们就一起做一番事情。但是每个行业都会有优胜劣汰的规则在，如果我们服务质量好了，生存的空间就大。”
<br/>
<br/>        “放心，我们会和你一起同甘共苦的。”
<br/>
<br/>        “好，我们有肉一起吃，有汤一起喝。”
<br/>
<br/>        郝景文在老家的分公司成立了，虽然开头有些艰难，但是结果还是令人满意的。
<br/>
<br/>        经过郝景文和一起来的同事的培训，大家基本都会了流程，陈子昂公司派人带来了电脑，并且培训大家学会了使用app，郝景文在福建的第一家分公司成立了。
<br/>
<br/>        他又教会了大家怎么去地市级县城拓展业务，社会在发展，人们的需求也在改变，不要小看那些小县城，他们的运购能力也是非常大的。
<br/>
<br/>        为了庆祝分公司顺利开业，郝景文将大家聚在一起，吃了顿饭庆祝他们公司成立。
<br/>
<br/>        “我呢是真的要感谢大家如此的支持我，你们对这个行业不是很了解，却义无反顾，并且还是在大家都不看好的情况下陪着我一起折腾，来，兄弟们，干了这杯酒。”
<br/>
<br/>        “文哥，我们是兄弟，爱拼才会赢，只要我们拧成一股绳一起努力，没有不成功的事情。”大家都是血气方刚的年轻人，愿意在一起做事情，无论前方是荆棘密布还是康庄大道，只要认准了就不会回头。
<br/>
<br/>        能入股的一般也是家世差不多的，能拿出来闲钱和人力的，这些年轻人也不是憨憨，家里的事业顺风顺水，努力一下开展一个全新的产业，成功了更好，不成功他们也会有退路，但是既然想干就一定要干好，他们做的这个工作，属于服务型的，只要细心，耐心，服务态度好，基本是没有任何问题的。
<br/>
<br/>        陈子昂的app上来之后，他们不但是送货而且可以收货，这样就形成了双向收益。
<br/>
<br/>        经过一段时间，他们已经在全国各地铺开了他们的网点。
<br/>
<br/>        但是，运输业之间的竞争也越来越激烈。
<br/>
<br/>        时代的发展催生了很多新兴的产业，同样的，大家看着有利可图的情况下，也会前仆后继踏足于某一个行业，经常拼得头破血流，于是一部分有恒心有毅力的人会坚持下来，也有的人呢会寻求突破口，寻找新的商机。
<br/>
<br/>        寒伯安兄弟三人和陈子昂属于双向型人才，他们会坚持初心，也会寻求新的商机，适时进行转型。
