　　此时远方的陈少爷已经开始了分流人物，那些扑面而来的丧尸群开始朝着不同的方向分散，最终会在不久后包围这座基地。
<br/>　　无数的丧尸就像是潮水，源源不断的依照分流从四面八方涌来，南署基地就像是这一片黑色海洋中的一座孤岛，即将被这恐怖的黑色洪流淹没。
<br/>　　楚涵站在指挥台的空地上，可以轻而易举的看到这一切，更能深刻感受到这场尸潮战最直面的压力。
<br/>　　身后的天照3000已经腾空，高少辉没有想到自己也会加入到这一战之中，甚至和狼牙战团的空军一同作战。
<br/>　　无数的士兵都抬着头，望向最高处的指挥台，将目光不约而同的看向这场战役的总指挥官楚涵上将。
<br/>　　楚涵没有做过多的停顿，只是看了四周那铺面而来的尸潮一眼，就直接走进了指挥室。
<br/>　　何峰等人全部退居一旁，冲着楚涵简单点头，他们知道战前的士气需要鼓舞，何峰之所以没有任何宣言，就是在等楚涵来做这件事。
<br/>　　楚涵没有看任何人，踏入屋内的第一件事就是拿起话筒，顿时明亮的声音就传递到了基地内的所有角落，也传递到了所有人的耳中。
<br/>　　“我是楚涵。”
<br/>　　这是楚涵打开扬声器后的第一句话，没头没脑更有些莫名张扬，他的自我介绍用的为‘是’这个字，并且没有添加任何后缀，哪怕他的名字后面有一系列官方头衔，可无论是上将还是狼牙统帅还是战役总指挥，他都没有用。
<br/>　　而出乎意料的，在这仅仅四个字一出口之际，整个南署基地就爆发了一阵爆炸般的震响。
<br/>　　那是九万人的齐声呼喊，振奋的情绪一下子到达高·潮。
<br/>　　只是‘楚涵’这一个名字，就可以到达无数效果！
<br/>　　何峰向来平淡的神色都微微触动，他感受着这处基地战场的火爆气氛，深深觉得楚涵此人就是一个奇迹，一瞬间就将原本压抑的气氛扭转。
<br/>　　当下的狼牙战团，当下的这九万人部队，好似已经强大的无所不能。
<br/>　　此时距离第一只丧尸触碰到城墙还有几分钟时间，所有人都已经听到了那恐怖尸潮呼啸而来的声音，都感受到了大地的拼命震动，这场战役就像是一场无法冲破的暴风雨，给人一种绝望的压力感。
<br/>　　楚涵利用这仅有的最后几分钟开口：“你们好，九万人的狼牙战团。”
<br/>　　又是一句极短的话，但却强势无比，将这九万人全部划入了狼牙战团之中，虽然训练时狼牙各军官都心中默认，但却从未有过真正的入团仪式，更不像在狼牙基地时那么正式。
<br/>　　但此时此刻在战斗前的这一刻，楚涵的承认，无疑是最大的认可。
<br/>　　谁都清楚狼牙战团的门槛有多高，无数人远赴狼牙却无功而返，更有无数人无比羡慕那座狼牙墓碑，那是真正的活着名震天下，死后无上荣耀！
<br/>　　这时候第一批飞到外围的空军，已经遇上了速度快的飞行丧尸，被和上改造过后的战斗直升机内安置着最新武器，飞行员专注于闪躲和纠缠，射击队成员专注于瞄准射击。
<br/>　　这么配合之中，南署基地的周围高空正在不断爆发着火和光，‘轰轰轰’的攻击声音更彻响不断。
<br/>　　待在外墙内坚守另一批陆地新武器的防守队，也已经做好准备，全神贯注的等待越来越近的尸潮进入攻击区域。
<br/>　　在这无数噪杂的声音响起之时，楚涵的声音也在继续：“我知道大家在这两个月过的很辛苦，狼牙的那些老兵一定没给你们好果子吃，军法处长刘玉定还动不动来个突袭。”
<br/>　　像是拉家常一样的话，在此时此刻与已经爆发的恐怖战斗形成强烈反差，更有种莫名的维和感，但这些话听在那些新兵耳中，却深有感触倍感亲切。
<br/>　　他们从来不知道楚涵是这样的人，更没想过传说出如同暴君一般的这名上将，说起话来如此平静亲切。
<br/>　　扬声器内传来楚涵轻微的笑声，好似是能够脑补这批新兵训练时的惨样，紧接着他继续说道：“不过你们也不用心里不平衡，当初狼牙刚成立时的第一批人是我亲自训练，他们受的苦可不比你们少，尤其是那会儿的刘玉定还不是一个成熟的军法处长，经常胡乱惩罚搅得整个战团不得安宁……现在想起来这些事情还历历在目，恍如隔日。”
<br/>　　说罢，楚涵也露出了一丝回忆的神态，听着这番话的众人也微微晃神。
<br/>　　可指挥室内的何峰等人，就一下子表情紧张起来，他们不明白战前的几分钟如此紧急的时刻，楚涵说这些干什么？
<br/>　　这不但对鼓舞士气毫无用处，还会让人分神，这可是大忌！
<br/>　　而就在何峰等人焦急无措，听着那番话的几万人晃神了几秒钟后，楚涵却忽然目光清明起来，眼神中隐隐浮上了另一重意味。
<br/>　　他忽然一改几秒钟前的平淡语气，声音无比宏亮又充满气魄：“狼牙成立时长不过两年，从无败绩！我们死过很多同伴，牺牲过很多优秀战士，但死在我们手中的敌军，却数不胜数！”
<br/>　　“印刻在狼牙墓碑上的那一个个名字，哪个不是战绩累累，哪个不是手中沾满了上万丧尸血，又有哪个不是名扬远播？”
<br/>　　“我们无数次逆袭，一次又一次的创造奇迹，我们屠丧尸，斩异种！让所有不看好我们的人哑口无言！”
<br/>　　“我们杀敌！我们远赴！我们征战！”
<br/>　　“生命不息，战斗不止！”
<br/>　　铿锵有力的声音，仿佛一次次重击，激荡在众人的心魂。
<br/>　　基地内一片鸦雀无声，所有人都被楚涵近乎于咆哮的宣言所镇，这是何等的气魄，在这一刻彻底爆发。
<br/>　　“千万丧尸又如何？”楚涵眼神坚定，口气狂野：“那就绝地生还！背水一战！”
<br/>　　话落，楚涵放下话筒。
<br/>　　这时候基地外的第一批尸潮恰好到达了防守队的攻击距离。
<br/>　　轰！
<br/>　　轰轰轰！
<br/>　　连续的好几声巨响在基地外的四周发出震耳欲聋的声音，好似是在回应楚涵的宣言结束。
<br/>　　紧接着下一秒，九万人的咆哮声近乎在同一刻响起，在整个基地回荡：“背水一战！”
