　　妖异青年三对一这种话傲慢的话一出，青龙跟牛妖王顿时勃然大怒，气息飙升。

<br/>　　离幽浑然一惧，帝皇之气愈演愈烈，让无数蛇种诚惶诚恐，来自灵魂深处就不自由的害怕惶恐起来。
<br/>　　同品种血统的压制，比官大一级压死人还要恐怖。

<br/>　　涂小安证道三阶，又到了要经历一次阶段性的进化，这是非常关键的系统好处，能得到实力的质变。

<br/>　　如今，这条超等帝皇烈阳蛇冒出来，到是有种给涂小安雪中送炭的感觉。
<br/>　　本来涂小安在遇见白家的那条祖奶奶蛇妖后，他就视对方为自己第三阶段性进化的蛇种。
<br/>　　毕竟血统高，实力强的蛇种不是大白菜。
<br/>　　但现在看来，显然眼前这条王蛇更加的适合涂小安。
<br/>　　他的帝皇之气，并不太逊色涂小安散发的蛇王气息，都对蛇类有着天生致命的压制。
<br/>　　如果将离幽给吞噬了，那么涂小安的蛇王威严更甚，实力又可以得到一次质变，最关键还能获得新的元素属性能力。
<br/>　　元素属性能力这种手段可技多不压身，纵然涂小安心中四种元素能力在手，也希望多多益善。
<br/>　　因为元素能力太好用了，进可攻退可守，攻击手段千变万化，威力还强大。
<br/>　　他也非常的期待，自己第五种元素属性力量会是什么。
<br/>　　若是真的完成了阶段性的进化，那么涂小安就真的不将妖灵山脉五大深山妖王放在眼中了。
<br/>　　如此一想，涂小安简直对眼前挑衅自己的这条王蛇越看越顺眼。
<br/>　　“本王接受你的挑战”涂小安此时嘴角微勾，浅笑吟吟，这送上门的上等蛇种，不吞白不吞啊。
<br/>　　话声一落，涂小安从泰坦巨蟒的头顶悬浮了起来，他们这种级别，自然要在天上打。
<br/>　　况且陆地是蛇鼠的战争场地。
<br/>　　刚悬浮起来，却见落子烟担忧的喊了声：“小安”
<br/>　　他看着绝美容颜的她，笑道：“你不会怕我输吧？”
<br/>　　落子烟立刻摇了摇头，嗔白了他一眼，轻柔的说：“我是希望你到时候能留离幽一命，虽然他自命不凡，目空一切，但无可否认，在你还没出现的时候，他是蛇族最出色的蛇种，天生帝皇气，一员虎将，他若死了，是整个蛇族的损失”
<br/>　　落子烟站在整个蛇族的角度去考虑大局，到是没有错。
<br/>　　同样是蛇，相煎何太急，但涂小安却摇了摇头：“不好意思，我无法答应你，因为蛇王威严岂容随便被挑衅，挑衅者死，况且如果我吞了它，自身修为会得到一次质变”
<br/>　　“你吞了离幽，实力会得到很大的一次提升吗？”落子烟闻言一愣的问。
<br/>　　“对，没错，若是成功了，我有把握战胜灰妖王”
<br/>　　落子烟顿时嫣然一笑：“那你就当我刚才那句话没说，你想怎么做就怎么做”
<br/>　　涂小安怔怔的看着她，这个女人，这种立场？
<br/>　　简直爱死了，完全已他第一位啊。
<br/>　　其实涂小安代表的就是蛇族了，王越强，种族自然会越强，也没什么毛病。
<br/>　　旋即涂小安深深的看了落子烟一眼，悬浮而上，高空而立。
<br/>　　“哼，我离幽会天下群蛇知道，谁才配当蛇族的王”
<br/>　　离幽也腾空的悬浮了起来，底气十足。
<br/>　　“白痴！”
<br/>　　青龙跟牛妖王碎口一骂，看妖异青年就跟看一个小儿智障一样。

<br/>　　“我猜这二货十分钟就会被主人吞到肚子里去”青龙淡淡的发表自己的言论。
<br/>　　牛妖王脱口而出：“这傻吊蛇，能撑过五分钟，就已经是走向人生巅峰了”
<br/>　　“什么情况啊”
<br/>　　猿猴王从战争场地飞了过来，看向高空的两道蛇类气息强大的男子询问。
<br/>　　“主人蛇家族有个二货，不服主人的蛇王地位，现在挑战呢”青龙说道。
<br/>　　猿猴王闻言，大声道：“岂有此理，这是以下犯上啊，君王一脉，岂容挑衅”
<br/>　　这要是在他的猴族，有泼猴出来挑衅，一巴掌猴脑都给拍出来。
<br/>　　种族势力，君王掌控生死大权，地位高高在上，子民必须无条件臣服。
