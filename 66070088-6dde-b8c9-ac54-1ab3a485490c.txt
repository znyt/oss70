　　嗤嗤哧！！！
<br/>　　无尽剑气斩过，血雨腥风，冲天的血箭如同喷泉一般，喷射在了阵法气罩之上。
<br/>　　两人都没有来得及发出一声惨叫，便被瞬间绞杀成了碎片，化为漫天残渣，伴随着体内倾泄而出的玄力，迸射八方。
<br/>　　苏莫眼疾手快，一步踏出，身形如剑一般，直接冲进了漫天气浪之中，瞬间将巫烈的烈焰石战魂吞噬了。
<br/>　　至于巫玉儿的战魂，融入长剑之中，在刚才的猛烈相击之下，已经彻底湮灭了。
<br/>　　苏莫也不担心被人发现，因为战台上一片混乱，气浪汹涌，魔气滚动，扫荡八方，他吞噬火焰石战魂只是一瞬间的事，根本不可能被人发现。
<br/>　　少倾，气浪缓缓消散，战台上彻底恢复了平静。
<br/>　　巫烈和巫玉儿两人，已经彻底不存在了，只剩下‘巫尘’瘦小的身躯，笔直的伫立在战台上。
<br/>　　整个武斗场，一片死寂，所有人都是一脸呆滞。
<br/>　　整个战斗过程，还不到一个呼吸的时间，居然就结束了。
<br/>　　而战斗结果，与众人的预料截然相反。
<br/>　　武帝境三重的巫尘，呼吸间就镇杀了武帝境四重的巫烈和巫玉儿！
<br/>　　这简直太出乎预料了！
<br/>　　少倾，整个武斗场一片哗然，呼喝声、惊讶声冲天而起。
<br/>　　“怎么可能？巫尘怎么会这么强？”
<br/>　　“他修炼的什么魔功？”
<br/>　　“这巫尘居然这么强？为何以前从未听说过？”
<br/>　　“好强烈的剑威！越级战斗，以一敌二，挥手间便解决了！”
<br/>　　“难怪他有恃无恐！
<br/>　　众人议论纷纷，均是不可思议的望着苏莫。
<br/>　　一个以前名声不显，天赋平平的人，突然崛起，给众人的冲击非常的大。
<br/>　　当然，除了震惊之外，很多人的脸色都阴沉了下来。
<br/>　　他们很多人都是希望巫尘死的，毕竟巫尘人面兽心，太过卑劣了。
<br/>　　但现在巫尘不仅没死，反而将巫烈和巫玉儿杀了，这如何是好！
<br/>　　巫天御面无表情，双眸微眯了下来，紧紧的盯着战台上的苏莫。
<br/>　　他也是没想到，这个巫尘居然有这等实力，因为就算是他出手，也没有把握如此快的解决巫烈和巫玉儿。
<br/>　　当然，他的修为只有武帝境二重境界，比巫尘还要低一重的境界。
<br/>　　但不管如何，巫尘的实力，比他也不遑多让了，只高不低。
<br/>　　目光凝视着巫尘的背影，巫天御心众突兀的升起了一丝熟悉的感觉，但具体哪里熟悉，他也说不上来。
<br/>　　战台上，彻底斩杀了巫烈和巫玉儿，苏莫心中松了口气。
<br/>　　这两人死了，他的身份暴露的可能性，就大大的降低了。
<br/>　　少倾之后，战台上的阵法气罩打开了，苏莫身形一闪，飞身走下了战台。
<br/>　　随即，苏莫也不停留，大步向武斗场之外走去。
<br/>　　“站住！”
<br/>　　眼看着苏莫即将走出武斗场，一声暴喝突然想起，震动视野。
<br/>　　苏莫闻言，转头看去，便看到看台之上一名高大的中年人站了起来。
<br/>　　此人的面容和巫烈有三分相似，苏莫心中唯一思索，便知道了此人的身份。
<br/>　　此人，是巫族中的一名普通长老，好像叫巫源，是巫烈的叔叔。
<br/>　　此刻，这名中年人身形微微颤抖，面色铁青至极，身上气息浩荡不休，望向苏莫的眼眸中，充满了强烈的杀机。
<br/>　　“巫尘，你杀了烈儿，难道就想这么走了吗？”巫源怒声喝道，他作为巫烈的叔叔，自然不能让巫烈就这么死了，不然等到日后大哥回来，该如何交代！
<br/>　　“怎么？武斗场公平比试，巫烈技不如我，死了也是活该，难道你还想违反族规，为他报仇不成？”苏莫冷笑着道，虽然对方是长老，但他也丝毫无惧。
<br/>　　巫族族规相当森严，对方根本不敢大庭广众之下违反族规。
<br/>　　当然，苏莫是在玩火，极易引火烧身，一个不慎就要遭受灭顶之灾。
<br/>　　但他也是豁出去了，此次来到巫族，他不达目的誓不罢休。
<br/>　　要么他惨死在巫族之中，要么是他在巫族之内崛起，然后将巫族搅个天翻地覆。
<br/>　　“哼！小畜生，你人面兽心，人人得而诛之，何需本长老出手！”
<br/>　　巫源冷哼一声，随即目光环视四周，朗声道：“巫尘这个畜生，活着也是我族的耻辱，有那个小辈愿意向他挑战，将他斩杀，本长老奉上下品灵晶一万！”
<br/>　　巫源此言一出，现场为之一静，不少人都倒吸一口凉气，峰分别呼吸急促了起来。
<br/>　　一万下品灵晶，这可是一笔巨富啊！
<br/>　　对于一般的武帝境武者而言，一万下品灵晶，足够使用几年了，几年之内都不用为灵晶操心了！
<br/>　　很多人心中大动，本来他们就厌恶巫尘的畜生行径，盼着巫尘被杀死。
<br/>　　现在有了巫源的奖励，很多人都动心了！
<br/>　　嗖！
<br/>　　一名身穿黑袍的巫族青年，一马当先，率先飞落到了战台之上。
<br/>　　“巫尘，我向你挑战！上来一战吧！”青年目光冷视着苏莫，厉声喝道。
<br/>　　苏莫扫视了对方一眼，顿时嗤笑一声，道：“你是白痴吗？武帝境五重修为，挑战我一个武帝境三重武者！”
<br/>　　“你……！”青年顿时语塞，他也知道对方不太可能应战，但对方的言语未免太侮辱人了吧！”
<br/>　　“我和你战！”
<br/>　　“我向你挑战！”
<br/>　　“敢否于我一战？”
<br/>　　“……”
<br/>　　随即，厉喝声四起，人影闪烁，足足有十几道身影飞落在了战台之上，均是要向苏莫挑战。
<br/>　　这些人均是武帝境三重修为，修为和苏莫相当，显然都是自认为战力强大之辈，即便知道苏莫实力超强，也丝毫没有畏惧。
<br/>　　苏莫见此，顿时心中无语，自己这是捅了马蜂窝了吗？
<br/>　　这些人，这么想要他死！
<br/>　　略一沉吟，苏莫心中冷笑了起来，既然这些人想战，他自然不会拒绝。
<br/>　　这些巫族的天才，能多杀一个都是赚的，而且他吞噬的越多，实力越是强大。
<br/>　　再者，和这些巫族天才一战，他的名声就能彻底打出去了。
<br/>　　到时候少司选拨，他也能提前进入族中高层的视野，成为重点关注的对象，这对于他日后成为巫族的少司，极为有利。
<br/>　　“今日累了，三日之后，想挑战我的人，我巫尘全部接了！”留下一句张狂的话语，苏莫转身大步离开了武斗场。
