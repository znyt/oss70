　　这古籍，正是楚风眠脑海之中的无数剑术所凝聚的。
<br/>　　这古籍之中所记载的，每一页都是远古时代精妙的剑术。
<br/>　　若是将这一本古籍拿出去，绝对是足以让九域之中的无数强者争夺，甚至是七大宗门都要争夺的宝物。
<br/>　　不过现在，逐渐的在楚风眠的脑海之中，这一本古籍，开始变的越来越薄，其中记载的无数剑术，只留下了其中精髓，剩下的一切杂质，都被楚风眠直接炼化。
<br/>　　在楚风眠的神识扫动之下，这剑术被不断的浓缩，逐渐的化为了一道剑术。
<br/>　　这一道剑术，从这古籍之中脱胎换骨出来，便是比起之前古籍之中所记载的任何一道剑术，都要强大的多。
<br/>　　楚风眠闭目养神，时间一天一天的过去。
<br/>　　这每一分每一秒之中，楚风眠都是在淬炼这九域剑术，将其不断的浓缩，虽然楚风眠脑海之中那古籍的大小，在不断的缩小，但是其中剑术的精妙，却是达到了另外一种截然不同的层次。
<br/>　　“没想到将这剑术彻底领悟，已经是过去了一周的时间。”
<br/>　　在洞府之中，楚风眠缓缓的睁开眼，距离他修炼之前，已经是过去了一周的时间。
<br/>　　在这一周的时间之中，楚风眠终于是将九域剑术，彻底的凝聚了起来，他自创剑术，风险极大，可以说一步走错，便是就会落得一个死无葬身之地的下场。
<br/>　　幸好的楚风眠提前发现了其中的不对，现在好好的将其中剑术的杂质去掉，将无数剑术，融为一体。
<br/>　　不然的话，这些剑术许多都是力量相克，一山不容二虎，在楚风眠的身躯之中，很有可能无数剑术同时爆发力量，足以是将楚风眠的身体都给炸毁，可以说是极为危险。
<br/>　　还好楚风眠现在从容应对，才消除了这些危险。
<br/>　　不过这也是给楚风眠提了个醒，他想要凝练这九域剑术，就必须是要步步小心才行。
<br/>　　“轰！”
<br/>　　就在楚风眠，逐渐将身上的剑意，全部平复下去的一刻，从外面，一阵灵力波动，突然响起。
<br/>　　“哦？这苏荷居然是练成了？不知道她所炼制出的灵器，到底如何？”
<br/>　　楚风眠之前交给苏荷的炼器之术，实际上也正是他们曾经天兵苏家的考核之术，用来考核苏家弟子炼器资质的考核之术。
<br/>　　利用这炼器之术，所炼制的第一件灵器，便是代表着他们的炼器资质如何。
<br/>　　能够炼成黄级灵器，便可以算是苏家弟子之中还算是有些炼器资质，可以培养的人物，若是玄级，则算是炼器之道的天才了，足以值得苏家重点培养。
<br/>　　而至于地级，天级，则都是苏家千百年来难遇的天才了，甚至是有可能继承苏家家主之位的人物。
<br/>　　现在这苏荷居然是足以炼成，这也就起码证明，她的资质并不算差。
<br/>　　“去看看吧。”
<br/>　　楚风眠起身，便是将这洞府之中的无数禁制，全部毁掉，他不慌不忙，慢吞吞的走了出去，一眼就看见了那苏荷。
<br/>　　苏荷的额头上，布满了豆大的汗珠，脸上尽是疲态，一次炼器，足足便是耗费了七日的时间。
<br/>　　对楚风眠来说，炼器七日，不眠不休，倒是算不得什么，不过苏荷仅仅只有锻体境的境界，这七日的时间，几乎是已经将她的所有力气都给用干了。
<br/>　　“服下吧。”
<br/>　　楚风眠出来，看到苏荷的样子，便是心神一动，从他的空戒之中，拿出了几枚丹药，交给了苏荷。
<br/>　　“是。”
<br/>　　看到楚风眠出来，苏荷本来想要起身相迎的，不过她的身躯却是虚弱无比，只好是先接过了丹药，吞服了下去。
<br/>　　她那疲态的脸色，如今才是逐渐的变的正常起来，在瞬息之间，便是恢复完，同时这丹药的灵力，居然是还没有散去，猛然之间在苏荷的身躯之中凝聚。
<br/>　　“轰！”
<br/>　　一股灵力波动，无数灵力，从苏荷的身躯之中爆发出来。
<br/>　　她的境界，在这瞬息之间，便是一下子达到了神海境的境界。
<br/>　　这让苏荷的眼神中，都满是不可置信的神色。
<br/>　　她的境界之前，才仅仅锻体境八重的地步，现在在吞噬了这一枚丹药之后，居然便是一下子突破到了神海境。
<br/>　　而且这丹药的灵力，温和绵长，绝非是一般丹药足以比拟的。
<br/>　　像是许多丹药，也都足以做到这一点，但是其中的药力极为霸道，十个人吞服下，甚至就死九个，只有一人可以成功活下来，而像是现在楚风眠给的这枚丹药，是几乎任何人都可以吞服的。
<br/>　　“多谢前辈赐予的丹药，小女日后必定偿还。”
<br/>　　苏荷站起身来，将那道谢道。
<br/>　　“无妨，一枚天级丹药罢了，不足为虑，你炼制的灵器呢？拿来给我看看。”
<br/>　　楚风眠随意笑了笑开口道。
<br/>　　对现在的楚风眠来说，一枚普通的天级丹药，也算不得什么了，从归墟天中楚风眠所得到的天级丹药，足足是有着上万枚，大部分的丹药已经是对楚风眠没有任何意义了。
<br/>　　拿出这一枚来，对楚风眠来说的确也算不得什么。
<br/>　　“天级丹药！”
<br/>　　苏荷也猜到了这，这丹药的品级绝不会低，却没有想到，这居然是天级丹药。
<br/>　　令无数圣者都恨不得去争抢的天级丹药，现在就被楚风眠这样的送给了她，让她服下，而且看楚风眠的样子，居然还是如此的轻描淡写。
<br/>　　“此人到底是什么人？我苏家先祖，居然是跟此人认识？”
<br/>　　苏荷心中都不由的感慨一声，如今碰到了楚风眠，绝对是她的大运了。
<br/>　　不过很快，苏荷便是反应过来，拿起了旁边的一件灵剑，递给了楚风眠。
<br/>　　“这是小女炼制的灵器。”
<br/>　　“地级极品？”
<br/>　　楚风眠接过这灵剑的瞬间，便是大约感觉到了，这灵剑的品级。
<br/>　　这苏荷所炼制的第一件灵器，居然就已经是达到了地级极品的层次。
<br/>　　这件灵器整体样子，已经算是不错，只是细节上，倒是粗糙一些，不过这已经是超出了楚风眠的想象了。
<br/>　　这样的资质，放在当初的天兵苏家，都算得上是绝代天才了。
