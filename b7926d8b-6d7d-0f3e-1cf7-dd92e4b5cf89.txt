　　诡异的黑色锁链，携着压迫性的力量席卷而来，慕清霄眼中依旧古井无波，神奇领域扩散而出，在他的视线中，锁链的速度就缓慢下来。
<br/>　　百米范围内自成一方领域，领域内剑意四射，领域内的空气都变得凝重起来，一剑斩出，剑气无匹。
<br/>　　黑色锁链在恐怖的斩击下，瞬间就出现一道裂痕。
<br/>　　站在空间内，感受到周围带来的压迫感，魂殿斗宗身躯一颤，惊骇失声：“空间领域！你是斗尊!不对，气息没有变化，你...你竟然在斗皇层次就悟出了空间领域！”
<br/>　　“...不得不说，你天资是我见过最恐怖的，如此绝世妖孽，可惜不是我魂殿中人，今天不管付出任何代价，都要将你留下！”
<br/>　　回过神来，魂殿斗宗语气凝重无比，隐藏在黑袍下的双眼，死死的盯着慕清霄，谁也不知道他此刻有多么的惊骇。
<br/>　　八星斗皇，领悟空间奥妙不说，居然还悟出了空间领域，如此妖孽一般的天赋，若是继续让他成长下去，简直不敢想象！
<br/>　　此时，慕清霄神色也变得颇为凝重，这是他第一次使用空间领域，效果比他想象的更强，在领域中，不仅是速度，力量与剑意都有明显的提升。
<br/>　　现在，不用化身成黑凤，他就能以人型姿态，挡下四星斗宗的全力一击，虽然有一些勉强。
<br/>　　没有理会魂殿斗宗的话语，慕清霄眼中也浮现浓郁的杀意，暴露如此多的底牌，哪怕是魂殿斗宗也必须永远的留在此地。
<br/>　　正当魂殿斗宗准备进攻的时候，空间压迫感突然消失，使得他为之一愣，随即望着慕清霄冷笑一声。
<br/>　　“桀桀，悟出空间领域又如何，任你资质妖孽又如何？这等神通根本不是你一个斗皇能掌控的，若真能掌控，我恐怕就要栽在这里，不过现在嘛！”
<br/>　　“弱者，总是啰嗦个没完没了，你清不清楚一个道理，话越是多，死的就越快，像你这种魂殿最底层的垃圾，也有资格说留下我，你也太高看自己了吧！”
<br/>　　冰冷的语气自嘴中吐露而出，慕清霄直接扯下黑袍，神色冷漠，毫不掩饰眼眸中浓郁的杀意。
<br/>　　听的慕清霄狂妄的语气，魂殿斗宗并不在意，他现在的任务就是为魂殿抹除一大祸患，只是听他口中说出魂殿两字，瞳孔微微一缩。
<br/>　　望着映入眼中的神骏青年，还有他眼中的杀意，魂殿斗宗心底升起一股强烈的危机感，后者居然清楚他的底细。
<br/>　　神纹闪烁间，黑色火焰如浪潮一般狂涌而出，周围空间被焚烧成一片虚无，嘹亮的凤鸣声，在枫城上空回荡。
<br/>　　“凤...！”
<br/>　　魂殿斗宗话语还未说出口，死亡的气息席卷而来，到嘴的话语吞入腹中，身形诡异的扭曲，下意识的想要躲开。
<br/>　　黑色的利爪携带着诡异的火焰，眨眼就拍了下来，利爪所过之处，空间都被撕开一道黑色的裂缝，由于天色昏暗的缘故，裂缝并不明显。
<br/>　　魂殿斗宗心头狂跳，自知躲避不开，连忙抬起锁链，横档在上空，四星斗宗实力，毫无保留的倾泻而出。
<br/>　　原本，他打算将慕清霄留在此处，但现在他心中唯一想的就是，找机会逃离，将此子的信息汇报上去，但现在看来，能不能离开还是两说。
<br/>　　“咔嚓...！”
<br/>　　他终究是高估了魂器的承受极限，伴随着一阵清脆的断裂声，那无坚不摧的黑色锁链，在凤爪下，宛如豆腐渣一般，竟寸寸断裂开来。
<br/>　　凤爪丝毫没有停止的趋势，恐怖的力量携带着诡异的火焰，直接劈在魂殿斗宗的身上。
<br/>　　“这是什么火焰!”
<br/>　　当九霄琉璃焱席卷而上，只见魂殿斗宗黑袍瞬间化作灰烬，露出一个惨白虚幻的身影，神火从他的手臂向全身蔓延。
<br/>　　在神火的燃烧下，魂殿斗宗的手臂已经焚烧殆尽，接下来将是他的整个灵魂，极致的恐惧，使得他脸色狰狞无比！
<br/>　　慕清霄双翅一振，丝毫不给他喘息的机会，凤爪直接将他的灵魂扣住，狭长的眼眸中杀意四射。
<br/>　　“不，你不能杀我，你要是敢杀我，身上会留下魂殿的灵魂印记，你将遭到整个魂殿的追杀！”
<br/>　　竭斯底里的惨叫声，自魂殿斗宗口中爆发而出，望着眼前两丈多高的黑色凤凰，眼中满是恐惧。
<br/>　　然而，九霄琉璃焱丝毫没有泯灭的趋势，慕清霄眼中满是不屑，他当然知道灵魂印记，那又如何？
<br/>　　凤爪一捏，恐怖的火焰就席卷而出，‘蓬’的一声，魂殿斗宗的灵魂就如气球一样炸裂，最后在神火中，连一丝渣都没有剩下。
<br/>　　整个世界都清静了下来，后者死亡的瞬间，一缕赤红色光点便快速钻进他的身体中。
<br/>　　漫天神火收拢，慕清霄重新恢复人形姿态，内视检查身体每一个角落，最后发现，心脏部位确实有一点赤红色的灵魂印记。
<br/>　　灵魂印记小的犹如灰尘，普通方式根本难以察觉，但他根本不能用人类来形容，体内有一丝的异样，就像喉咙中恰着一根刺，慕清霄能够清晰的感应到。
<br/>　　“想依靠灵魂印记追杀我，可惜这种东西对我来说，根本不足为道。”
<br/>　　慕清霄眼底上过一丝不屑，对于别人来说，心脏是最关键的部位，印上灵魂印记后，根本很难消除，但对他来说就完全不同了。
<br/>　　哪怕将心脏毁了，他都能够活下去，然后再造，更别说小小的印记，心念一动，九霄琉璃焱便从心脏中狂涌而出，流淌在四肢百骸中。
<br/>　　在恐怖的神火下，心脏上的赤红色印记，眨眼间就被抹除，体内异物消失，慕清霄舒爽的伸了个拦腰。
<br/>　　只要在给他一点时间，别说魂殿，魂族能奈我何？
<br/>　　朝四周瞥了一眼，刚才的凤鸣已经引来了许多气息，皆是斗皇级别，但刚才战斗的气势太恐怖，他们都不敢接近。
<br/>　　慕清霄也不理会他们，韩枫死了，对药尘的承诺也完成了，最重要的是异火榜排名第十五的海心焰到手。
<br/>　　有了海心焰，他的实力又能提高一些，神火的威力也会随之提高。
