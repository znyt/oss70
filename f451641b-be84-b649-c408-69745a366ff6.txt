　　老者看着楚风眠对着内丹感兴趣，目光闪烁。
<br/>　　直接是喊出了一个天价。
<br/>　　“一个亿，你怎么不去抢！”
<br/>　　拓跋鼎在一旁，不由的怒声说道。
<br/>　　这内丹，虽然看起来极为奇怪，但是其中并无任何妖神的气息，最多只是一枚御风境妖兽的内丹。
<br/>　　价值个几百万九鼎丹，已经是极限了，一个亿，简直就是在坑人。
<br/>　　“怎么？买不起别买，这内丹可不是寻常东西。”
<br/>　　老者冷哼一声，也是毫不客气的说道。
<br/>　　“就一个亿，拿得出来就买走，拿不出来自己离开。”
<br/>　　“一个亿是吗？好。”
<br/>　　听到老者的话，楚风眠倒是轻描淡写之间，挥了挥手。
<br/>　　楚风眠打开空戒，在他的手中，突然便是出现了一个亿的九鼎丹，直接对着那老者扔了过去。
<br/>　　同时楚风眠便是手一动，这一枚内丹，也是直接落入到了楚风眠的手中来，被楚风眠仔细把玩起来。
<br/>　　听到楚风眠开口的如此痛快，老者本来还想要在加上一些价格的，但是如今看到楚风眠如此快的拿出了丹药，拿走了内丹。
<br/>　　他也只能是接过了那一亿九鼎丹。
<br/>　　“说吧，这内丹到底是什么来路？”
<br/>　　楚风眠一边观察这内丹，一边向着老者询问道。
<br/>　　这内丹的力量，看起来也就是一只御风境巅峰妖兽的内丹，不过在其中，楚风眠却是足以感觉到一种不寻常的力量。
<br/>　　“这其实是一枚独牛内丹。”
<br/>　　听到楚风眠的询问，这老者也是缓缓开口道。
<br/>　　“这是我从一处诡异遗迹中，遇到的一头修炼到了御风境巅峰的独牛，杀了他的内丹。”
<br/>　　“独牛？”
<br/>　　楚风眠听到这老者的话，心中也不由的有些惊讶。
<br/>　　他所惊讶的，不是这独牛的稀少，而是这独牛，是在算不上是什么稀奇妖兽。
<br/>　　甚至可以说，独牛是最为常见的妖兽之一，位于妖兽的底层。
<br/>　　寻常独牛的实力，最为强大的也就达到淬骨境巅峰，能够达到锻体境实力的独牛，都是少数。
<br/>　　而这老者，居然是说着是从一头御风境巅峰的独牛身上得到的内丹？
<br/>　　独牛可以修炼到御风境的难度，简直是比起一个没有任何灵脉的废物，修炼到圣者一样。
<br/>　　几乎是绝不可能发生的事情。
<br/>　　“老家伙，你不是在胡说吧。”
<br/>　　拓跋鼎听到老者的话，不由的眉头一皱说道。
<br/>　　“这是独牛的内丹？”
<br/>　　“哼，若是你们不信，自然可以查看一下这内丹的力量，的确是独牛的力量。”
<br/>　　老者冷哼一声，开口道，语气中倒是也不是很清楚的样子。
<br/>　　“的确是不知道为什么，这独牛才会成长到如此强大的地步，不过独牛的如此变异，肯定是得到了其中一些奇遇，这内丹也是大的出奇，很有可能其中有些什么奇物。”
<br/>　　“哦？吞服了什么东西？”
<br/>　　听到老者的话，楚风眠突然眼睛一亮，一个神识，便是笼罩其中。
<br/>　　就在这独牛内丹之中，隐藏了其中最深处，却是的确有着一股极为诡异的力量。
<br/>　　这一股力量，一直是隐藏在独牛内丹深处，甚至是被独牛的重重力量笼罩着，这才是楚风眠第一时间没有察觉到。
<br/>　　如今听到老者的提醒，楚风眠特意探察，这独牛内丹中，果然有诈。
<br/>　　“这是，天晶？”
<br/>　　楚风眠的灵识，深入到这独牛内丹之中，突然他眼神一变，察觉到了其中这一股精粹到极点的灵力。
<br/>　　这一股灵力，突然让楚风眠想到了一个东西。
<br/>　　天晶。
<br/>　　天晶，并非是九域产物，而是来自与域外，是一种最为精粹灵力的凝聚体。
<br/>　　九域中的任何奇物，在灵力方面，都无法跟天晶媲美。
<br/>　　天晶，是九域中所有武者近乎梦寐以求之物，一小块天晶，都足以改改善武者的身躯，净化武者身躯血脉中的杂质。
<br/>　　像是楚风眠的身躯血脉，虽然是有着战龙决的原因，但是他吞服过一些丹药，身体内肯定是会有着一部分杂质的。
<br/>　　而天晶的能力，则是将这些杂质全部驱除。
<br/>　　足以让武者得到一个完美的身躯，一个指甲盖大小的天晶，已经是足以让寻常武者净化一次身躯了。
<br/>　　这天晶可是来自与域外，并非九域之中的东西，所以罕见到极点。
<br/>　　楚风眠也有些想不到，居然是碰到天晶。
<br/>　　在这独牛内丹中中，却是既然有着一小块的天晶碎片。
<br/>　　虽然这一小块天晶碎片，只有指甲盖大小，但是价值，只怕抵得上万亿九鼎丹的价格。
<br/>　　更何况这天晶，本身便是有市无价的东西，想要买都根本买不到。
<br/>　　“这可真是一个大便宜。”
<br/>　　楚风眠心中极为激动，天晶，他也只是听说过，这还是第一次见到。
<br/>　　他心中虽然激动，但是脸色依然还是保持平静，免得被人看出来，令人看不出楚风眠的表情。
<br/>　　一个亿的交易，在黑市中也算是一个大交易了，引起了周围不少人的注意。
<br/>　　一尊身影，刚刚走入到黑市之中，这是一尊年轻男子。
<br/>　　这人，看起来跟楚风眠年纪相仿，看起来也是来参加北邙学院考核的青年才俊，一身实力，达到神力境二重的境地。
<br/>　　算得上是一位资质不错的天才。
<br/>　　他一进入其中，便是看到了楚风眠手中的独牛内丹，眼神不由一亮，直接便是走到了楚风眠的面前，开口道。
<br/>　　“小子，将这内丹卖给我吧，这是两个亿九鼎丹。”
<br/>　　说着，这年轻男子便是直接大手一挥，拿出来了两亿九鼎丹，直接便是向着楚风眠递了过去。
<br/>　　楚风眠抬头，瞥了一眼那年轻男子，并没有理会他。
<br/>　　至于这年轻男子递过来的九鼎丹，他更是根本没有去接，完全无视了这年轻男子。
<br/>　　这年轻男子，看着楚风眠丝毫不理会他，脸上不由的露出了几分怒意。
<br/>　　直接便是站在了楚风眠的面前来，大声的吼道。
<br/>　　“小子，本少跟你说话呢，听不见吗？”
<br/>　　楚风眠看了一眼这年轻男子，眼神中露出几分冷色，口中冰冷的开口道。
<br/>　　“不卖。”
