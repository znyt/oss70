　　此时蛇王归位，普天同庆，现场山呼海啸上维持了很久很久。
<br/>　　直到道袍男子李顺走到蛇王雕像前，一举手，现场的欢呼声才慢慢的落下，渐渐安静下来。
<br/>　　李顺脸上端着典型信徒的神态，望着黑压压的几十万民众道：“蛇王大人被大家的虔诚所感动，特赐下珍贵的蛇王水，但凡来到现场的，都可以领一杯蛇王水，大家喝了蛇王水，可消灾去病，身体安康”
<br/>　　李顺的话一落，现场顿时砸开了锅，蛇王水早就名声在外，甚至还救过不少捕蛇人的性命，另外蛇王水，可不是你想喝就能喝到。
<br/>　　如今，蛇王与民同庆，恩泽天下，赐蛇王水，这可是用钱都买不到的，顿时间，一个个跟中了大奖一样的兴奋起来。
<br/>　　况且如今，已蛇王的威望，无人会怀疑蛇王水的效果，那喝了就算不能长命百岁，也能百病不缠身。
<br/>　　“蛇王大人万岁...”
<br/>　　现场的气氛一下子被掀开了，高潮一浪接着一浪，天气炎热，民众的情绪更是炽热的。
<br/>　　随后，有三位壮汉一起奋力抬着一口大水缸气喘吁吁的立于蛇王雕像前。
<br/>　　别看这口大水缸形象是不怎么样，但是里面却装满了所谓的蛇王水，而且有一股异香弥漫出来，使人闻了精神一震，毛孔舒展。
<br/>　　外表不重要，重要的是内涵，好不疑问，蛇王大人赐下的蛇王水是价值千金的，一个个冒着贪婪的目光看着这口不太好看的大水缸。
<br/>　　看似这口大水缸要三个大汉抬出来，里面全是蛇王水，但你要知道，现场多少民众啊，如此一对比这口大水缸简直就不值一提了。
<br/>　　一张巨大的长形桌子也摆放起来，一个个一次性的杯子摆上，有专门人来将杯子装满蛇王水。
<br/>　　人那么多，人人都想喝到蛇王水，自然是敬老爱幼，第一杯谁来喝，无容置疑，自然是李老爷子李得丰。
<br/>　　道袍男子尊敬的将一杯蛇王水端到李老爷子的面前，李得丰接住，顿时间，一道道目光汇聚在他老人家的身上。
<br/>　　蛇王水真正没多少人喝过，大家都非常想知道喝了蛇王水的人会有什么惊人的效果。
<br/>　　蛇王显化归位，赐下的蛇王水简直太令人期待了，信徒的崇拜是疯狂的，是没有逻辑的。
<br/>　　就算现在跟他们说，这是蛇王的尿，恐怕也会有人会抢着喝。
<br/>　　李得丰也是当仁不让，慈眉善目的一笑，然后朝着蛇王雕像微微鞠躬，然后一口而净，一点不拖泥带水。
<br/>　　“老爷子，蛇王的蛇王水味道如何，您老有什么感觉没有”
<br/>　　“老爷子，快说说，这蛇王水是苦是甜！”
<br/>　　现场的人都盯着李得丰的反应，连连的询问，非常的迫不及待。
<br/>　　蛇王神威赫赫，赐下来的圣水不可能平庸。
<br/>　　“甜，很甜，喝了之后全身都暖和，好像感觉自己都变年轻了，呵呵...！”
<br/>　　李老爷子喝了一杯蛇王水后，原本佝偻的身子居然笔直起来，面色红润，容光焕发。
<br/>　　他的满头白发既然微微的转黑了起来。
<br/>　　“大家看，老爷子的头发变黑了，真的好神奇啊”
<br/>　　不少人发现了老爷子的变化，顿时惊叹不已，就算心里早有准备，但是看到第一个喝下之人的真实变化，还是被震惊到了。
<br/>　　“谢谢蛇王大人赐圣水，让我这半截身子迈入土里的老头，还能多活几年”
<br/>　　外人就看到李得丰的白发微微转黑，但最真实的感受跟变化，只有老爷子自己心里明白。
<br/>　　他现在觉得自己的身体换发生机，好像一下子年轻了十几岁，一双深陷的眼睛明亮有光泽起来。
<br/>　　“师公，您会长命百岁的”
<br/>　　隐藏在暗处的涂小安微微一笑，这第一杯蛇王水，可是经过他的特殊定做，里面有非常足分量的杀人蜂蜂蜜，另外更是在里面加了蛇莓的果汁，由于担心一颗蛇莓的能量太充沛，涂小安就加了少许的果汁。
<br/>　　但效果已经非常的显而易见。
<br/>　　李得丰的身份非同小可，别的都不说，光自己父亲的师父，这一个身份，就足够他尊敬。
<br/>　　当年，涂小安知道自己的父亲是非常非常尊重自己的师父，一身捕蛇水平都是来源李得丰。
<br/>　　另外自己的师父何震南，也是李得丰的徒弟。
<br/>　　只是涂小安跟这位德高望重的老人家到是接触的不多，那时候他也还小。
<br/>　　除了第一杯蛇王水，大缸中的都是普通的山间泉水，在加了大量的蜂蜜进去，甘甜润喉，说不上去百病，但多多少少强身健体，至于蛇莓的果汁，也加了不少。
<br/>　　由于蛇莓会暴涨体型，能量霸道，所以涂小安也分外的慎重。
<br/>　　蛇莓他很多，没有不舍得，就怕人吃了一下自己长到二米多高，不知道是帮他还是害他。
<br/>　　但总而言之，现在每一杯蛇王水分量都是恰到好处的，可以做到强化人的身体，有病去病，有恶疾顿消。
<br/>　　只有蛇王的威望越高，涂小安才能有渊源不断的信仰之力。
<br/>　　李老爷子喝了蛇王水的样子，大家都欣喜的看在眼中，紧接着，现场开始排队，一个一个人来领取蛇王水。
<br/>　　另外除了蛇王水，挨家挨户，每一户可以免费领到一座蛇王的小型雕像，供奉在家中。

<br/>　　整整不下十万个小型蛇王雕像，做工精美，甚至都具备了涂小安的意识烙印。
<br/>　　当然，涂小安也是为了做长久的打算，一次把事情做完满了，以后就坐等收成。
<br/>　　接下来，但凡喝到蛇王水的人，都是一阵惊讶惊叹，另外顺手一个蛇王雕像就带走，简直让人流连忘返。
<br/>　　一时之间现场的气氛再次的高潮起来，喝过蛇王水的人都惊叹自己全身充满了力量，什么蛇王水啊，简直就是圣水。
<br/>　　“老天爷，我十几年的咳嗽居然被这个水喝好了，蛇王万岁！”
<br/>　　一位老妇女激动的跪在了蛇王雕像面前，连连的磕头感谢。
<br/>　　一位坐轮椅中风瘫痪了五六年的老爷爷，神奇的从一轮上站了起来，将他们的一众儿女给惊的大喊蛇王万岁。
<br/>　　一桩桩，一件件事情不断的发生，很多人身患多年的恶疾，立刻百病全消，让人不得不感叹蛇王的伟大善举。
<br/>　　功德无量呀，这才是真正的造福一方。
