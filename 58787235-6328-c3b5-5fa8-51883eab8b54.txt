　　果然，漫天紫炎之下。
<br/>　　那恐怖的炙热高温，让得数十闲散长老，连忙后退。
<br/>　　随后，纷纷脸色为难地看向木青云。
<br/>　　木青云，顿时脸色难看无比，“紫炎易霄，果然名不虚传。”
<br/>　　“众闲散长老退下，九大长老随我上。”
<br/>　　九大长老，自然是指长老头衔前带数字的二、三、四、、、九长老。
<br/>　　药王谷九大长老，皆是天元中期修为。
<br/>　　加上自己这个天元后期。
<br/>　　木青云有百分百把握拿下易霄。
<br/>　　十人，再次联手而上。
<br/>　　萧逸一道紫炎屏障，护在叶铭之上。
<br/>　　随后身影一闪，与他拉开了距离。
<br/>　　天元境的战斗，无比强悍，哪怕些许余波，都不是叶铭能承受的。
<br/>　　那漫天紫炎，轻松就困住了各位长老。
<br/>　　只有木青云一人，能突破紫炎的封锁，直攻萧逸而来。
<br/>　　萧逸，自是不惧。
<br/>　　拳头上包裹着被极度压缩的紫炎，一拳轰出。
<br/>　　木青云，冷笑一声，“不自量力。”
<br/>　　‘轰’的一声巨大轰鸣。
<br/>　　萧逸被震退数十步，木青云则纹丝不动。
<br/>　　“易霄，你虽声名在外，却不是我的对手。”木青云不屑说道。
<br/>　　“我没兴趣与你慢慢战斗，束手就擒吧。”
<br/>　　“木界封印。”
<br/>　　木界封印，药王谷地阶巅峰武技。
<br/>　　由木青云这位天元后期武者使出，威力极强。
<br/>　　无数深黑色的粗壮藤蔓，凭空而现。
<br/>　　随后，以极其玄奥的方式，互相组合，排列，形成一个个巨网。
<br/>　　巨网，似乎就是因为这种排列，变得威力极其强大。
<br/>　　巨网，从四面八方涌来，封锁了萧逸的所有退路。
<br/>　　萧逸甚至还未反应过来，四张巨网以连接在一起，形成了一巨网牢笼。
<br/>　　“给我破。”萧逸大喝一声。
<br/>　　正欲一拳轰出。
<br/>　　木青云的速度更快，低喝道，“火界封印。”
<br/>　　一道道图案玄妙的火焰，顷刻间打向巨网。
<br/>　　霎那间，巨网上火焰涌动。
<br/>　　巨网牢笼，也成了火焰牢笼。
<br/>　　牢笼之外，始终被一股玄奥的力量所加持，极其强大。
<br/>　　萧逸一连轰了数十拳，根本奈何不了牢笼分毫。
<br/>　　两种封印术结合而成的手段，加上乃是木属性与火属性相配合。
<br/>　　这个巨网牢笼，已经超出来了萧逸现今能打破的范畴。
<br/>　　这便是天元境后期强者，若是铁了心要对付萧逸。
<br/>　　一个照面，就能将萧逸拿下。
<br/>　　木青云作为药王谷大长老，乃是炎武王国上层的巅峰强者之一。
<br/>　　他的修为与实力，可想而知。
<br/>　　“哼。”木青云冷哼一声，看着牢笼内的易霄，满脸不屑。
<br/>　　“易霄，你私自吞服了我药王谷最珍贵的丹药。”
<br/>　　“那是我药王谷储存了千百年的无数珍稀天材地宝而成。”
<br/>　　“我只能将你击杀。”
<br/>　　“此事，哪怕猎妖殿也无法多说什么。”
<br/>　　说罢，木青云没有再看萧逸一眼，身影一闪，来到叶铭身旁。
<br/>　　大手一挥，轻松便破去了保护叶铭的紫炎屏障。
<br/>　　“至于你。”木青云脸色一冷，“害我药楼遭此大祸。”
<br/>　　“虽错不全在于你。”
<br/>　　“但，你也活罪难逃。”
<br/>　　“废去修为，便当是小惩大诫了。”
<br/>　　“老匹夫，你敢？”牢笼内的萧逸，怒喝一声。
<br/>　　“叶铭少一跟头发，我要你整个药王谷鸡犬不宁。”
<br/>　　话音刚落。
<br/>　　一股毁灭气息，陡然间笼罩了整个空间。
<br/>　　让得在场所有人脸色大变。
<br/>　　那股气息的来源，正是牢笼内的易霄。
<br/>　　此刻的易霄手上，正漂浮着一股四色火焰。
<br/>　　“那是什么东西？”木青云大惊失色。
<br/>　　哪怕是他这个修为的强者，也从中感受到致命的威胁。
<br/>　　此刻自是惊骇无比。
<br/>　　“破。”萧逸大喝一声。
<br/>　　手中四色火焰，轰然爆炸。
<br/>　　那原本坚固无比的火焰牢笼，瞬间破碎。
<br/>　　萧逸飞跃而出，速度快到了极致。
<br/>　　下一秒，直接来到叶铭身旁。
<br/>　　“想在老夫手上救人？”木青云冷喝一声。
<br/>　　但看到易霄手中再次凝聚的四色火焰，心头猛地一慌。
<br/>　　就是这一慌的瞬间，萧逸瞬间拎起叶铭。
<br/>　　身影一闪，离开了33层空间，往药楼外飞离。
<br/>　　“该死，休想跑。”木青云当即追击。
<br/>　　一众长老，也立刻追击。
<br/>　　唯独木长老，脸色难看，但在原地迟疑了一下，也是脚步一动，跟随而去。
<br/>　　萧逸的速度，确实极快。
<br/>　　不过短短时间，已到达药楼之外。
<br/>　　当然，木青云等人，速度也极快。
<br/>　　几乎是同一时间，再次包围了萧逸。
<br/>　　“给我让开。”萧逸冷喝一声。
<br/>　　“不可能。”木青云强硬道，“我药王谷千百年珍稀药材的药力，皆在你身上。”
<br/>　　“你休想安然离去。”
<br/>　　“你们…”萧逸咬了咬牙。
<br/>　　“好，很好。”
<br/>　　“哈哈哈哈，这就是所谓的药王谷，这就是所谓的炼药圣地。”
<br/>　　他蓦地想起来，当年的易老，也曾大闹药王谷。
<br/>　　虽不知当年之事如何。
<br/>　　但，就观此刻木青云的态度，萧逸便对药王谷再无好感。
<br/>　　本来，要他归还那颗巨大丹药，并非难事。
<br/>　　但木青云的霸道，以及那种黑白不分地将事情赖在叶铭身上。
<br/>　　却让萧逸绝不接受，更不会平白受这莫须有之气。
<br/>　　当然，或许在木青云看来，丹药被服下，肯定已经被消化了。
<br/>　　他并不知道，丹药其实还一直好好的。
<br/>　　但，他万万不该这般对萧逸。
<br/>　　强者，都是有傲气的。
<br/>　　“好，很好，我倒要看看，药王谷今日能否拦得住易某。”
<br/>　　萧逸大喝一声，一手护着叶铭，一手凝聚出四色火焰。
<br/>　　“一起出手。”木青云大喝一声。
<br/>　　数十长老，同时气势爆发。
<br/>　　“木界封印。”
<br/>　　“火界封印。”
<br/>　　木青云，联合数十长老，所施展的火木双属性封印，其威力，必定更加骇人。
<br/>　　一张更为巨大的火焰巨网，凭空朝萧逸袭去。
<br/>　　萧逸手中四色火焰，脱手而出。
<br/>　　“爆。”
<br/>　　剧烈爆炸，与火焰巨网碰撞。
<br/>　　火焰巨网，顷刻间被炸得摇摇欲散，但为彻底破碎。
<br/>　　木青云以及一众长老，再次合力。
<br/>　　火焰巨网，不断靠近萧逸。
<br/>　　萧逸，不断后退。
<br/>　　每退一步，便是一道四色火焰打出。
<br/>　　每次火焰爆炸，便是药王谷内千米之地沦为齑粉。
<br/>　　“该死。”木青云脸色难看道，“不能再这样拖下去了。”
<br/>　　“否则，整个药王谷，都会被毁掉。”
<br/>　　“木长老，你去将传承元器取来，老夫亲自拿下易霄。”
<br/>　　木长老话音落下。
<br/>　　恰在此时，一道威严的苍老之声，响彻整个药王谷。
<br/>　　“不必了。”
<br/>　　一道流光，从远处疾速而来。
<br/>　　来人，是个老者。
<br/>　　刚刚落下，在场中人，无比脸色一变，纷纷行礼。
<br/>　　“参见谷主。”
<br/>　　不错，老者正是药王谷谷主。
<br/>　　“嗯。”老者点了点头，随后看向萧逸。
<br/>　　“这位便是最近风头正盛的易霄分殿主吧。”
<br/>　　“不知所谓何事，今日要大闹我药王谷？”
<br/>　　萧逸冷笑一声，“此事，你大可问问木青云。”
<br/>　　木青云脸色冷冷地瞥了眼萧逸，随后对着老者拱了拱手，将事情一一道来。
<br/>　　老者，也就是药王谷谷主，脸色不断变幻。
<br/>　　最后，恢复平静。
<br/>　　“原来如此。”药王谷谷主点点头。
<br/>　　“不知易霄小友，可有兴趣当我药王谷客卿长老？”
<br/>　　“嗯？”萧逸一愣，随后眉头一皱。
<br/>　　周遭众人，包括木青云在内，有些反应不过来。
