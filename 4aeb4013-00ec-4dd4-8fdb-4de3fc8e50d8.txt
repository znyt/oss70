        &lt;h3&gt;第三卷第一百四十九章  做一个有良心的企业&lt;/h3&gt;
<br/>
<br/>        元旦过后，大家都处于忙碌的状态中，寒伯安的公司也不例外。
<br/>
<br/>        上一次开会时强硬的表态在公司内部是起到了作用，但是对方一直不愿意挪东西，他们只想给对方留一点余地，以免撕破脸皮。
<br/>
<br/>        可是他接到了一个电话：“寒总，你们一直压的那批货不发是什么意思？那个是我一个朋友的。”
<br/>
<br/>        “你是？”寒伯安并没有听出来对方是谁。
<br/>
<br/>        “我是吴家的.....”对方语气生硬，一副命令的口气。
<br/>
<br/>        “哈哈哈，吴家的？吴家是什么样的人，我是知道的，别打着吴家的名义做坏事了。”寒伯安打断对方的话，说完这句话直接就挂断电话了。
<br/>
<br/>        能打到他手中的电话也不是简单的电话。
<br/>
<br/>        “长卿，通知媒体立马曝光这件事情。就说有人打着吴家的名义威胁我们。”寒伯安内线电话叫来了李长卿。
<br/>
<br/>        “好的，马上去办。”李长卿在寒伯安身边这么多年，知道寒伯安的做事风格。
<br/>
<br/>        下午两点李长卿召开了记者招待会，带着记者到了现场，将那一批货物曝光在大众的眼前。
<br/>
<br/>        “这批货已经压了我们好几天了，我们一直和对方交涉着，希望对方能够明白我们做事情的规矩，也希望对方自己站出来向有关部门自检，但是对方一直拖各种关系来让我们运送这批货物，今天竟然用吴家的名义来压我们了！大家都知道吴家是一个清明廉正的家族，怎么可能做这些事情？我们和吴家曾经因为寒二公子的关系有一点来往，我们深深知道吴家不会支持别人做这中违纪，违法的事情。为了不给吴家抹黑，也为了我们的道义，我们选择曝光这件事情。请大家做个见证！”
<br/>
<br/>        这件事情被媒体曝光，引起了不小的轰动，也引起了有关部门的注意，他们整个行业带起了一阵清查风。
<br/>
<br/>        李长卿敲了敲总裁办公室的门，推开门，寒伯安正在低头批文件。
<br/>
<br/>        “搞定了？”寒伯安头也不抬的问。
<br/>
<br/>        “搞定了。”李长卿欲言又止。
<br/>
<br/>        “你是怕还有后手？”
<br/>
<br/>        “唉，心里的想法总是瞒不过你。”
<br/>
<br/>        “兵来将挡水来土掩，我们应该和吴家终将会有一次较量，吴家现在可能视我们为眼中钉肉中刺。”寒伯安抬起头，看着窗外，虽然是冬天，但是风淡云轻。
<br/>
<br/>        “是啊，二公子和落雪暖昧不清，落雪一天不嫁吴家，寒家都是被打击的对象。”李长卿忧虑的说。
<br/>
<br/>        “我们好好做事情吧，现在关系错综复杂，谁也不知道后面怎么样，但是我们一定要行的端走得正！看见了没有？墙上的字！”寒伯安掷地有声。
<br/>
<br/>        “做一个有良心的企业，做一个有良心的中国人！”李长卿念出了声。
<br/>
<br/>        “是的，只要我们照章行事，心系我国，以小家助大家，为我们这个社会的发展出一份自己的力，虽然我们弱小，但是每个企业，每个人都贡献自己的一份力量，何愁我们国不强大？我们这一代人是中坚力量，我们的努力不但是为了我们自己的腰包能鼓起来，更是努力让我们的国家富强起来！”寒伯安站起来看着墙上的字。
<br/>
<br/>        “无论是陈子昂的信息化、耗材、还是我们的物流、租赁以及我们现在的运输业、互联网的快速发展，还有很多优秀的企业都是在推动我们社会朝前发展着，为着我们的富强之梦在努力着。做一个有良心的企业，做一个有良心的中国人是我们这一代人必须具备的信念！
<br/>
<br/>        所以，长卿，我们一定要坚持，坚持自己的底线，坚持道德，不能和有些不好的风气同流合污，哪怕很艰难，哪怕有压力，我们一定要做自己，无论是对内还是对外！万众齐心方能取胜！”
<br/>
<br/>        “是，我们一定要坚持自己的底线，谢谢你，老大，我虽然是个助理吧，但是心中还是有一杆秤的，要不是你有原则，我怕我早都不在你身边了。”长卿声音有点哑，每个人都有梦想，每个人都希望自己是一个栋梁，无论是那种职业，那种身份。
<br/>
<br/>        寒伯安发现自己的助理都有些激动了，于是拍了拍长卿的肩膀：“放心吧，不会让你失望的。”
<br/>
<br/>        工作之余，安男拿起手机，刷了一下网页，发现有寒伯安他们公司的新闻，其实有新闻也很正常，但是今天的新闻有些不一样。
<br/>
<br/>        她拨通了寒伯安的电话：“琪，公司没有事情吧？”
<br/>
<br/>        寒伯安心里一暖，“没有问题的，只是一些小事情。”
<br/>
<br/>        安男：“牵扯到了那样的一个家族，怎么可能没有事情呢？你这样做一定是迫不得已。”
<br/>
<br/>        寒伯安：“宝贝，放心，我都会处理好的。”
<br/>
<br/>        安男何其聪明，她对他们家的事情看得一清二楚，虽然她没有做生意，但是安林曦是研究经济学的专家，代凉是政，治学的专家，安男虽然选择的学科和父亲母亲都是不一样的，但是从小耳染目睹怎么能够不明白其中的厉害关系呢？
<br/>
<br/>        “有想我吗？”寒伯安为了缓和安男的担心笑着问。
<br/>
<br/>        李长卿，随时随地都会吃到狗粮。
<br/>
<br/>        “怎么会呢？要是想你就不会给你打电话了。”安男在电话那头调皮的笑着。
<br/>
<br/>        “知道老婆的意思了，想我的时候一定是我在身边，不想我的时候才打电话。今天的天气很好，但是不能和老婆看蓝天白云，总是很遗憾。”寒伯安对着电话笑得一脸温柔。
<br/>
<br/>        李才卿：这个平时霸道而冷峻的男人只有在他小妻子面前才是春暖花开，什么时候才能这样对我？
<br/>
<br/>        寒伯安似乎感受到了李长卿的怨念，回头冰凉凉的看了一眼李长卿，李长卿拿起文件转身就走。
<br/>
<br/>        寒伯安：真是没眼色，没看见我和我家的安男在聊天吗？
<br/>
<br/>        但是，转眼，李长卿又回来了。
<br/>
<br/>        寒伯安看了一眼长卿，对着电话说：“下班在家里等我，我得让你想着我。我这里还有点事情，晚上回家说，再见，宝贝。”
<br/>
<br/>        寒伯安等着安男挂了电话，他才挂电话。
<br/>
<br/>        “怎么了？”
<br/>
<br/>        “最新邮件，你看看，陈子昂发过来的，你们约定的会议明天就到了，你确定你今晚要去苏州？”李长卿忐忑的问。
<br/>
<br/>        “你怎么不早说？”
<br/>
<br/>        “你那眼神恨不得杀了我，我敢说？我不就是想学学经验嘛！”
<br/>
<br/>        “你还用学？那骚气的嗓音一唱起来，迷倒一大片。”寒伯安开了个玩笑。
<br/>
<br/>        “别怼我了，赶紧给安男小姐姐重新打个电话吧，车已经在门口等着了，要去机场了。”
<br/>
<br/>        寒伯安无奈，一高兴竟然忘记了要去北京了。
<br/>
<br/>        其实，像他这么忙碌的人，行程都是助理安排，他又怎么能记得住？
