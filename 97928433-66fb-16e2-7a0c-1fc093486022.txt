　　场上的时间慢慢的走着，等到香蕉这边操控着手中的奶妈回家补给了一个鬼书的回蓝小件后，他这边在线上打的更加的轻松了起来。
<br/>　　等到时间到达10分钟的时候，他这边手中已经是96刀在手，而反观对面的楚南，此刻却是只有63刀，十分钟的时间香蕉无压力补刀，而且在利用q技能的aoe伤害补刀的时候，他还可以利用自己的技能消耗对面的武器。
<br/>　　弄的楚南根本不敢上前，只能在塔下补刀，而且就算是在塔下，他也会面临着香蕉的消耗。
<br/>　　仗着自己有着w技能的回血，而且视野做的好，还有就是夜雪狂歌在上路，他这边倒是打的非常的凶悍。
<br/>　　“完了，蓝队这一局已经非常的难打了，south这一个点才十分钟就被压了30刀，武器的发育可是被大大拖后了。”
<br/>　　“而且香蕉这个点现在真的没人能制的了他了，要伤害有伤害，要回复能力有回复能力，最重要的是还有沉默，真的很难办啊。”
<br/>　　“最重要的是奶妈这个英雄越到后期就会变的越强，到了后期奶妈的回血能力可是非常的强的，尤其是他的大招，这个是群体回血，这一局奶妈是ap出装，到了后期他的回血可是非常的多的。”
<br/>　　“是的，现在奶妈发育的这么好，蓝队这边根本不敢找机会发动团战，如果他们要去下路gank，只要奶妈一个大招就可能扭转局势，这一局大魔王可是带的传送，他到时候完全可以传送到下路的。”
<br/>　　“这一局大魔王打的也非常的好，在前期这段时间没有让暗夜占到丝毫的便宜，现在多这么发育一分钟，蓝队这边的胜率就会下降一点，白队这个阵容可是越打越强的。”
<br/>　　解说席上米粒和林雅楠两人在那里讨论着，而比赛室内，看着双方的局面，秦霄几人面色也是有些凝重。
<br/>　　“17分钟，准备小龙团吧，那个时候咱们这边的核心装备基本上都差不多要做出来了。”秦霄开口说道。
<br/>　　“嗯，本来以为拖后期，咱们有武器这个英雄会很好打呢，不过他们这个阵容也是越打越强的，尤其是奶妈的沉默，这个真的是太恶心人了。”楚南跟着开口。
<br/>　　“奶妈的技能都很恶心人啊，w技能的回血、大招的回血、e技能的沉默，q技能还可以远距离消耗，除了脆一点之外，这个英雄真的有些强啊。”蒋念一脸的赞叹。
<br/>　　“准备团吧，小龙团如果输了，那这一局就没什么希望了，大家加油。”秦霄继续说着，不过说道小龙团的时候，就是他的心里也没有多少底气。
<br/>　　但是这一波却是必须要团的，因为他们这个阵容打到后期是完全没法和白队打的，如果继续这么发育下去，俺么他们其实就是在自寻死路，所以这一波团是他们唯一的希望。
<br/>　　当时间到达17分钟的时候，白队这边在叶帝的指挥下，五人已经是向着小龙这边聚集了过来，而蓝队那边几乎在看到白队向着这边汇聚的第一时间，也是向着这边敢来。
<br/>　　“双方这是要打小龙团了，不过这一波蓝队真的能打赢么？”看着两边的动作，米粒有些疑问的说道。
<br/>　　“应该有机会吧，他们这边只要寒冰的先手开的好，配合上妖姬的伤害，蓝队是可以率先秒掉一人的，而且娜美还有控制，这一波娜美先开团同样可以的。”林雅楠跟着开口说道。
<br/>　　“是啊，现在蓝队这边有着妖姬这么一个点，而且还有这么多控制，说不定真的能打出一波优势。”
<br/>　　听着两人的对话，直播间里众多蓝队的粉丝再次燃起了一丝希望，也许真的会如他们两个所说的那样，这一波蓝队打出一个0换5，然后直接反转局势呢。
<br/>　　双方在小龙处僵持着，大概过了30秒钟左右，蓝队这边终于是率先发起攻势，娜美的大招呼啸着向着叶帝五人奔袭而来。
<br/>　　紧接着寒冰的大招冰箭也是朝着这边飞射而来，并且还是后发先至，先娜美的大招射到了螳螂的身上。
<br/>　　“寒冰的大招中了，这一波蓝队有机会。”看到寒冰的大招命中螳螂，米粒眼睛一亮，大声喊道。
<br/>　　同一刻蓝队众人的眼睛也都亮了起来。
<br/>　　“秒螳螂、秒螳螂！”几人不约而同的大声喊道。
<br/>　　下一刻在无数观众的注释下，楚南他们几人便是冲了过来。
<br/>　　蒋念手中的妖姬直接w技能位移上前，想要最快速的将白队这个输出非常恐怖的螳螂给秒掉。
<br/>　　然而还不待等他开始输出，便是看到奶妈已经一个闪现冲了过来，摇手一指，妖姬顿时被沉默了下去。
<br/>　　“奶妈沉默，不过没有用啊，寒冰、挖掘机、武器他们的输出应该足够秒杀掉这个螳螂了。”看着香蕉闪现沉默，林雅楠有些叹息的喊道。
<br/>　　果然在蒋念被沉默之际，一旁的楚南看到香蕉这边终于是交出自己的沉默，这才敢开着自己的e技能反击风暴冲了上来。
<br/>　　之前他已经无数次先开启反击风暴，被奶妈给沉默下去，然后空放了，所以这个时候也是由不得他不谨慎。
<br/>　　在娜美这边刚刚将螳螂击飞的时候，武器的眩晕便是接了上来，下一刻挖掘机又接上，几乎短短的一瞬间，夜雪手中螳螂的血量便是掉到了半血。
<br/>　　“完了，这一波螳螂真的要死了，蓝队这一波机会抓的很好。”米粒开口喊道。
<br/>　　而场下这个时候蓝队的众多粉丝，已经是开始欢呼了起来，之前一直在被白队压制着，现在终于是轮到他们反击的时候了。
<br/>　　然而就在下一刻，当他们看到奶妈一个w技能直接将螳螂的血量回复到四分之三左右，紧接着扇子妈又来了一个群体大招的时候，场下刚刚响起来的欢呼声便是沉默了下去。
<br/>　　而叶帝这边在扇子妈开启了自己的群体护盾的时候，已经是操控着手中的卡萨丁一脚向着对方的人群中踩了过去。
