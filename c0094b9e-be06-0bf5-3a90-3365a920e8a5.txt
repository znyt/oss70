　　半日后，萧逸与长老们早已离开了紫云城，而且身在百里之外。
<br/>　　至于其他族人们，他们分成各个队伍，分批往不同方向离开，就算慕容家追击，也无法将他们一网打尽。
<br/>　　而各个队伍，会在真正安定下后，重新联络，并聚集起来，倒不必担心族人们会失散。
<br/>　　另一边，巧的是，萧逸等人不断赶路，却忽然遇上了一同逃亡的四长老等人。
<br/>　　他们从矿脉出发，沿着陨星山脉饶了个弯，没想到最后竟双双碰上了。
<br/>　　四六长老身后跟着数十个子弟，前是最出色的子弟；而矿脉的其他人，也同样分成不同方向逃离了。
<br/>　　“少家主，你没事，太好了。”萧壮和萧子木一见到萧逸，激动了起来。
<br/>　　四六长老则面露伤心，但遇上长老们后渐渐恢复了笑容。
<br/>　　“先别激动，等我们真正安全离开了再说吧。”萧逸微微一笑，道。
<br/>　　众人继续快速赶路。
<br/>　　萧逸背着依依，顺口问道，“大长老，我们可有撤离的目的地？”
<br/>　　大长老摇摇头，道，“没有，也只有这样，慕容家才很难寻得到我们的踪迹。”
<br/>　　二长老道，“等我们安定下来了，会立刻接回其他族人，休养生息后，终有一天会回紫云城的。”
<br/>　　又过了几个时辰，众人已经远离紫云城很远了。
<br/>　　这时，萧逸背上的依依柔声道，“少爷，你出汗了，是不是依依太重？还是放依依下来自己跑吧。”
<br/>　　说着，依依给萧逸擦了擦额头的汗。
<br/>　　萧逸摇摇头，笑道，“你这小身板，我背几天几夜都不会累，安心趴着吧。”
<br/>　　大长老看了眼其他子弟，发现萧壮等人脸色疲惫，停了下来，道，“已经离开紫云城很远了，慕容家估计很难追上了，我们稍作休息一番再赶路。”
<br/>　　其他子弟毕竟修为不强，长时间快速赶路，确实吃不消。
<br/>　　闻言，众人停下休息。
<br/>　　不一会儿，正当众人准备再次赶路时....
<br/>　　忽然，远处一道剧烈的破空声袭来。
<br/>　　萧逸眺目远望，远处天际，竟有一道模糊的人影。
<br/>　　人影的速度极快，而且越来越靠近，也越来越清晰。
<br/>　　最重要的是，一股威压，竟莫名其妙地笼罩了所有人，让众人难受至极。
<br/>　　“御空飞行，那是洞玄境武者。”大长老站了起来，忽然惊呼道，“不好，是慕容墨。”
<br/>　　他的话音刚落，下一秒，天地灵气陡然间暴走。
<br/>　　方圆数百米的灵气，竟一瞬间凝聚，仿佛受某种控制一般，重重地压在了萧逸等人身上。
<br/>　　一瞬间，所有人都动弹不得，包括大长老和二长老。
<br/>　　仿佛，有一个无形的囚牢，将所有人困了起来。
<br/>　　“灵气牢笼，遭了，慕容墨果然是洞玄境武者。”大长老大惊失色。
<br/>　　嗖，一道人影从天而降。
<br/>　　人影是个老者，须发皆白，一身绿袍，那瘆人的眼睛，如同鬼魅，仅仅瞥了众人一眼，便让众人不自觉地打了个冷颤。
<br/>　　“萧离火，多年不见，怎么急着离开呢？”老者笑着问道，笑容中充满了阴冷。
<br/>　　“慕容墨。”大长老咬牙看着面前老者。
<br/>　　显然，面前老者就是慕容家大长老，慕容墨，更是一位洞玄境武者。
<br/>　　“哈哈哈哈。”慕容墨大笑起来，“萧离火，你们以为远离了紫云城，我就找不到你们了吗？”
<br/>　　“洞玄境手段莫测，对灵气的掌控超乎寻常。你们的气息会残留在离开路上的灵气中，我只要掌控灵气，便对你们的一举一动了如指掌。”
<br/>　　“而且，我能御空飞行，你们如何能逃得出我的手掌心，哈哈哈哈。”
<br/>　　慕容墨笑容中充满了阴鸷和高高在上的狂傲。
<br/>　　“该死。”大长老愤愤地说了一声，面容中充满了着急，他不怕死，怕就怕家族子弟们无法离开。
<br/>　　“萧离火，你很着急，也很害怕。”慕容墨得意地说道，“从我回归紫云城的那一刻，就注定了你们萧家族人将死绝殆尽。”
<br/>　　“慕容墨，有什么冲我来，放了我的族人。”大长老怒吼道。
<br/>　　“哟呵，曾经高高在上的萧家大长老萧离火，如今你是在求我吗？”慕容墨阴笑道。
<br/>　　慕容墨继续道，“还记得很多年前，你我都是紫云城一代天才。但，你的天赋却总压我一头，你的炼药天才之名，更是让我无地自容。”
<br/>　　“现在呢，你却只如一只卑劣的蝼蚁，我随时都可以捏死。”
<br/>　　慕容墨说到最后，已经面露杀意。
<br/>　　灵气牢笼内，萧逸试了试挣扎，却徒劳无功。
<br/>　　方圆数百米凝聚成了天地灵气，太过强横了。
<br/>　　这就是洞玄境武者的强大，所过之处，天地灵气受其掌控。他甚至不用任何动作，单靠这一手能耐，就能将萧逸等人压迫至死。
<br/>　　“慕容墨，算我求你，你可以杀了我，但请放了我的族人。”大长老放下了尊严，哀求起来。
<br/>　　“哈哈，萧离火，你也有要求我的一天。”慕容墨越发得意。
<br/>　　“别急。”慕容墨冷声道，“你们今日杀了我们这么多族人，我要你们十倍奉还。慕容山他们，很快就会追上来，到时候，他们会好好招待你们的。”
<br/>　　“我乃堂堂洞玄境武者，杀你们这群蝼蚁，只会脏了我的手。”
<br/>　　“慕容墨，是你们慕容家挑起战斗在先，何必说得这么冠冕堂皇。”二长老愤怒地骂道。
<br/>　　“我不管。”慕容墨蛮狠道，“总之，我要你们死。”
<br/>　　说罢，慕容墨凌空飞行，来到萧逸面前，冷声道，“你就是萧家那个天才吧？就是因为你，才坏了我们慕容家那么多计划，逼得老夫亲自出手。”
<br/>　　萧逸怡然不惧，冷眼瞥着他。
<br/>　　“好凌厉的眼神，好坚定的神智，竟不惧老夫的直视。”慕容墨微微吃惊，而后笑得更加阴冷，“呵呵，对付你这种天才，更有意思呢。”
<br/>　　“待会儿，一刀一刀把你的肉割下来，慢慢废了你，听着你的惨叫声，必然会很美妙。”慕容墨舔了舔嘴唇。
<br/>　　“你这个变态，休想伤害少爷。”依依愤怒地说道。
<br/>　　“哟，还有个小女娃，倒是长得水灵，啧啧，老夫好久没有见到这样的美人了。”慕容墨眼中露出了**的目光。
<br/>　　“老怪物，你找死。”萧逸顿时满脸杀意。
<br/>　　澎湃的杀意倾斜而出，令得慕容墨大惊。
<br/>　　“好恐怖的杀意。”慕容墨皱起了眉头，随即对萧逸的杀机更为浓郁，“你这种天才，若是成长起来，日后定是一方强者。”
<br/>　　“可惜了，你这个天才今日便要陨落，而且是死在老夫手中。”慕容墨举起了手掌，欲要击杀萧逸。
<br/>　　恰在此时，忽然，一道凌厉的攻击出现。一只干枯却有力的手掌，一掌打向慕容墨。
<br/>　　慕容墨猝不及防，竟吐出一口腥血，并被一掌击飞。
<br/>　　攻击之人，竟是大长老。
<br/>　　而后，大长老双手一挥，那禁锢众人的灵气牢笼瞬间消失。
<br/>　　大长老一把拎起萧逸的衣领，全力抛飞。
<br/>　　“萧逸，快跑。”大长老大喝一声。
<br/>　　这时，慕容墨站稳了身子后，擦了擦嘴角的鲜血，冷冷道，“萧离火，你怎么可能破开我的灵气牢笼？”
<br/>　　“嗯？不对。”慕容墨反应过来，冷笑道，“萧离火，好本事阿，竟然隐藏了实力。没想到你竟是半步洞玄，难怪三个先天九重都无法攻破你萧家。”
<br/>　　萧逸已被远远抛飞，大长老远远喝道，“萧逸，不要回头，速速离去。萧家有你便有未来，不要让我们白死。”
<br/>　　“休想逃。”慕容墨冷喝一声，欲要追击萧逸。
<br/>　　“休想追。”三长老、四长老、七八九长老瞬间联手，意欲挡下慕容墨。
<br/>　　萧壮、萧子木等子弟，一个都没跑，一字排开，打算用身体挡住慕容墨。
<br/>　　“滚开。”慕容墨大手一挥，各大长老毫无抵抗之力便被拍飞。
<br/>　　“萧逸，快跑。”
<br/>　　“萧家有你便有未来。”
<br/>　　“不要让我们白死。”
<br/>　　“不要让我们白死....”
<br/>　　众人怒吼着，再次爬起来抵挡慕容墨。
<br/>　　大长老大手一挥，“灵气牢笼。”
<br/>　　刚才禁锢众人的方圆数百米灵气，反而凝聚起来，禁锢起慕容墨，挡下他追击的步伐。
<br/>　　远处，萧逸咬了咬牙，眼角泛出一滴泪光。前世今生，他头一次因为其他人而落泪。
<br/>　　然后，他毫不犹豫地转身离去，速度飙升到极致。
<br/>　　但，他不是贪生怕死；他的意图是....
<br/>　　“慕容墨老怪，有本事来追小爷....”
<br/>　　“今日你若杀不了老子，他日，我要你整个慕容家鸡犬不留，血流成河....”
<br/>　　萧逸已经远去，但那带着威胁意味与杀意的声音，清楚地传到了慕容墨耳中。
<br/>　　“臭小子，休想跑掉。”慕容墨咬牙切齿。
