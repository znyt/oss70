　　第114章
<br/>　　寥寥十二个字便勾勒出一遍北国画卷，唐如才这些人都是懂行的，立刻就领略到词中的妙处。
<br/>　　程大雷继续握笔。
<br/>　　『望长城内外，惟馀莽莽，大河上下，顿失滔滔』
<br/>　　『山舞银蛇，原驰蜡象，欲与天公试比高』
<br/>　　……
<br/>　　唐如才倾着半个身子，许多人都向这边望来，一句『欲与天公试比高』激得众人心生豪横，似乎要有十大碗酒才能浇透心中块垒。这不是江南水乡的和风细雨，而是北国的大山大河；这不是才子佳人的小情小爱，这是燕赵男儿的慷慨之歌。
<br/>　　胸中有山，诗中才看得见山，胸中有水，笔下才走得了水。唐如此看着程大雷，用难以置信的目光：他的胸怀里有什么，出得下这样的诗句。
<br/>　　像是有整个江山。
<br/>　　『江山如此多娇，引无数英雄竟折腰』
<br/>　　唐如才激动得眼眶都红了，一种情绪在怀里酝酿，只等随着程大雷笔下的墨字倾泻而出。可奇怪的是，程大雷手中的笔悬在半空，下阙只开了一句，迟迟没有下笔。
<br/>　　程大雷僵硬的举着笔，额头似乎浮现三缕黑线。此刻他终于意识到一件事：这词不能再写下去了。
<br/>　　下面就是秦皇汉武，唐宗宋祖……可这世界根本没有唐宗宋祖，更不用提后面的一代天骄。
<br/>　　他这篇抄错了。
<br/>　　怎么办，好尴尬。
<br/>　　你说某肚子里这么多诗词，随便抄一首不就好了么，为什么抄了一首不能抄的。假若自己把这首涂了，重新再写一首，是不是更加尴尬。
<br/>　　在线等，挺急的。
<br/>　　“阁下怎么不写了？”唐如才试探着问道。
<br/>　　程大雷直起身，随意把笔抛在桌上，道：“这就够了。”
<br/>　　“够了，什么够了？”
<br/>　　“只有上阙就够了，半篇惊江南，全篇震天下，所以够了。”程大雷又把糊弄李行哉的那一套拿了出来。
<br/>　　“呃……”
<br/>　　唐如才被噎得七荤八素，他刚被词勾起情绪，正打算随着词中意慷慨激昂，可程大雷忽然告诉他没有下阙。这种感觉就类似与看了一部大片，最后二十分钟却被掐了，憋得是分外难受。
<br/>　　“现在我们可以走了吧？”
<br/>　　“当然，当然。”
<br/>　　唐如才目送着目送着程大雷下楼，看目光中的意思，颇有种要做程大雷门下走狗的感觉。
<br/>　　他刚回过神，往桌子上扫了一眼，忽然间暴跳如雷：“字呢，你们谁把字藏起来了？”
<br/>　　楼上几人眼观心，心观口，都不说话，假装自己没瞧着。
<br/>　　程大雷刚下楼，就撞见客栈掌柜牛本善。牛本善陪着笑道：“客官是要去参加甲字号院子里的辩宝大会么，我可以让手下领路。”
<br/>　　“不了，今天有些累，明天再去看看，反正明天还有。”程大雷说了一声，忽然想起什么，冲牛本善道：“今晚可能有人来找我，你记得告诉他们我住在丁字号。”
<br/>　　“那我一定给您记着，人来了肯定告诉他们。”说罢，牛本善又随口问了一句：“什么人呀？”
<br/>　　“不必麻烦，我们已经到了。”
<br/>　　正这个时候，一群人呼啦啦闯进客栈。人头数足有七八十个，瞬间占满客栈大厅。
<br/>　　“漕帮做事，无关人等散开。”李佳明道。
<br/>　　牛本善吓了一跳，漕帮的煞星怎么到了这里。他忙拦住李佳明：“李爷，我们和漕帮的关系一直不错，每月的供奉也从没少过，今天是怎样道理。”
<br/>　　“不管你的事，躲开。”
<br/>　　漕帮的人开始清场，很快整个大厅就只剩下程大雷几人，徐神机、赵子龙也闻讯来到大厅。
<br/>　　随后，一台小轿停在客栈门口，有人掀开轿帘，一白发无须的老者缓缓走入大厅。
<br/>　　“是孙老爷子！”
<br/>　　牛本善在柜台内吓得一跌，漕帮的孙老爷子年事已高，平常人想见他一面都不易，他今天怎么也来了。
<br/>　　目光看了看程大雷一伙，他们是怎么闯下这塌天之祸。
<br/>　　“我听说有个要给码头立规矩的人来了，原来就是你这个黄口小儿？”老者抬起头，以浑浊的眼球看着程大雷。
<br/>　　程大雷抿了口茶：“那还能是你这个老不死的？”
<br/>　　哗！
<br/>　　雅雀无声，漕帮孙老爷子什么地位，谁敢同他说这种话。就连那孙老爷子表情也明显怔了怔，脸上挤出一抹僵硬的笑。
<br/>　　“初生牛犊不怕虎呐……”
<br/>　　“帮主，别和他废话，先让我剁了他。”李佳明大喝。
<br/>　　“漕帮还真是没规矩啊，什么人也可以说话。”程大雷将茶杯上的浮沫吹去。
<br/>　　孙老爷子嘴角抽搐两下，狠狠瞪了李佳明一眼。
<br/>　　“人是在这里嘛！”
<br/>　　“老爷，就是这里，我亲眼看到他们进去的。”
<br/>　　“走，绝不能饶了他们！”
<br/>　　客栈外，响起一阵洪钟般的声音。有漕帮的手下跑进来，在孙老爷子耳边低语几句。
<br/>　　“帮主，金龙镖局的洪天吉到了。”
<br/>　　“我们和金龙镖局向来井水不犯河水，他们来干什么……”老者目光放在程大雷身上，见他仍不紧不慢的喝茶，心里思索道：难道此人是金龙镖局放出来落我漕帮面子的。
<br/>　　哼，真如果是这样，漕帮也不怕你金龙镖局。
<br/>　　“让他们进来。”老者合上双目，如老僧入定。
<br/>　　“孙德龙，怎样，你要给那小子出头嘛！”
<br/>　　随着洪钟般的声音，一个年过半百的胖子走进来，此人就是金龙镖局的东家黄天吉。如今是年纪大了不再走镖，曾经年轻时，也是南国绿林道响当当的人物，金龙镖局的镖旗挂出来，南国几州，无论水道陆道，便没有一家山寨敢动手。
<br/>　　此刻孙德龙睁开眼睛：“出什么头，不是你要给这小子撑腰么？”
<br/>　　“撑什么腰，不是你要给这小子出头？”
<br/>　　咦！
<br/>　　二人同时一楞，目光闪烁，最后将目光放在程大雷身上。
<br/>　　二楼，唐如才几个人探出脑袋往下面望。刚才漕帮到他们就吓了一跳，如今金龙镖局也到了。一个人究竟要怎么作死，才能同时把这两家势力都得罪到。
<br/>　　程大雷坐在那里，仍不紧不慢的喝茶，他忽然抬起茶壶，问：“谁给添水？”
<br/>　　
<br/>(本章完)
