        郎安又重复了一遍：“你，得跟我走。”
<br/>
<br/>        “我……”为什么要跟你走？
<br/>
<br/>        连星儿的话还未说完，眨眼睛，郎安便跃过了防护网，直接跳到了她的跟前。
<br/>
<br/>        “有了你，我才能有选择的余地。”郎安意味不明地打断了连星儿的话。
<br/>
<br/>        “？！”连星儿一脸懵，完全没听懂，更加不知道他要她跟他走是何种意思。
<br/>
<br/>        下一秒，郎安猝不及防地抬起手来，一掌将连星儿打晕了过去。
<br/>
<br/>        明司寒再次抬起头来时，并未看到连星儿的身影，心里莫名其妙地一阵紧张。
<br/>
<br/>        他立即放下了手中的活，只身往森林边缘走去。
<br/>
<br/>        “星儿呢？”
<br/>
<br/>        走到那几个负责捡柴的同学身边，明司寒凝眉问道。
<br/>
<br/>        “在那儿啊！”其中一个女生指向林子里不到五米远的地方，在没看到连星儿的人影后，顿时有些惊慌了起来，“诶？她人呢？我刚刚还看到她就站在防护网那边捡柴的。”
<br/>
<br/>        明司寒二话不说，直接往林子里冲去。
<br/>
<br/>        他来到防护网旁，只见地上洒落着一堆柴，还有一根他起初在路边的小卖部里重新给她买的棒棒糖。
<br/>
<br/>        “星儿！星儿！”明司寒意识到连星儿可能出事后，对着森林深处大喊了两声，回应他的却只是一阵回音。
<br/>
<br/>        下一刻，他不再做无用地挣扎，而是掏出手机，给裴萱妍打了通电话。
<br/>
<br/>        “星儿失踪了，你叫人来我这儿。”明司寒焦虑不安地命令道。
<br/>
<br/>        裴萱妍也大吃一惊地反问：“失踪？你们不是去露营了吗？怎么会失踪？而且，你们露营的地方很安全的。”
<br/>
<br/>        “我怀疑，带走星儿的人，是你们一直在找的郎安！”明司寒剑眉微蹙。
<br/>
<br/>        上次他跟裴萱妍达成协议后，便成了他们组织的外编制人员。
<br/>
<br/>        对于他们组织的事情，他虽然了解得不多，但是他知道，他们在追踪郎安。
<br/>
<br/>        而且，他们希望他替他们写一款无信号追踪码，用在纳米追踪器上。
<br/>
<br/>        “如果是郎安的话，你就不用担心连星儿的安危了。他不会伤害她。”裴萱妍不痛不痒地说道。
<br/>
<br/>        明司寒眉头紧蹙，沉默了。
<br/>
<br/>        此时裴萱妍忽然感慨道：“怪不得我们一直找不到郎安，原来那家伙躲进原始森林里去了。”
<br/>
<br/>        “如果你不叫人过来找，那么我自己进森林里去找！”明司寒冷冷地说完后，直接挂断了电话。
<br/>
<br/>        明司寒刚挂了裴萱妍的电话后，手机屏幕上闪出了连星儿的来电。
<br/>
<br/>        他欣喜若狂地按下接听键，将手机重新举到了耳边，唤道：“星儿。”
<br/>
<br/>        “我会把她平安送回来，所以，你不用进这片森林里来找。就算你进来了，你也是找死。林子里的野兽，会吃了你。”郎安的声音，从听筒里淡淡地传出来。
<br/>
<br/>        明司寒再次皱紧眉头，质问道：“你到底要对星儿做什么？”
<br/>
<br/>        “明天早上，太阳升起来的时候，你到你现在站的这个地方来等她。”郎安说完，直接将电话给挂断了。
<br/>
<br/>        明司寒再次回拨过去时，提示对方已关机。
<br/>
<br/>        夜幕降临。
<br/>
<br/>        连星儿迷迷糊糊地睁开双眼，只看到眼前有火把在模糊的视野里跳动。
<br/>
<br/>        她揉了揉眼睛，待视野清晰后，这才看清楚面前是火堆，而火堆旁还有用树枝叉着的两只烤得上了焦糖色的野兔。
<br/>
<br/>        连星儿从一堆疙手的树叶中坐起身子，四下看了看后，只见那边洞口突然窜进来一个黑影又将她吓了一跳。
<br/>
<br/>        当她定睛望去时，只见郎安怀里抱着一些果子朝她走了过来。
<br/>
<br/>        郎安随后在连星儿身旁坐下，拿起一只烤好的野兔，递向连星儿。
<br/>
<br/>        “先吃肉，再吃果子。”
<br/>
<br/>        郎安说完后，连星儿怔愣地接过树枝叉着的烤野兔。
<br/>
<br/>        她拿在手里吹了吹，然后撕下一块兔肉塞入嘴中。
<br/>
<br/>        因为没有盐味，令她难以下咽，却又不敢当着郎安的面吐出来，只好囫囵吞枣式地吃下去。
<br/>
<br/>        “你说，我是选择当一匹狼，就在这片森林里生活下去，还是选择当一个人，回到你外公身边去，听你外公差遣？”郎安忽然黯然神伤地问。
<br/>
<br/>        连星儿再次吞下嘴里的食物，一本正经地说道：“当然是当一个人，替自己活着！”
<br/>
<br/>        “你的血，可以解除我身体的变化。这就是小时候，我为什么攻击你的原因。”郎安低沉道。
<br/>
<br/>        连星儿苦笑：“我知道，虽然只是猜的，但也八九不离十了。”
<br/>
<br/>        “你不怕我吗？我是实验室里出来的变异人。”郎安的眼底，有金色的眸光波动。
