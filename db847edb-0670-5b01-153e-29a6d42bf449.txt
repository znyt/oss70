　　轰！轰！
<br/>　　苏莫率先出手了，掌中魔气涌动，双掌齐出，猛然拍出。
<br/>　　刹那之间，两道硕大的掌印，分别向巫烈和巫玉儿袭杀而去。
<br/>　　两道掌印均是呈现墨黑之色，乍一出现便裹挟汹涌的魔气，魔威浩荡，滚滚涌动，如同怒海狂涛。
<br/>　　由于双方距离极近，两道掌印如同两道黑色的磨盘，瞬间便杀到了巫烈和巫玉儿的身前。
<br/>　　“什么？”
<br/>　　所谓高手一出手，便知有没有，苏莫乍一出手，所有人便是心中一惊。
<br/>　　一是惊讶苏莫的实力，这两掌明显远超武帝境三重武者的战力，比之一般的武帝境四重武者，都丝毫不弱，甚至稍强两分。
<br/>　　二是惊讶苏莫居然修炼的是魔功，虽然族中不禁止修炼其它种族的武学，但霸道狠辣的魔功很少有人修炼。
<br/>　　“怎么可能？”巫烈和巫玉儿最为吃惊，两人都很是了解巫尘，一个多月之前也交过手，但现在的巫尘和一月之前，明显判若两人。
<br/>　　不仅气质大变，连武学体系都完全不同了，这是怎么回事？
<br/>　　“破！”
<br/>　　但攻击已经近在眼前，两人来不及多想，立刻出手抵挡。
<br/>　　巫烈的拳头赤光耀眼，犹如是烧红的烙铁，一拳狠狠的轰向袭来的掌印。
<br/>　　巫玉儿双手一扬，剑影闪烁，幻化出十数柄虚幻的长剑，向掌印迎击而去。
<br/>　　轰！轰！轰隆隆！！
<br/>　　瞬息之间，三人的攻击碰撞在了一起，发出惊天动地的爆响，掌印、拳劲、剑影纷纷崩灭，化为滔天的乱流，席卷了整个战台。
<br/>　　乱流席卷到战台上的阵法气罩之上，顿时被气罩吸收，无法波及到战台之外。
<br/>　　嗖！嗖！
<br/>　　巫烈和巫玉儿的身形微震，立刻身形向后飞退。
<br/>　　“怎么可能？”两人满脸的震惊之色，巫尘的实力居然如此的强大，比之他们都不弱了！
<br/>　　这是怎么回事？
<br/>　　短短月余的时间，怎么可能会有如此大的变化！
<br/>　　难道巫尘得到了什么超强的魔功？
<br/>　　才导致实力大增！
<br/>　　两人心惊不已，当然他们只是心惊巫尘的改变，到没有什么惧怕，毕竟就算巫尘有了武帝境四重的实力，也不可能打败他们两人联手。
<br/>　　“你们实力不错！但仅此而已！”
<br/>　　苏莫冷漠的声音响起，一击试探之后，再次出手了，他的手中剑光一闪，出现了一柄雪亮的长剑。
<br/>　　这柄长剑正是廖云剑的宝剑，反正此地也没有人认识，他可以肆意使用。
<br/>　　“斩！”苏莫厉喝一声，手中剑光爆闪，不断的斩击，无数的墨黑色的剑气，密密麻麻，铺天盖地的斩击而出。
<br/>　　苏莫可没有心思与这两人缠斗，准备以雷霆之势斩杀两人。
<br/>　　这一击，他暗暗催动了体内的白色长剑战魂，以及十种长剑武魂，让得他的剑威暴增。
<br/>　　白色长剑战魂，自然是吞噬郭剑青得来的战魂，这战魂的威力着实不弱，都快赶得上另外十种武魂之合了。
<br/>　　咻咻咻！！
<br/>　　铺天盖地的墨黑色剑气，裹挟惊天魔气，魔威滚滚，威势霸道森然，剑威浩大无疆，瞬间将巫烈和巫玉儿全部笼罩。
<br/>　　“这不可能！”巫烈放声怒吼，声音中充满了恐惧，这铺天盖地的剑气让他全身颤抖。
<br/>　　他，怕了！
<br/>　　他无论如何也没有想到，巫尘的实力居然强大到了这等地步。
<br/>　　虽然这让他感觉不可接受和满心疑惑，但此时根本来不及考虑了。
<br/>　　“战魂！”
<br/>　　战台上空间有限，逃无可逃，巫烈心念一动，顿时背后虚影升腾，释放出了自己的战魂。
<br/>　　他的战魂是一块火红色的大石，高达一丈有余，通体赤红，燃烧着熊熊的烈焰。
<br/>　　这种火石，名为烈焰石！
<br/>　　这是烈焰石战魂！
<br/>　　“赤炎神拳！”巫烈怒喝一声，双拳齐出，拳劲如同一团团硕大的火球，裹挟磅礴的威势，疯狂的砸向迎面而来的无数剑气。
<br/>　　另一边，巫玉儿同样面色巨变，她的实力还不如巫烈，面对的压力可想而知。
<br/>　　“怎么会这样？”巫玉儿脸色苍白，心中恐惧不已，巫尘的实力让她感到恐惧。
<br/>　　难道？
<br/>　　一瞬间，一个惊人的想法在她心中升起。
<br/>　　一个月之前，巫尘被巫烈杀了，就算侥幸没死的话，如此重的伤势想要恢复过来，起码也要一两个月的时间吧！
<br/>　　但是，这一个月的时间，巫尘不仅伤势恢复了，连实力也暴涨了数倍。
<br/>　　实力暴涨也就罢了，整个人都变了，气质变了、气息变了，连修炼的武学都变了！
<br/>　　这说明什么？
<br/>　　说明眼前的巫尘，很可能根本就不是巫尘！
<br/>　　这个想法，让得巫玉儿大惊失色，但是，铺天盖地的剑气即将临身，她根本来不及多想。
<br/>　　唰！
<br/>　　电光火石之间，巫玉儿不敢有丝毫的保留，手中剑光一闪，背后虚影升腾，同样瞬间释放出了自己的战魂。
<br/>　　巫玉儿的战魂是一柄两尺来长的娟秀小剑，战魂乍一出现，便融入了她手中长剑之中。
<br/>　　唰！
<br/>　　长剑一挥，一道匹练般的剑气挥洒而出，撕裂一切，向漫天剑气斩击而去。
<br/>　　巫烈和巫玉儿两人，几乎都展现出来巅峰的实力，来阻挡苏莫的袭杀。
<br/>　　可以说，两人的实力非常不错，两人虽然不算天才，但联手之下，和一般的武帝境五重的武者，也能斗上两三招。
<br/>　　但是，他们偏偏遇上了苏莫，纵然实力不错，也难以完全发挥出来。
<br/>　　就在两人出手的刹那，苏莫微微催动了自己的吞噬战魂，强大的吞噬之力，瞬间将两人笼罩。
<br/>　　因为战台上有阵法气罩隔绝，苏莫倒也不怕动用吞噬战魂，只要不是太明显，巫天御等人根本发现不了。
<br/>　　当然，巫天御等去过苍穹世界的人，就算见到吞噬战魂，也不一定能认出来。
<br/>　　他们那些人，之感受过吞噬之力，并没有见过吞噬战魂的庐山真面目。
<br/>　　无形的吞噬之力将巫烈和巫玉儿两人笼罩，两人顿时全身一震，身形、气血、玄力、战魂等等一切，都受到了吞噬之力强烈的影响。
<br/>　　一瞬间，两人发出的攻击，威力爆降大半。
<br/>　　“不！”
<br/>　　两人发觉这一切，顿时惊恐的大喝了起来。
<br/>　　不过，他们根本反应不过来，下一息，铺天盖地的剑气绞杀而过，火焰拳劲和匹练般的剑气纷纷被绞碎，发出剧烈的爆炸，化为冲天的气浪横扫八方。
<br/>　　密密麻麻的墨黑色剑气，瞬间将巫烈和巫玉儿的身躯淹没。
