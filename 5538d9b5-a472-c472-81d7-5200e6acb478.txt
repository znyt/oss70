　　当苏莫和龙腾联袂走出祖龙塔的这一刻，数百万道目光，几乎是齐刷刷的落在苏莫的身上。
<br/>　　现场一片死寂，所有人的目光，均是死死的盯着苏莫，仿佛苏莫是一件绝世宝物一般。
<br/>　　这些目光之中，包含着震惊、羡慕、嫉妒、怀疑、好奇等种种不同的情绪。
<br/>　　因为，从目前的情况来看，苏莫十有八九，真的闯过了第十层，得到了祖龙精血，不然不会有如此的情况。
<br/>　　而龙帝的目光，没有落在苏莫的身上，而是第一时间落在了龙腾的身上。
<br/>　　见到龙腾和苏莫一起出来，龙帝的眉头微不可查的一皱，心中升起了一丝不好的预感。
<br/>　　不过，他并没有什么动作，因为，若是祖龙精血在屠霄的手上，自然会是他囊中之物。
<br/>　　不然，他现在出手，也无济于事。
<br/>　　“见过龙帝！”
<br/>　　龙腾带着苏莫，来到了龙帝的身前，向对方抱拳行礼。
<br/>　　两人心中暗暗惊讶，虽然已经能猜到，祖龙塔外可能已经惊动了大量的龙族之人，但是，见到周围铺天盖地的龙族之人，两人还是难免心中惊讶。
<br/>　　“嗯！”
<br/>　　龙帝背负着双手，微微颌首，目光落在了苏莫的身上，道：“屠霄，适才是你闯进了祖龙塔第十层？”
<br/>　　龙帝目光灼灼，如同是两道光束，极为刺目。
<br/>　　此刻，所有的人，都紧紧的凝视着苏莫，虽然大家都知道，除了屠霄之外，不可能是其他人闯进第十层，但还是要屠霄亲口确认。
<br/>　　“禀龙帝，是我闯进了第十层！”苏莫面色郑重的点了点头。
<br/>　　“那你得到了祖龙精血？”龙帝再次问道，此言一出，所有人的心中都充满期待，期待苏莫确认，是否得到了祖龙精血。
<br/>　　万众瞩目！
<br/>　　所有人的目光，都充满了极度的好奇，好奇苏莫有没有得到祖龙精血。
<br/>　　面对龙帝的询问，面对数百万龙族之人的目光，苏莫并未回答，而是看向了龙腾。
<br/>　　众人见此，心中纷纷一惊，祖龙精血在龙腾的手上吗？
<br/>　　只见龙腾微微一笑，身形缓缓的腾空而起，这一幕让众人心中惊疑不定。
<br/>　　呼吸之间，龙腾的身形飞到了千丈高空，停止了下来。
<br/>　　昂！
<br/>　　而后，一声嘹亮的龙吟之声，响彻天地，只见龙腾的身躯，陡然只见化为了庞大的彩色神龙。
<br/>　　彩色神龙，身上夺目的彩色光芒照耀八方，气势威严、霸道、尊贵无比。
<br/>　　“祖龙？”
<br/>　　“这是……？”
<br/>　　“怎么可能？”
<br/>　　所有人看到这一幕之人，无不心神震颤，面色呆滞，就连龙帝都不例外。
<br/>　　龙腾居然化为了洪荒祖龙，这是什么情况？
<br/>　　很多弱小的龙族之人，见到龙腾的本体，甚至忍不住想要跪拜。
<br/>　　这是来自血统的压迫，这是来自精神上的压迫，洪荒祖龙作为龙族始祖，是所有龙族之人的领袖。
<br/>　　“龙帝，诸位前辈，我已经融合了祖龙精血，进化了血统，现在拥有和洪荒祖龙同样的天赋和能力！”
<br/>　　洪亮的声音，从龙腾所化彩色神龙的口中响起，浩浩荡荡，传遍了整个龙都城。
<br/>　　此言语，如同天地雷音，在众人的耳边炸响，震的所有龙族之人脑子嗡嗡作响。
<br/>　　现场，彻底的死寂了下来，所有人都惊呆了。
<br/>　　龙腾融合了祖龙精血，进化了血统，成为第二个祖龙。
<br/>　　龙帝面无表情，眸中却是闪过一丝阴霾，果然，他最不希望看的的情况出现了。
<br/>　　祖龙精血被龙腾得到，他就失去了机会。
<br/>　　苏莫面色淡淡，他在暗暗的观察所有人，其它龙族的神王级别强者，眼中只有震惊与不可思议，唯独龙帝不同。
<br/>　　这让苏莫暗暗的警惕了起来，看来此事，让得龙帝不喜了。
<br/>　　司空炎倒是没有苏莫这么观察入微，他见到龙腾这般情况，满脸的喜色，根本没有注意龙帝的神态。
<br/>　　这时，只见龙族大长老，面带激动之色，朗声道：“当年，祖龙离开之时，曾留下意志，龙族中谁能融合他留下的精血，将成为龙族新的祖龙，领袖天龙神朝！”
<br/>　　大长老激动的声音，响彻全场，震动八荒，再次让所有人为之大震。
<br/>　　新的祖龙？
<br/>　　的确，无数万年之前，洪荒祖龙离开之时，的确是留下了这种意志。
<br/>　　“见过祖龙！”
<br/>　　“见过祖龙！”
<br/>　　“见过祖龙！”
<br/>　　霎时之间，无数的龙族之人，纷纷向高空中的龙腾躬身行礼，面色恭敬无比。
<br/>　　“拜见祖龙！”
<br/>　　大长老亦是没有因为自己的身份，而有所怠慢，同样向龙腾行礼。
<br/>　　他从太古末年，便跟随洪荒祖龙，见证了洪荒祖龙的传奇一生，也见证了荒界从弱小走上强盛，再到如今的衰落。
<br/>　　可以说，他就是洪荒祖龙最忠实追随者，自然不会违背洪荒祖龙的意志。
<br/>　　“拜见祖龙！”
<br/>　　“拜见祖龙！”
<br/>　　“拜见祖龙！”
<br/>　　随着大长老行礼，龙族中数十位神王级别的强者，有一大半也随之行礼，显然认可了龙腾新的祖龙的身份。
<br/>　　这是洪荒祖龙的意志，他们没有理由反对，再者，龙腾本就是万古天骄，天赋强大的天才人物。
<br/>　　不过，虽然很大一部分龙族神王认可，但是龙帝和少部分的神王级别强者，却是无动于衷。
<br/>　　而这少部分的神王级别强者，均是向龙帝投去询问的目光，显然，他们以龙帝马首是瞻。
<br/>　　这一幕，都看在苏莫的眼中，让他心中无奈。
<br/>　　看来，龙腾成为新的祖龙，得到了龙族大部分人的尊敬，但是却不会让龙帝臣服。
<br/>　　不过，让他稍稍放心的是，以目前的情况来看，大多数的龙族之人，还是认可龙腾的，那么他和龙腾就不会有危险。
<br/>　　当然，这些人之所以认可，也是因为龙腾已经进化，事情已经无法改变了。
<br/>　　若是祖龙精血还在，龙腾并没有融合的话，这些人出手抢夺都不出奇。
<br/>　　“龙帝，我进化成了祖龙血统，继承了洪荒祖龙的传承，成为龙族新的祖龙，您觉得够不够资格？”
<br/>　　龙腾依旧保留着本体，巨大的龙眸望向吞虚龙帝，声音低沉的问道。
