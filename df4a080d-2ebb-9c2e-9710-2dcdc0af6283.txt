<br/>				欧阳家族，其实在西北昆仑山附近的一个小镇上。小镇子所有的人，基本上都是欧阳家族的。<br/>　　<br/>　　其实自然环境，对于后天武者来说，并不会影响什么。但是对于先天武者来说，还是有影响的。<br/>　　<br/>　　先天武者，已经是另外一个层次的武者，他对于感应自然，和利用自然之力，达到了另外的一个境界。先天，还可以通过天地之气来壮大己身，这样修炼，次啊能达到抱丹的境界。其实这个天地之气，其实也就是灵气。<br/>　　<br/>　　当然，在这里，武者都是称为天地之气，而不是灵气。武者对于灵气的利用，是非常简单和粗暴的，并不像修真者利用灵气，还可以利用其他手段，来聚集和使用灵气。<br/>　　<br/>　　在这个星球上，灵气可以说非常的匮乏。一边也就在一些特定的场所，或者是一些山川等等，才会有灵气。而外边，由于城市化进程等等，灵气已经变成污浊了。<br/>　　<br/>　　这也说不上好或者不好，毕竟对于星球上绝大多数人类来说，都不是武者，也不需要灵气，需要的是金钱和利益。<br/>　　<br/>　　欧阳家族在西北昆仑山脚下，其实也是这个原因。在很久之前，欧阳家族的先祖，发现昆仑山里面灵气很不错，而且这里还盛产一些药材，所以就居家搬到了这里。<br/>　　<br/>　　这一居住，也不没有挪窝的打算了，反正都是习惯了！再说了，每个先天高手，都需要一个比较好的修炼环境。现在，欧阳家族还有一个两个先天高手，在闭关修炼中。<br/>　　<br/>　　这些，都是在路上的时候，欧阳若曦讲给陈默听的。自然，家族里先天高手，自然是一笔带过，并没有讲给陈默听。<br/>　　<br/>　　他们一路开车，然后到机场乘坐飞机，直接去了西北省会以后，在哪里在开车，一直到昆仑山脚下。<br/>　　<br/>　　当然，欧阳若曦也没有回家的打算，而是在接近家族的一个地方小城待着，等待父亲的到来。毕竟陈默在前面提了那么多条件，自家的环境有很令人无语，有很多无法给陈默解释的关系，干脆就在离家不远的地方，给父亲看看病。<br/>　　<br/>　　只要将病看完，如果治疗好了，那么由父亲出面，带着陈默去家族的药库取一株澜海花心，也就没有啥问题。要是治疗不好，那么也没有什么好说的，各回各家就成，反正出诊费用已经给陈默付过了！<br/>　　<br/>　　在陈默和欧阳若曦往家族赶的路途上，欧阳家族里也有人在议论着欧阳若曦。<br/>　　<br/>　　“爸！关于若曦最新的情报，送过来了！”欧阳若原走进四合院的后堂书房，见到欧阳轩后，有些急切地说道。<br/>　　<br/>　　欧阳若原，欧阳若曦的堂~哥，他的父亲欧阳轩，也就是欧阳若曦的堂伯，他们这一支，一直以来都想将族长的位置抢过去。<br/>　　<br/>　　对此，欧阳轩也是殚精竭虑的，花了极大的心思。<br/>　　<br/>　　欧阳靖，也就是欧阳若曦的父亲，这次练功岔气，导致功力后退，其实也是他在里面捣鼓的。不过家族里很少有人知道，知道的，也就一个人，就是这个走进房间的欧阳若原。因为这个事情就是他具体执行的。<br/>　　<br/>　　“给你说了多少次了，无论什么事情，都要沉住气，天还塌不下来！”欧阳轩一边写着毛笔字，一边缓缓的训斥道。<br/>　　<br/>　　“哦！父亲，对不起！”欧阳若原赶紧道歉，并且静心站在了欧阳轩的身边，看着他将一幅字写完。<br/>　　<br/>　　欧阳轩写的一幅字是：<br/>　　<br/>　　‘有志者、事竟成，破釜沉舟，百二秦关终属楚;<br/>　　<br/>　　苦心人、天不负，卧薪尝胆，三千越甲可吞吴。’<br/>　　<br/>　　写完之后，将毛笔轻轻放入笔架，然后缓缓的读了一边。这幅字，其实也是他的心声，计划了这么多年，眼看着自己离族长的位置，逐渐的接近，心情也是比较好的，所以今天写下这幅字，也是对这些年自己心声的一个表达，也是对自己的一个肯定！<br/>　　<br/>　　计划了这么多年，自己的族弟，也是现在的族长，欧阳靖，一身功力已经开始缓缓减退。如果功力下降到后天七层，他还能做族长么？<br/>　　<br/>　　不禁心中会心一笑，多年心愿终于要实现了。<br/>　　<br/>　　转头看了看儿子，对于这个大儿子，他还是比较满意的。不过由于年轻，心思还是有些沉不住！<br/>　　<br/>　　“嗯！你刚才不是有话要说么？说吧，什么事情让你沉不住气？”<br/>　　<br/>　　“父亲，是关于若曦的信息！”<br/>　　<br/>　　“哦？她的信息，那个方面的？”随手拿起一块汗巾，边擦手边走向茶台。<br/>　　<br/>　　欧阳若原赶紧上前，给茶壶中接水并烧上。至于说茶叶的拿取，放入等，他就没有插手，因为父亲对于喝茶有种仪式感，丝毫不让其他人插手，都是自己亲自动手。<br/>　　<br/>　　等将茶叶准备好，等着水烧开的时候，才示意欧阳若原继续说。<br/>　　<br/>　　“她又找了一个医生，可能已经在往家族回来的路上了。”<br/>　　<br/>　　“嗯!她还真是百折不挠啊！”欧阳轩倒是有些佩服这个侄女了，没有想到她能这么努力的为父亲奔走。<br/>　　<br/>　　“不算这个医生，她已经找了十几个国手了吧！”欧阳轩问道。<br/>　　<br/>　　“是！有十三个人了，这个算是第十四个了。”欧阳若原说道。<br/>　　<br/>　　“十四个了，还是不死心么？哎！”欧阳轩重重的叹了一口气。<br/>　　<br/>　　对于这个侄女这么努力，也许换成自己是她的父亲，那么心中就只有无限的幸福，女儿孝顺啊！<br/>　　<br/>　　但是现在，已经到了紧要关头，两者又是敌对关系，自然不想让欧阳靖有一丁点的恢复！<br/>　　<br/>　　那么，此刻欧阳若曦的行为，可就是有些令人讨厌了。<br/>　　<br/>　　“父亲，我安排些人手，将欧阳若曦阻挡住吧！”欧阳若原说道。<br/>　　<br/>　　此时，茶壶水已经烧好。欧阳轩拿起茶壶，给茶叶冲洗。<br/>　　<br/>　　一套行云流水的动作下来，给他和欧阳若原分别到了两杯茶。然后缓缓端起茶杯，轻抿而下，欧阳若原也是如此。<br/>　　<br/>　　“好茶！”欧阳轩说道。<br/>　　<br/>　　对此，欧阳若原却没有尝出来什么，只是感觉有股苦味，还有香甜的味道，咂砸嘴，最后也没有给出个什么评价。<br/>　　<br/>　　欧阳轩也不指望这个家伙能给个什么评价，老子给儿子弄个功夫茶，让他喝喝，已经是不错了，要是还要评价一二，绝对上前一顿洗码！<br/>　　<br/>　　“人啊！总是不撞南墙不回头啊！看来有些事情，终归是要让她明白的！”放下茶杯，欧阳轩意有所指的说道。<br/>　　<br/>　　“父亲，你是同意我的说法了，找人阻止欧阳若曦？”欧阳若原也是若有所思的问道。<br/>　　<br/>　　欧阳轩看了看儿子，还是年轻啊！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
