<br/>				晚饭的时候，陈默和李瑞，是和吴老、何老分开吃的饭，至于双方也就是点头认识了一下。<br/>　　<br/>　　陈默的药膳，经过小舅的一下午的制作，在晚上的时候受到了大家的欢迎。真的没有想到，药膳里用到的中药，可以说是非常的立竿见影。<br/>　　<br/>　　当然不是那种白发边黑发，皱纹消除等等的效果，而是那种喝了药膳后，就感觉浑身上下暖洋洋的不说，还能让人精神清爽。<br/>　　<br/>　　药膳只是配置了乾坤珠内的中药，主要是应用相关的食物和药膳治疗调养，而且陈默配置的主要是既将药物作为食物，又将食物赋以药用，药借食力，食助药威，二者相辅相成，相得益彰；既具有较高的营养价值，又可防病治病、保健强身、延年益寿。<br/>　　<br/>　　何老和吴老有各自的药物服用，所以药膳都是采取通用的规则，都是以养生为主的，这样熬制好以后，自家的人也能喝一些。<br/>　　<br/>　　李瑞这个吃货，晚上就狠狠的喝了两大碗。<br/>　　<br/>　　等晚上又聊了很长时间后，李瑞才去睡觉。陈默没有想到李瑞过来打乱了自己的修行，但是对于这种打乱，他也并不在意。<br/>　　<br/>　　如果修行修到孤家寡人的话，那还不如不去修炼，一个人孤独的活着，又有什么意思呢？<br/>　　<br/>　　进入练功房以后，已经接近了午夜十二点。为了防备声音传出去，陈默给整个练功房布设了隔绝法阵，一个非常简单但是也很实用的阵法。<br/>　　<br/>　　这两天有很多事情，一件接着一件，所以他都没有好好修炼。但是练气三层的瓶颈已经清晰可见了，感觉只要加把劲就能达到练气三层，所以不想再拖延下去，有的时候修炼就是这样，不进则退！<br/>　　<br/>　　准备好一切，尤其是灵液！在嘴里含一滴后，直接盘膝做好，缓缓运行玄天真诀，沿着平时修炼的路线运功。乾坤珠也在功法运行后，直接悬浮在了他的手心上部，与他的真元遥遥呼应，一进一出之间，增加着陈默的真元。<br/>　　<br/>　　和达到练气二层是的情景差不多，也是“轰！”然之间，整个精神体犹如蛋壳破碎后重新建立，他感觉到自己的精神又顿时大了一圈。<br/>　　<br/>　　瓶颈突破了，浑身上下洗涤一新，全身都像是在一个温暖的液体中游荡。每一个细胞都在欢呼，精神也像是升华般，让他欲罢不能，这种感觉，还真是令人舒爽难忘。<br/>　　<br/>　　等陈默又巩固了练气三层的精神力和真元，这才缓缓收功。闻到浑身散发的淡淡腥味，陈默也是感叹，每次升级，身体都要排出一定的废物，不过现在已经没有像是最开始洗筋伐髓那个阶段了，简直排除的东西是臭不可闻的。<br/>　　<br/>　　现在也就是淡淡的灰色物质，有着淡淡的腥味，看来身体内的杂质已经少多了。<br/>　　<br/>　　先给练功房用了一个清洁术，将异味全部消除。但是对自己却没有使用，回到卧房后，直接拿着换洗的衣物去了洗澡间，想好好泡泡澡。虽然可以用清洁术，但是却感觉总是少了点什么似得，所以陈默还是有时间就泡泡澡。也许这就是，那传说中贱人就是矫情的概念吧。<br/>　　<br/>　　等洗完澡，浑身上下都干干净净的，站在镜子前面，顿时有种自己又变帅气的感觉。每次升级，都是对身体的一种升华，所以外形也会随之变得更加柔和，更加自然。<br/>　　<br/>　　陈默不是那种帅气男人，也不是小鲜肉级别的人。但是自从修炼一来，他感觉自己渐渐在朝小鲜肉的不归路上走去。<br/>　　<br/>　　虽然自己有肌肉，看上去还非常的协调好看，但是在外面的时候除了成露出两只胳膊外，也就是穿个短袖，其他什么也露不出来啊！<br/>　　<br/>　　所以他已经到了小白脸的程度了，谁让他的皮肤这么好？这次练气三层，让他的皮肤也更加的白皙红润，并且有种细腻光滑的感觉。<br/>　　<br/>　　哎!照这样的状况修炼下去，自己可能不想引起注意也要引起注意了，因为自己实在是在逆生长啊！<br/>　　<br/>　　而且陈默现在的脸型也更加的圆润，更加的贴合自然。也就是说他虽然还是他，但是他的五官看上去更加的协调自然，也就是平时大家说的顺眼。以前长的普通，但是却变得越来越顺眼，也就慢慢的变的帅气起来。<br/>　　<br/>　　“感觉自己越来越萌萌哒！”陈默对着镜子臭美了一下，不错呦！<br/>　　<br/>　　等一切都收拾好了，再次返回练功房。他还想制作一些符箓，以防万一。上次被人阻击算是幸运，但是这种幸运又能持续多久呢？还是需要自身硬才行。<br/>　　<br/>　　要不是金刚符箓，要不是雷暴符箓，自己绝对是打不过王琦的，说不定就把命都给丢了。所以自己有才是真的有。<br/>　　<br/>　　自己虽然已经到了练气三层，但是很多符箓依然没有办法炼制，不过自己所拥有的一些可以补充全了，每样准备个十张百张的最好。<br/>　　<br/>　　闲话少说，开始绘制，当然乾坤珠内的灵液是不可少的，早就准备好一滴后，稀释放好，等自己真元不继的时候饮用。<br/>　　<br/>　　乾坤珠禁制的第二层，还有很多手法没有学会，所以进度有些慢，但是也解开了近三分之一的程度。等解开乾坤珠第二层禁制，面积也能扩大一倍以上，时间也能加快的更多。<br/>　　<br/>　　符箓的绘制是个水磨功夫，还有着失败率的制约。现在陈默绘制熟悉的符箓，能够达到4层的成功率，也就是说他绘制十次符箓，能有四次是成功绘制，其他六次都会失败。<br/>　　<br/>　　这在灵气匮乏的这里，却实在是非常高的成功率了。<br/>　　<br/>　　等绘制了几张符箓后，陈默又开始打坐恢复真元。一直到天色破晓后，才缓缓收功，又是新的一天来临了。<br/>　　<br/>　　陈默来到了葫芦谷，找了一个空地，开始练习王家长拳。这是他从王琦哪里抢夺过来的，虽然是体修的拳法，但是他也不介意去学习。主要是传功玉符中根本没有什么拳法，所以只能参考体修的拳法了。<br/>　　<br/>　　他和王琦对战的时候，人家王琦使用的长拳像模像样的！可是他呢，用的却是小儿打架的功夫，要不是他动作快，反应也快，仗着金刚符箓的护体，才能够和王琦对战。还是那种笨拙的出拳方法，真的有够丑的。<br/>　　<br/>　　王家长拳虽然没有多少招式，也就三十六招，但是攻击防御面面俱到，还是不错的拳法，要不王琦也不会珍惜的放到怀里了。<br/>　　<br/>　　其实，这个拳谱之所以能被王琦带出王家，还是因为他修炼天赋不错，另外完成了一些王家的任务，才能得到这样的奖励。但是由于记挂王强说的人参，又害怕弄丢这个拳谱，才随身带着，没有想到却便宜了陈默。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
