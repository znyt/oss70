　　“怎么回事？”
<br/>　　“这第九城之中到底是什么情况？”
<br/>　　“太古怪了，完全没有一丝的动静！”
<br/>　　公孙长绝等人，都是惊奇不已，这第九城的情况，让得所有人都摸不着头脑。
<br/>　　此时此刻，不仅是苏莫等人奇怪，其它所有城池之人，都是心中惊疑。
<br/>　　这诡异的一幕，让得所有人都不敢轻易攻击第九城，未知代表着危险。
<br/>　　接下来，各个城池都暂时沉寂了下来，变的一片平静。
<br/>　　整整几个时辰，都没有任何的动静。
<br/>　　苏莫有些坐不住了，这么干等下去，也不是办法。
<br/>　　“我准备出去探探其它几座城池的虚实！”苏莫向众人说道。
<br/>　　“带多少人？”公孙长绝问道。
<br/>　　“二十余人足矣！”苏莫沉声说道，其实，他是想自己单独出去的。
<br/>　　毕竟，就算遇到别人的袭杀，他一个人也比较好撤退。
<br/>　　但是，若是他一个人出去，公孙长绝等人再使出什么幺蛾子，那就麻烦大了。
<br/>　　所以，他便准备带着二十余人出去。
<br/>　　“嗯！”公孙长绝点了点头，并未反对，苏莫出去探探情况也好。
<br/>　　随后，苏莫挑选了二十余人，将龙双双以及两名人王宫的天才带上了。
<br/>　　包括他在内，一共二十二人，缓缓的飞出了第三城。
<br/>　　“苏莫！”
<br/>　　“是苏莫！”
<br/>　　“原来苏莫在第三城！”
<br/>　　当苏莫现身的这一刻，各大城池之中，很多人均是面色一凝。
<br/>　　第二城之中，贺兰殇眼眸微眯了下来，他早就猜测苏莫可能是在第三城，看来果然如此。
<br/>　　第八城之中，古魔太子眸中杀机一闪，苏莫终于现身了。
<br/>　　苏莫离开第三城之后，便带着人向第一城飞去，他的速度并不快，随时注意着其它几座城池的动向。
<br/>　　他一次出来，就是想要探出第一城、第五城、第六城和第七城都有什么人。
<br/>　　最主要的是，找到龙腾和叶凌天的所在。
<br/>　　眼见着苏莫就要飞到第一城，第六城之中，突然有一群人飞了出来，直扑苏莫而来。
<br/>　　这群人速度极快，越有四十余人，其中有魔族、有不死族、还有星辰族，个个气息强大。
<br/>　　当然，这些人之中，有一人苏莫极为熟悉，那就是楚天墟。
<br/>　　“嗯？”苏莫发现此景，立刻停了下来，遥望着急速而来的楚天墟等人。
<br/>　　“苏莫，他们人多，我们先撤吧！”龙双双面色凝重的说道，他们才二十余人，而对方有四十余人。
<br/>　　“不用！”苏莫脸色冷漠，微微的摇了摇头，他倒不是自大，而是无所畏惧。
<br/>　　在群战之中，他的吞噬之力作用极大，而且，无论是龙羽、龙腾、还是叶凌天，想必都不会看着他被围攻。
<br/>　　以龙腾和叶凌天的实力，无论在那一座城池之中，必然是统领者。
<br/>　　所以，他留下来，就能搞清楚每个城池的情况。
<br/>　　嗖嗖嗖！！
<br/>　　几个呼吸之间，第六城的四十余人纷纷到来，立刻将苏莫等人包围了起来。
<br/>　　“苏莫，你被淘汰了！”
<br/>　　为首的是一名魔族之人，眸中暗含杀机，面色冷漠的说道。
<br/>　　不仅是这位魔族之人，包括楚天墟在内，很多人眼眸中都蕴含着凌厉的光芒。
<br/>　　现在，苏莫被他们包围了，正是解决苏莫的时机。
<br/>　　“就凭你们，还不够资格！”苏莫冷笑一声，立刻便催动了吞噬元神，强大的吞噬之力席卷而出，敌方的四十余人全部笼罩在了其内。
<br/>　　“出手！”楚天墟等不及了，率先暴喝一声，向苏莫出手了。
<br/>　　拳如龙，耀眼的拳茫，如同一颗巨大的陨石，横贯长空，直击苏莫。
<br/>　　轰！轰！轰隆隆！
<br/>　　其他第六城之人，亦是没有闲着，纷纷出手了，各种攻击眼花缭乱，魔气沸腾、源力浩荡，铺天盖地的向苏莫等人袭来。
<br/>　　“杀！”
<br/>　　苏莫振臂一呼，立刻挥剑还击，一瞬间，他将血脉之力催动到了极致，全身光芒刺目。
<br/>　　唰！
<br/>　　一剑挥斩而出，无可匹敌的混沌剑气，直击楚天墟。
<br/>　　龙双双等人，亦是全力出手，抵挡这众多的攻击。
<br/>　　轰！轰！轰！！
<br/>　　双方的攻击碰撞，爆响声震动天地，大量的攻击爆开，形成升腾的蘑菇云，极为恐怖。
<br/>　　苏莫等第三城之人，人少势微，只是第一轮攻击，便有几人被重创，口中鲜血狂喷，仓皇逃窜。
<br/>　　这还是在有苏莫的吞噬之力影响之下，不然更加难以抵挡。
<br/>　　至于苏莫，丝毫无伤，他的防御力极为逆天，根本不可能这被这些人击伤。
<br/>　　就在此刻，异变再生，第一城之中，突然有大批的人飞了出来，径直向苏莫和楚天墟等人所在之地飞了过来。
<br/>　　这群人数量众多，足有八十余人，为首之人正是人王宫的叶凌天。
<br/>　　由于第一城距离很近，叶凌天等人几乎是瞬息而至，立刻在外围又形成了一个更大的包围圈。
<br/>　　“什么？”楚天墟等一众第六城之人见此，顿时个个面色一变，他们还未来得及发动第二轮攻击，已经陷入了包围之中。
<br/>　　“剿灭第六城之人！”叶凌天面色冷漠，二话不说，朗喝一声，便立刻开口了。
<br/>　　轰轰轰！！
<br/>　　霎时之间，铺天盖地的攻击，疯狂的向楚天墟等第六城之人攻去。
<br/>　　“杀！”龙双双等第三城之人，不用苏莫指挥，便纷纷出手，内外夹击第六城之人。
<br/>　　大战再次爆发，第六城之人面临内外夹击，只能拼命还击，抵挡内外的攻击。
<br/>　　场面混乱不堪，爆炸声、惨叫声不绝于耳，一轮攻击之下，第六城之人足有一大半被重创。其中更是有几人肉身崩溃，元神在混乱的冲击波之中，无法逃走被苏莫直接吞噬了。
<br/>　　逃！
<br/>　　剩下的第六城之人，哪敢多留，个个身形爆闪，开始急速逃窜。
<br/>　　楚天墟的实力很强，并未受伤，立刻就放弃了袭杀苏莫，同样开始逃窜。
<br/>　　不过，此时此刻，任谁都没有想到的是，古魔太子再次出城了，率领七八十人，向苏莫所在之地杀了过来。
<br/>　　一群人气势腾腾，速度如风似电。
<br/>　　“剿灭第一城和第三城之人！”古魔太子朗声暴喝，他此行的目的，是擒拿或者击杀苏莫，至于剿灭第一城和第三城只是幌子而已。
<br/>　　那些溃逃的第六城之人，一见古魔太子前来相助，顿时纷纷停了下来。
<br/>　　而这时，第六城之中，又出来了三十余人前来增援溃败的楚天墟等人。
<br/>　　局势，变得彻底不可控了起来，第一城、第三城、第六城、第八城，共计有二百余人。
