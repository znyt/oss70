　　第643章
<br/>　　和珅一张脸瞬间变得红润起来，连夜劳作的疲惫此刻也烟消云散。能管理一座城池，他自己的野心也迈出了第一步。
<br/>　　程大雷给了他一个鼓励的眼神，问：“物资清点得如何？”
<br/>　　“禀大当家，兵器船只倒是小事，关键是粮食。有了这批粮食当解了燃眉之急，朔方城至少能撑到今年秋收。”
<br/>　　程大雷点点头，道：“也不可太过大意，朔方城的人口不会局限在目前，随着流民不断迁入，人口越来越多，所消耗的粮食必然也越来越大。还是要让朔方城能自给自足，对了，垦荒的事做得如何了？”
<br/>　　“河渠都已挖好，原先荒废的农田已经播种下去。”
<br/>　　程大雷站起身：“好，我去看看。”
<br/>　　“我陪大当家一起去。”和珅立刻道。
<br/>　　“嗳，和大人劳累一夜，还是好好歇息，我自己去看看就好。”
<br/>　　程大雷一个人走出城主府，见城中一派忙碌气象，他们正在修葺城池，许多坍塌的房屋要将废墟清理掉。目前朔方城还是房多人少，原先的朔方城能容纳三十万人口，目前却不足十分之一。
<br/>　　他信步走出城门，来到城外的农田，这里倒是清净些。在播种之后，大家就等着种子出土，然后除草施肥等诸多事务，在经过一年忙碌之后，方才能在秋天收获一些可怜巴巴的粮食。
<br/>　　程大雷深吸一口气，使用了系统奖励的技能。系统奖励的作物加速生长技能，程大雷只等着现在才使用。
<br/>　　眼前也无太多变化，程大雷所期待的白光一闪，种子便蹭蹭生长的场面并没有出现。
<br/>　　不得不说，程大雷此刻心中有些失望。
<br/>　　忽然，他脸上一凉，抬起头看到天空彤云密布，细细秘密的雨丝层层叠叠飘落下来。
<br/>　　一场春雨扑面而来。
<br/>　　程大雷立刻转身往城中跑，但最后跑进城门时仍然被雨打湿了肩膀。
<br/>　　他和几个老兵在城门下躲雨，见天空黑得透不过气来，守门老兵一开始没认出程大雷，聚在一起聊天。
<br/>　　程大雷盯着外面的雨帘，有了这场春雨，朔方城多少有了些起色。自己可以将关鱼的鱼字军调来。关鱼与和珅一正一邪，正好互相钳制。
<br/>　　想到此处，程大雷发出一声长叹：事到如今，这些人心苟且自己也算是无师自通了。
<br/>　　嘟，收到任务：将朔方城人口扩充到十万。
<br/>　　嘟，收到任务：修复破败的城池。
<br/>　　这时程大雷收到两条系统任务，扩充人口的任务也非一时一日之功，但为程大雷洗白名声的队伍已经散出去，只要朔方城慢慢发展起来，不愁没有流民迁入，一切只是时间问题。
<br/>　　而修葺城池的工作一直在做，却也用不了太长时间。
<br/>　　所以这两项任务，不用程大雷多加甘于，迟早都会完成。
<br/>　　此刻几个老兵都认出了程大雷，却是没想到会和大家一起躲雨，各人慌张跪倒：“大当家……”
<br/>　　程大雷摆摆手让他们起来，每个人却都激动得不行。蛤蟆城现在兵力超过三万，程大雷也不可能保证每个都见过。蛤蟆城中不了解他的人大有人在，所以程大雷不仅在外的名声很糟糕，在蛤蟆城内部也不是很好。
<br/>　　程大雷也被搞得有些尴尬，冒雨回了城主府，心里盘算着如果这里的事情迈入正轨，自己也可以回蛤蟆城了。
<br/>　　恰在此时，雨中传来急促的马蹄声，刘发财身穿蓑衣，骑着一匹快马赶来。来到城主府门前，他跳下马后马也不拴，便大步走进城主府。
<br/>　　程大雷正在花厅下观雨，见他过急匆匆过来，心中也有些奇怪。
<br/>　　“大当家，出事了。”
<br/>　　“什么事？”
<br/>　　刘发财步入花厅，将脸上的雨水抹去，道：“东海反了。”
<br/>　　程大雷皱起眉头：“别着急，慢慢说，东海谁反了？”
<br/>　　东海一地，是诗书礼乐之乡，程大雷去年还在琅琊城折腾一遭。听刘发财一一言来，程大雷才明白事情经过。
<br/>　　造反的倒不是东海王，乃是一伙草寇。他们打着正义教的旗号，在徐州拉起数万人马，在东海一地烧杀抢掠，官兵竟不能形成有效阻拦。
<br/>　　如今他们占了琅琊城，自立为王。
<br/>　　这种事虽然是第一次发生，但也是迟早会发生的事情，程大雷倒也不觉得太奇怪。问道：“他们的首领是什么人，有消息没有？”
<br/>　　“名叫刘顶天，据说原先是海上一伙海盗，具体消息还没有查清楚。”
<br/>　　东海与凉州相距太远，委实也差不到什么准确消息。他点点头：“反了就反了吧，却也不知他们会折腾起多大风浪。你急匆匆过来，就是为了这件事？”
<br/>　　“华大夫的家人……怕在这次事情中遭了殃……”刘发财低声道。
<br/>　　呀！
<br/>　　程大雷惊呼一声，他倒是将这件事遗忘了。自从华正则来到蛤蟆城后，程大雷一直在想办法留住他，这些天窝在朔方城不回去，未必没有避而不见的原因。却是没有想到，不用程大雷找，理由自己找上门来。
<br/>　　东海一乱，华正则成了无家可归之人，也不用程大雷想什么办法了。
<br/>　　“华大夫的情绪可还好？”程大雷问。
<br/>　　刘发财摇摇头，他一路随着华正则过来蛤蟆城，二人自然有些交情。
<br/>　　“华大夫知道这件事事后，痛心得直接昏厥过去，据说他有一位青梅竹马怕也在这次事情中遭了难。如今寻死觅活，由刘军师陪着。”
<br/>　　程大雷无可奈何的叹口气，那华正则并不属于乱世，心里只想着治病救人，安稳过日子。可逢上这样一个糟糕的世道，其实谁也身不由己。
<br/>　　“啊呀，你刚刚说什么，刘军师陪着他？”
<br/>　　“是呐，刘军师也是心热之人，看不得华大夫如此。”
<br/>　　程大雷吓了一跳，刘悲不在还好，刘悲在的话，活人也能被他折腾死了。更何况是华正则这寻死之人。
<br/>　　“快快，牵我的牛来，我要回蛤蟆城，事不宜迟。”
<br/>　　程大雷跨上黑牛，不敢停留片刻，冒雨返回蛤蟆城。
<br/>(本章完)
