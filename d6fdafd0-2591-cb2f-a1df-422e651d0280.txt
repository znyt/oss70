　　“诸位道友，百年时间将至，老师即将开讲大道，紫霄宫内，不得喧哗，尔等速速进来，不可怠慢！”
<br/>　　“道友此言大善！”
<br/>　　众人一听，纷纷点头称是，向着宫内走去。
<br/>　　元始淡漠地看了众人一眼，当先带着老子与通天进入宫殿，在他们之后，女娲伏羲，鲲鹏冥河，镇元子，红云老祖等尾随而进。
<br/>　　一入宫殿，众人只觉得天地为之一变，原本灰蒙蒙的空间瞬间变化成一个鸟语花香，仙气缭绕的仙境洞天。在洞天的最前面的虚空，有一个巨大的广场，广场之上，空旷的地面洁白如玉，高耸着周天三百六十五个古朴的柱子，每棵柱子上都镶嵌着星辰般的璀璨纹路，玄妙无比！
<br/>　　当然，这还不是重点，重点是，在广场的最前方，大殿的主位，讲道云台之下，有着六个蒲团，仅仅六个，这蒲团看似寻常，实则仿佛蕴含着一股玄奥的气机一般。这么一来，众人如何还不明白，这蒲团必定是代表一种大机缘啊。
<br/>　　只是此处是紫霄宫，不好斗法，众人只能冲上前，谁争到就是谁的。
<br/>　　元始带着老子与通天，一步跨出，便到了蒲团之边，三人第一时间坐到最前方的三个座位，之后睁着眼睛，饶有趣味地看着其他人如何争抢。
<br/>　　有空间法则，就是那么任性。
<br/>　　至于其他人就没有那么好运了，紫霄宫中乱哄哄一团，众修士争抢蒲团，女娲与伏羲靠着在元始三人之后进入宫殿的优势，在元始三人坐下后不久，也飞到了蒲团旁边，这时，其他人也快要到了。
<br/>　　“哥哥，你坐吧。”女娲对着伏羲说道。
<br/>　　这时，伏羲眼尖，顺势就将女娲推坐在第四个蒲团上面，他刚想要去坐地五个，却被擅长速度的鲲鹏道人与红云道人抢夺了下来，坐稳了。伏羲无奈，向着女娲摊了摊手，只好在女娲后面坐了下来。
<br/>　　而这个时候女娲则是很气愤，怒视着两人，这人怎么能如此无耻呢，竟然抢别人的座位，太可恶了。
<br/>　　鲲鹏满脸阴沉，脸色低沉得可怕，好像谁都欠他多少灵宝似的，周身散发着令人十分不爽的气息，面对女娲的怒视，他直接无视掉。而红云道人倒是有些不好意思，脸上有些尴尬，不过，他也没有起身的意思。
<br/>　　“不想却是被他们几人抢了先机。哎？”
<br/>　　众修士见状，纷纷无奈。
<br/>　　无奈，众修士只好抢占第二排的位置，虽说第二排没有蒲团，但此处却是除了第一排之外最靠近讲道云台的地方。镇元子、帝俊、太一、冥河等人，马上找到了适合自己的位置坐下，不多久，紫霄宫就坐满了人，寂静下来。
<br/>　　众人都在闭目深思，修养好精神，等着鸿钧道人的出现，之后，也有陆陆续续地几人，来到了紫霄宫，见没有座位，他们无奈，只好坐在最后面。
<br/>　　就快要到了百年期限之时，就在此时，宫门外一阵急促脚步声传来，所有人神情一动，元始也扭头望去，却看见两个道人，衣衫褴褛，神色焦急，浑身上下很是狼狈地踏入紫霄宫。
<br/>　　这俩道人，眼见紫霄宫大殿前全是修士，而且最前方的六个蒲团，一个个更是坐满了，不由脸色难看起来。
<br/>　　其中一人眼睛一转，顿时大哭起来，其状甚是悲惨，大声嚎叫道：
<br/>　　“师兄啊，想不到你我师兄弟二人不远亿万万里只为听道而来，本想着为西方众生，求得大道超脱，不再受那魔气的影响，可事实却是如此的残酷。
<br/>　　我俩到了这紫霄宫，竟然连个座位都没有，这是不是太欺负我们西方修士了？我们西方虽然贫瘠，但人穷志不穷，如此，我还有何脸面去见西方修士啊。如此，还不如死了算了。”
<br/>　　声音之悲切，让人听后潸然泪下，感动至极。那道人悲呼之中，一头撞向旁边的柱子，想要一死了之。元始冷笑，看着他们表演。
<br/>　　“师弟，切不可如此啊，你我随便找个角落坐下，亦可听得大道啊！”
<br/>　　另一个道人见状，急忙阻拦，脸上亦露出疾苦之色，一副死了爹娘的模样。让人看了直呼悲惨。
<br/>　　“道兄，不可如此！”
<br/>　　虽然大多数人冷眼旁观，只要是与自己无关的事儿，自己毫不理会，就当是看热闹好了。但，亦有人感觉他们是真的很惨，纷纷劝道。
<br/>　　至于元始等人，更是连口都懒得开，脸上露出冷笑，纷纷表示看戏。元始突然看向最后一个蒲团上的红云道人，只见红云道人脸上亦露出同情的表情，但他也没有起身的想法。
<br/>　　元始感觉，这不对啊，传说就是最后红云道人起身让座的啊，怎么如今与他知道的不符啊？
<br/>　　陡然，原本还不想起身的红云道人眼中出现瞬间的呆滞，但片刻就恢复了正常，只是他脸上的同情之色更甚了，他突然起身，大声道：
<br/>　　“道兄万万不可，贫道此位让与你也是无妨。”说着，他还上前去拉住那道人。
<br/>　　“多谢道兄。”那道人见状大喜，急忙坐下，随口道谢道。那样子，哪还有半点悲惨之状啊。
<br/>　　红云道人脸色尴尬，大殿之内的众多修士亦是脸色顿时僵硬，看着红云，如同是在看傻子一般。
<br/>　　元始瞳孔微微一缩，眼中闪过一丝精光，他十分确信自己刚才并没有看错，那红云眼中的确是出现了瞬间的呆滞，尽管那十分短暂，短暂到让人无法察觉，就连元始，若不是他仔细地盯着红云，他也发现不了。
<br/>　　这让元始十分震惊，刚刚红云道人的状态绝对不对，那不是他的本意，但之后他又为何为改变主意？到底是谁？鸿钧？还是天道？亦或者另有他人？
<br/>　　元始只感觉自己发现了什么，但又没有发现什么，红云道人的事他不想管，当然，他也管不了。事不关己，高高挂起，他只是来听道的，其他的吗，就相当于来看戏。
<br/>　　而就在这时，坐了红云的蒲团的那道人见自己师兄还没有座位，便动了其他的心思。
<br/>　　他打量着前五个坐在蒲团上的修士，元始三兄弟，一看就是三人连在一起，不好对付，直接过了。女娲，是一个女子，他也不好意思去欺负一个女子，她也过了。最后，也就只剩下了一个满脸阴沉的鲲鹏道人。
<br/>　　一番权衡后，他目露果断之色，骤然向着身旁的鲲鹏道人发难，大声喝道：
<br/>　　“你是何人，怎有资格与我同坐，还不退下！”
