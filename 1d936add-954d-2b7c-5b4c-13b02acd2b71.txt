　　当然，就算巫族不来找苏莫的麻烦，他也会去找巫族的麻烦。
<br/>　　若是宏青璇遭遇不测，将来他必会屠尽整个巫族。
<br/>　　巫族之事可以先放在一边，目前，苏莫最主要的还是学习炼器之术。
<br/>　　片刻之后，苏莫开始检查自己的储物戒。
<br/>　　学习炼器，总要有炼器材料才行，他杀了那么多巫族，获得了大量的储物戒，也有一些零散的炼器材料。
<br/>　　再加上，他的身上本身也有一些材料。
<br/>　　一番搜寻之下，苏莫找到了几百种炼器材料，不过，能用于炼制帝级宝器炼制的材料，不足二十种。
<br/>　　而且，这二十中材料中，只有一种是炼制重渊斧的材料。
<br/>　　这让苏莫非常无奈，他是准备炼制重渊斧的。
<br/>　　毕竟，重渊斧是炼器传承中最容易炼制的兵器，他只要能炼成重渊斧，就说明有帝级的炼器水平，加入炼器师公会便不成问题了。
<br/>　　“炼器师公会中应该不缺材料！”苏莫暗道，不过帝级材料的价格肯定不低，他现在并没有太多的灵石购买。
<br/>　　而且，他以前的炼器熔炉级别太低，已经不堪大用，必须重新购买。
<br/>　　好在他身上宝物众多，各种战利品无数，都能兑换成灵石。
<br/>　　少倾之后，苏莫整理出十件帝级下品兵器，准备全部换成灵石，用于购买炼器材料。
<br/>　　随后，苏莫没有急着去行动，而是意念一动，进入了虚界神图之中。
<br/>　　神图空间。
<br/>　　大地一片狼藉，山峰也尽数倒塌，整片空间已经遭到了严重的破坏。
<br/>　　百万苍穹神宫之人，正聚集在一起，嘈杂的声音此起彼伏。
<br/>　　四个月了，苏莫还没有进来，所有人都是急了。
<br/>　　众人正将巫娴以及临两名巫族天才围在中间，询问情况。
<br/>　　这三人，都被苏莫收进了他的神图空间。
<br/>　　不过，巫娴三人盘膝而坐，双眸紧闭，根本不理会众人。
<br/>　　嗡~~~
<br/>　　空间一阵颤动，苏莫的身影突兀的出现在了高空中。
<br/>　　“苏莫！”
<br/>　　“宫主！”
<br/>　　“宫主终于来了！”
<br/>　　见到苏莫出现，众人顿时大喜，纷纷围了上来。
<br/>　　“苏莫，怎么样了？离开苍穹世界了吗？”龙腾率先问道，其他人均是个个睁大了眼眸，满脸期待之色。
<br/>　　这么长时间了，应该早已离开了苍穹世界了吧！
<br/>　　苏莫目光扫视四方，掠过一众苍穹神宫的长老和弟子，朗声道：“诸位，现在我们已经离开了苍穹世界，来到了天荒星域！”
<br/>　　“噢！”
<br/>　　“太好了！”
<br/>　　“哈哈！终于出来了！”
<br/>　　“好期待啊！不知道外面是什么样的世界！”
<br/>　　无数的苍穹神宫弟子，欢呼出声，振奋不已。
<br/>　　就算是帝释、海明江等老一辈强者，也是难掩喜色，个个神情激动。
<br/>　　这一切，简直都像做梦一般，他们并没有出什么大力气，苏莫就已经完成了目的。
<br/>　　“不过，现在我正在遭受巫族尊者的追杀，外面太危险，待稳定下来之后，再安排你们出去！”苏莫继续道。
<br/>　　“什么？尊者的追杀？”
<br/>　　众人闻言纷纷心神剧震，遭到尊者的追杀那还得了？焉能活命？
<br/>　　“苏莫哥哥，那你怎么办？”夕儿上前，满脸担忧的问道。
<br/>　　“没事！”
<br/>　　苏莫拍了拍夕儿的手，然后对众人道：“我自有保命的办法，大家无需担心！”
<br/>　　随即，苏莫安慰了众人一番，让大家安心修炼便可，等待他的好消息，并且看好巫娴。
<br/>　　众人虽然不知道苏莫有什么办法，但也帮不上任何的忙，个个满脸担忧之色。
<br/>　　而后，苏莫便飞向了不远处倒塌的山峰。
<br/>　　山峰倒塌，芊寻月的灵柩也被埋在了乱石之中。
<br/>　　不过，苏莫也没有担心，有玄天灵柩保护，芊寻月的躯体不会有损。
<br/>　　很快，苏莫便将玄天灵柩从乱石之中取了出来，玄天灵柩丝毫无损。
<br/>　　随后，苏莫便将玄天灵柩收进了储物戒之中。
<br/>　　储物戒中虽然不能存放活物，但目前的芊寻月并没有生机，可以轻易存放。
<br/>　　最后，苏莫便离开了虚界神图，再次回到了外界的房间之中。
<br/>　　房间中，秦不死正在打坐调息，修养声息。
<br/>　　苏莫将对方的空间玉佩还给了对方，然后向对方打了声招呼，便离开了房间。
<br/>　　离开房间之后，苏莫再次来到了一楼，找到了卢云，并邀请对方回房间一叙。
<br/>　　卢云没有拒绝，毕竟现在的苏莫，很可能是帝级炼器师，说不定过几日就是公会的炼器师，他一个小小的炼器学徒哪敢明面上得罪！
<br/>　　苏莫和卢云在房间内谈了很久，他给了卢云十件帝级神兵，让其售卖成灵石，然后购买二十份炼制重渊斧的材料。
<br/>　　当然，苏莫也不会亏待对方，结余下来的灵石就算是给对方的劳务费了。
<br/>　　十件帝级下品兵器的价值，足以购买三十份重渊斧的材料了，所以，苏莫已经算是非常的大放了。
<br/>　　卢云见此顿时大喜，立刻就应承了下来，剩余的那些灵石，足够他自己购买大量的炼器材料，挥霍一段时间了。
<br/>　　当然，苏莫也没有忘记打探消息，他从卢云的口中，问到了很多关于这无尽星空的信息。
<br/>　　这无穷无尽的星空，统称为洪荒宇宙，洪荒宇宙分为四大星域，分别是神荒星域、古荒星域、玄荒星域以及他目前所在的天荒星域。
<br/>　　天荒星域，大到无边无际，拥有上千星河。
<br/>　　所谓的星河，便是有大量生命星辰聚集的区域，而所谓的生命星辰，便是灵气浓郁，适合生存与修炼的星辰。
<br/>　　每一个星河范围都非常的广阔，至少都有数千乃至上万个生命星辰，囊括了很大一片星空。
<br/>　　这让苏莫震撼不已，一个星河便如此之大，那整个天荒星域，乃至整个洪荒宇宙呢？
<br/>　　他的心灵感觉到了深深的震撼，更有对这无尽星空产生了极大的好奇与向往。
<br/>　　洪荒宇宙之大，武道发展到了何种地步呢？
<br/>　　而苏莫目前所在的星河，名为古灵星河，他现在所踏足的星辰，名为北陵星。
<br/>　　北陵星在古灵星河数千生命星辰之中，算是中等大小的星辰，武道非常昌盛，武尊境强者无数，甚至还有少量超越武尊的存在。
