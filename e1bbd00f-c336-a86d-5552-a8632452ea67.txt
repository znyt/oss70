<br/>    两年之后。
<br/>    
<br/>        两年的时间，陈明已经存够了不少的师德，现在距离三百枚道种倒是差距不远了，门下的弟子也有上千突破了王级，燕山的实力已经是暴增。
<br/>    
<br/>        陈明在这两年之内，自然也不是无所事事，他已经在燕山全境安插了摄像头，额，银镜阵法，确保整个燕山的安全。
<br/>    
<br/>        对于燕山发生的一切，陈明可谓是了若指掌，闲暇之时，就是找了熊大熊二加上老龟搓麻将。
<br/>    
<br/>        修行？不存在的。
<br/>    
<br/>        有那个功夫还不如找紫霞仙子探讨一下人生呢。
<br/>    
<br/>        如今大陆之上的局势可谓是风云变化，不过在一个月之前，魔宗已经承认王级战场输了，彻底龟缩，而这一次的功臣，卓青瑶和凌仙，算算日子，今天也该回来了。
<br/>    
<br/>        而且也奇了怪了，我陈仙师风流倜傥，玉树临风，为啥没有妹子主动找我呢？
<br/>    
<br/>        还真是奇怪啊！
<br/>    
<br/>        卓青瑶走在上山的路上，莫名打了个哈欠，“怎么了，我的身体，不可能会着凉啊。”
<br/>    
<br/>        凌仙笑道:“可能是因为师父在想你呢，你也知道，师父这个人，就在意我们几个，可惜，我还是没有把三师弟带回来。”
<br/>    
<br/>        卓青瑶道:“走吧，上山去吧，师父可能已经等的着急了。”
<br/>    
<br/>        待得两人来到院子外，就听到陈明大笑的声音，“哈哈哈，清一色十八罗汉，大满贯，给钱给钱，快点，不就是一千个铜子吗，你们至于吗？”
<br/>    
<br/>        凌仙看向卓青瑶，“师父，应该记得我们今天回来吧？”
<br/>    
<br/>        卓青瑶直接推开门，可以看得到陈明正在兴冲冲地数钱，满面笑容。
<br/>    
<br/>        陈明听到开门声，看向门口，嗯，凌仙回来了，小伙子不错，长得更加帅气了，不过他旁边这个是谁？
<br/>    
<br/>        一身红衣，明眸丽齿，唇红齿白，婀娜多姿，发若流碧，肤若凝雪，这个容貌，可谓是倾国倾城，闭月羞花，就是陈灵鱼，在这个女孩面前都略显青涩。
<br/>    
<br/>        陈明皱了皱眉头，“美女，你谁啊，你是怎么混到我们燕山来的？”
<br/>    
<br/>        卓青瑶的双眼逐渐变得寒冷，自古只见新人笑，哪里见到旧人哭，这才多久，师父就把我忘了，等等，我为什么会想到这句话。
<br/>    
<br/>        “师父！”
<br/>    
<br/>        陈明恍然大悟，女大十八变，果然名不虚传，这才两年的功夫，已经漂亮的我认不出来了
<br/>    
<br/>        陈明打了个哈哈，“青瑶啊，两年不见，已经漂亮的我认不出来了，既然已经回来了，还不赶快去给师父做碗面。”
<br/>    
<br/>        凌仙抬头看天，两年以来，大师姐的脾气越来越大了，不知道师父怎么哄好了，不会吧，师父直接让大师姐去做面？
<br/>    
<br/>        然后凌仙就看到卓青瑶盈盈一笑，一挥衣袖，已经朝着厨房去了。
<br/>    
<br/>        师父当真是鬼才，也唯有师父，才能降服住大师姐了。
<br/>    
<br/>        熊大熊二老龟，见到凌仙和卓青瑶回来，也是自觉地离开，自觉的将陈明桌子上的铜钱全部拿回自己的腰包，退去了。
<br/>    
<br/>        凌仙看向陈明，拱手道:“师父！弟子，弟子还是没有将三师弟带回来，魔宗已经将三师弟保护起来了。”
<br/>    
<br/>        陈明点了点头，“无妨，回来了就好，你三师弟，等等，我还有个三徒弟？哦哦哦，对，素衣嘛，我记得的，你三师弟的事，为师会想办法的。”
<br/>    
<br/>        李素衣到魔宗，反正不是受苦，而且干坏事也以失败告终，惦记那么多干嘛？
<br/>    
<br/>        凌仙不禁感动的涕泪横流，“师父思念三师弟成疾，一定是想到三师弟便泪流不止，为了避免自己思念过度，所以才选择暂时忘记三师弟，师父对三师弟的关爱，弟子铭记于心！”
<br/>    
<br/>        陈明淡淡挥手，然后道:“别说了。”
<br/>    
<br/>        凌仙道:“弟子明白，弟子不会再提及三师弟了，对了，师父，弟子还有一件事要告诉师父。”
<br/>    
<br/>        陈明无所谓地道:“说吧！”
<br/>    
<br/>        凌仙咬了咬牙，反正都是死，干脆死的壮烈点，“师父，我与魔宗圣女两情相悦，已经私定了终身，没有禀报师父，特来请罪！”
<br/>    
<br/>        凌仙在陈明的身前跪下，低头，不敢去看陈明的表情，过了一会儿，却见陈明哪儿还是没有什么动静，凌仙小心翼翼地抬头，看向陈明，陈明的脸上，表情似乎是凝固了一般，双眼瞪圆。
<br/>    
<br/>        在凌仙的眼中，陈明分明是气得不轻啊！
<br/>    
<br/>        然而陈明现在正在想的是，这个剧本我好像看过，里面不都是喜欢写什么正道天才爱上了魔宗圣女之类的情节吗？
<br/>    
<br/>        然后正道天才和魔宗圣女的不伦恋被正道和魔宗排斥，视为禁忌之恋，然后的剧情就应该是正道天才被逐出宗门，沦为魔道，而后挑起整个大陆的战争，而且极为仇视原来的宗门。
<br/>    
<br/>        这个剧本不太对啊，三徒弟已经离家出走了，现在这个架势，如果按照剧本走，二徒弟也要离家出走了啊。
<br/>    
<br/>        还好我陈仙师从来不按照剧本走，凌仙以为陈明已经气糊涂了，连忙拱手道:“师父，不要因为弟子的事气坏了自己，这就是徒弟的不对了！”
<br/>    
<br/>        让凌仙没有想到的是，陈明却是一脸惊喜地问道:“有孩子了吗？生了吗？几个月了？”
<br/>    
<br/>        凌仙愣了愣，师父是不是气糊涂了，“师父，你不生气啊？”
<br/>    
<br/>        陈明很是满意地点头道:“这有什么好生气的，魔宗无疆抢我一个徒弟，你抢他一个圣女，简直干得漂亮！”
<br/>    
<br/>        陈明继续道:“对了，既然两情相悦，那就赶紧把婚事定下来，如果还没成婚就有孩子了，也没问题，先把孩子带回来给我看看，没有想到，我居然也是做师公的人了。”
<br/>    
<br/>        凌仙有些吃惊地看向陈明，脸上刷红刷红的，“师父，这个，我们还没有孩子。”
<br/>    
<br/>        他们现在就停留在拉拉小手的阶段，哪儿来的什么孩子。
<br/>    
<br/>        陈明很是意味深长地看了凌仙一眼，淡淡道:“阿崽，为师对你很失望。”
<br/>    
<br/>        凌仙生怕陈明继续问东问西，飞也似的跑了，这个时候，卓青瑶已经是端了一碗面来到了陈明的身前，陈明扫了一眼卓青瑶，脸上还有着些许面粉，伸手落在卓青瑶的脸颊上，入手丝滑，“多大的人了，还弄得一脸都是的。”
<br/>    
<br/>        卓青瑶的脸上闪过一抹羞红之色，“哎呀，师父，吃你的面！”
<br/>    
<br/>        “叮，你触发了任务，补全，卓青瑶已经意识到了自己的天劫将至，请帮助卓青瑶补全，帮助她成功度过天劫。奖励，师德:十万点。”
