　　老郭侃侃而谈：“这问题，我还真研究过，原因就在于，泰山的灵气实在是太旺了，如果有门派在这里住下，长期以往，势必影响巨大，当皇帝的自然不允许这种事，历代皇帝都会封禅泰山，把泰山当成是皇家专注，哪里有人敢在这里立派。要说道观佛寺也有一些，都是古代遗存的一些装点门面，不成气候。”
<br/>　　快过年了，天又冷，再热门的景点也没什么人了，两人在山脚下的客栈订了三间房，先住下来，老郭就开始打电话，找当地法术界的朋友，因为是秘密行动，叶少阳提前交代别说出自己名字，老郭电话里就说自己是来旅游的，见面一起吃个饭之类的。
<br/>　　挂上电话，老郭告诉叶少阳，自己找了两个当地法术界的朋友，回头去见个面。毕竟是要在这边办事，之后布阵什么的，材料当然只能从本地找，必须得有当地法术界朋友帮忙。
<br/>　　“我是真佩服你，哪里都有你朋友。”叶少阳感慨。
<br/>　　“都是生意上的朋友，进货的时候认识的。”老郭嘿嘿一笑。
<br/>　　“哦，他们不认识黎山老母吧？”
<br/>　　“那哪能啊，就他们的身份，想告密都没地方，我就说是来玩的，晚上去见下。”老郭看了下手表，“下午还有时间，我们要不要先去下泰山，熟悉下环境？”
<br/>　　叶少阳觉得也好，问了瓜瓜他们，也都想上山看看，于是一起出门，买了票一起上山。
<br/>　　根据林三生的说法，泰山虽然大、周围的山不少，但封禅仪式只能在主峰进行——否则就不叫封禅泰山，于是他们直接爬上主峰。
<br/>　　泰山的风景还好，细致地观赏起来，比不上黄山、庐山这些秀美风光，但是山势雄壮巍峨，连绵百里，看上去极为雄壮，是别处都感受不到的气势。
<br/>　　几人一边上山，一边讨论起东岳大帝。说起来，东岳大帝最早并不是道教的神仙，而是华夏民间供奉的神，后来道教正式册封，虽然名头很大，是个“大帝”，但是没什么实权，一向守护着泰山生死地。
<br/>　　从人间到鬼域，有好几处自然的虚空裂缝，南方在酆都，北方就在泰山，泰山一直设有生死道，东岳大帝一门，就负责引渡整个山东地区的生魂，从泰山生死地进入鬼域。
<br/>　　后来，阴司进一步完善鬼域，封锁了人间一些大的空间裂缝，避免出问题，泰山的生死地也被封锁，之后便是佛门介入，转轮王接手轮回司，与崔府君并立，人间所有生死之事，都是这二位主持，东岳大帝就在泰山清修，后来诸神之战，与东皇太一和尸王混战，开辟了青冥界，带着几个徒弟搬了进去，后来留下他们，自己不知道去了哪里。
<br/>　　东岳大帝的下落，法术界一直都是迷，这次黎山老母他们的行动，叶少阳乍一听的时候还挺意外的，仔细一想，黎山老母和摇光仙子是东岳的徒弟，对他的下落自然比一般人知道的清楚。
<br/>　　会当凌绝顶，一览众山小。
<br/>　　爬到泰山山顶，俯瞰四周，真的有这种感觉。
<br/>　　老郭累的气喘吁吁的，跟在叶少阳身边，不住感慨，二十年没来，连山路都变得不一样了。
<br/>　　“泰山很大，有不少山峰，不知道他们祭司封禅，是在那一座山？”叶少阳挠着头。
<br/>　　“我想应该就是这主峰了。只有这里才是真正的泰山，你看这些碑文什么的，说明历代封禅祭祀都是在这里。”
<br/>　　休息了一会，一行人沿着山路在山顶一带转悠，在一座被灌木和乱草掩盖的峰头上，找了一座破败的寺庙，连匾额都没了，四周围着一圈栏杆，有牌子上写着“历史文物、禁止入内”的标语，边上竖着一块石碑，记录着有关这座庙的信息，叶少阳一干人看完之后，眼睛都亮起来。
<br/>　　这居然是东岳帝君庙！
<br/>　　据碑文上所写，这是宋代建的庙，明代修缮和扩充，是过去帝王家的官庙，用来祭祀的。一行人披荆斩棘，绕着庙宇走一圈，发现这座庙占地还挺大的，差不多有一个标准足球场的大小，只是年久失修，外墙成片地都倒塌了，中间有一个很大的道场，中间有个九层高台，也塌得不成样子，到处长满了荆棘乱草，难以下足。
<br/>　　叶少阳就让包子和瓜瓜进去看一眼，两人进去之后，很快就出来了，表示里面也乱得不行，只有正殿还稍微成形，里面有一尊神像，看上去像个皇帝，虽然庙宇不行了，但这神像盛气凌人，他们两个本是邪物，几乎不能靠近。
<br/>　　“这八成是东岳大帝的像！”老郭看着叶少阳说道。
<br/>　　叶少阳明白他的意思，命瓜瓜和包子一人一个方向，搜查整个泰山，看还有没有庙宇祠堂，自己跟老郭和碧清就在附近溜达，叶少阳一边走，一边暗自观察周围的地貌特征，记在心里。
<br/>　　后来瓜瓜和包子相继回来，表示庙宇之类的还有不少，有些还有道士或和尚在里头住，不过都没什么灵气，至少没有东岳帝君庙这样灵气充盈。
<br/>　　“那些庙，都不是修行门派，估计很多还都是官方弄的，忽悠游客的。”老郭下了判断，指着帝君庙，压低声音的道：“祭祀仪式，怕是就在这了。”
<br/>　　叶少阳也没说什么，招呼大家一起下山，虽然是淡季，游客还是有的，一路上也看到不少人，叶少阳一开始有点不解，法术界的行动，一向都是背着人的，更不要说是黎山老母这样的阐教金仙，更不能被人看到，那么祭祀活动怎么进行呢？造个鬼打墙，把人都拦在外面？
<br/>　　跟老郭一说，老郭表示这是想多了，泰山上白天人多，到了晚上就没人了，更不要说现在是冬天，就算是起早看日出的，最早也得五点多才起床。
<br/>　　由此推算，黎山老母他们应该会在半夜展开活动。
<br/>　　下山回到客栈，老郭去见朋友了，叶少阳洗完澡坐在阳台的躺椅上。
