陆寒霆去了书房，他伸手将办公桌上的文件全部拂落在了地上。
<br/>
<br/>        将脖间的领带扯下来，用力的甩在地毯上，他坐进了沙发里，挺括的后背慵懒的抵靠进去，整个人弹出了一股巨大的阴郁戾气。
<br/>
<br/>        闭了一下眼再睁开，眼眶里全是红血丝。
<br/>
<br/>        她现在在干什么？
<br/>
<br/>        会不会哭？
<br/>
<br/>        他真的很混蛋，混蛋透顶了！
<br/>
<br/>        忘了他吧！
<br/>
<br/>        陆寒霆觉得自己现在的精神状态很不好，他又有发病的迹象了，精硕的胸膛上下起伏，可怕的像是野兽的喘动。
<br/>
<br/>        他现在想冲回卧室里，回到她的身边，他身体有点难受，脑袋里全是她穿着香槟色吊带裙的样子，他想回去将她狠狠的压住，将她给揉碎了，这样他就不会那么难受了。
<br/>
<br/>        陆寒霆出了一身的冷汗，为自己这个可怕的念头，现在好像每次发病的时候他都想回去找她，然后伤害她。
<br/>
<br/>        陆寒霆觉得快控制不住自己了，他起身打开了抽屉，拿出了一瓶安眠药。
<br/>
<br/>        打开瓶盖，他倒了三颗安眠药放在嘴里，生咽了下去。
<br/>
<br/>        吃下安眠药他就不会去找她了，不会打扰她了，就这样吧，让她走，让她恨他吧。
<br/>
<br/>        ……
<br/>
<br/>        翌日清晨。
<br/>
<br/>        夏夕绾下楼的时候就看到了陆老夫人，老夫人慈爱的笑道，“绾绾，你醒啦…哎哟，你这个眼睛怎么了，怎么肿肿的？”
<br/>
<br/>        夏夕绾昨晚哭了很久，整个眼眶都肿起来了，虽然她提前拿热毛巾敷过了，但还是肿肿的。
<br/>
<br/>        “奶奶，没事，我昨晚写论文熬夜了，大概是没有休息好。”夏夕绾亲昵的挽住了陆老夫人的胳膊撒娇道。
<br/>
<br/>        “什么论文非得熬夜，可以留到白天写嘛，以后不许熬夜，多伤身体啊，快，吴妈，用个热鸡蛋给绾绾敷敷眼睛。”陆老夫人疼爱的埋怨道。
<br/>
<br/>        夏夕绾并不想让奶奶知道她和陆先生之间的问题，就算昨晚他没有警告她，她也不会说的。
<br/>
<br/>        喵喵~
<br/>
<br/>        这时小圆圆跑了过来。
<br/>
<br/>        夏夕绾弯腰将小圆圆抱在了怀里，撸了撸小圆圆身上的软毛，这还是陆先生去国外出差的时候送给她的礼物，没想到现在时过境迁了，他对她所有的呵护和宠爱都变成了厌烦和薄情。
<br/>
<br/>        “对了绾绾，寒霆怎么还没有起床，他很少起这么晚的？”陆老夫人问。
<br/>
<br/>        他还没有起床么？
<br/>
<br/>        撇开两个人的感情不谈，夏夕绾还是很担心他的病情的，不知道他晚上还能不能睡着觉？
<br/>
<br/>        “奶奶，我去看下陆先生。”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾抱着小圆圆上了楼，来到了书房，她抬手“叩叩”的敲了敲书房的门。
<br/>
<br/>        陆寒霆还没有醒，昨晚吃的安眠药药量有点大了，不过听到敲门声，他迅速睁开了眼，醒了。
<br/>
<br/>        昨晚在沙发上睡的，坐起身的时候他觉得头痛欲裂，不用照镜子都知道自己现在有多糟糕。
<br/>
<br/>        这时“叩叩”的敲门声又响起了，然后门被打开，夏夕绾那道纤柔的身影出现在了门边，澄亮的水眸看了过来。
