　　第466章
<br/>　　徐神机本来抱着必死的心情来，但走入大帐后，明显感觉情形有些不对。他心中一动，忽然想到一种更糟糕的情况。
<br/>　　“找到程大雷了？”
<br/>　　陆承和崔相同时点点头，尤其是崔相，一脸欣慰的看着徐神机，指着徐神机的鼻子道：“从我第一眼看见你，就知道你不简单！”
<br/>　　“抓到了？”徐神机问出一个更关心的问题。
<br/>　　陆承一脸惭愧：“是我们疏忽了，没有做好准备，两军冲杀，我军损兵折将，大败而归。”
<br/>　　徐神机心中松了口气，突然装出勃然大怒的表情：“我说什么来着，你怎么就不听我的，现在怎样，让程大雷逃了吧。”
<br/>　　陆承羞愧得无地自容，经过这件事，他再不敢质疑徐神机的智谋。瞧瞧，什么叫真人不露相，什么叫隐在云雾中，偶尔露峥嵘，为什么其他人满天下找不到程大雷，人家随手一指就能指出来。这是运气么，不，这是本事。
<br/>　　崔相呵呵笑了一声，道：“徐军师莫要生气，我已狠狠责怪了他，这次请徐军师归来，就是下一步的计划。”
<br/>　　徐神机叹口气：“崔相勿怪，只是杀不了程大雷，我实在不放心呐。”
<br/>　　崔相越看徐神机越高兴，这根本就是天上掉下来的宝贝。得到这样的人才，杀不杀程大雷都显得不重要了。
<br/>　　“放心，一定有机会的。”崔相摆摆手：“来人请薛将军进来。”
<br/>　　不多时，薛丁山来到帐内，崔相道：“这次就由薛将军带队，徐军师辅佐，程大雷现在一定还未跑远，找到他，杀掉他。”
<br/>　　“末将接令。”薛丁山双手抱拳。
<br/>　　陆承忽然想起一事，道：“今日从贼兵口中得知，程大雷似乎正在寻薛将军。”
<br/>　　“他听说过我？”薛丁山眼睛一亮，冷漠的脸上浮现一抹喜色。
<br/>　　人的命树的影，不管程大雷是恶名还是威名，但他在帝国军方还是有些名声的，尤其是在年军人中间。既然程大雷知道自己，那岂不是说明自己也有些名声。
<br/>　　崔相摆摆手：“好啦，既然人家那么看重你，你也莫要让他失望。现在就点起三千人马，由徐军师辅佐，这次一定不能让他逃了。”
<br/>　　徐神机也想随着出战，碰到程大雷自己直接反水。虽然这几日他在军中待遇不错，但也不是能随便乱走，最起码走出这军营还是万万不能的，毕竟自己是被抓来的俘虏。
<br/>　　薛丁山和徐神机带队出发，由陆承引路，一路摸着程大雷的尾巴追去。在月上中天时，咬到了程大雷的尾巴。
<br/>　　“薛将军，人就在前面。”陆承。
<br/>　　“追！”初生牛犊不怕虎，薛丁山还想杀了程大雷扬名立万。
<br/>　　全速行军，想要咬住程大雷，只不过跟不上程大雷的速度而已。
<br/>　　“大当家，回头杀吧？”单雄信。
<br/>　　“杀什么，人家三千，我们三百，怎么杀。”
<br/>　　程大雷道了一声，忽然勒马停住，想了想道：“也未必不可杀。”
<br/>　　对方人数虽多，但大多是步兵，仗着骑兵的优势，给他们个下马威也不是不可。
<br/>　　“听我号令！”程大雷举起斧头：“随我冲杀过去，不可恋战。”
<br/>　　身后是银眸扛旗，程大雷带着三百义字军冲杀过去，这次没有直扑对方主阵，攻击的是敌人侧翼。
<br/>　　两军擦肩而过，程大雷多造伤亡，大斧又不知砍下了多少人头。
<br/>　　在擦肩而过的过程中，程大雷瞥见了徐神机，对方冲他一直点头。令程大雷诧异的是，徐神机穿着儒生长衫，似乎在帝国军中还很受尊重。
<br/>　　难道他叛变了。
<br/>　　如果真是这样，那就是天大的好消息。不管徐神机加入谁，肯定没多少时间就把人家搅合黄了。
<br/>　　也就是程大雷这一愣神的光景，一杆银戟向程大雷袭来，对方招式很凌厉，程大雷不得不认真对待。
<br/>　　“程大雷，纳命来！”银戟小将吼道。
<br/>　　程大雷没有和对方缠斗，卖了一招后就带着弟兄冲了过去。不过这小将还是给他留下深刻印象，他回头望了一眼，查看这员将的信息。
<br/>　　不看则以，一看大惊失色，对方的名字是——薛丁山。
<br/>　　程大雷冲杀过去，停下黑牛，本来程大雷想要撤退的，但现在不想退了。一来，是要把徐神机捞出来，二来，是因为程大雷非杀薛丁山不可。
<br/>　　两军对阵，隔着五百步对峙。
<br/>　　“大当家，我们不跑了么，对方的兵力可是远强过我们？”单雄信。
<br/>　　“怕什么！”程大雷冷哼一声：“本大家神通盖世，吹口气就把他们卷到天上了。”
<br/>　　“那大当家你为何还不吹？”
<br/>　　“你管我。”程大雷白他一眼，还是没想到如何破敌。
<br/>　　帝国军中，薛丁山也不知道该如何进攻，刚才与程大雷短暂交手，就知道程大雷名不虚传，本事并不弱于自己。当然，年轻人么，不踩着成名人物的脑袋怎样往上爬。如今天时地利人和，不正是杀他的好机会。
<br/>　　徐神机拍马过来，道：“薛将，不若让我上前，劝他归降。”
<br/>　　“他有可能归降么？”薛丁山问。
<br/>　　“不怕，我有三寸不烂之舌，对其晓明厉害，不怕他不降。”
<br/>　　“好，就请徐军师上前试试。”
<br/>　　徐神机心中松口气，立刻拍马出阵。
<br/>　　陆承担忧道：“他不会跑回去吧？”
<br/>　　“不怕。”薛丁山摇摇头，卸下背上铁弓，张弓搭箭瞄准徐神机的背心：“他敢跑，我就敢杀。”
<br/>　　陆承吓了一跳：“薛将不可莽撞，崔相十分看重此人，不能如此做。”
<br/>　　薛丁山目不转睛，紧紧盯着徐神机的背影，口中道：“将在外，军令有所不受。”
<br/>　　陆承心中一激灵，感觉到薛丁山身上流漏出的杀意，虽然还是一头幼虎，但已有咆哮山林之威。
<br/>　　徐神机一开始骑马行得很慢，在快要接近程大雷阵前时，突然拍马冲来，口中大吼：“大当家，我回来啦，大当家……”
<br/>　　徐神机这几天当真是被憋坏了，在军营中每天装神弄鬼扮高人。今日终于可以重获自由，他难免也激动起来。
<br/>　　薛丁山嘴角抿起，口中吐出两个字：“果然！”
<br/>(本章完)
