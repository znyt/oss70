整个幽兰苑的人都在收拾东西，准备出发去帝都城，其实也没有什么要带的，帝都遍地奢华，什么都有，但是毕竟在这里生活了七年，对很多东西有了感情，要收拾了带走。
<br/>
<br/>        陆寒霆处理了一下紧急文件，然后回到了自己的卧室里。
<br/>
<br/>        不知为什么，他觉得心里空空的，像是丢失了一样很重要的东西。
<br/>
<br/>        要离开的时候，他竟然生出了很多不舍，不舍这座海城，不舍这个幽兰苑，更加不舍这个卧室。
<br/>
<br/>        他不是一个会对一个地方产生感情的人，但是这里好像冥冥之中有着什么东西在吸引着他，让他深深的不舍和眷恋。
<br/>
<br/>        但是，他又不知道这个东西是什么。
<br/>
<br/>        喵~
<br/>
<br/>        喵喵~
<br/>
<br/>        这时耳畔传来了两声小猫叫。
<br/>
<br/>        陆寒霆低眸，浑身雪白的小圆圆已经来到了他的脚边，抬着一双乌溜溜的大眼睛望着他。
<br/>
<br/>        陆寒霆的心头突然一软，像是一片羽毛轻轻的划过了他的心湖，让他平静的内心荡起了一圈圈的涟漪，他低腰，将小圆圆抱在了自己的手臂里。
<br/>
<br/>        他记得这只小猫咪是他出差的时候买回来的，他觉得挺奇怪的，他应该不会买这么一个柔软的小东西。
<br/>
<br/>        喵~
<br/>
<br/>        喵喵~
<br/>
<br/>        小圆圆将脑袋埋在他的臂弯里，安静的蜷成一团，很不开心的样子。
<br/>
<br/>        陆寒霆摸了摸小圆圆的脑袋，低声道，“你怎么了，怎么好像很伤心，我感觉奶奶，福伯，吴妈…你们所有人都很伤心的样子…”
<br/>
<br/>        小圆圆注定不能回答他了，他好像是自言自语。
<br/>
<br/>        ……
<br/>
<br/>        三个月后。
<br/>
<br/>        帝都城。
<br/>
<br/>        今天的帝都城非常热闹，因为今天是厉家掌上明珠厉嫣然的生日。
<br/>
<br/>        厉家富可敌国，厉家现在的家主厉君墨是全球第一首富，厉嫣然是厉家的独女，首富千金，是真正的天之贵女，从出生就在鲜花掌声和所有人的宠爱里长大。
<br/>
<br/>        今天她的生日并没有大力操办，而是给厉嫣然做主办了一个小型生日派对，邀请来参加的都是她的朋友们，也就是帝都的名媛千金和豪门太子爷们，年轻人一起潇洒狂欢，给厉嫣然这个真正的小公主庆生。
<br/>
<br/>        豪华party上，孙进将生日礼物送给了被众人围绕的厉嫣然，“嫣然表妹，生日快乐。”
<br/>
<br/>        今天厉嫣然穿了一件轻青色的薄纱长裙，肌肤赛雪，她生的很美，小小的鹅蛋脸漂亮夺目，清纯的乌发散落肩头，很有几分动人的灵气。
<br/>
<br/>        三十年前帝都第一美人是柳招娣，其实也就是柳璎珞，现在帝都第一美人就是厉嫣然。
<br/>
<br/>        厉嫣然开心的收下了礼物，“谢谢表哥。”
<br/>
<br/>        厉家和孙家是有亲戚关系的，孙进是孙家的公子哥，换女人像换衣服一样，现在他怀里搂着娱乐圈新晋的嫩模姗姗。
<br/>
<br/>        厉嫣然看了一眼姗姗，“表哥，你怎么现在还换女朋友啊，你就不怕你的未婚夫吃醋？”
<br/>
<br/>        大家迅速起哄道，
<br/>
<br/>        “是啊孙少，这孙夏两家可是有婚约的，夏家以前有个小女儿，小时候走丢了，让你快活了这些年，可是最近夏家把那个走丢的小女儿给找回来了，也就是你的未婚妻，你可是名草有主的人，以后不可以乱来了。”
