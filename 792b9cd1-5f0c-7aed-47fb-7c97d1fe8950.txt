　　第159章
<br/>　　这时，杨龙停的大军已杀到黑石城，在离城十里处扎营。
<br/>　　兵分四路。
<br/>　　一路兵由李善言率领，统兵一万，围东门。
<br/>　　二路兵由齐德龙率领，统兵一万，围西门。
<br/>　　三路兵由齐德强率领，统兵一万，围北门。
<br/>　　中军两万由杨龙停率领，攻南门。
<br/>　　所谓围城必缺，杨龙停是征战沙场的老油条，这个道理不会不懂。明知如此，他依旧摆出这样的阵势，很明显连逃跑的机会都不给薛半川。
<br/>　　五万人每天的人吃马嚼就是一个天文数字，所谓三军未动，粮草先行，每天都有源源不断的物资运达。
<br/>　　杨龙停他心里很清楚，战争比的不仅仅是军事，还包括经济，政治，人心等等。后勤若能做的好，未开战先赢了一半。
<br/>　　于是他格外关注粮草的问题。
<br/>　　今日的运粮队来得晚了些，而对方并没有给杨龙停带来所需的粮草，带来了罗铁的尸体。
<br/>　　看着大帐内罗铁的尸体，杨龙停沉默良久，周围的空气似乎都冷下来许多。
<br/>　　运粮失利的小队，跪在帐外良久，身体瑟瑟发抖。
<br/>　　“杀！”
<br/>　　这是杨龙停见到罗铁尸体后说得第一个字，一挥手便是成片的人头落地。
<br/>　　然后，杨龙停给程大雷写了一封信，命人送到蛤蟆寨。
<br/>　　信中他只写了五个字：“汝不畏死乎！”
<br/>　　很快，他便收到程大雷的回信，回信是六个字。
<br/>　　“汝要战，便来战。”
<br/>　　看到信后，杨龙停勃然大怒，立时就要点兵，先破蛤蟆寨，再杀薛半川报仇。
<br/>　　然后，他很快便冷静下来。
<br/>　　程大雷是肯定要杀的，无论他杀不杀罗铁，这件事都不会改变。而现在杀，肯定不行。
<br/>　　大军刚刚扎营，再返回来攻打程大雷，总显得有些尴尬。而且，若薛半川趁机从背后偷袭，杨龙停就会陷入腹背受敌的地步，这乃是兵家大忌。
<br/>　　况且，程大雷手中还握着自己的粮道，若他切断自己的粮道，五万大军就会陷入绝境。
<br/>　　这样一说，这程大雷似乎还不太能得罪。
<br/>　　杨龙停既狠得下心肠，自然也弯得下腰，没有这种程度的厚黑学造诣，他成不了今天的幽州王。
<br/>　　于是，杨龙停又给程大雷写了一封信，信中内容简略来说：我知道你把罗铁杀了，杀得好，杀得对，杀得呱呱叫，以下犯上，按律当斩，没什么道理好讲。你不杀，我也正要杀他，我想着赏你一些什么，那些粮食就对你的赏赐吧。当然你能来我身边做事就更好了。
<br/>　　信送出去后，很快杨龙停便收到程大雷的回信，信中程大雷又恢复之前的态度。
<br/>　　『造谣北方的光明·帝国的脊梁·我永远崇拜的幽州王殿下：
<br/>　　收到您的信之后，我让山寨所有人一起看，无不为您的英明而折服，多少人以泪洗面，以阅读到这封信为荣耀』
<br/>　　『……当然，我很是想去王上身边效力的，奈何重疾缠身，连床都下不了，只能在心里为您祈福，祝您旗开得胜』
<br/>　　落款是：您忠实的小弟·幽州王永远的崇拜者·幽州青牛山陷阵营游骑将军程大雷。
<br/>　　看这封信时，杨龙停没有李善言那种恶心的感觉，反而隔着信纸感受到淡淡的寒意。
<br/>　　他闭上双目，回忆起第一次也是唯一一次和程大雷见面的情景。他弯着腰裹着破袄，言谈举止里透着谄媚。还包括递到幽州王城胡言乱语般的书信。像极了城里种种善于拍马屁，钻营投机的小人。
<br/>　　毫无疑问，自己被他骗了。
<br/>　　那些递到幽州城的信，人人视作笑话，包括杨龙停也是如此。但在视作笑话的同时，潜意识中就轻视了程大雷。
<br/>　　活到今天，杨龙停比任何人都明白，这世上不缺少敢硬碰硬的猛人。同时，乱葬岗也从不缺少无名尸体。
<br/>　　这世道，硬汉子是活不长的，可一味低头，也没人会瞧得起。
<br/>　　怕的就是该强时强，该低头时低头。
<br/>　　杨龙停就是这种人。
<br/>　　难道程大雷也是这种人……或许，他是一个比薛半川更值得杀的人。
<br/>　　不管那么多，先安抚好这狗贼，等破了黑石城，回去时顺道再除掉此贼。
<br/>　　现在杨龙停要做的事情是攻城。
<br/>　　……
<br/>　　城内的薛半川，并不是没有防备。照道理来说，杨龙停大军压境，早该有城内的大户，打开城门迎接杨龙停进城，他们可不想和薛半川共存亡。
<br/>　　但这件事直到现在都没有发生，原因是，在这之前，薛半川就把城里的大户请到城主府，举办了一场晚宴。
<br/>　　宴席上，薛半川举杯未饮先泪，说：杨龙停不是个东西啊，要把咱们都杀死啊，不过你们也别担心，我会保护你们的，以后大家就住在城主府，我和你们共存亡。
<br/>　　美其名曰保护，其实就是羁押。把城里大户的妻儿老小都囚禁起来，薛半川狠下心来，就一个意思：要活大家一起活，要死大家一起死。
<br/>　　他们子弟有在军中任职的，此时也没办法反叛，毕竟父母妻儿都在薛半川手中的。
<br/>　　真投靠杨龙停，毫无疑问薛半川会将自己的妻儿杀尽。
<br/>　　狠，是这个时代大人物谁也不缺乏的一种素质。
<br/>　　面对杨龙停的大军压境，薛半川只有一个态度：不战。
<br/>　　反正我这里老婆孩子热炕头，要嘛有嘛，而你则是远道而来，一天没粮你就抓瞎。大家耗吧，看谁耗得过谁。
<br/>　　除了不战之外，薛半川还有一条策略：造谣。
<br/>　　在城内大张旗鼓宣扬：杨龙停不是个东西啊，从小就好色如命，他的姐姐妹妹，姑姑婶婶……凡是家里的女人，他一个都没放过。据说他因为从小贪色过度，长大就不行了，老婆都是给别人睡觉，他在一旁看，满足某种刺激感。不然，他怎么会到现在都没儿子呢……
<br/>　　这种话通过各种方式传到杨龙停都大营中，谣言这种东西，真传得没边了，也就没人信了，可总有一两个没脑子的，听到后说：喔，怪不得怪不得。
<br/>　　谣言不伤人，但他恶心你。
<br/>　　毫无疑问，这条计策是程大雷出的。
<br/>
