　　“得到了此物，不知道能不能证明我来到了第八层？”
<br/>　　苏莫微微的叹了口气，找了这么久都找不到火元精，他只能把此物收取了。
<br/>　　随即，苏莫不再多留，立刻向上方返回。
<br/>　　他的混沌之力，已经支撑不了多久了，必须立刻离开了。
<br/>　　很快，苏莫到达了第十七层，他又停了下来。
<br/>　　他准备在第十七层抓取一只火元精，这样更保险一些。
<br/>　　毕竟，他若是没有抓取火元精，万一得到的那金色光团不能证明他到达了第十八层，那他岂不是白跑一趟了！
<br/>　　所以，他必须再抓取一只火元精，有备无患。
<br/>　　第十七层空间不大，很快苏莫就在岩浆之，找到了一只火元精。
<br/>　　此火元精，幻化成了一条银色的小龙，长牙舞爪，甚是凶恶。
<br/>　　嗖！
<br/>　　发现了苏莫的存在，银色小龙如同一支银色利箭，速度奇快，向苏莫扑了过来。
<br/>　　“过来吧！”
<br/>　　苏莫大手一探，掌中混沌之力喷涌，立刻形成了一只混沌大手，向银色小龙狠狠的抓了过去。
<br/>　　毫不意外，混沌大手，一把便攥住了小龙。
<br/>　　轰隆隆！！
<br/>　　银色小龙实力极强，即便被混沌大手抓住，依旧疯狂的挣扎，响起无尽岩浆涌动。
<br/>　　“给我死！”苏莫大手狠狠的用力，强大的力量猛烈的挤压银色小龙。
<br/>　　昂！
<br/>　　银色小龙哀嚎一声，便不再挣扎，化为了一团银色的火焰。
<br/>　　“不错！”苏莫暗暗点头，随即，他便将银色火焰收了起来。
<br/>　　这一次，应该万无一失了，就算得不到十八层的奖励，十七层的奖励还是稳了。
<br/>　　随即，苏莫不再停留，身形冲天而起，向上方山洞之中飞去。
<br/>　　……
<br/>　　熔岩窟，洞穴之中。
<br/>　　褚彦面色阴沉的望着众人，沉声问道：“那段惊天真的没有出来？”
<br/>　　“褚彦大哥，段惊天真的没有出来，我们没有骗你！”
<br/>　　“他的确没有出来，应该还在熔岩池内！”鲁东道。
<br/>　　“估计他知晓你要亲自下去，隐藏了起来！”魏仑说道。
<br/>　　众人纷纷开口，对于褚彦没有找到段惊天，他们也很是无语。
<br/>　　褚彦闻听众人之言，面色难看，沉默不语。
<br/>　　段惊天居然没有出来，依旧在熔岩池之下，这让他郁闷不已，自己白跑了一趟。
<br/>　　或许，真的是段惊天猜到自己会下去，特意隐藏了起来。
<br/>　　褚彦心中思忖，自己要不要再下去，毕竟，在熔岩池之下，他能随意教训段惊天，没有人来插手。
<br/>　　但是，若是等到对方出来，此地有执事镇守，宫内有执法长老，他并不好随意出手。
<br/>　　“那个小子，倒是躲过了一劫！”不远处的中年执事，心中暗叹一声。
<br/>　　本以为褚彦下去，那个段惊天不死也差不多了，但是，他没想到，段惊天居然如此好运。
<br/>　　哗啦！
<br/>　　中年执事暗叹之时，就在褚彦思索要不要再下去之时，熔岩池之中冲出了一道身影。
<br/>　　这道身影身穿青衣，普普通通，正是苏莫。
<br/>　　“段惊天？”
<br/>　　“他出来了！”
<br/>　　“这家伙出来了！”
<br/>　　众人一见苏莫，不少人纷纷惊呼出声。
<br/>　　褚彦的目光，瞬间便锁定在了苏莫的身上，眸中厉芒闪动。
<br/>　　不过，他并未出手，毕竟，此时此刻，一见不适合出手了。
<br/>　　当着执事的面，攻击其他弟子，这不仅是违反宫规，还是藐视执事。
<br/>　　唰！
<br/>　　飞出熔岩池之后，苏莫径直飞到了中年执事坐在的石台前，落在了地上。
<br/>　　“执事大人，我来领取奖励！”
<br/>　　苏莫对于褚彦等人，连看都未看一眼，目光凝视着眼前的执事，抱拳行礼。
<br/>　　“嗯，你到达了第几层，是否抓取了火元精？”执事打量了苏莫一眼，微微点头问道。
<br/>　　这位中年执事，身穿褐色长身，留着一缕山羊胡，面容和煦，并不是引领苏莫入门的那位执事。
<br/>　　“禀执事……！”苏莫正欲开口，褚彦带着一帮人走了过来，立刻打断了他的话。
<br/>　　“段惊天，你目无法纪，连伤两名弟子，你可知罪？”
<br/>　　褚彦来到了苏莫的身边，面色阴沉的说道。
<br/>　　而鲁东和魏仑，均是跟着褚彦的身边，脸色俱是显得有些苍白。
<br/>　　“褚彦，明日不说暗话，我为何伤他们，你心中有说，此事就算捅到执法堂，我也不惧！”苏莫面色冷漠的说道。
<br/>　　“你……！”
<br/>　　褚彦顿时语塞，他本想以此胁迫苏莫低头，没想到苏莫不吃这一套。
<br/>　　“好，很好！”
<br/>　　褚彦怒急而笑，厉声道：“刚才在熔岩池之中，不知道你藏在了哪里？但是，你躲得了今日，躲不了来日！”
<br/>　　“褚彦，够了，此事到此为止了！”
<br/>　　这时，石台后的中年执事站了起来，面带威严的说道。
<br/>　　他身为人王宫的执事，褚彦当着他的面，威胁其它弟子，他若是再不出言，岂不是毫无颜面了。
<br/>　　褚彦见执事出声，面带沉默之色，他知晓自己在执事面前，确实有些过了。
<br/>　　“躲你？”
<br/>　　苏莫闻言嗤笑，沉声道：“我何需躲你？”
<br/>　　“你没躲我？”
<br/>　　褚彦闻言冷笑一声，道：“笑话，我从第十三层找到第十七层，都没有找到你，难道你不是躲藏起来了？”
<br/>　　褚彦心中不屑，他认为苏莫之前定然躲藏了起来，但现在居然还不承认。
<br/>　　“不好意思，我在第十八层！”苏莫淡淡的说道，这人简直无知，他在第十八层，对方在上面怎么可能找的到。
<br/>　　“你在第十八层？”
<br/>　　此言一出，褚彦一楞，众弟子亦是纷纷一愣。
<br/>　　就连中年执事，都不禁微微一怔。
<br/>　　“哈哈哈！！”
<br/>　　陡然，朗笑之声响起，只见身形高大的鲁东突然哈哈大笑了起来。
<br/>　　他目光望着苏莫，仿佛听到了某中笑话一般，大声道：“段惊天，你未免太会吹了吧？你居然说你在第十八层？”
<br/>　　“吹牛也要有个限度吧！”
<br/>　　“你认为我们会相信？”
<br/>　　“简直可笑！”
<br/>　　其他人纷纷回过神来，顿时一片哗然，没有人相信苏莫在十八层。
<br/>　　因为，这简直不可能！
<br/>　　熔岩窟存在了几十万年，以外宫弟子的身份，能进入十八层的，目前只是十二人。
<br/>　　这种情况，可谓是人王宫万年都难出一个这种天才。
<br/>　　所以，没有人相信苏莫在第十八层。
