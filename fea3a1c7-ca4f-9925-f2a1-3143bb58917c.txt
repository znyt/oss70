　　炎撼世离开了，楚风眠便是调转身形，手持那七彩晶石，走向这七彩霞光。
<br/>　　这七彩霞光，绝对是一尊极为可怕的强者，布置而出的，数万年的时间，都没有削弱上面多少的力量。
<br/>　　楚风眠看得出来，这七彩霞光，只怕是九劫古帝都难以逃得出，这还是经历了数万年岁月的流逝后，剩下的力量。
<br/>　　这曾经布置下七彩霞光的强者，绝对是一位天人，甚至是在仙人九阶之中都算是强者的存在。
<br/>　　在如今的时代之中，这七彩霞光化为的屏障，也几乎是所向无敌的存在，没有几个人可以打破。
<br/>　　不过楚风眠捏着七彩晶石，接近走过去的一刻，这七彩霞光，便是纷纷退避。
<br/>　　这七彩晶石就是进入这七彩霞光内的钥匙。
<br/>　　等到楚风眠踏入其中的一刻，这七彩霞光便是再度封闭起来，跟从来没有人进入过一样。
<br/>　　“这里是？”
<br/>　　楚风眠踏入七彩霞光，一眼望去，面前便是一座座的宫殿，林立在山峦之上。
<br/>　　这一座座宫殿，跟楚风眠想象之中的破损却是截然不同，每一座宫殿，都是完好无损的。
<br/>　　十万年的时间，仿佛是让这些宫殿，没有任何的变化。
<br/>　　“这怎么可能？”
<br/>　　楚风眠眼神之中闪过几分惊讶。
<br/>　　这样的古老遗迹，楚风眠在归墟天之内也探索过不少了，至少都是会有着几分破旧毁灭的迹象才对。
<br/>　　可是这些宫殿，居然是完好如初？
<br/>　　在最前方，足有一座高达百丈的巨大铜门，在这铜门上的牌匾上，写着两个巨大的“天元”二字。
<br/>　　这天元二字，威严无比，十万年时间，这天元商会虽然没落，可他的威势依然存在。
<br/>　　“这里的确是天元商会的遗迹。”
<br/>　　看到这天元二字，楚风眠便是放心了下来，看来跟他猜测的不错，这里就是天元商会曾经的遗迹。
<br/>　　楚风眠打量了一眼，便是向着遗迹内部走去。
<br/>　　除了那七彩霞光之外，楚风眠还没有发现在这遗迹之中还有着其他的危险。
<br/>　　就当楚风眠这么想的时候，一股风从这遗迹内部吹了过来。
<br/>　　在这一股风上，居然隐隐约约带有着几分紫色的迷雾。
<br/>　　“毒？”
<br/>　　楚风眠目光闪烁，他发现这紫色迷雾在接近到楚风眠的一刻，便是猛烈的向着楚风眠的身躯之中钻，试图钻入到楚风眠的身躯之中去。
<br/>　　这些紫色的迷雾，绝对是一种剧毒，从这天元商会的遗迹内飘散了出来。
<br/>　　“这种剧毒，只怕是古帝一旦沾染，都难以招架。”
<br/>　　楚风眠打量了一眼这毒物，也是目光无比冰冷，这种毒物，古帝一旦沾染，都是非死即伤。
<br/>　　不过楚风眠的肉身强横到了一种不可思议的地步，在他的身躯之中，十二万九千六百枚血肉晶体紧紧的凝聚在一起。
<br/>　　使得楚风眠的肉身，完美无瑕。
<br/>　　任由这毒物在厉害，也无法侵入到楚风眠的身躯之中。
<br/>　　现在的楚风眠，可谓是百毒不侵。
<br/>　　“看来这天元商会的毁灭，跟这毒雾脱不开关系！进入看看！”
<br/>　　楚风眠身形闪烁，踏入大门之内，进入到了遗迹之中。
<br/>　　踏入大门，这毒雾的威力，更是猛烈了不少，这种毒雾的威力，比起许多天下间的剧毒还要厉害不少。
<br/>　　幸好是楚风眠练就了半步不朽真身，不然的话，只能够是以灵力抵挡的话，楚风眠最多是在这其中待上几分钟的时间，就必须要退出去。
<br/>　　这里面的毒雾，只怕是半步天人境级别的强者，都未必敢硬抗。
<br/>　　“好强的毒，这天元商会里到底是有什么东西，散发出如此之毒偶的毒雾？”
<br/>　　这毒雾，几乎是弥漫在了整个天元商会之中，每一座宫殿内，都是一阵紫色雾气笼罩，烟雾缭绕，处处是毒。
<br/>　　这里说是一处遗迹，但是简直就是一座毒城，只怕是被其他人发现，都不敢涉足其中。
<br/>　　楚风眠一路是来到了最近的一座宫殿之中。
<br/>　　刚刚踏入其中，楚风眠就看见了在这宫殿之内，躺着的几座骸骨。
<br/>　　这些骸骨，呈现出了一种黑色，都是位于一座立柱之下，半躺着的姿势。
<br/>　　“毒气入骨，这些骸骨的主人生前，应该都是被这毒气所毒死的！”
<br/>　　楚风眠观察了一眼，就看了出来，这些骸骨主人的死，也跟这毒气脱不开关系。
<br/>　　楚风眠走过去，轻轻一触碰，这骸骨便是轰然破碎，化为了一谈烟尘，飘散再天地之中。
<br/>　　“这些骸骨的主人，生前至少可都是古帝级别的人物，现在居然一碰就碎？”
<br/>　　古帝纵然是陨落他，他留下的骸骨之中也会蕴含无比强大的力量，一些强者的骸骨，比起许多诸天奇物都要强大的多。
<br/>　　就算是曾经楚风眠得到的十方天魔，青铜天魔的骸骨，十方天魔可是传说之中的人物了，距离如今的时代数十万年，甚至是数百万年。
<br/>　　但是他的骸骨，被楚风眠得到吞噬，都足以助楚风眠一步登天，肉身达到了半步不朽真身的地步。
<br/>　　古帝骸骨纵然没有这么珍贵，但是坚韧程度也不亚于寻常的天级灵器，可眼前这骸骨，楚风眠只是轻轻触碰，就破碎了。
<br/>　　“这些骸骨，至少经历了数万年，甚至是十万年的风化，才会如此的脆弱！”
<br/>　　楚风眠又碰了一下旁边的一尊骸骨，这骸骨也是一样，至少是碰到一点点的外力，就会破碎成灰。
<br/>　　“看来这些人，不像是外来的武者，多半都是天元商会的弟子，只是后来都被毒死在了其中，没有任何一人可以逃出生天。”
<br/>　　楚风眠看了一眼，便是明白了其中的玄妙。
<br/>　　只是让楚风眠有些不能理解的是，在这天元商会内，到底是如何出现如此厉害的毒气。
<br/>　　在这宫殿之内，甚至是都没有打斗的痕迹，这天天元商，显然不像是外部强者入侵导致的衰败。
<br/>　　这毒气的根源，是来自于天元商会遗迹之中最中心的主殿之中。
<br/>　　“先看看这宫殿之中，有没有什么天元商会留下来的丹药！”
