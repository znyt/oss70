<br/>　　周围不少人都愕然的看着林峰，这可太疯狂了。

<br/>　　林峰第一个任务就是去一个明月小区拿结婚纪念照。
<br/>　　林峰马不停蹄就直接奔到了明月小区。
<br/>　　吼吼吼
<br/>　　顿时七八个丧尸全部冲了过来，林峰一刀直接给他们全砍了。
<br/>　　冲入小区内，小区内还有十几个丧尸，看到林峰，闻到了活人的气息也都张牙舞爪的冲来。
<br/>　　青莲剑歌
<br/>　　林峰一刀下去，瞬间刀芒把他们全部绞杀，颇为的犀利。
<br/>　　林峰根据任务提供的资料直接来到了发布任务者之前住的屋子，门自然是锁着的，林峰一脚下去，直接给他踹开了。

<br/>　　林峰把这个照片取下来然后丢入物品栏内，然后又扫荡下这屋子里，基本上吃的喝的都拿走了。

<br/>　　第二个任务则是纯粹的猎杀任务。
