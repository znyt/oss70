　　西域，归墟城。
<br/>　　归墟天开启的这一个月的时间，在归墟城外许多各大势力的强者，都没有离开，都是安静的等待着归墟天中的武者离开。
<br/>　　尤其是七大宗门的强者，如今更是全部都在外面等待着，交谈着，不少强者的眉目之间，都是露出了几分急躁之色。
<br/>　　“这归墟天都快要关闭了，怎么还没有人出来？”
<br/>　　一尊身上蕴含凌冽剑意，天剑宗的强者，望着归墟天，眉目之间都是露出了几分紧张之色。
<br/>　　这归墟天的裂痕，已经是越来越小，再过一两个时辰的时间，便是很有可能就会关闭了。
<br/>　　可现在，他们天剑宗的弟子，却是没有任何一人，从这归墟天之中出来。
<br/>　　进入归墟天中的武者，有些会陨落在里面不错，可是这一个人都没有出现，绝对是足以让他们感到极为奇怪的。
<br/>　　从归墟天开启之后，还从来没有发生过这样的事情。
<br/>　　“不止是你们天剑宗的人，我们荒古门的人，也没有出来。”
<br/>　　在那天剑宗强者的身边，又是一尊武者开口道。
<br/>　　“不知道这一届归墟天到底是发生了什么事情？莫非是开启了什么远古传承宝藏？才让他们都没有出来？”
<br/>　　“可是，这就连一个人都没有出现。”
<br/>　　哪些七大宗门的武者，都在交谈这。
<br/>　　可是除了七大宗门之外，其他势力的弟子，不少人都是出来了。
<br/>　　单单是他们七大宗门的弟子，没有任何一人出来了。
<br/>　　“嗖！”
<br/>　　就在众人交谈之际，一道遁光，突然是从中飞了出来。
<br/>　　“好快的遁光！”
<br/>　　“此人是谁？”
<br/>　　一些视线，纷纷看了过去，这遁光的速度，远比寻常生死境武者要会的多。
<br/>　　“是楚风眠？北域百绝榜的第一人！”
<br/>　　一尊强者，突然认出了楚风眠的身份，大声的喊道。
<br/>　　“北域百绝榜第一？一个新人？居然会有着如此遁光？”
<br/>　　“此人，肯定是在归墟天中，得到了不小的机缘！”
<br/>　　一个这一届才进入归墟天的新人，居然就有着如此遁光，肯定便是代表着，楚风眠的这归墟天中得到了不少的机缘。
<br/>　　不过楚风眠的遁光，一闪而过，并没有停留，很快的便是离开了归墟城。
<br/>　　楚风眠已经是感觉到，在归墟天周围还隐藏有着不少的强者，这些隐藏的强者，一道道目光看向楚风眠，已经是蕴含了几分杀意。
<br/>　　楚风眠这样从归墟天中离开，实力大增的，便是直接代表着，楚风眠肯定是从归墟天中得到了不小的好处的。
<br/>　　匹夫无罪怀璧其罪，这些武者不少人都打着要杀了楚风眠的注意，楚风眠虽然出身自北邙学院，但是这些强者不少人背后的势力，可都不亚于北邙学院的。
<br/>　　他们想杀楚风眠，自然也不会忌惮楚风眠背后的势力。
<br/>　　这些人的实力虽然还不是楚风眠的对手，但是要是跟这些人起了冲突的话，必然会暴露出他的真正实力来，免得麻烦，倒不如现在赶快离开。
<br/>　　“魔将大人，我们不杀了此人？”
<br/>　　在归墟城外，几尊身影目光闪烁，看着楚风眠的离开，其中一人不由的开口道。
<br/>　　“此人杀了我们地魔门无数天才，现在他又在归墟天中得到了不小的奇遇，不杀了他，终究会成为一个大患。”
<br/>　　“这里毕竟是西域，不好动手，等到了北域，我们有的是办法对付他。”
<br/>　　那尊地魔门的魔将，冷哼一声开口道。
<br/>　　“此子现在是我们地魔门的心腹大患，门主已经是制定好了计划，等此人回到北域，便是他的死期！”
<br/>　　一边交谈着，一边这一群地魔门武者，便是纷纷离开。
<br/>　　而另一边。
<br/>　　楚风眠一路飞行，已经是向着北域的一座城池，东升城飞去。
<br/>　　这东升城距离归墟城不远，其中有着可以通往其他城池的传送法阵，西域之大，以现在楚风眠的实力，想要御空横穿，也是极为勉强的一件事情。
<br/>　　所以只能是先前往一座城池，在利用传送法阵，才可以回到北域之中。
<br/>　　东升城，并不算大，也就相当于是曾经武胜国度那边的偏远之地，有着几尊圣者，便就算是其中的最强者。
<br/>　　“站住，进城者，需要缴纳一枚下品灵石。”
<br/>　　楚风眠落入到东升城门前，几尊武者便是围了上来，其中一名为首的武者开口道。
<br/>　　“给。”
<br/>　　楚风眠心神一动，空戒之中便是拿出了一枚下品灵石，直接便是给那一尊武者扔了过去，他便是直接步入到了这东升城之中去。
<br/>　　以他现在的实力，若是展现出自己的气势，这些小小的护卫，根本不可能敢阻拦楚风眠。
<br/>　　就算是这东升城主，也要对楚风眠毕恭毕敬的。
<br/>　　不过仅仅一枚下品灵石罢了，对楚风眠财富来说，九牛一毛都算不上，楚风眠也懒得麻烦，直接缴纳灵石，进入其中。
<br/>　　进入到这东升城中，一阵阵的叫卖声，便是从耳边响起。
<br/>　　其中街道两边有着不少的店铺，都在买卖灵药，灵器等，旁边也有着一些摊位，这些摊位的主人，大多都是一些实力不强的武者。
<br/>　　偶然得到一些东西卖掉，来换取一些修炼所用的丹药，灵石罢了。
<br/>　　这些买卖的东西，楚风眠扫视了一眼，便是都没有任何的兴趣了，他现在空戒中的任何一件诸天奇物，都要比起这些人买卖的东西要好上千倍，万倍。
<br/>　　楚风眠也就是闲来无事，才随意看看。
<br/>　　“恩？这倒是一件好东西。”
<br/>　　楚风眠扫视过去了一眼，突然在一座摊位上，看见了三棵金色的竹子。
<br/>　　这三棵金色竹子，竹叶上，都是蕴含着点点金斑。
<br/>　　“粹金竹，好东西。”
<br/>　　楚风眠脚下一动，便是向着这摊位走了过去，这摊位的主人是一名十七八岁的少女，实力不强，仅仅只有锻体境的境界。
<br/>　　就算是在这东升城中，都算不上是什么天才，只能算是资质平平。
<br/>　　唯一引人注目一点的便是，这女子的容貌，虽然穿着普通，却是有着一尊尊贵气质，似乎是有着极为尊贵的血脉来历。
<br/>　　“这三棵金色竹子，怎么卖？”
<br/>　　楚风眠走到摊位前，直接便是开口询问道。
