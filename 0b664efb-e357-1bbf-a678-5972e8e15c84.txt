　　到处都是尸体，从中堂到道场，从道场到前厅，地上算是碎肉和血迹，就在一个时辰前，道风来了，单枪匹马，一个人杀上了他的北斗观，偏巧他去轩辕山主峰开会去了，他的那些弟子，除了一部分精锐，被自己带走，在极北远方静修，大部分留在北斗观的弟子，差不多全被杀了。
<br/>　　然后，道风大概是凭着自己与几件法器之间的感应（他猜的，因为法器都锁在法阵加持的神龛里），找到藏着法器的神龛，然后把神龛给砸烂了，杀人无数，下山的时候，正赶上神女峰的人前来救援，也尽数屠了，连神女峰两位长老也没放过……
<br/>　　简直人神共愤！
<br/>　　清长风咬牙切齿，冲出山门，正赶上一道人影从山下掠上来，却是许久不见的星月奴。
<br/>　　“师兄，去哪里？”
<br/>　　“去杀道风那贼子！”清长风大声呼喊，“你一路上山难道没看到，我山门被他屠了！”
<br/>　　“我问过神女峰的人，真没想到，他竟如此不留余地！”星月奴也是愤恨不已，随即劝道，“事已至此，你不要冲动。”
<br/>　　“冲动？敢情杀的不是你门人，你是不冲动！对了，今日之祸，还是你惹来的，若不是为了帮你，怎么会这样，惹上这么个无法无天的煞星魔头！连影魅也死在他们手上！”
<br/>　　星月奴面色一沉，道：“你这话说得好没道理，我不也是为了轩辕山，难道是为了我自己？”
<br/>　　“对，你是为了你自己，结果我被屠了山门，影魅被杀，你自己却证道混元了！”
<br/>　　星月奴咬了咬牙，说道：“师兄你冷静些，现在不是斗气的时候，你想去找道风算账是吧，可你去哪里找他，就算找得到，你也不是他对手，想要报复，办法多的是。”
<br/>　　星月奴好说歹说，总算让清长风冷静下来。
<br/>　　“太阴山围了酆都城，你可听说了？”
<br/>　　“跟我有甚关系，难道你要去帮阴司？”
<br/>　　星月奴冷哼道：“我巴不得他们打得更热闹，我们不帮阴司，也不帮太阴山，我等了许久，就是要等这样一个机会，让他们好生打去吧，我们趁机在人间闹出些动静，直接对付道风是不行的，但我们可以跟他学。”
<br/>　　“学什么？”星月奴不解。
<br/>　　“他屠了你的山门，你也可以去他的山门……”
<br/>　　“茅山？”清长风皱眉，“屠杀茅山弟子？”
<br/>　　“屠杀自然不行，会引起人间强烈的抵触，但我们可以控制茅山弟子，逼道风和叶少阳露面。”
<br/>　　清长风一听，眉头皱的更紧了，“这好没劲，不是我做事风格！”
<br/>　　“师兄！”星月奴正色看着他，“那道风心狠手辣，叶少阳又诡计多端，正面对付他们，不是办法，我们要的是成事，不是有劲没劲，这一点，你可要记住了。”
<br/>　　清长风很不爽她居高临下的语气，想要反驳她，又想到而今她实力在自己之上，自己山门又几乎被灭，再无可用之人，想要报仇只能跟她合作，只好答应下来。
<br/>　　“好，我同你去！”清长风咬紧牙关，看着山路上堆积成山的尸首，双拳捏得咯吱响，浑身紫气萦绕。
<br/>　　“道风，此仇不报，我清长风是你孙子！”
<br/>　　一声惨叫，撕裂了空气，从黑色的浓雾中传出。浓雾是圆球形的，其中好像笼罩着什么东西，打斗声不绝，使这团浓雾一般的球形不断震颤着，看上去随时都要涨破，但却一直维持着形状。
<br/>　　打斗声更加频发，其中还夹杂着一些闷哼、怪叫，以及能量相撞的声音，在寂静的夜晚的山谷中不断响彻。
<br/>　　“啊！”
<br/>　　又是一声惨叫，接着是两人重重喘息的声音，然后噗的一声巨响，黑气炸裂，汇集成一共五道，降落在了中间一个人的身上。
<br/>　　身穿蓝衫，长发飘飘的道风，站定在草丛里，表情有点疲惫，但两只眼睛却放着光彩，低头望着地上，在草丛中间，躺着一只巨大的花蛇，头上长着两只鹿角一样的东西，浑身抽搐着，两只眼睛里却射出凶光，跟道风对视着。
<br/>　　道风收回了五朝元气，然后移动目光，望向站在巨蛇边上的芮冷玉。
<br/>　　芮冷玉脸上有几道血污，不知道是她的还是蛇的，瘫坐在地上，看上去比道风要疲惫地多，大口穿着粗气。
<br/>　　“还能行吗？”道风问。
<br/>　　芮冷玉点点头，把右边脱落下去的肩带网上提了提，缓缓站起来。
<br/>　　黑蛇雪白的肚皮已经翻上来了，一动不动地仰面躺着，道风和芮冷玉站在它身边，相比之下显得十分渺小。
<br/>　　但这个庞然大物，却几乎死在了他们手上。
<br/>　　道风走过去，低头看着大蛇，说道：“我与你无仇，我不杀你。”
<br/>　　大蛇眼中现出犹豫，突然张开嘴。
<br/>　　道风一步上前，捏住了它的嘴巴，道：“我不要你认主，我不伤你魂魄，你往生去吧。”
<br/>　　大蛇眼睛湿润了，用眼神哀求道风，道风不理。大蛇无奈之下，只有认命，一道蛇形魂魄从躯体中飞出，朝远处遁去了，身体立刻就软下去，死亡了。
<br/>　　芮冷玉目送大蛇的魂魄飞远，转头问道风：“为什么不收它做妖仆。”
<br/>　　“说了给你当补品，我要它做什么。”
<br/>　　“那就多谢了。”芮冷玉嫣然一笑，蹲在大蛇身前，手中小剑插进了蛇皮中，然后手伸进去，挖出了一个血淋淋的东西，在草地上蹭掉血污，手上的东西立刻放出绿色的光，是一枚椭圆形的珠子，大小跟个鸵鸟蛋似的。
<br/>　　芮冷玉用手颠了颠，道：“这化蛇的内丹，蕴含着他体内至少七八成的修为，是极好的东西，你拿去炼了吧。。”
<br/>　　“我已证道，要这做什么，你用吧。”
<br/>　　又抛回去，芮冷玉伸手接着，道：“我先吸血。你躲开点。”
<br/>　　道风也不知道吸血有什么好害羞的，转过头去，芮冷玉这才咬破化蛇的脖子，血还没有凝固，还是热的……
