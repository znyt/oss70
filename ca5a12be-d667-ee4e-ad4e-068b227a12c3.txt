　　　　中间和大家聊了很长一段时间，最后直播快要结束的时候，叶帝将他准备好的另外一首吉他曲南山南给直播间里的众多观众唱了出来。
<br/>　　这首歌曲一出，着实再一次震惊了所有人的眼球，无数人为之欢呼雀跃，而在当天晚上这两首歌曲便是被无数的粉丝给推到了新歌榜冠亚军的位置上。
<br/>　　一夜无话，第二天中午叶帝和林雅楠两人赶到了机场来接柳清月，昨天夜里柳清月已经给他打过电话了，告诉叶帝她订的航班信息。
<br/>　　中午11点45分的时候，在机场接地大厅内，叶帝和林雅楠便是看到一个穿着热裤、上身一件白色半截袖，带着一款黑色墨镜，头上还有一个白色鸭舌帽的靓丽女孩走了出来，两人一眼便是看出来这是柳清月。
<br/>　　而柳清月的这一副打扮，着实吸引了不少人的眼球，回头率爆高。
<br/>　　看到叶帝两人，柳清月拉着自己的行李箱便是向着这边走来，而直到这个时候众人才将注意力集中在叶帝和林雅楠两人的身上。
<br/>　　“那不是大魔王么，这个女孩和他认识？”虽然叶帝同样也带着墨镜，但是这个时候已经是有人将他给认出来了。
<br/>　　“那个女孩我好像见过，她是不是经常在火神直播来着，以前我记得大魔王去她的直播间查过房，叫什么清月的。”
<br/>　　“大魔王这美人缘真的好啊，身边都是美女，一个个都是祸国殃民的主，羡慕啊。”
<br/>　　虽然众人都在低声轻语，不过这个时候倒是少有人过来打扰叶帝，大家也都明白，在这种情况下，叶帝肯定不愿意被别人打扰的。
<br/>　　“没想到你们两个还挺准时的么！”将行李箱扔到叶帝的手里，柳清月笑着说道，看的出来能够来到这里，她非常的开心。
<br/>　　“赶紧走吧，老秦他们还等着你吃饭呢。”叶帝催促道，时间已经快要12点了，基地内那些人可都在等着呢。
<br/>　　有着柳清月的加入，基地内的气氛越发的活跃了起来，不说其他的，有着两个大美女在这里，最起码的也能养眼啊。
<br/>　　而且这里的男士知道两人都是单身后，或多或少的都有一些那种想法，林雅楠和柳清月绝对算的上是他们眼中最理想的拍档了。
<br/>　　都是从事同一个行业，而且赚的也可以，两人在直播界的地位俨然已经是一姐和二姐的地位了，至于谁是一姐，自然是林雅楠了，毕竟她从事的解说行业，曝光率很高的。
<br/>　　接下来的几天柳清月彻底的在这里安定了下来，而随着柳清月的到来，杨小艺这边终于是有了个伴，因为柳清月都是夜里9点钟开始直播，然后直播到凌晨2点。
<br/>　　加上柳清月本人非常的活泼，所以几天的时间她倒是和杨小艺处的非常的好，这倒是让叶帝等人有些诧异。
<br/>　　他们这些人和杨小艺在一起这么长时间了，居然还没有一个刚过来几天的和她聊的来，这人比人真的是没法比啊。
<br/>　　当然了，柳清月这样的节奏，倒是让蒋念几人大失所望，因为她夜里直播，那就代表着白天要睡觉了，大家接触的时间自然是十分的少了。
<br/>　　当然了这些他们都是在心里想想，谁都不好意思说出来，毕竟如果被柳清月拒绝的话，那就有点太尴尬了。
<br/>　　几天的时间一晃而过，转眼间距离夏季赛开赛仅剩下一天的时间了。
<br/>　　“明天就开始夏季赛了，大家都准备的怎么样了？”坐在客厅吃晚餐的时候，叶帝笑着问道。
<br/>　　“我们蓝队这实力又提高了很多，你们白队想要打败我们可真的有点难咯，感觉现在就是清浅上来打辅助，你们都没机会。”显然蒋念对于那天叶帝说他蹭人气的事还是耿耿于怀，所以在叶帝开口说话的时候，他便是嘲讽道。
<br/>　　“切！”然而还没等到叶帝开口说话，一旁便是传来一道不屑的声音，不用看都知道是谁，能够这么挤兑蒋念的除了小艺之外，恐怕还真没别人了。
<br/>　　“怎么，不服啊！不服solo！”听着杨小艺那不屑的声音，蒋念同样斜倪着自己的眼睛，开口说道。
<br/>　　“solo就solo，一会你可别哭鼻子！”杨小艺直接应道。
<br/>　　本来大家都以为杨小艺会拒绝呢，毕竟她才刚开始接触这个游戏没多久，可是听到她突然应了下来，这一下众人的好奇心都起来了。
<br/>　　这一段时间杨小艺已经将自己的段位打到了国服的钻2，虽然众人都知道她的天赋很高，但是具体高到了什么水准，大家心里都没有数。
<br/>　　对于叶帝总是说杨小艺比他强，众人也只是以为是叶帝对于杨小艺的鼓励，心里倒是没有当真，不过今天杨小艺和蒋念solo，这倒是可以好好看一看杨小艺的表现了。
<br/>　　“你行么？”听着杨小艺的话，叶帝开口偷偷的问道。
<br/>　　“怎么可能不行，小艺可厉害了，这两天都是她带着我排位的，简直6的飞起啊，我那个钻4的大号被她带的全胜上到了钻2。”还没等杨小艺开口，一旁的柳清月便是已经兴奋的说了起来。
<br/>　　“带你全胜上钻2？你也就钻4的水平吧，这么看来小艺还真的不简单啊，哈哈，蒋念你可小心了啊，要是输了可就真的太丢人了。”楚南开口说道，当然了话语里更多的则是调侃，说杨小艺能够solo击败一个世界王者，这一点他还是不信的。
<br/>　　起码今年杨小艺不能，如果击败那也是按照叶帝所说的，等到明年杨小艺的实力到达巅峰的时候。
<br/>　　“我要是输了，我直播吃键盘！”蒋念自信满满的开口说道。
<br/>　　“你能重说一遍么？”一旁的杨小艺面无表情的说道。
<br/>　　“说什么？”蒋念诧异。
<br/>　　“刚才那句吃键盘，我好录下来。”杨小艺比了比自己手中的手机。
<br/>　　看着杨小艺的这个动作，蒋念顿时笑了起来，不过他倒是不怂，直接开口再次说了一遍。
