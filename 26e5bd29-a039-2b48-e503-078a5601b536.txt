　　　　他们不下来，涂小安自然有办法让他们下来，所谓卧榻之侧岂容他人鼾睡，堂堂登基坐殿的蛇王头顶站着几个人，像话吗。
<br/>　　涂小安能容他们在自己的头顶趾高气扬？
<br/>　　“五雷轰顶”
<br/>　　涂小安不动声色，但体内的雷电之力已经悄然的运转。
<br/>　　霎时间，比武台的上方雷鸣声乍起，嘶鸣，似有劫云翻滚如潮，只见东南西北中凌空而立的五位大神学长的头顶上，均出现了一道白光雷电，顺势劈下。
<br/>　　他们脸色大变，青天白日，何来的雷电。
<br/>　　雷电之快，无人能反应，顿时间，东南西北的四位措不及防的中招了，直接身形不稳的朝着下方落去。
<br/>　　但，毕竟他们都是古学院一等一的高手，每个人实力都很强，就算被雷电劈了一下，还是能稳稳的落到地面，并未受多少影响。
<br/>　　可是，他们的面子却过不去。
<br/>　　说好的不下来呢，却被一道雷电赶下来。
<br/>　　“大家看，大神学长们被雷电给劈了下来了”
<br/>　　“大小姐邪七七还在上面，她身中雷电，既然一点事情都没有”
<br/>　　风华绝代的邪七七，目光如电的看到下方的涂小安，：“你居然掌控雷电能力？”
<br/>　　“下来吧”
<br/>　　涂小安只是回了她三个字。
<br/>　　邪七七似有所感，曼妙的身子左右摇曳，翩翩起舞，下一秒，连续三道雷电劈向了她的位置，既然均被她躲了过去。
<br/>　　比武台上，出现了三个冒着黑烟的坑。
<br/>　　“那么凶干嘛，讨厌...”邪七七莲步轻移，竟如鬼魅般的飞离比武台的位置，不等接下来的雷电劈下。
<br/>　　也算识趣。
<br/>　　须根之间，整个空间变得寂静无声，天上出现雷电，将大神学长都给劈了下来，紧接着，所有人撑着惊骇的目光，看着比武台上的那道清瘦身影。
<br/>　　最强新生拥有掌控雷电的能力？
<br/>　　那瞬间，大家直瞪瞪的发呆，露出怎么也抓不住要领的神情。
<br/>　　掌控雷电，是怎样的手段呀，想想都可怕。
<br/>　　底下的副院长摸了摸小胡须，暗道：“都传白镇蛇王，掌控雷电，原来这条蛇精是拥有了元素雷电能力，不得了”
<br/>　　“臭小子，你们敢阴我们”
<br/>　　几个大神学长落地之后，怒火中烧的看着涂小安，心中也处于震惊的状态。
<br/>　　尤其是白君然，脸色难看了起来，这个夺自己蛇王石的小子，看来还真的不是泛泛之辈。
<br/>　　“你们最好给我乖乖的在台下看着，乱动小心被雷劈”涂小安呵呵一笑，微勾唇角，妖冶的眸子中化过一丝玩味：“要是对我有意见，等我打完这场，你们可以上来挑战我，任何人都可以”
<br/>　　涂小安此话一出，现场瞬间再次寂静，空气都凝固了，好牛逼的话，让大神学长挑战他？
<br/>　　底下所有学员痴痴的看着那张清秀的脸庞，好大的口气，这才叫目空一切呀。
<br/>　　“哈哈...你当你是谁呀，区区一个新生既然如此口出狂言，不知道天高地厚”
<br/>　　人群中也不知道是哪位大神学长的拥护者，看傻子一样的看着他。
<br/>　　紧接着，很多人看他都能看一个死人一样，一人挑战天才班前五的高手，这牛比吹得，让在场的其他人都脸皮燥热了。
<br/>　　涂小安摇了摇头，先把正事做了再说吧。
<br/>　　看着对面的凌岩航，道：“我们可以开始了”
<br/>　　凌岩航还在沉寂在之前的几道雷电之中，这小子真的掌控雷电？
<br/>　　一举将上空五位天才班最牛比的学员打落下来，虽然他是出其不意，让大神们有点措手不及，但是也足够震撼了。
<br/>　　凌岩航太清楚上面五个人的实力了，一个个都是变态中的变态。
<br/>　　都是天赋异禀的修武者。
<br/>　　比学院的高级教官还要强。
<br/>　　看来这场比武，棘手了，这位最强新生是个难缠的对手。
<br/>　　凌岩航不想被淘汰出天才班，出了天才班，高人一等的荣誉没有了就罢了，连每个月的灵石都会锐减。
<br/>　　自己必须全力以赴了。
<br/>　　微微的思忖，凌岩航俨然了下来，从背后抽出一把寒芒的刀，整个人霍然一变，一股强悍的气息迸发出来，周身有狂风席卷出去。
<br/>　　“真正的比赛要开始了”
<br/>　　“小学弟，当心了，刀剑无眼”
<br/>　　凌岩航强悍的气息下满是杀气，大刀在手，也不见他做了什么，好像是刀影一掠，比武台的半空当下突然闪现出了一条银白色的月牙，然后凌厉无比的刀气四下蔓延开来。
<br/>　　骤然之间，死亡之气，突兀地浓郁了起来，比武台的地面被刀气刮裂，刀气只逼涂小安。
<br/>　　要是中了，一个人会顷刻间一分为二。
<br/>　　涂小安眼孔收缩，这个人用刀，很显然是修炼了什么厉害的刀法绝技，一出手刀气凌然，出手便是杀招。
<br/>　　所谓的刀气，是个人在刀法上面有很高的成就，才能办到。
<br/>　　“破...”
<br/>　　涂小安一脚震地，根本没有要闪躲的意思。
<br/>　　只见蔓延而来的刀气轰然的爆炸了。
<br/>　　“你输了”
<br/>　　涂小安震散了蔓延的刀气，旋即开口。
<br/>　　这一开口，台下的人都懵比了，拜托，你虽然破了对方的刀气，又不代表赢，出手都还未真正出手，就喊对方输了。
<br/>　　就算你是最强新生，也不带那么装逼的吧。
<br/>　　可凌岩航却惊奇得像半截木头般愣愣地戳在那儿，把嘴张得像箱子口那么大，接着他咽了两三口唾沫，难以置信。
<br/>　　没错，他好像是输了。
<br/>　　台下的人，或许还没看清楚，他也没看清楚，但他感觉到了。
<br/>　　感觉到了瞬间的死亡。
<br/>　　他一动不动，后背冷汗涔涔而下，一双眼睛圆瞪，看怪物一样的看着涂小安。
<br/>　　“认输吗”
<br/>　　“我...”凌岩航张了张嘴，很憋屈，就这样输了？
<br/>　　他就挥出了一道刀气，这不过开场的热身，真正的手段还没用出来的。
<br/>　　但他不得不认输了，此时台下的很多人终于看明白了，露出了恍然大悟的神情。
<br/>　　原来凌岩航的喉咙眼处，有一炳薄如蝉翼的飞刀，就宛如被一双无形的手握住，抵在他的喉眼上。
<br/>　　这....
<br/>　　他是怎么做到的，什么时候出手的，没有一个人看清。
