　　噗通！
<br/>　　丧失了修为之后的灰袍老者，直接从高空坠落下去，好在他的肉身也不弱，倒也没有落得活活摔死的下场。
<br/>　　但周围众人看到这一幕，顿时一个个全都彻底傻眼了。
<br/>　　这灰袍老者可是汉天王朝的国师，修为比国主宇文邦还要高，在整个毁灭大陆上，也是最最顶尖的强者！
<br/>　　如今居然就这么败了？
<br/>　　眼前这个叶欢，究竟是人还是魔鬼，怎么会有人如此强大的能耐？
<br/>　　而那宇文邦反应过来之后，哪里还敢多待，慌忙转身拔腿就要逃窜。
<br/>　　叶尘却不慌不忙，随手一指弹出，
<br/>　　嗖！
<br/>　　一道无形气劲，直奔宇文邦爆射而去，
<br/>　　彭！
<br/>　　宇文邦才只逃出数千米，整个人就突然爆炸开来，甚至连惨叫都没有来得及发出，就彻底消失不见。
<br/>　　哗啦啦！
<br/>　　周围那些皇宫的金甲将军和银甲侍卫，乃至整个皇城的人，看到叶尘如此轻易就斩杀了国师和宇文邦，顿时一个个全都纷纷跪俯在地上，吓得连头都不敢抬起。
<br/>　　“也该离开！”
<br/>　　叶尘却根本没有理会，随手一挥，前方空间出现一道漩涡，他直接迈入进入其中。
<br/>　　“恭喜你，传承者！你不但领悟到了毁灭法则，居然还能参悟到，毁灭法则和造化法则之间的联系，很不错！”
<br/>　　那道一直无比淡漠声音，再次在叶尘脑海之中响起，而且十分罕见的，对叶尘由衷的发出一道赞美。
<br/>　　“接下来，你将进入第三层世界，在这个世界里，你要参悟的是自然法则……”
<br/>　　……
<br/>　　叶尘在第一层世界呆了三个多月，便领悟了造化法则。
<br/>　　而第二层世界，领悟毁灭法则所用的时间更多，连一天都没有用到。
<br/>　　这让叶尘产生了一种，也不过如此的感觉，所以他本以为，第三层世界应该也不会耽搁太久。
<br/>　　然而，
<br/>　　让他万万没有想到的是，在这个世界，他足足耗费了七年时间，才勉强领悟到了自然法则。
<br/>　　紧接着，第四层世界，是阴阳法则，叶尘又足足用了十年，方才成功领悟。
<br/>　　……
<br/>　　一晃眼，
<br/>　　叶尘进入轮回路后，已经经历了数百年岁月！
<br/>　　在这数百年的时光之中，叶尘先后参悟了灵魂法则、力量法则、秩序法则、混乱法则、本源法则、虚幻法则、七情法则、无情法则、能量法则、永恒法则……
<br/>　　一次接着一次的轮回，周而复始，连叶尘自己都已经不记得具体时间，甚至都快要忘记自己最初的使命，感觉就像是在走一条永远都没有尽头的路。
<br/>　　若非叶尘经历过两世轮回，意志无比坚定，恐怕早就已经在迷失在，其中的某个世界之中了。
<br/>　　终于，
<br/>　　那道淡漠的声音，再次在叶尘脑海中响起，
<br/>　　“传承者，恭喜你来到第四十九重世界，在这个世界里，你要参悟的是时空法则……”
<br/>　　叶尘听到这道声音，那犹如一潭死水般的深邃眼眸中，终于出现了一道涟漪。
<br/>　　“整整四百多年了！这一切，终于是要结束了吗？”
<br/>　　这一路走来，叶尘也终于明白，为什么说这条轮回路九死一生，除了第一、第二层世界，叶尘没有待太久以外，其余的四十六层世界，叶尘就像是经历了四十六世不同的人生。
<br/>　　姑且不说，参悟这整整四十多道规则之力，难度有多大，光是这长达数百年的苦修，没有一颗坚定之心，也绝对坚持不下来！
<br/>　　“最后一层世界，是时空法则，对我来说，应该不会太难吧？”
<br/>　　叶尘在心中暗暗嘀咕了一句，
<br/>　　哗啦~
<br/>　　周围空间一阵变幻，等到叶尘再次睁开双眼，发现自己在一张温软的床上，而在他的旁边，还躺着一名容貌绝美的女子。
<br/>　　“也不知道这一世，我又是什么身份？”
<br/>　　经历过四十八世的轮回，叶尘对这种场景，早就已经习以为常，很快就适应了下来。
<br/>　　然而，
<br/>　　就在他准备从床上爬起来的时候，却发现自己此刻的身体，居然是一个婴儿！
<br/>　　“我去！什么情况？”
<br/>　　叶尘牟足了力气，却也未能从床上爬起，又重重栽倒了下去，顿时将旁边那名绝美的女子惊醒。
<br/>　　绝美女子见状，慌忙将叶尘搂在怀里，柔声道：
<br/>　　“孩子，别怕，妈妈在呢！”
<br/>　　叶尘先是哭笑不得，旋即又顿时一愣，两眼不由得死死盯着眼前这个绝美的女人片刻，脑袋轰的一下，瞬间呆滞了，
<br/>　　“妈妈？”
<br/>　　原来眼前这个女人，赫然正是叶尘的母亲林婉容啊！
<br/>　　叶尘想喊，可此时他还只是一个婴儿，喊出口之后，就变成了咿咿呀呀的哭声，根本还说不了话。
<br/>　　但是，他此刻的心里，却早已经掀起了惊涛骇浪。
<br/>　　“难道这一世，我居然穿越到了自己的身上？而且还是我刚刚出生之时？”
<br/>　　呆呆了片刻之后，望着眼前这张绝美的脸庞，叶尘忍不住放声大哭起来。
<br/>　　前世，叶尘在九岁那年，母亲林婉容便病逝而亡，对于这张脸庞的记忆，已经无比遥远，但是每当想起，都无比的温暖。
<br/>　　而如今，他终于又有机会，再次见到自己的母亲，虽然不确定眼前的世界是真是假，但是这种感觉，简直无法用语言来形容。
<br/>　　“嘎吱~”
<br/>　　就在这个时候，响起了房门打开的声音，一名丰神俊朗的青年人，从外面快步走了进来。
<br/>　　“容容，孩子这是怎么了？怎么哭的这么厉害？”
<br/>　　叶尘听到这道声音，不由得扭头望去，顿时再次呆住，停止了啼哭。
<br/>　　眼前这人，赫然正是他的父亲叶天歌，只不过此时的叶天歌，约莫二十七八岁，还十分的年轻，而且身上锐气十足，和叶尘印象中的叶天歌，气质大不相同。
<br/>　　说话间，叶尘已经被叶天歌抱起，冲着他笑眯眯道：
<br/>　　“果然是我的好儿子，一看到老爸就不哭了！就是表情有点严肃，要是能笑一笑，那就更好了！”
<br/>　　一旁的林婉容，伸手在叶天歌身上拍了一下，笑道：
<br/>　　“他还是个婴儿，哪里听得懂这些？你粗手粗脚的，别弄疼了孩子，赶紧给我！”
<br/>　　叶天歌只好一脸不情愿的将叶尘，又送到林婉容的怀里。
<br/>　　林婉容将叶尘抱在怀里，一边轻轻摇晃着，望向一旁的叶天歌道：
<br/>　　“天歌，我希望咱们的孩子，将来能平平凡凡的度过一生，和其光，同其尘，不要再牵扯进那些家族争斗之中，不如就给他取名叫叶尘，你说好不好？”
