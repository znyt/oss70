        &lt;h3&gt;第三卷第一百章  失落的相见&lt;/h3&gt;
<br/>
<br/>        凌辰原本也是山西的一个世家子弟，只是后来家族人口单薄，在加上老一辈的人思想比较固执，一直不愿意进行改变，守着自己的老本行，慢慢的家里的生意一代不如一代，到了凌辰的手中，也勉强维持不亏本。
<br/>
<br/>        现在的社会不进行改变就会被社会的潮汐卷走，就像是浪潮中的沙砾，在潮来潮往中被卷起，能沉淀下来的都是经过浪潮的洗礼。
<br/>
<br/>        凌辰大学毕业便回去接手家族事业，经过多年的打拼也没有多少起色，后来凌辰狠狠心，重新拓展了一项新的业务，他经过市场的考察，偷着瞒着家里的人做了一个全国总代理，重新开始了他的创业之路。
<br/>
<br/>        创业之初的艰难不是一般人能想象得到了，每天全国各地到处飞，每天拓展业务，寻找合适的代理商，经过几年的打拼，慢慢的有了起色，现在也是到处飞，只是比最初好了很多，现在的他只要管理好手中的经销商就好。
<br/>
<br/>        凌辰到了陈子昂公司，见了宁雅，因为人多，连话也没有多说几句。心里微微有些失落，这次到北京出差是一回事，但是最重要的还是想见一见宁雅。
<br/>
<br/>        坐在豪华的酒店房间，看着酒店外青翠的万年青扑上了一层薄薄的雪花，凌辰的心里也像是下起了雪。
<br/>
<br/>        “小雅，我在某某酒店。”他拿起了电话，拨通了宁雅的电话。
<br/>
<br/>        “我在加班呢，今天事情特别多，要赶着把事情做完。”宁雅的专业学的是经济管理，但是阴差阳错的进了陈子昂的公司当起了办公室主任。
<br/>
<br/>        “那么拼干嘛，你也就是过度一下而已。”凌辰对着电话看着宁雅的模样，他将宁雅的照片设置了屏保，以此来表达他对宁雅的爱恋。
<br/>
<br/>        “不管过不过度，干着一份工作就需要一份责任，如果在别人的公司都干不好，如何能为自己做事情？”宁雅说。
<br/>
<br/>        “有时间能一起坐一坐吗？”凌辰终究还是想单独见一见宁雅。
<br/>
<br/>        宁雅在电话里沉默了很久。
<br/>
<br/>        凌辰也没有挂电话，似乎能听到彼此的呼吸声。
<br/>
<br/>        “算了，我知道你忙，那你就先忙吧。”凌辰过了很久才开口。
<br/>
<br/>        “凌辰，谢谢你今天为公司出了一份力，我知道你都是为了我。”宁雅拿着电话看着窗外飘飘洒洒的雪花，就像是她现在的人生，有点飘忽有点虚幻。
<br/>
<br/>        她现在似乎不适合谈情说爱，家里条件是很好，她家可以说是一方首富，只是每个家庭都有每个家庭的幸和不幸。
<br/>
<br/>        她有弟弟和妹妹，只不过弟弟和妹妹是阿姨生的。
<br/>
<br/>        在大学的时候，凌辰就追求她，但是两家家境悬殊差别太大，就算她同意，怕是她的父亲也不会同意。
<br/>
<br/>        他们俩个保持着一种奇怪的关系，在她心神疲惫的时候，她想到的是凌辰。
<br/>
<br/>        “也不算是完全为了你，陈子昂那人值得人出手相帮，你说的很对，无论是干什么就要认真负责。你先忙吧，有时间了你给我打电话。”凌辰主动结束了电话。
<br/>
<br/>        宁雅说她很忙，肯定是忙，她是一个事业型的人，之前时，现在也是，曾经她在她家担任着要职，为了一个项目可以几天不睡觉，陪着部门同事一起加班赶工期。
<br/>
<br/>        只是后来因为她的弟弟接手公司的事情，她才从重要的位置上退下来，那一段时间是宁雅最苦闷的日子，她父亲等于是放弃了她，只是因为她是个女孩子，女孩子再能干也不能为家族传宗接代。
<br/>
<br/>        宁雅因为苦闷，他们俩个经常电话联系，宁雅也因为心情不好的原因，就会喝点酒，一喝酒就会给凌辰诉说自己的快乐和忧伤。
<br/>
<br/>        凌辰的手机似乎是为宁雅二十四个小时开着的，只要是宁雅打来的电话，凌辰都会接。
<br/>
<br/>        挂了电话，凌辰给宁雅发了一句话：“我就像那飘落的雪花，附身在你的掌心，你的温暖融化了我，我与你相依相随。”
<br/>
<br/>        宁雅看着qq消息，望着外面飘落的雪花，伸出手，触手而来的凉沁入心扉，凌辰的情话很甜，只是她却感觉不到温暖。
<br/>
<br/>        她并没有给凌辰回信息，没有结果的爱恋都是徒增烦恼而已。
<br/>
<br/>        只是不知道为什么，她总是会在忧伤的时候想找他去诉说。
<br/>
<br/>        凌辰盯着手机看了好一会，最后将手机扔到床上，也将自己摔在床上，弹起来又摔下去，不管怎么说，今天见到了宁雅就是快乐。
<br/>
<br/>        对啊，见到宁雅就是快乐。
<br/>
<br/>        在学校的那些快乐的时光也飞了回来，虽然宁雅在学校也是不怎么搭理他。
<br/>
<br/>        他们都算是名门之后，家里都是将他们送去读金融管理，大概也是希望他们能够担起家族的事业吧。
<br/>
<br/>        宁雅在学校就是出了名的女神，她有着一头柔顺的披肩短发，染成淡淡的黄，身高1.72，高挺的鼻梁，水灵灵的大眼睛，现实版的芭比公主，一个洋娃娃。
<br/>
<br/>        宁雅的美是那种张扬的美，只要人看见了就会挪不开眼。
<br/>
<br/>        凌辰从见到宁雅的那一天，宁雅便走进了他的心里，而且生了根发了芽，扒都扒不走。
<br/>
<br/>        大学时的生活多姿多彩，宁雅的周围也是有着很优秀的人存在，凌辰永远都是那个在外围看着的哪个人。
<br/>
<br/>        凌辰拿起手机，他想看看宁雅有没有忙完，有没有给他回信息。
<br/>
<br/>        扒拉着手机里的信息，终于等到了宁雅的信息：“工作还没有干完，早点休息，勿念！”
<br/>
<br/>        凌辰嘴角噙着笑：“看到了你的信息，就像是看到了美丽的你，跳动的心脏告诉我，活着就是因为爱着你。”
<br/>
<br/>        宁雅此时放空着自己的思想，凌辰是不是真的爱着她？他的情话就像是源源不断的源泉，总是时不时的冒出来一句，让人心动不已，又让人捉摸不透。
<br/>
<br/>        他们认识也很长时间了吧，从何时起，凌辰的情话已经是一种习惯了？
<br/>
<br/>        她拿起手机，想发一个信息过去，但是叹了口气，只要每次和凌辰聊天的时候她都会纠结很久。
<br/>
<br/>        他们有将来吗？
<br/>
<br/>        答案是否定的，既然没有将来，为什么还要彼此放不下？
<br/>
<br/>        只是心里的那一块地方，还有一个凌辰，他静静的呆在哪里，时不时的跳出来，拨弄一下她的神经和心情。
<br/>
<br/>        “怀着美丽的心情对你说：晚安！在梦里也许能相见，因为梦里永远都是安宁的天。”
<br/>
<br/>        夜晚是多情的，也是无情的。多情的是温柔的眼，是那天上的星辰，无情的是夜里的暗！
<br/>
<br/>        宁雅关了手机，今日份的工作还没有赶完，陈子昂那个疯子是拼命三娘，在她的身边你得打起十二分精神，跟上她的步伐，要不然怎么能当上办公室主任。
<br/>
<br/>        还好，宁雅有过之前的工作经验，要不然这份工作对她来说就是艰难的挑战，看着外面灯火阑珊的夜，宁雅握紧了手中的鼠标，她，虽是女儿身，但是她也有一颗冲天的心！
<br/>
<br/>        优秀的人有一种能力，他们能相互吸引，相互欣赏，相互帮助，随着陈子昂事业的拓展，她身边的人也越来越多，单打独斗的日子已经过去了，他们凝聚在一起，为了各自心目中的梦想努力奋斗着。
