　　第564章
<br/>　　“张六，出来。”
<br/>　　一声大喝，从山脚传到山顶，天上的云朵似乎也被冲散几分。
<br/>　　以山门为界，人群哗然。来万卷山瞻仰的游子以及书院的学子都处于极大的震惊中。
<br/>　　燕不归将重剑扛在肩头，脚下搁着一具被砍成两段的尸体，地上血流了一片。有些年纪轻的读书人，脸色已变得苍白，便是有些见识的人，此刻却也是紧皱眉头，很显然是克制自己不要吐出来。
<br/>　　燕不归负剑上山。
<br/>　　书院的弟子从山上涌下来，人人穿白袍，持宝剑，但面对一步步上山的燕不归，却都停下脚步，不敢近前。
<br/>　　只要一个人不瞎，就能看出来燕不归已经走火入魔。哪怕他是个瞎子，也能嗅到燕不归身上的血煞。
<br/>　　天子尚且避醉汉，正常人还是不要和疯子计较太多。
<br/>　　燕不归开始闯山，如一头暴怒的犀牛，用重剑开路，似乎要冲翻一切。
<br/>　　宋寒之正在筹备白沙岛的事情，将书院的高手转移到海岛上。已运过去一部分，但这件事毕竟急不得，目前还未全部完成。
<br/>　　这时候，有人急匆匆将消息报到他这里。宋寒之当时一激灵，提着剑冲出屋子。
<br/>　　刚出门就撞上马猛，二人对视一眼，异口同声爆出一个字。
<br/>　　“走。”
<br/>　　二人纵跃下山，速度都是极快。同时，书院大批高手正往山下赶，当赶到现场，看到眼前的惨状时，宋寒之双目通红，心里恨不得将燕不归千刀万剐。
<br/>　　燕不归一心闯山，书院的弟子不得不拦。可又有谁挡得住燕不归手中的重剑，一条蜿蜒的石阶，稀稀拉拉摆着尸体，死的当然大部分是书院的人。也要一些无心上山的游客，只因不小心挡住燕不归的路，如今也死在当场。
<br/>　　燕不归出手极干净，干净的意思是直接了当。重剑大开大合，这些人的死相极惨，有人尸首分离，有人被从中间劈开，有人虽然还活着，可两条腿却被齐生生斩断……
<br/>　　宋寒之将牙齿咬得嘎吱嘎吱响：“好贼。”
<br/>　　燕不归停住了脚步，因为他已认出宋寒之。知道这人在书院身份不低，是说话能管用的人。
<br/>　　“叫张六与我出来。”
<br/>　　宋寒之吩咐众弟子退下，这样的魔头的确不是他们有资格阻拦的，而对付燕不归这样的人，书院也并不是没有对策。
<br/>　　“六爷不在，你有什么话，可与我说，我让你死得明明白白。”宋寒之冷冷道。
<br/>　　昨夜黑贞已经死了，燕不归的悲痛可想而知。他想了一天，终于想明白了。有几个人是一定要死的，一是打伤黑贞的张六，二是书院的诸人，三就是给黑贞治伤的华正则，他为什么没把人救活，四，就是差点害的黑贞被砸成肉酱的程大雷。
<br/>　　当然，目前黑贞还不知程大雷是何许人也，不过，感觉应该也是书院的人。所以，想明白的燕不归就提着大剑来了书院。
<br/>　　反正该死的人太多了，那么，自己就干脆把他们都杀光吧。
<br/>　　而宋寒之也并没有对燕不归撒谎，张六目前的确没在书院。他做最早的船出发，此时当已到了白沙岛。
<br/>　　“呔，与他废话做什么，让俺一刀劈了他。”
<br/>　　马猛大吼一声，他人如其名，天生火爆性格，上次与燕不归碰上，他手中没有兵器，十成本事还没能发挥出五成，打得是分外憋屈。此刻仇人相见，分外眼红，他哪里还忍得了。
<br/>　　提着半人多高的斩马刀便冲了过去，一个虎跳出现在燕不归面前，双手握刀，向燕不归脑袋上砍去。
<br/>　　“来得好。”燕不归一声爆喝：“你也与我死。”
<br/>　　刀与剑撞在一起，都是大开大合的打法，一时打得难解难分。
<br/>　　宋寒之并不太担心，书院没了张六，还有马猛。这里的高手，可不仅张六一个。燕不归既然脑子抽疯，敢来送死，宋寒之就不介意成全他。
<br/>　　程大雷在偏僻处远远瞧着，燕不归和马猛都是绝世等级的人物，而场上的绝世高手还不止他们两位。
<br/>　　“啧。”程大雷轻叹一声：“今天还真是群英汇呐。”
<br/>　　这种场面，程大雷自量也没把握平安脱身。当然，看燕不归的意思，也没打算平安脱身。如果没有一腔必死之心，他也不会脑袋抽疯来闯书院的山门。
<br/>　　现在所有人的注意力都集中在燕不归身上，也没人会想到，还有一头恶贼埋伏在附近。程大雷自然没打算现身，他正寻找机会，悄悄摸到山上，把刘发财救出来。
<br/>　　而此时，燕不归和马猛的战斗已快分出胜负，马猛毕竟年老体衰，再加上燕不归一心杀人，招招都是拼命的打法。一心强攻，竟压得马猛抬不起头来。
<br/>　　“马老爷子，小心。”
<br/>　　宋寒之突然喝了一声，持剑冲了上去。马猛也不想一条命搁在这里，见情形不对，便想抽身撤出战团。
<br/>　　但有些事，并不是他想就能做到的。
<br/>　　“与我死。”
<br/>　　燕不归将剑投了出去，身后一道杀机突至。他下意识回过头，重剑不偏不倚扎如他的身体。一股热血从口中喷了出来，身体咚地一声倒地。
<br/>　　这招脱手剑还真是防不胜防，程大雷发出一声感慨。
<br/>　　宋寒之也是一惊，向前冲的脚步挺住了。手中的剑一挥，口中道：“布阵。”
<br/>　　书院的书生意气剑阵，号称无人可破。但今天书院一部分高手在岛上，也布不成完整的七十二剑阵，目前书院留下的人，只能凑成三十六之数。
<br/>　　程大雷还是很想见识一下书山剑阵的威力，可他实在没时间耽搁。如今书院大半高手都被燕不归吸引在此地，此刻山上防守正是空虚。
<br/>　　此时不走，更待何时。
<br/>　　程大雷悄悄摸摸，鬼鬼祟祟向山上摸去。
<br/>　　燕不归已经陷入包围中，书山剑阵已成，三十六持剑书生将他围在当中。宋寒之是今日的阵胆，主持今日的大阵。
<br/>　　书院的剑阵威力很强，不过今日这样的大场面却也是很少出现。
<br/>　　但燕不归脸上没有一丝惧色，只有要杀人的猛烈欲望。
<br/>　　“你们……都与我死。”
<br/>(本章完)
