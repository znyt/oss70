　　可是不管李承乾是否纠结于武家兄弟的身份，该发生的事情还是会发生，对于这种败家玩意儿来说，没有什么事情是他们不敢干的“你们还愣着干什么，把他们从本公子的马车上赶下来。”早已经没了耐心的武元爽挥手对手下的四五个家将说道。
<br/>　　“喏！”家将们显然不是第一次干这种事，答应一声便下了马，向着李承乾的坐驾围了过来。
<br/>　　“嘭”，手弩声响，一溜乌光带着残影钉到了武元庆与武元爽的脚下，夜魅手里拿着不知从什么地方掏出来的手弩，阴着脸说道：“谁敢过来，下一次某会射他们的头。”
<br/>　　“手弩？”武家兄弟的脸色变了变，对视一眼，感觉这一次好像是惹到不该惹的人了。
<br/>　　被手弩指着，对于武家兄弟来说也是第一次，看着乌黑的三棱弩矢，两兄弟的背后隐隐有些发寒，感到惹了麻烦的同时，冷汗也开始不断的往外冒。
<br/>　　大唐民间不禁弓箭，但弩却是在这个范围之外，不要说寻常百姓人家，就是一些顶级豪门的少爷公子想要带着强弩出门，那也是不大可能的事情。
<br/>　　“把武器全都放下，靠墙站好。”得到李承乾眼神示意的夜魅根本不给武家兄弟说话的机会，靠着对武家兄弟的威胁，逼退了一众家将之后便等在了原地，就那么静静的注视着两个油头粉面的家伙，直到等来了巡街武候。
<br/>　　“怎么回事儿？”看着剑拔弩张的场面，巡街武候也有些紧张。
<br/>　　长安城虽说是天子脚下，但是纨绔们斗起来的时候可不管这些，仗着家里老子娘的后台，除了不敢当街杀人之外，其他什么事情都干得出来。
<br/>　　“把这些人带回去，通知他们家里人来领人。”将手弩交给苏猛，夜魅掏出一块腰牌，在巡街武候前面晃了晃。
<br/>　　“喏！”金光闪闪的腰牌在面前闪过，巡街武候甚至都没有看清上面写的是什么，只是那耀眼的金光足以证明这是宫里的东西，巡街武候就是再大的胆子也不敢和有这种腰牌的人做对，所以在答应了一声之后，立刻呵斥着手下将武家兄弟以及众家将围了起来。
<br/>　　“误，误会，几位兄弟，这是误会！”说话的是刚刚拦路的家将，至于说武家兄弟……这俩货已经被吓傻了。
<br/>　　凭借一块牌子就能调动巡街武候，这是什么样的权势？什么样的地位？他们两个的老子调任工部尚书，已经算是品级够高了，可也没有这样的待遇，这足以说明对方是他们家老头子都惹不起的。
<br/>　　意识到这个问题之后，兄弟俩自然有些发毛。
<br/>　　不到江南不知道自己钱少，不到长安不知道自己官小，在利州那个小地方，老头子上利州都督，他们两兄弟完全可以在那里称王称霸。
<br/>　　现在到了长安才知道，敢情就算是三品大员家的公子，也特么屁都不是，遇到狠人照样也要吃亏。
<br/>　　“哥……”即将被押走的瞬间，武元爽叫了一声。
<br/>　　“闭嘴！”武元庆似乎知道弟弟要说什么，厉喝一声把他后面的话堵了回去。
<br/>　　妹妹可是郑重的警告过他们两个，不要接着她的身份在外面惹事，如果他们两个现在把妹妹的身份抖落出来，闹不好结果比现在还要惨。
<br/>　　小武的身份，小武后台的身份，这些武家兄弟都知道，但是这种身份和地位上的较量只能私底下进行，如果扯到官面上，一个弄不好被大义灭亲的可能性远比私了的可能性要大。
<br/>　　盏茶时间之后，武家兄弟以及一众手下被巡街武候带走，长街只上只剩下李承乾一行。
<br/>　　“哥哥，为什么不好好教训教训他们？”李治眨着眼睛，有些郁闷的问道。
<br/>　　“一架马车而已，你能把他们怎么着？”李承乾撇撇嘴说道。
<br/>　　“可是他们要灭人全家呢。”
<br/>　　“吹个牛而已，当不得真，就像我总说要把你屁股打开花，却一直没有动手一样。”
<br/>　　回宫的路上，李承乾给小屁孩儿李治解释着什么是威胁，什么是震慑，不知不觉间说了很多。
<br/>　　等到把小屁孩儿送回长孙那里之后，李承乾这才空出时间叫过杨雨馨问道：“新阳公主这段时间在什么地方？都在忙些什么？”
<br/>　　“殿下，三天前新阳公主已经回应国公府了！”杨雨馨回答道。
<br/>　　应国公就是武士彟，这老家伙前段时间被调回了长安，继任工部尚书，这也是武家兄弟为什么会出现在长安的原因。
<br/>　　正在摆弄着一架望远镜的李承乾诧异的扭过头：“啥？回家了？我怎么不知道？”以小武和他的关系，按理说没可能回家不告诉他一声。
<br/>　　杨雨馨觉得李承乾的反应似乎有些大了，但最后依旧说道：“呃，新阳公主只是回去家去看看，要不了几天就会回来，所以也就没和殿下您说。”
<br/>　　“哦哦！”听说小武只是回家去探亲，李承乾心中涌起的那股子酸涩才慢慢平息下去。他对小武所投入的关心可并不比长乐少，所以如果小武真的不告而别，李承乾怕是要伤心很长间。
<br/>　　“太子哥哥，你是不是想我啦？”就在李承乾打算再说些什么的时候，出乎预料的，门外探进了一个满是辫子的小脑袋，不是小武却是谁来。
<br/>　　“你个小丫头，不是回家了么？怎么又跑回来了？”李承乾把对杨雨馨的问题咽回肚子里，对小武招手说道。
<br/>　　“还不是因为我那两个不争气的哥哥！”已经十来岁的小姑娘像大人一样叹了口气，走进来靠到李承乾的身边，扯着他的衣袖说道：“太子哥哥，刚刚是不是我那两个不争气的哥哥惹到您了？我已经把他们带来了，您看着处罚吧。”
<br/>　　“处罚？”李承乾看了小武一眼，伸手在她头上弹了一下：“小丫头片子，跟我玩心眼儿来了是吧。”
<br/>　　“哪有啊，我那两个哥哥的确太不像话了，冲撞了殿下，若不好好惩罚，只怕将来还会惹出更大的麻烦。”提到武家兄弟，小武的语气中满是不屑，似乎有种打心眼里瞅不起的意思。
