　　　　一只二十九级的猫精反而被一块蛇王石所控制，可想这块石头有多么的变态了。
<br/>　　此时，两股蛇类气息不断的此消彼长，狭路相逢勇者胜。
<br/>　　涂小安愈战愈惊，五分钟后，他既然还没办法彻底压制这股特殊蛇类气息，心中不得不猜测，这不会真的是证道蛇王体内的内在金丹所化吧。
<br/>　　不然不可能强到这般的变态。
<br/>　　猫精吞了蛇王石，所到之处，群蛇跟随，多达十几万的大军。
<br/>　　如果猫精在一个地方停留个十天半个月，那么不知道会吸引多少蛇类涌现。
<br/>　　光靠自己的蛇王气息既然都压制不了，涂小安心中一狠，将系统赋予的蛇类统治能力启动，身上同时迸发两股无与伦比的恐怖气息。
<br/>　　有了统治能力的加入，霎时间，这场肉眼不可见的战斗开始了一边倒的趋势，空间冷冽寒芒，整个山洞寒冬腊月。
<br/>　　仅仅半晌，涂小安全面压制，一鼓作气的将蛇王气息彻底的笼罩到了猫精的身上，而猫精体内发光发亮的蛇王石变的黯淡了不少，不在作妖。
<br/>　　紧接着，猫精海棠就恢复了清醒，体型缩小，恢复到了正常猫的可爱样子，只是浑身毛发还是一片焦黑。
<br/>　　当海棠清醒之后，看到不远处的一条通体紫金色小蛇悬浮着，顿时诧异不已。
<br/>　　涂小安一声大喝：“海棠你别动”
<br/>　　他要将海棠体内的蛇王石给逼出来，可旋即，海棠像是也意识到了这一点，居然开始奋力的挣扎起来，帮着蛇王石一起来抵挡涂小安身上的恐怖气息。
<br/>　　“不要夺我的蛇王石，不要”
<br/>　　海棠的眸不在妖邪血红，反而楚楚可怜气息，眼前的这条无与强大的蛇精在夺它的宝贝。
<br/>　　此时，猫精发出来的声音也不在沙哑可怖，好像一位柔声细语的小姑娘在恳求。
<br/>　　涂小安冷淡无波，眸光冷电：“你应该知道自己控制不了这块蛇王石，何必贪恋”
<br/>　　蛇王石里面蕴含一股强大而霸道的力量，助石之人可提升实力，但持石愈久，反会产生石控人心之象，魔念丛生，难以自拔。
<br/>　　这也是这种石头的变态之处。
<br/>　　但归根结底，是你的实力不够，得不到这块石头的认可。
<br/>　　“不，我需要它，我需要它帮我突破到三阶生灵”
<br/>　　海棠眸露坚定，有种豁出去，奋不顾身的决心。
<br/>　　涂小安微微一怔，为了突破三阶，连命都可以不要了？
<br/>　　要知道就算没有涂小安出手，也有别人，这块蛇王石是猫精从白家人的手中抢的，白家人不会就此罢休。
<br/>　　最棘手的就是，海棠藏无可藏，都会被人找出来。
<br/>　　忽然，涂小安身上恐怖的气息一消，风淡云轻，然后沉声的问：“告诉我为什么，你难道不清楚，自己已经开始被这块石头所控心智了吗，别说什么要成为三阶生灵，我想还有其他的原因吧”
<br/>　　没错，三阶生灵超凡脱俗，凌驾普通生物之上，但也没必要如此豁出去也命都可以不要吧。
<br/>　　它这是赌博，而且这场赌博已经在输了，如果继续下去，它会一无所有，连自己的这具身体都会输的不是自己的。
<br/>　　猫精海棠闭口不语，显然没有去相信眼前这条恐怖的蛇精。
<br/>　　它甚至不知道自己怎么惹出了这条蛇精。
<br/>　　“我是陈高阳的朋友”涂小安只能打这张牌：“他说他的女朋友叫海棠，半个月前失踪了，想必是你假扮的吧”
<br/>　　海棠目瞪口呆，忽然，它脑海中出现了之前实验室的那位清秀少年，那声音，不就是这样吗。
<br/>　　可一位人类，怎么会又忽然变成蛇精。
<br/>　　虽然在这个过程中，海棠的神志断断续续，时而清醒，时而混乱，但一个是人，一个是蛇，两者完全八竿子打不到。
<br/>　　倏然，海棠激动的说：“你莫非已经是三阶生灵，可以化为人形？”
<br/>　　也唯有这种解释了。
<br/>　　涂小安闻言，也不否认，也不承认，到是反问：“你跟陈高阳的事情，我都知道了，所以，你无需隐瞒，告诉我，我或许可以帮你”
<br/>　　“我没有隐瞒，我就是想靠这块蛇王石的力量，助我突破三阶生灵”海棠此时乖巧的趴在地面，浑身抖了抖，毛发上的焦黑立刻散去，露出了一身纯白胜雪。
<br/>　　涂小安看着如今的猫精，很漂亮，一点都不凶煞，它的猫族血统应该很高，否则不可能成长到这种地步。
<br/>　　况且这只猫精可是有一百六十二年的寿命，活的比人还长。
<br/>　　“借助蛇王石突破，可你失败了，既然失败了，为什么还执迷不悟”
<br/>　　猫精海棠不甘心道：“我想在试一试，我不想放弃”
<br/>　　“为了陈高阳？”涂小安灵光一闪，按照他所知道的，这只猫精能用声音假扮人类，充当陈高阳的女朋友，他们两个人的感情也很好。
<br/>　　那么他为什么突然离开陈高阳，就是为了突破三阶？
<br/>　　“没错，我不想离开高阳哥哥，他需要我，我要一辈子守护他”
<br/>　　此话一出，你会看到一只猫的眼神中流露出极为人性化的情感，有眷恋，有不舍，更有生死别离的痛苦。
<br/>　　“难不成，你不突破到三阶生灵，你就没办法守护他了？”涂小安说着眼睛一瞪：“我知道了，你想名正言顺，你想突破三阶生灵，化为人形？”
<br/>　　这是人妖恋，就像聊斋里的故事，狐妖化为人形跟人谈恋爱。
<br/>　　但不同的是，这只猫精纵然灵智大开，可以口吐人言，但它并不具备化为人形的能力。
<br/>　　也胜在陈高阳是个天生的瞎子，虽然能哄骗一时，当如何能哄骗一辈子。
<br/>　　所以，海棠只剩下一条路，那么就是自己必须成为三阶生灵，不但可以口吐人言，还可以化为人形。
<br/>　　那么就能如童话般的故事一样，拥有美好的结局，它跟陈高阳一辈子在一起，不怕谎言揭穿。
<br/>　　“不，我只想安安静静的守护我的高阳哥哥，成不成人形根本不重要”
<br/>　　海棠眼中闪过一丝柔情，但却摇了摇头。
<br/>　　嗯？
<br/>　　涂小安蹙眉，不是这样，那么它为什么一定要拼死也要突破三阶生灵？
<br/>　　涂小安有点懵逼了，要突破三阶生灵，那可是九死一生，甚至十死无生也有可能。
