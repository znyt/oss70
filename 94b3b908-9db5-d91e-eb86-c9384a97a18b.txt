        &lt;h3&gt;第三卷第三百一十六章  兄弟三人行&lt;/h3&gt;
<br/>
<br/>        江俞轩回到了北京，他很纠结是先去看陈子昂呢，还是联系张倩楠。
<br/>
<br/>        张倩楠和张函从西城走后就不和他联系了，原来都是张倩楠主动给她发信息，俩个人似乎从各自的眼前消失，你不闻我不问。
<br/>
<br/>        在这场游戏里，江俞轩似乎成了一个局中旁观者，只是这几年白白浪费了很多时间，不过对他来说似乎没有损失什么，他还是他，大家都还在，尤其是陈子昂还单身。
<br/>
<br/>        在他纠结的时候，郝景文为他提供了机会。
<br/>
<br/>        他在群里接到了郝景文的信息。
<br/>
<br/>        帅得掉渣的文文：“三哥，回北京了吗？如果回来了能不能先来公司一趟？”
<br/>
<br/>        帅得掉渣的轩轩：“我还在机场，我回去放下东西就过去吧。”
<br/>
<br/>        帅得掉渣的黑虎：“我也到公司，我们碰碰面。”
<br/>
<br/>        帅得掉渣的子昂：“我下午还有事情，就不和大家碰面了，明天见。”
<br/>
<br/>        帅得掉渣的轩轩艾特帅得掉渣的子昂：“我先去文文和大哥那里了，晚上一起吃饭？”
<br/>
<br/>        帅得掉渣的子昂：“不，下午加班，晚上会很晚！”
<br/>
<br/>        帅得掉渣的轩轩：“那好吧，明天见！”
<br/>
<br/>        帅得掉渣的文文：“害，打脸的速度很快，三哥，晚上不准备和我们一起吃饭吗？”
<br/>
<br/>        帅得掉渣的轩轩：“我不是喊陈疯子和我们一起吃饭吗？”
<br/>
<br/>        帅得掉渣的黑虎：“哟咿，我晚上不能和你一起吃饭，我要陪安男。”
<br/>
<br/>        帅得掉渣的文文：“哟咿，我晚上不能和你一起吃饭，我要陪小甜心。”
<br/>
<br/>        帅得掉渣的轩轩：“你们。。。。。。没爱了！”
<br/>
<br/>        他能陪谁？
<br/>
<br/>        江俞轩赶到兄弟三人的公司，寒伯安和安男已经在公司了，还有寒伯安的助理李长卿。
<br/>
<br/>        “长卿，接替了文文的工作干得很顺手吧？”江俞轩笑着问李长卿。
<br/>
<br/>        “顺什么手，我天天扮娘娘。”李长卿一脸憋屈，人家是堂堂七尺男儿，刚阳帅气，照这样下去他都会怀疑自己了。
<br/>
<br/>        “扮娘娘？”江俞轩和郝景文的八卦心起，同时问。
<br/>
<br/>        “不得已而为之，角色变换而已，怎么了长卿，你还觉得委屈了你？”寒伯安捏着安男的手，安男的手真软和。
<br/>
<br/>        “怎么了长卿，还委屈你了？”江俞轩和郝景文哈哈大笑，他们真的很想看李长卿扮娘娘的样子。
<br/>
<br/>        “说实话，你也别笑，我们真的是不得已而为之，有人盯上我们了。”寒伯安的声音有些冷。
<br/>
<br/>        “谁盯上你们了?”江俞轩和郝景文齐声问。
<br/>
<br/>        “我还说不上来，这也就是我将你们叫到一起的原因，我们三个人可以说是一体的，我们这个公司是文文在负责，这几年的发展势头也很好，文文也有可能会被关注，俞轩，你也是，注意点。”寒伯安看了一眼江俞轩。
<br/>
<br/>        “轩轩，还有，陈子昂有可能也会卷入一场漩涡里，我们现在可以说和她是一体的了，大家都要做好思想准备。”
<br/>
<br/>        江俞轩是知道陈子昂和张家、陆家的一切瓜葛，在他的心里其实是知道陆家肯定不会善罢甘休的，只是看时间的早晚，当初那么圣母心的想促成陈子昂和张函，现在想一想是多么可笑的一件事情。
<br/>
<br/>        郝景文自从见了张函之后，对陈子昂和张函之间的故事也多了一些关注，当然也知晓了其中的缘由，他并不看好陈张能复合，只不过大家都在极力的促成这可怜的一家人，他也希望他们能在一起，只是往往事如违愿，他们的促进只是加速了陈子昂的困境。
<br/>
<br/>        “目前局势不太明朗，陈子昂和我们这里并没有什么不顺利的事情出现，我们会加倍注意的。再说，我们对于陈子昂的事情也只是猜疑，陈子昂看起来很稳妥。”郝景文看着寒伯安夫妇，这一对走哪儿都是绞着状态，安男的手一直被寒伯安握在手心。
<br/>
<br/>        “注意点是没有坏处的，我在美洲的时候被人强制要求签了合同，而来人说他是甸城谷家，谷家后面到底有着怎么样的关系，我们似乎都不太清楚，虽然谷家是慈、善家，作为一个仁义道德的人是不会使用手段的，相反，他们在美洲威胁了我，而且他们的货物存在瑕疵。”寒伯安将他在美洲的遭遇给江俞轩和郝景文说了。
<br/>
<br/>        “这一次我们参加陈子昂公司的推广会，无疑是站了陈子昂的队，后期我们真正的会被绑在一起，也许有人会想办法分解我们，我觉得团队的力量还是很大的。”郝景文喝了一口茶，他特别喜欢喝茶。
<br/>
<br/>        “我们江家会和陈子昂在一起的。”江俞轩看了一眼寒伯安，寒伯安不用说了，安男是陈子昂的表姐。
<br/>
<br/>        “你们江家不是你说了算哦，我们寒家可是我说了算，不，是我们安男说了算。”寒伯安又不忘撒一地狗粮。
<br/>
<br/>        “哈哈哈。”郝景文不厚道的笑了，能让三哥无语的大概只有大哥了。
<br/>
<br/>        “放心，我不但江家说了算，陈家我也说了算，陈子昂过两天去甸城，公司里的一切都交给我了，她说了我就是她坚强的后盾。”江俞轩慢悠悠的说。
<br/>
<br/>        “所以嘛，大哥，文文，你们说谁说了算，谁是赢家？”
<br/>
<br/>        “不是，你不是已经从公司滚出来了吗？还官宣？合着你们闹着玩呢？”郝景文大跌眼镜。
<br/>
<br/>        “给别人看的，我们公司内部已经宣布我执行总裁，以后有什么事情都是我说了算，嗯哼！”江俞轩很嘚瑟。
<br/>
<br/>        “哇，陈子昂可以啊，你们俩个人连我们都骗过去了。不过你们这样一做，给别人看不就不起作用了吗？”寒伯安插话。
<br/>
<br/>        “我们开会都是内部的高层会议，很多人都明白我们的用意，不用解释的。”江俞轩骄傲的一笑。
<br/>
<br/>        “你就这么信任他们？”
<br/>
<br/>        “用人不疑，疑人不用，这是我和陈子昂一贯的作风，当然，我们会防患于未然的。再说了，那些高层哪一个没有拿公司的股份，早就和公司为一体了。这就是陈子昂的聪明之处，陈子昂曾经说过，我们公司任何一个人都可以独挡一面，离开任何人都可以运转。”江俞轩伸了个懒腰：
<br/>
<br/>        “大哥，文文，晚上和我一起吃饭吗？不和我一起吃饭我就回去睡觉了，明天要回去帮陈子昂。”
<br/>
<br/>        郝景文和寒伯安相互看了一眼，他们的狗粮撒下去没有任何结果，合着人家情比金坚。
<br/>
<br/>        “你要是想和我们一起吃饭了，那晚上就一起吃，反正你现在回去也是单身狗，没有人陪你。”郝景文笑着说。
<br/>
<br/>        “那就勉为其难的一起吃顿饭吧，长卿不是还单着吗？哈哈哈！”江俞轩看了一眼在旁边坐着的李长卿心情特别好。
<br/>
<br/>        “长卿，宫阳小姐姐什么时候到？”
<br/>
<br/>        “应该快了。”李长卿看着江俞轩微微一笑“不好意思，我女朋友一会就来了。”
<br/>
<br/>        江俞轩：“.......我还是回家，不对，我回去陪陈子昂加班！”
