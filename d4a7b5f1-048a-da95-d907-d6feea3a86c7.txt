　　面对十余道小型飓风的攻击，苏莫心念电转，准备暂时放弃对付巫天御。
<br/>　　虽然他还有底牌、比如大日武魂和大日神体的力量，但根本不足以击败对方。
<br/>　　此人之强，远超他的预料，他完全没有必要把时间都浪费在对方的身上。
<br/>　　虽然对方抓去了宏青璇，但他现在最主要的目的，还是擒拿巫族的试炼者，这关乎他和无数苍穹神宫之人的生死，孰轻孰重，苏莫还是知道的。
<br/>　　待擒拿了巫娴，以及一些巫族天才，他才能放心对付巫天御。
<br/>　　唰唰唰！
<br/>　　下一刻，苏莫身边陡然出现了三具三色身影，赫然是三具魔影分身。
<br/>　　魔影分身苏莫已经很久未用了，但他早将这么秘术修炼到了圆满的境界，每一具魔影分身，都能有他九成的修为实力。
<br/>　　三具魔影分身，手中各自凝结出了一柄数丈长的玄力长剑，而后齐齐出手，奋力抵挡十几道小型飓风的攻击。
<br/>　　而苏莫，将他修炼的帝级身法《无影魔宗》施展到极致，整个人如同一道幻影，急速向万余里外的巫娴冲了过去。
<br/>　　轰隆隆！！
<br/>　　一阵惊天爆响，三具魔影分身根本不是那些飓风的对手，一轮碰撞，其中一具魔影分身直接炸开了。
<br/>　　而另外两具魔影分身，直接被击飞了出去，飞到了九霄云外。
<br/>　　咻！
<br/>　　苏莫身形划破空间，瞬息数千里，只是眨眼之间，他便临近了巫娴。
<br/>　　轰隆！
<br/>　　苏莫大手凌空一抓，一只玄力大手碾碎虚空，向巫娴镇压而去。
<br/>　　巫娴的实力，苏莫也看到了，对方要比巫天御弱不少，他有把握几招之内擒拿对方。
<br/>　　巫娴见此，柳眉一挑，不过她却并未躲避，而是玉手一拍，一道浩大的掌印如同一座横亘的小山，迎击而去。
<br/>　　在巫娴的周围，还有大量的巫族天才，众人见此纷纷出手，各种强大的攻击，五光十色，排山倒海一般，全部杀向苏莫。
<br/>　　苏莫幻化的玄力大手，被无数的轰击击中，几乎没有任何悬念，便被瞬间碾成了碎片。
<br/>　　而后，无数的攻击威势不减，铺天盖地的向苏莫袭杀而去。
<br/>　　苏莫见此全身汗毛一炸，这么多攻击，就算是他实力再强，也抵挡不住。
<br/>　　他立刻停止了身形，方向一转，便向上方冲去，企图避开这些攻击，再行擒拿巫娴。
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　下一刻，喊杀声四起，苍穹神宫一方的人，自然不会让苏莫独自战斗，纷纷冲了过来，再次混战了起来，各种攻击眼花缭乱。
<br/>　　“土著，你的对手是我！”
<br/>　　苏莫刚刚避开无数攻击，巫天御的声音在身后响了起来。
<br/>　　只见巫天御速度极快，裹挟着通天彻地的飓风，向苏莫杀了过来。
<br/>　　轰隆隆！！
<br/>　　狂风肆虐，轰响震天，又是十几道飓风呈扇形向苏莫袭杀而来。
<br/>　　这飓风扫荡虚空，威势无敌，足以撕碎一切。
<br/>　　“可恶！”苏莫心中怒火中烧，看来此人不会给他擒拿巫娴的机会，他必须要想办法解决此人，他才好动手擒拿巫娴。
<br/>　　嗡~~
<br/>　　下一刻，苏莫的背后再次有虚影升腾，如同一轮曜日升空。
<br/>　　这是大日武魂！
<br/>　　大日武魂一出现，苏莫也立刻催动了大日神体的血脉力量，瞬间形成了一个巨大的大日领域。
<br/>　　大日领域笼罩方圆数千里，将十几道飓风，以及巫天御都尽数笼罩在了其内，炙热无比的温度，顿时汹涌爆发。
<br/>　　大日领域虽然不足以抵挡住巫天御的攻击，但能大量削弱对方的攻击威力。
<br/>　　“大苍穹剑法——日月转！”
<br/>　　苏莫双手握剑，一剑横斩虚空，刹那之间，犀利而又浩大的三色剑气，向十几道飓风拦腰斩去。
<br/>　　一剑出，苍穹寂灭，日月无光，仿佛日月的精华，都蕴含在了这一剑之中。
<br/>　　轰轰轰！！
<br/>　　瞬息之间，恐怖的一剑和十数道飓风相击在了一切，剑气撕裂了一道又一道飓风，直到一连斩灭了九道飓风之后，剑气才崩溃消散。
<br/>　　而后，苏莫身形一闪，便避开了剩下的几道飓风。
<br/>　　“什么？”巫天御见此，顿时惊呼一声，心中再次狂震。
<br/>　　他倒不是震惊苏莫的实力，依旧是震惊苏莫的武魂。
<br/>　　十一种武魂？
<br/>　　巫天御感觉现在仿佛是在做梦一般，一切都是那么的不真实。
<br/>　　刚才苏莫动用十种武魂之时，他已经震惊莫名，但现在苏莫居然还有其它武魂！
<br/>　　这实在是太不真实了！
<br/>　　苏莫到底有多少武魂？这完全不符合常理啊！
<br/>　　其他的巫族天才，同样是心中狂震，个个心中掀起惊涛骇浪。
<br/>　　好在众巫族天才，心中的承受力已经变得强大了起来，没有再出现之前那种愣神的情况。
<br/>　　“我就先灭了你！”苏莫怒喝一声，十一种武魂催动到了极致，二十八柄灵剑也再次回到了他的身边。
<br/>　　他准备不惜一切代价打败巫天御，即便是动用大三宝术，也在所不惜了。
<br/>　　只要打败了巫天御，哪怕身体虚弱，实力大降，他和众人联手，也足以扫荡其他巫族天才了。
<br/>　　啾！
<br/>　　就在苏莫准备动用大三宝术之时，一声尖锐刺耳的鸟鸣声突然响起。
<br/>　　这声鸣叫极其的尖利，吩咐能洞穿人的耳膜，刺进人的脑海。
<br/>　　而后，天空突然黑了下来，只见一只庞大无比的怪鸟，遨游虚空，急速而来。
<br/>　　此鸟全身呈灰白之色，长约百余里，翼展足有近两百里，遮天蔽日，威势无匹。
<br/>　　此鸟的利爪尖锐无比，寒光熠熠，它还长了一只巨大的口器，如同一柄巨大的神剑。
<br/>　　“终于来了！”苏莫见此顿时心中一喜，不死鸟，秦不死！
<br/>　　苏莫虽然不知道秦不死的具体实力，但据他的观察，对方绝对不会弱。
<br/>　　只要对方能缠住巫天御三个呼吸的时间，他就有把握趁着这点时间，杀死其他所有的巫族天才。
<br/>　　唰！
<br/>　　庞大的怪鸟速度无与伦比，瞬息而至，其巨大的利爪从天而降，一爪向巫天御狠狠的抓去。
<br/>　　“给我拖住他！”
<br/>　　苏莫见此，顿时暴喝一声，而后转身向巫娴所在之地急速冲去。
