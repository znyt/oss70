　　地上躺着的人，正是当下黑芒战队的队长，同时也为狼牙中将之一的江凌若。
<br/>　　作为三战队中为数不多的女性，还是战力超绝的队长，江凌若此时与其他成员一样，毫无形象的躺在一堆腐肉黑血的肮脏地面，与黑芒战队的成员挤在一起，熟睡的样子没有清醒时的那股锋芒，看起来有些恬静。
<br/>　　洛冥充满感慨：“我一手将这些小姑娘找出来，每一个都是华夏顶尖的人才，各个有着在不同领域超出常人的天赋，哪知还没机会让这个特种战队正式成立，世界就变了。”
<br/>　　“我给了她们每人不同的任务，江凌若当初的任务地点是东南亚，可就在任务时限内末世爆发。”洛冥说着，陷入了回忆：“当我得知南方那片地带已经没有人类活动的迹象时，就已经放弃了，却没想到有一天，江凌若竟然自己找到了上京基地。”
<br/>　　段江伟没想到江凌若还有这样的故事，不禁好奇：“也就是说，江凌若是自己从南方那片尸海杀了出来？”
<br/>　　洛冥点头：“当她出现在我面前的时候，我甚至都以为自己在做梦，这个还未成形的预备特种战队，江凌若论综合能力最强，队长当之无愧。”
<br/>　　段江伟有些惊讶，再次看向江凌若的目光也随之变化：“难怪能当上狼牙三战队之一的队长，也难怪楚涵会将黑芒交到她手上。”
<br/>　　洛冥笑道：“我也没料到江凌若会和楚涵有交集，这世间果然巧合不出不在。”
<br/>　　段江伟望着周围一片黑芒战队成员，神情收敛了在杀羽那的调侃味道，变得有些谨慎认真：“论杀羽为战力最强，但以前我最忌惮的就是黑芒战队，不过现在黑芒已经重生，与过去肖坤率领的模式有所改变。”
<br/>　　“于是不再忌惮了？”洛冥随口一问。
<br/>　　“不。”段江伟神色隐晦：“更忌惮了，因为了解甚少。”
<br/>　　洛冥看了段江伟一眼，忽然笑起：“你怎么不忌惮神隐？金阳基地当初有多盛极一时，就是被神隐区区一百人给灭掉的。”
<br/>　　段江伟目光看向不远处躺着的神隐战队成员，摇摇头道：“关于这三个战队，段氏基地的参谋部有过系统分析，段氏战团也有针对性的训练，并且时间还不短，我们不怕激进代表的杀羽，也不怕阴谋诡计的神隐，就怕黑芒的暗中一击，因为根本防不胜防。”
<br/>　　听到这番话，洛冥微微好奇：“你很防备楚涵啊？竟专门为此训练段氏战团。”
<br/>　　段江伟忽然有些窘迫的挠了挠头，而后眼神认真：“如果不是因为人类共同的敌人太多也太强，我和楚涵压根没有机会决一胜负，否则我一定会竭尽全力跟他争锋相对的比拼一次。”
<br/>　　洛冥挑了挑眉，忽而转移话题道：“奇怪，怎么没看到楚涵？”
<br/>　　“随同作战的参谋部也不在。”段江伟说着，自然而然的望向旁边的高台：“如果我是指挥官，定然会在改造战场的时候，建立一座可以纵观全局的指挥台。”
<br/>　　“上去看看。”洛冥说着就率先朝着那里走去。
<br/>　　两人没花多长时间就走到了高台顶部，进入指挥间后更没有任何意外的看到了七横八躺的参谋部成员，全部都在熟睡。
<br/>　　见到这副景象的洛冥彻底无语，这狼牙战团从上到下，从军官到士兵，竟然就这么睡的连人来了都不知道！
<br/>　　尤其是何峰竟然也这么不着调，这一点让洛冥有种无力感，这种放纵可不是他教出来的徒弟。
<br/>　　“起来了臭小子！”气愤之下，洛冥一掌把何峰拍醒。
<br/>　　被惊醒的何峰下意识反击，一柄锋利的匕首已然出鞘，毫无任何犹豫的就朝着自己的前方刺去！
<br/>　　叮——
<br/>　　一声脆响，匕首被段江伟手中的紫色长剑挡下，同时也让后者后怕不已，连带着望向何峰的眼神都犀利起来。
<br/>　　刚才那一瞬间，何峰就差点刺中洛冥！
<br/>　　洛冥愣了愣，看着睡眼惺忪的何峰心下复杂，哪怕是下意识的一刺，都如此快狠准，看来何峰在狼牙战团的这段时日，进步的不止一星半点啊！
<br/>　　在声音刚响起的刹那，指挥间的参谋部成员就都接二连三的惊醒，也全体下意识的拔·出武器。
<br/>　　一时间指挥间的气氛就此冷凝，变得一触即发。
<br/>　　而然在他们睁开眼后，便集体呆住。
<br/>　　眼前这两人看起来好眼熟啊……
<br/>　　被惊醒的何峰望着眼前两人先是没反应过来，紧接着便吓了一跳，连忙起身朝着洛冥敬礼。
<br/>　　周围的其他参谋部成员也回过了神，稀里哗啦的敬礼。
<br/>　　他们的动作不整齐，甚至还有人因为没睡醒而不规范，这要是在平时少不了军法处的一顿责罚，但在此时此刻却让洛冥和段江伟都心绪复杂。
<br/>　　这场大战打的到底有多艰难，才会让这些参谋部的军官都累成这个样子？
<br/>　　洛冥先是沉默了良久，紧接着才开口道：“赢了？”
<br/>　　何峰顿了一秒后点头：“大胜。”
<br/>　　洛冥还想问什么，话到嘴边却咽了回去，道：“楚涵呢？”
<br/>　　何峰心中一紧，眼神在刹那间变换了好几次，道：“在狼牙。”
<br/>　　“恩？”洛冥有些惊讶：“他这时候怎么会在狼牙？”
<br/>　　何峰知道这件事瞒不过去，但碍于段江伟在场，便说道：“他受了伤，具体情况我也不清楚，三天前就送走了，眼下应该是尚九娣在照顾。”
<br/>　　“受伤？”段江伟从何峰的话语里听出了一丝不同寻常，忍不住反驳：“据我对楚涵的了解，别说受伤，哪怕思维都不清醒，都会战斗到最后，你说他三天前就被送到了狼牙基地，而这四天之内你却对他的情况毫不了解，说明这期间南署基地的战事，楚涵都没有过问过……”
<br/>　　何峰心神一紧：“你想说什么？”
<br/>　　段江伟眼神深邃：“楚涵不会放任这里如此重要的大战不管，除非……他昏迷不醒，根本管不了！”
<br/>　　话一出，何峰眼神瞬变，直视着段江伟一言不发。
<br/>　　洛冥却是一颗心顿时下沉，楚涵昏迷不醒，足足四天之久？！
