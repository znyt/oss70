　　所有人全都愣住了，
<br/>　　几乎都怀疑，是不是自己的耳朵出了问题？
<br/>　　这家伙，
<br/>　　居然要让十七公主当他的婢女！
<br/>　　他是疯了吗？
<br/>　　“你找死！”
<br/>　　那北山铁男，再也按捺不住，顿时大吼了一声，拔出腰间的一把巨剑，就朝叶尘狠狠劈了过来！
<br/>　　轰！
<br/>　　在北山铁男拔出巨剑之后，顿时剑气冲天，那巨剑，犹如泰山压顶般，朝叶尘砸落。
<br/>　　就连周围的围观者们，也被那惊人的剑气所惊，吓得连连倒退，个个脸色大变，不由得纷纷惊呼，
<br/>　　“据说北山公子今年才刚满三十岁，就已经凝血境八重境界的高手！”
<br/>　　“不愧是我罗刹城年轻一代中，最为杰出的天才啊！”
<br/>　　“这等天赋，莫说整个罗刹城，便是整个魔界，也是少有的天才！”
<br/>　　“这个狂妄的小子，竟然敢对十七公主不敬，死了也是活该！”
<br/>　　……
<br/>　　众人议论纷纷，显然没有人看好叶尘，可以从北山铁男的手下活命。
<br/>　　就连那十七公主，虽然觉得这个侍卫，有些与众不同，不过并不认为，他会是北山铁男的对手。
<br/>　　“这么有趣的一个人，倒是可惜了！”
<br/>　　十七公主小声嘀咕了一句，便准备转身离去。
<br/>　　然而，
<br/>　　就在这个时候，
<br/>　　接下来的一幕，却让她愣在了当场。
<br/>　　只见，
<br/>　　北山铁男的巨剑，斩落在那侍卫的头顶之上，非但未能将他劈成两半，反倒是北山铁男，连人带剑，直接倒飞了出去，重重栽倒在地上，然后狂喷了一口鲜血，显然已经受了不轻的内伤。
<br/>　　“什么！”
<br/>　　在场的众人，除了戴安娜事先已经猜到了结果，其他人全都万万没有想到，竟然是这么一个局面。
<br/>　　罗刹城年轻一辈的第一天才北山铁男，竟然败在了一名普通侍卫的手上，而且还是以如此诡异的方式！
<br/>　　“你……你是御空境高手？你到底是什么人？”
<br/>　　北山铁男捂着胸口，指着叶尘一脸惊骇道。
<br/>　　要知道，就连他们罗刹城的城主，也不过只是御空境巅峰而已。
<br/>　　放眼整个魔界，御空境绝对属于一流强者，而年轻一辈之中，能达到如此境界的，更是屈指可数。
<br/>　　眼前这小侍卫，看样子也不过才二十来岁，竟然有如此修为？
<br/>　　这不得不让北山铁男，重新审视叶尘的身份了。
<br/>　　叶尘嘴角微微一扬，不置可否，直接缓缓抬起手臂，冷冷道：
<br/>　　“我说过，再敢出言不逊，死！”
<br/>　　等到叶尘‘死’字落下，手掌也已经缓缓拍落。
<br/>　　轰！
<br/>　　一道惊人的金色巨掌，陡然出现在北山铁男的头顶上方，然后狠狠砸落下来！
<br/>　　“不！！你不能杀我！你不能……”
<br/>　　北山铁男的哀嚎声，戛然而止，整个人已经完全淹没在那道金色巨掌之下，顷刻化成了一滩肉泥！
<br/>　　“嘶~~”
<br/>　　所有人看都这一幕，纷纷倒吸了一口凉气，
<br/>　　“北山公子，竟然真的……被他杀了？”
<br/>　　“这家伙到底是什么人？真的只是一名普通侍卫吗？”
<br/>　　“刚刚那金色巨掌，是御空境强者才能具备的手段吧？”
<br/>　　……
<br/>　　众人早已经远远的躲开一旁，开始低声议论起来。
<br/>　　“大胆！你竟然敢杀了本公主的贴身亲卫，你可知道他是谁吗？”
<br/>　　十七公主从震惊中反应过来之后，顿时美眸一瞪，冲着叶尘娇喝道。
<br/>　　尽管她对于北山铁男的纠缠，一直非常不爽，可是他毕竟是自己的贴身亲卫。
<br/>　　打狗还得看主人呢！
<br/>　　更何况，北山铁男本身背景也不简单，乃是罗刹城四大将之首，北山家族的嫡子。
<br/>　　然而，
<br/>　　面对十七公主的质问，叶尘却连眼皮都没有抬一下，
<br/>　　“再敢废话，我连你一起杀！！”
<br/>　　“你！……”
<br/>　　十七公主顿时气得怒火冲天，尽管知道对方修为极高，可是她身为罗刹城的公主，何曾被人如此无礼对待过？
<br/>　　倒是戴安娜眼疾手快，当即一把拉住十七公主，低声道：
<br/>　　“公主殿下，不可冲动！此人不简单！”
<br/>　　十七公主顿时心头一凛，望着地上，那被拍成肉泥的北山铁男，下面的话，无论如何也说不出口了。
<br/>　　可是她堂堂公主之尊，若是就这么退缩了，又实在不甘心。
<br/>　　就在十七公主进退两难之时，忽然一道苍劲有力的声音响起，
<br/>　　“是什么人胆敢在我罗刹城闹事？”
<br/>　　声音落下，一名气势雄厚的中年男子，自城楼之上跃下，然后踏空而行，缓缓朝众人走了过来。
<br/>　　看到此人出现，十七公主顿时喜形于色，众人也纷纷惊呼起来，
<br/>　　“是东门统领来了！”
<br/>　　“东门统领乃是我罗刹城八大统领之一，也是御空境强者！”
<br/>　　“东门统领出手，定可以制服这个凶徒！”
<br/>　　……
<br/>　　之前被叶尘的手段所震慑的众人，顿时一个个又重新找回了自信，喜笑颜开。
<br/>　　“拜见公主殿下！”
<br/>　　那位被称作东门统领的中年男子，看到十七公主之后，当即走上前，拱手行了一礼。
<br/>　　“东门统领不必多礼！此人杀了北山铁男，而且还对本公主无礼，你快点给我出手拿下他！”
<br/>　　十七公主一下子找到了靠山，当即指着叶尘恨恨道。
<br/>　　“什么？北山公子死了？”
<br/>　　东门统领听到这话，顿时两眼一瞪，目光这才转移到叶尘的身上，脸色又浮现出一抹狐疑之色，
<br/>　　“区区一个低级侍卫，竟然可以杀死北山公子？让本统领来试试你！”
<br/>　　说完这话，
<br/>　　东门统领直接朝叶尘遥空一拳砸出，
<br/>　　轰！
<br/>　　一枚硕大的拳头虚影，朝叶尘呼啸而去，连同周围都犹如掀起一场飓风！
<br/>　　众人只感觉热浪滚滚袭来，不由得再次惊叹，
<br/>　　“不愧是御空境的强者啊！举手之间便能够操控天地之力！”
<br/>　　“两人都是御空境，这下有好戏好了！你们说谁会赢？”
<br/>　　“当然是东门统领！他可是我们罗刹城八大统领之一，那小子虽然也是御空境，但毕竟太年轻了！”
<br/>　　……
<br/>　　众人议论纷纷，就连戴安娜的眼眸之中，也露出一抹期待之色。
<br/>　　她和众人几乎也是相同的看法，只要东门统领打败这个人族男子，她也就不用再受制于人了。
<br/>　　然而接下来的一幕，却让她目瞪口呆。
