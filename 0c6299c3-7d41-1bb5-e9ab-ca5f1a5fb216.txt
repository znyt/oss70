<br/>“振邦哥，等一下！”众人刚离开宿舍，还没有出宿舍区，身后有人喊了起来。<br/>　　<br/>　　众人止步回首，发现有三个人疾步跑了过来，两男一女，不是别人，正是一百零八号平民小队中的林秋山和林婉儿兄妹，以及外号野狼的霍利。<br/>　　<br/>　　“你们三个这是……”李振邦疑惑的问道。<br/>　　<br/>　　“我们来是想问一下你们报名升班考核了吗？”林秋山追上来有些急切的问道。<br/>　　<br/>　　“我们还没有呢！正准备去报名！”李振邦似乎明白了他们的来意。<br/>　　<br/>　　“太好了，那你们是报名二年级还是三年级的考核？”林秋山有些紧张的问道。<br/>　　<br/>　　“我们准备直接报名三年级考核，你们不会也是准备来入伙的吧？”张大友窜了出来，嘿嘿笑着问道。<br/>　　<br/>　　“你们……已经有人了？”林秋山有些愕然的问道。<br/>　　<br/>　　“是啊！我们有人了，不过你们三个要是也参加三年级考核的话，我们倒是不介意再加上你们！不过也只能再加上三个人了，多了我也没办法了！”张大友耸了耸肩膀，摊开手，撇了撇嘴，一副无奈的样子。<br/>　　<br/>　　“真……真的？”林秋山激动的问道，他有些不敢相信，自己真的有机会，他一开始不过是抱着试试看的想法。以李振邦等人的实力，报名三年级升班考核基本是板上钉钉的事儿，他们离开小队也是因为他们准备报名三年级考核。<br/>　　<br/>　　“当然是真的，你没看到我们现在已经七个人了？考核队伍最多只能十个人，多了我也爱莫能助了！”张大友看了看对面的三人，脸上依然挂着微笑。<br/>　　<br/>　　这三个人的实力他们都是见识过的，想当初他们还是高级战士的时候，林秋山就已经是一名青铜战士了，就算他现在还是青铜战士，相比实力也不容小觑。<br/>　　<br/>　　妹妹林婉儿是一名金土双系魔法师学徒，现在想必也是一名一级魔法师了。双系魔法师，那可不是一加一等于二这么简单。<br/>　　<br/>　　野狼这小子可是让他佩服的人，光是战斗意志就有成为一名强者的可能，看这小子炯炯有神的眼睛，估计现在应该也是一名青铜战士了。<br/>　　<br/>　　以他们这样的实力完全可以三人组队参加三年级考核了，自己队伍里面加上这三个人不但不会降低队伍实力，反而会有一个很大的提高。<br/>　　<br/>　　看他们三人的意思，很显然是希望加入自己的队伍的，既然利大于弊，张大友自然也乐享其成。<br/>　　<br/>　　“我们就只有三个人，我们一定不会拖大家后腿的。”林婉儿听到张大友的话，有些激动的说道。<br/>　　<br/>　　野狼霍利并没有说话，只是冲着李振邦露出一丝微笑，李振邦冲着他点了点头作为回应。<br/>　　<br/>　　其实三个人完全没有必要加入李振邦的队伍，不过他们都知道李振邦是一名召唤兽法师，召唤兽法师想要完成升班考核，主要还需要团队的力量。他们相信以自己的实力，一定会对李振邦有所帮助的，算是对以前的事情报恩。<br/>　　<br/>　　这一次的团队空前的强大，满编十人的队伍，最低都是半只脚迈进青铜战士的高级战士。<br/>　　<br/>　　当十个人浩浩荡荡走进报名处的时候，也确实让人为之侧目。一般情况下，人数越少越容易通过，所以多数人都选择四五人左右组队进行升班考核。<br/>　　<br/>　　看着来到面前的是一个十人小队报名，负责登记的老师有些诧异，好心提醒道：“你们确定要十人一起参加考核吗？十个人的话考核难度会提高很多的，我建议你们最好还是分成两个队伍。”<br/>　　<br/>　　“谢谢老师，不过我们已经决定好了，十个人一起报名参加升班考核。”张大友身为舍长也是这个小队的队长，自然成为了这个小队的发言人。<br/>　　<br/>　　看着张大友坚定的眼神，负责登记的老师无奈的叹息着摇了摇头，说道：“十人小队，二年级升班考试。”<br/>　　<br/>　　“老师，不好意思，我们要报名的是三年级升班考试！”张大友赶忙纠正道。<br/>　　<br/>　　“什么？三年级升班考试？你不是逗我吧？一年级十人小队选择的竟然是参加三年级升班考试？”负责登记的老师惊呼道，他实在是不敢相信自己听到的。<br/>　　<br/>　　听到他的惊呼，登记处的所有老师和学生都惊讶的看了过来。老师们还好，不过是遗憾的摇了摇头，很快恢复了镇定继续登记。可是屋子里面的学生却都炸开锅了，纷纷和身边的人议论了起来。<br/>　　<br/>　　有几个认识他们的人，好心的过来劝导一番，不过显然都没有什么效果，最后只好不了了之了。不过大家看他们的眼神都各不相同有的人是同情、有的人是不忍、有的人是遗憾、有的人是幸灾乐祸。<br/>　　<br/>　　负责登记的老师显然意识到了自己的失误，赶忙降低了声音劝说道：“你们不要拿自己的前途开玩笑好不好？你这明显就是一个不可能完成的任务啊！”<br/>　　<br/>　　张大友一副信心十足的说道：“老师，你放心，如果我们的队伍还过不去的话，恐怕就不会再有队伍能过去了！”<br/>　　<br/>　　负责登记的老师虽然有劝说的权利，但是却没有强制的权利，所以在劝说无果的情况下，只好无奈的将众人的信息登记下来了。<br/>　　<br/>　　登记完成以后，负责登记的老师又一次被震惊了。十人的队伍里，只有两名高阶战士，其余的八人分别是四名青铜战士，一名水系一级魔法师，一名土系二级魔法师，还有一名金土双系的一级魔法师，当然李振邦的召唤兽法师被他自动无视了。<br/>　　<br/>　　负责登记的老师心中释然了，难怪这么有自信，这个队伍的实力确实是非同小可啊！如果分成两队的话，至少有一队是非常可能升班三年级成功的，另一队大概率失败的原因是因为有个召唤兽法师。<br/>　　<br/>　　登记完成以后，众人只要等待考核的那天来到就可以了。最重要的事情已经完成，众人心中也算是放下了一块石头。<br/>　　<br/>　　考核结束以后就是暑假了，众人已经开始计划暑假的规划了，仿佛他们已经升班考核成功了一般。<br/><br/><br/>
