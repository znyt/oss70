　　苏莫之所以想要斩杀巫玛和巫幽兰等人，并不是为了他们的储物戒，而是为了特殊体质。
<br/>　　这两人一人为元灵体，一人为琉璃宝体，虽然这两种体质不是什么逆天神体，但也是名列太古异体榜七百多名的体质。
<br/>　　所以，苏莫才不惜冒险，也要斩杀这两人。
<br/>　　只要吞噬了这两人的血脉，就能拥有两人的特殊体质，到时候他拥有四种特殊体质，天赋就能更上一层楼。
<br/>　　苏莫催动了吞噬战魂，开始吸纳神丹中的玄力。
<br/>　　他已经在巫族待了近一年的时间，虽然大部分时间本体都存放了起来，但当初在地灵塔修炼室的时候，本体一直在修炼。
<br/>　　所以，修为足够稳固，突破应该不成问题。
<br/>　　他没有直接吞噬神丹，只是通过吞噬之力，将神丹中的玄力吸纳。
<br/>　　十几枚神丹之中，玄力快速流逝，全部进入了苏莫的体内。
<br/>　　苏莫体内各色玄力狂涌，他早已运转了大混沌无形神诀，极力的炼化了起来。
<br/>　　大量的玄力被炼化，化为了无形玄力，汇聚到了苏莫的五枚神丹之中，他的修为开始了快速的增长。
<br/>　　没过多久，十几枚神丹中的玄力便消耗一空，神丹尽皆化为了齑粉。
<br/>　　苏莫再次取出了几十枚神丹，继续吸纳。
<br/>　　时间缓缓流逝，几个时辰之后，苏莫的修为达到了武帝境三重的巅峰，距武帝境四重只剩下一步之遥了。
<br/>　　这时候他停了下来，没有立刻冲击境界，而是沉寂了几个时辰，将暴涨的修为好好的稳定了一番。
<br/>　　随后，他一挥手，大量的神丹出现在了面前，足足数千枚武皇境的神丹。
<br/>　　苏莫洗劫各大小镇之时，虽然大多数的巫族之人，直接爆体而亡了，但也有大量实力强一些的武者，被他吞噬进了战魂之中。
<br/>　　所以，他的身上除了近千枚武帝境的神丹之外，武皇境的神丹不下数万枚。
<br/>　　轰！
<br/>　　苏莫释放出了吞噬战魂，将数千枚神丹直接吞噬了。
<br/>　　霎时之间，神丹中蕴含的无尽玄力，在战魂之中爆开，而后疯狂的冲进苏莫的体内。
<br/>　　“我去！”苏莫大惊失色，吞噬的太猛了，磅礴浩瀚的玄力，差点直接将他撑爆了。
<br/>　　他立刻极力运转大混沌五行神诀，拼命的炼化玄力，冲击武帝境四重境界。
<br/>　　轰隆隆！！
<br/>　　苏莫体内神丹剧烈震荡，风起云涌，如同火山沸腾，又犹如大海咆哮。
<br/>　　海量的玄力，给予了他突破境界足够的支撑，仅仅十几个呼吸的时间，就让他的境界瓶颈开始松动了起来。
<br/>　　咕咚！
<br/>　　不远处，聂乾和韩盈目瞪口呆，两人不约而同的吞咽了一口口水。
<br/>　　他们那里见过这么修炼的，不仅是能吞噬神丹，居然还一次性吞噬几千枚神丹。
<br/>　　两人差点以为，苏莫要爆体而亡了。
<br/>　　“真乃神人也！”聂乾呢喃自语，随即暗暗的摇了摇头，自从认识了苏莫，他感觉自己的人生观就要改变了。
<br/>　　“能认识苏兄，是我们之幸！”韩盈低声道。
<br/>　　“以苏兄的天赋，将来成为学府的圣子之一，都有可能！”聂乾唏嘘道。
<br/>　　“这我相信，而且，等将来学府的神子离开了学府，苏兄甚至有天赋冲击神子之位！”韩盈如是说道。
<br/>　　两人将苏莫的天赋和战力，均是看在眼中，战魂强大的过分，越级战斗如同吃饭喝水，不敢说将来冲击神子之位，圣子之位可能性非常的大。
<br/>　　两人低声闲聊，聊苏莫的天赋，聊自己活得的战利品，聊日后的修炼。
<br/>　　不知不觉，时间便过去了一个时辰。
<br/>　　就在此刻，一直在闭目修炼的苏莫，顿时身形一震，体内一股浩大的气息升腾而起。
<br/>　　而后，仅仅是半个呼吸的时间，一切回归平静，浩大的气息也尽皆返回了苏莫体内。
<br/>　　又过了片刻之后，苏莫缓缓睁开了眼眸，眸中精芒爆射十丈，璀璨夺目。
<br/>　　“恭喜苏兄突破境界！”
<br/>　　“恭喜苏兄突破境界！”
<br/>　　聂乾和韩盈见此，顿时来到了苏莫的身前，满面微笑的抱拳道喜。
<br/>　　两人心中感慨不已，刚认识苏莫之时，苏莫还只是准帝境界，这才多久的时间，就已经将他们完全甩开了。
<br/>　　这个修炼速度，当真天下无敌了。
<br/>　　“呵呵！两位客气了！”苏莫亦是满脸笑意，修为顺利突破，接下来的就好办多了。
<br/>　　“苏兄，你的天赋世所罕见，就算是在学府中也是最顶级的存在！”聂乾笑着说道。
<br/>　　“是啊！估计就算是神子，也不过如此吧！”韩盈赞同的说道。
<br/>　　“你们过奖了！”
<br/>　　苏莫摇了摇头，略作沉吟，道：“接下来的战斗，你们就不要现身了，我独自解决！”
<br/>　　“苏兄，你有几成把握？”韩盈和聂乾异口同声的问道，说到正事两人脸色变郑重了起来。
<br/>　　“八成以上吧！”苏莫自信说道，他并没有十足的把握，不过并不是没有十足的把握对付巫玛等人，而是大祭司。
<br/>　　此事，还需好好的斟酌一番，搞不好会有大麻烦。
<br/>　　“那就好！”韩盈两人闻言点了点头，只要苏莫有足够的把握就行，他们实力太低，也帮不上什么忙！
<br/>　　和两人随意聊了两句之后，苏莫便不再理会两人，独自低头沉思了起来。
<br/>　　他是准备做完这一票就离开了，但分身呢？
<br/>　　这最后一战，他有办法瞒过分身身上的少司令牌上大祭司的意念，但之后分身如何逃离？
<br/>　　分身一旦逃离，或者是丢弃少司令牌，估计立刻就会被大祭司发现。
<br/>　　但是，这个分身他却是不想舍弃。
<br/>　　思索了片刻，苏莫也没有什么好办法，便立刻沟通阎魔鬼圣，询问对方有没有好办法。
<br/>　　“苏莫，汝只能舍弃分身了！”阎魔鬼圣道。
<br/>　　“没有别的办法了吗？”苏莫不甘心的问道。
<br/>　　“有，但风险太大，一旦引起巫族大祭司的注意，天上地下再没有你逃生之地！”阎魔鬼圣郑重的说道。
<br/>　　苏莫叹了口气，随即重重的点了点头，为了安全起见，只能舍弃分身了。
<br/>　　到时候大祭司发现问题，也已经晚了，他已经逃离了天冥星。
<br/>　　而后，苏莫便不再多想，静静的巩固修为，等待分身将巫玛等一众巫族天才引来此地。
