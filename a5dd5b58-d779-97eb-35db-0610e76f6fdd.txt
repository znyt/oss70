        童筱颖心里顿时觉得委屈，但强忍着不让自己的眼泪掉下来，而是放慢语速，温柔地回答道：“昼景，对不起，让你担心了……”
<br/>
<br/>        其实，她就是不想让他担心她，她才不告诉他的啊！
<br/>
<br/>        结果，还是害他替她担心了。
<br/>
<br/>        “算了，你人没事就好。”听到筱颖那柔柔弱弱的声音，傅昼景的火气顿时就消了，缓了缓心情后，才和颜悦色地说，“你搬去我大哥的私宅里住吧！大哥私宅里很安全，而且还有佣人可以照料你的生活起居。”
<br/>
<br/>        “我……”童筱颖欲言又止，本想抗拒，但为了不让傅昼景再担心自己，最终还是弱弱地答应了，“嗯。”
<br/>
<br/>        “我还有事，挂了。”
<br/>
<br/>        “好，拜拜。”
<br/>
<br/>        “嘟——”
<br/>
<br/>        “昼景，你——”要早点回！
<br/>
<br/>        童筱颖接下来的话还未说完，傅昼景那边便已经挂了电话。
<br/>
<br/>        傅夜沉就站在卧室门边，看着童筱颖那多愁善感，娇弱又受了委屈的模样，心疼不已。
<br/>
<br/>        童筱颖放下手里的移动座机电话，转身走到衣柜前，打开衣柜，开始收拾行李。
<br/>
<br/>        她不想去大哥的私宅住，哪怕是有原因的，但这事若被人传出去了，怎么着都不成体统。
<br/>
<br/>        毕竟，她只是他的弟妹。
<br/>
<br/>        童筱颖忽然间想到了什么，一个转身，才发现大哥傅夜沉就站在卧室门口等她。
<br/>
<br/>        “大、大哥……”童筱颖柳眉微微蹙动，顿了顿后，接着问道，“漫雪她……也住在私宅里来了吗？”
<br/>
<br/>        “她不在滨江城，而是留在了临海城的傅家大庄园里，每天陪着我爷爷和我母亲。”傅夜沉淡淡地回答道。
<br/>
<br/>        最近，他有听他母亲高兴不已地在电话里跟他念叨，说他给她找了个懂事又上进还讨人喜欢的好媳妇。
<br/>
<br/>        苏漫雪住在傅家大庄园里的这段日子，参加完高考后，每天都没闲着。她不仅跟着城里的顶级大厨学习了烹饪，而且还跟着一个一门技术承传了百年之久的老师父学了口技。
<br/>
<br/>        现在的苏漫雪，不仅会下厨做美食讨家里那两位长辈的开心，而且还会模仿鸟兽虫鱼的叫声，甚至是男女老少的声音，和那老师父搭戏在幕后配合得惟妙惟肖，把傅老爷子逗得开心不已。
<br/>
<br/>        苏漫雪打得什么主意，傅夜沉心里一清二楚。
<br/>
<br/>        她是眼瞧着在他这里不受待见，又想着保住她未来的傅家大少奶奶的地位，她能不变着花招去讨家里那两位长辈的欢心吗？
<br/>
<br/>        傅夜沉警告过苏漫雪，只要她安分，不去骚扰筱颖，他便给她这个有名无实的傅大少爷的未婚妻身份。
<br/>
<br/>        童筱颖眼珠子转了转，抿了抿唇，别扭地恳求道：“大哥可不可以……跟漫雪说，我这当弟妹的想邀请大嫂来滨江城玩？”
<br/>
<br/>        傅夜沉知道筱颖此刻跟他提苏漫雪，是什么意思，无疑是为了避嫌。
<br/>
<br/>        苏漫雪若是来了滨江城，她童筱颖又住在他的私宅里，那么她苏漫雪必定也得住他的私宅。
<br/>
<br/>        傅夜沉很不乐意让苏漫雪住自己的私宅里，但此刻为了筱颖，他不得不收敛一下自己的我行我素。
<br/>
<br/>        童筱颖见傅夜沉不吭声，还以为是她自己太自以为是，正苦恼着要不要收回刚刚句话的时候，傅夜沉却闷闷地应了声：“好。”
<br/>
<br/>        “谢谢大哥！”童筱颖释然一笑，心里也跟着松了口气。
<br/>
<br/>        傅夜沉待童筱颖收拾完行李箱后，便主动帮她提箱子，带着她离开了家上了自己的车。
<br/>
<br/>        童筱颖坐在副驾驶座上，看着这条前往他傅夜沉的私宅的似曾相识的路，脑海里不经意间闪过一些不好的回忆画面。
<br/>
<br/>        要去傅夜沉的私宅，就必须路过一片名为“花园”的别墅区。
<br/>
<br/>        之前，苏漫雪在这片别墅区里买了一幢别墅，而且还是她童筱颖为她设计的室内装修。
<br/>
<br/>        童筱颖记得那天，她去了花园别墅区里，替苏漫雪的新别墅量完房出来后，天空下起了大雨。
<br/>
<br/>        她被淋成了落汤鸡，同时被车辆碾过水洼溅了一身泥水。
<br/>
<br/>        而那个时候，傅夜沉从一辆霸气的越野车上下来。
<br/>
<br/>        他二话不说，就搂住她热吻了起来。
<br/>
<br/>        在他得知她是他的弟妹之前，他对她做过许多令她费解的亲密举动。
<br/>
<br/>        他亲吻她，抚摸她，跟她说一些暧昧又霸道的话。
<br/>
<br/>        童筱颖对他的所作所为，总是恼羞成怒，难以理解。
<br/>
<br/>        现在，再看着正在专心致志地开车的傅夜沉，帅气依旧，高冷中又透着禁欲的气息，完全和那个时候对她百般热情的样子是天壤之别。
<br/>
<br/>        路过花园别墅区时，傅夜沉突然开口说道：“漫雪在这里买了一幢别墅，装修都已经搞好了，并且已经过户到你的名下，作为送给你和阿景的新婚贺礼。”
<br/>
<br/>        “诶？！”童筱颖惊愕。
<br/>
<br/>        苏漫雪会这么好心？！
<br/>
<br/>        当然，她想得一点都没错。
<br/>
<br/>        苏漫雪的这份“好心”，是被傅夜沉给“逼”的。
<br/>
<br/>        其实，这幢别墅，本就该属于她童筱颖。
<br/>
<br/>        不，更确切地来说，苏漫雪曾经所拥有的一切，本来都是童筱颖的。
<br/>
<br/>        如果不是苏漫雪从中作梗……
<br/>
<br/>        傅夜沉想到这里，握着方向盘的手，又紧了紧，剑眉微蹙，发泄着内心的不满与愤恨。
<br/>
<br/>        童筱颖缓过神来后，弱弱地说了声：“谢谢。”
<br/>
<br/>        到了傅夜沉的私宅，开门迎接他俩的是一位中年妇女。
<br/>
<br/>        童筱颖记得这个妇女，那次，还是她替她烘干的衣服。

<br/>        中年妇女一见到他俩，就欠身颔首，毕恭毕敬地唤了声：“大少爷、大少奶奶。”

<br/>        “她不是大少奶奶，以后你唤她大小姐便是。”傅夜沉脱了鞋，双脚一前一后，趿进了早已准备好摆在地上的男士拖鞋里。
<br/>
<br/>        “是，大少爷。大小姐，您好！”中年妇女微笑着看向傅夜沉身后的童筱颖，重新问候了一句。
<br/>
<br/>        童筱颖换了拖鞋，怔怔地回以一个灿烂的笑容，温柔地问道：“你好！我该怎么称呼你？”
<br/>
<br/>        “叫我刘婶便是。”刘婶和蔼可亲地回答道。
<br/>
<br/>        刘婶是傅夜沉的私人管家刘管家的姐姐，同样受过专业的管家服务训练，不仅会将家里打理的有条不紊，而且还对家主忠心耿耿。
<br/>
<br/>        以前，傅夜沉很少回私宅，所以刘婶一个星期来一次打扫一下卫生即可。
<br/>
<br/>        现在傅夜沉回滨江城接管傅氏集团，回私宅的次数也稍微多了些，刘婶为了方便伺候傅大少爷，便在这段时间里一直留在私宅里住下了。
<br/>
<br/>        “刘婶，你好！”童筱颖对刘婶非常尊敬，完全没把刘婶当佣人对待。
<br/>
<br/>        “大小姐的房间，我已经收拾干净了。我现在就带大小姐过去。”刘婶接着说道。

<br/>        当刘婶准备伸手去拿童筱颖腿边立着的行李箱时，童筱颖立即自己提起行李箱放在身前，微笑着谢道：“我自己提就好。”
<br/>
<br/>        傅夜沉见童筱颖跟刘婶客气，转身利索地夺走了童筱颖手中的行李箱，往走廊尽头走去。
<br/>
<br/>        刘婶看着傅夜沉这一举动，显些忍俊不禁。
<br/>
<br/>        这男人宠不宠女人，只要一个小细节，明眼人就能一清二楚。
<br/>
<br/>        他们回来之前，傅夜沉就已经跟刘婶打过电话了，让刘婶收拾他房间隔壁的那间卧室，腾出来给这位大小姐住。
<br/>
<br/>        只不过，刘婶觉得奇怪。
<br/>
<br/>        大少爷为什么要让她称呼大少奶奶为“大小姐”呢？
<br/>
<br/>        安置好童筱颖后，傅夜沉主动给傅家大庄园里的管家打了一通电话，让管家派人把苏漫雪接到他滨江城的私宅里来。
<br/>
<br/>        谁知，他这通电话打过去后没多久，他的母亲便回了一通电话过来，命令他亲自回去接苏漫雪。
