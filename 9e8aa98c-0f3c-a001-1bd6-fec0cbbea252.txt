
<br/>　　<br/>　　    两天之后，茅小草和王惊蛰回到了陇西，在临故地心头别有一番滋味，上一次来陇西他的节奏几乎是跟抢亲差不多，这一次，就化干戈为玉帛了，不管以前跟小草父母撕的有多不愉快，那他总得要面对一个事实就是，他和小草的感情已经被摆到面上来了。<br/>　　<br/>　　    到了陇西之后，王惊蛰买了些礼品，然后直奔李家大宅，他俩到门口的时候李兰荣和茅清水还有那老管家已经在门前等候了，礼数整的非常周到。<br/>　　<br/>　　    双方见面，王惊蛰初时略显尴尬，不过小草的父母反倒是好像当什么事都没有似的，完全没有什么不自在的地方。<br/>　　<br/>　　    “来了，惊蛰？路上挺累的吧，来，进屋先歇会吧，喝点热茶”李兰荣熟络的拉着他的手，挺埋怨的说道：“你看你，来就来呗，都是自己家客气什么，带东西干嘛呢，下回不许这样了哈”<br/>　　<br/>　　    “礼数不能缺啊，毕竟您和阿姨都是长辈”王惊蛰被拉的都有点不好意思了，讪笑着回了一句。<br/>　　<br/>　　    茅清水笑呵呵的点头说道：“一回生二回熟，下次再来就好了……”<br/>　　<br/>　　    小草在家里的这段时间，其实给李兰荣和茅清水都已经做好功课了，她的态度首先就是，自己肯定非王惊蛰不嫁了，再一个呢，王惊蛰的身份摆在那呢，找上这么一个女婿，肯定是陇西李氏的幸事，你们没有啥可拒绝的理由。<br/>　　<br/>　　    上一次王惊蛰上门提亲，再加上王冬至的横空出世，已经给小草的父母狠狠的上了一课，那就是有些事你不能看表面，得往了深了去看，王惊蛰看起来一穷二白啥也没有，可你架不住他是个潜力股吧？<br/>　　<br/>　　    李兰荣和茅清水也明白了一个道理，这个女婿不管从哪个角度来说，他们似乎都得认定了。<br/>　　<br/>　　    来到李家大宅，王惊蛰和李兰荣在客厅里喝茶闲聊，小草和茅清水去后面厨房准备饭菜。<br/>　　<br/>　　    “他这一进门，我和你爸要是点头了，那你俩的事可就彻底定下来了”茅清水洗着菜叶，忍不住的又问了一句：“草啊，你真是想好了？”<br/>　　<br/>　　    茅小草麻利的拾掇着一条鲤鱼，头也不抬的“嗯”了一声，说道：“我和他之前有些事你可能不知道，不过我可以告诉你的是，当一个男人什么都肯为你付出的时候，你要是还错过了他，那就是一辈子的惋惜了……妈，这么和你说吧，没认识惊蛰之前我不知道什么叫喜欢一个人，认识他后，如果再错过了，我更不会知道什么叫喜欢了，明白么？”<br/>　　<br/>　　    茅清水抿了抿嘴，叹了口气，也不再接着往下说什么了，她这女儿犟的，你不管用什么道理都是难以拉回头的。<br/>　　<br/>　　    客厅里，王惊蛰正襟危坐端着茶杯，李兰荣靠在椅子上，歪着身子问道：“惊蛰啊……”<br/>　　<br/>　　    “唉，您说”<br/>　　<br/>　　    “你和小草呢，感情稳定了，年岁也摆在这里了，这次认了门之后，这个双方家长？”李兰荣比划了下，笑呵呵的说道：“家长也得见见面了，谈谈细节么”<br/>　　<br/>　　    王惊蛰略微一懵，问道：“啥细节啊？”<br/>　　<br/>　　    李兰荣手指点着他，笑道：“装糊涂呢，当然是结婚的细节啊，你俩的事什么时候办，你家是什么意见啊”<br/>　　<br/>　　    王惊蛰当即就愣住了，心里顿时“咯噔”一下。<br/>　　<br/>　　    当年，王风饶和小草的爷爷为他定下这门婚事，无非是为了他三十岁那年的天堑埋下的一道伏笔，小草这女人命相贵气，福运深厚，属于旺夫那一类的女人，她还是李氏皇族的后代，娶了她的结果虽然是达不到逆天改命吧，可却也能为王惊蛰平添一缕运气。<br/>　　<br/>　　    想到过了那天谴，各方因素都得具备齐全了，这小草就是其中一个。<br/>　　<br/>　　    可真要说到娶她，王惊蛰就有点犹豫了，犹豫的关键就是自己万一真有一天过不去天谴的话，那小草岂不是要孤守空房了？<br/>　　<br/>　　    爱一个女人，这么干那就是坑她了，所以王惊蛰后来也想了，在自己没有八成左右的把握过了那道天谴的时候，他宁可不娶了。<br/>　　<br/>　　    所以，结婚这个事，此时是没办法提上日程来的。<br/>　　<br/>　　    王惊蛰沉默半晌，然后故作尴尬的挠了挠脑袋说道：“叔，我俩好上才不到一年，时间有点短，磨合磨合吧，二十来岁也不着急，是不？”<br/>　　<br/>　　    “呵呵，你这态度可以，嗯，不错，挺负责的……”<br/>　　<br/>　　    一个多小时后，饭菜上来。<br/>　　<br/>　　    席间，李兰荣一直拉着王惊蛰喝酒，但此人的酒量就属于那种，挺愿意喝，但是一喝就多的状态，基本上半斤白酒下肚，王惊蛰刚有点晚微醺脸色微红的时候，李兰荣就开始上头了。<br/>　　<br/>　　    小草在桌子底下，捅咕着王惊蛰，然后歪着身子小声说道：“那个什么，喝差不多就行了，你在让他喝的话，到时候就刹不住车了”<br/>　　<br/>　　    “哎呀，你没看出来么，我有心思浅尝辄止，但你爸明白还正在兴头上呢，你是不知道喝酒的人啊，上劲了的话，你要是拦着他，都容易跟你急眼”<br/>　　<br/>　　    “那你自己悠着点吧……”小草隐晦的点了他一句。<br/>　　<br/>　　    “唉，惊蛰，不是，你干什么呢，喝酒啊”李兰荣给王惊蛰面前的杯子满上，然后突然一把搂住他的肩膀，端着酒杯说道：“老弟啊……”<br/>　　<br/>　　    “哎呀我擦”王惊蛰当即就懵逼了。<br/>　　<br/>　　    李兰荣醉眼朦胧的说道：“大哥跟你说，上一次你来吧全是误会，也不能怪我们啊，李家是什么底子你也知道，这可是李氏皇族的后裔，可是呢到了我这一辈，底蕴全都糟蹋没了，大哥心里有苦但是大哥说不出啊，所以我才想着要和清河崔氏联姻的，就想借着崔家让我们陇西李氏在辉煌一下，在当皇帝那是不可能的了，可是做个大富之家的话，我这个想法有毛病么？大哥的心意，你明白吧？”<br/>　　<br/>　　    王惊蛰抹了把额头的冷汗，点头说道：“懂，懂”<br/>　　<br/>　　    “来，跟大哥把这杯干了……”<br/>　　<br/>　　    小草捂着脸，叹了口气，在桌子底下踩了王惊蛰一觉，瞪着眼睛说道：“都喝差辈了，你赶紧给我停下！”
