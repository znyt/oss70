顾夜瑾拎着袋子回到了西江月的别墅里，可是他没有找到叶翎的人。
<br/>
<br/>        整栋别墅还是他走前的样子，但是楼上楼下都找不到叶翎了。
<br/>
<br/>        “翎翎！翎翎！”
<br/>
<br/>        顾夜瑾翻遍了整个房间，连衣柜都没有放过，可是找不到人，精健的胸膛上下起伏，他清寒的眼眶里迅速布满了红血丝。
<br/>
<br/>        她去哪里了？
<br/>
<br/>        她是不是已经走了？
<br/>
<br/>        顾夜瑾心头涌上一股莫名的失落和恐慌，刚才她的样子很不正常，他不应该冷言冷语而是应该将她抱在怀里哄的。
<br/>
<br/>        他应该告诉她，她僵硬的模样让他舍不得去强迫她，还有跟着他，他不会让她吃药的，他有分寸。
<br/>
<br/>        他应该多跟她科普一下生理知识的，她很小就没有了妈咪，上学发了一本生理书，她害羞的将那本书压在了书包的最下面，不敢让他瞧见，他也没有教过她怎么样才会怀孕，刚才那样她是不会怀孕的。
<br/>
<br/>        顾夜瑾心头乱成一团麻，他拎着车钥匙就想车门去找她，但是这时他突然看到了她穿的那只粉色拖鞋。
<br/>
<br/>        有一只粉色拖鞋掉在了客厅的地毯上，往前看，落地窗的角落里好像坐着一个人，不过被拉起来的窗帘给挡住了，刚才他就没有看到。
<br/>
<br/>        顾夜瑾迅速走过去，伸手拉开了窗帘。
<br/>
<br/>        叶翎抱膝坐在松软的地毯上，蜷在角落里，她将小脸埋在膝盖里，一只脚上有鞋，另一只脚上的鞋掉了。
<br/>
<br/>        她静静的坐在这里，蜷着自己，茶色的卷发乖巧柔软的披散了下来，安静的她周身都萦绕着一股淡淡的难过和悲伤。
<br/>
<br/>        顾夜瑾觉得自己的心被一只大手给狠狠的揪住了，他伸手，轻轻的摸上了她的脑袋。
<br/>
<br/>        叶翎抬头，看向了他。
<br/>
<br/>        顾夜瑾以为她的脸上会有泪，但是她的脸上干干的，眼里也干干的，什么泪都没有，只不过她的瞳仁有点涣散。
<br/>
<br/>        很快，她将目光聚焦在了他的身上，“药买回来了？”
<br/>
<br/>        顾夜瑾嗓音发涩，点头道，“买回来了，多大的人了，怎么走路还会掉鞋？”
<br/>
<br/>        他蹲下身，将手里的粉色拖鞋穿在了她雪白的小脚丫上。
<br/>
<br/>        叶翎看着他，浅浅的勾了一下红唇，“我突然改变主意了，不想吃药了，把药扔了吧。”
<br/>
<br/>        刚才她坐在这里的时候突然想起一件事，流产的那一天，等她从昏迷中醒来的时候已经被送到医院了，早晨时分屋里的血已经渗透到了外面，别的租客发现了迅速打电话报了警，将她送到了医院。
<br/>
<br/>        她还记得有一个穿着白衣大褂的女医生拿一种很鄙夷的目光看着她，对她说道，“看着年纪这么小竟然这么不自爱，下面撕裂好些天了都不来医院处理，现在流产又留了一夜的血，子宫受到重创，以后你都不会再怀孕了。”
<br/>
<br/>        叶翎还记得那一天生日他将她弄得很疼，不知道什么叫撕裂了就被他一个耳光打了过来，又被顾夫人赶出了顾家。
<br/>
<br/>        她身边一个人都没有，腿上的疼痛让她羞耻，所以只能忍受，后来流产了，医生告诉她很多名词，比如撕裂，流产和再难受孕。
