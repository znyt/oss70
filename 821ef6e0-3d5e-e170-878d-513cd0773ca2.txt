　　“的确不凡！”苏莫眼眸闪亮，望着急速袭杀而来的硕大雷球。
<br/>　　虽然每一层空间，龙灵的实力只增加一成，但这一成的实力，却是巨大的暴涨。
<br/>　　这说明，龙灵巅峰十成的实力无比的强大，远非一般的同阶万古天骄可比。
<br/>　　唰！
<br/>　　苏莫的手中，剑光一闪，出现了一柄大剑，此剑只气息，骇人无比，恐怖的锋锐之气，粉碎虚空，真是终极神剑。
<br/>　　“灭！”
<br/>　　苏莫一剑斩出，一道犀利的混沌剑气，瞬间暴斩长空，直接便击中了硕大的雷球。
<br/>　　轰！
<br/>　　一声爆响，整动无尽的虚空，巨大的雷球爆开，化为万千细小的雷蛇，迸射八方。
<br/>　　不过，这一剑虽然斩灭了雷球，混沌剑气也同样崩碎，四散在无尽的虚空之中。
<br/>　　昂！
<br/>　　龙吟声震天，巨大的太虚雷龙，全身弥漫出无尽的雷光，整个身躯都化为了雷霆，划破虚空，径直向苏莫扑了过来。
<br/>　　“灭！”苏莫面无表情，再次出剑，不过，这一剑他动用了万化神剑元神的力量，一剑便将雷龙灭杀。
<br/>　　太虚雷龙消散，白色的光门再次出现，苏莫身形一闪，进入了光门之内，进入了第七层的空间之中。
<br/>　　第七层空间之中，出现的是紫血天龙，此龙长达五千丈，全身血红，和龙洌的本体一样，无比的霸气。
<br/>　　此龙的气势极为暴炸，铺天盖地，如同整个苍天镇压而下，恐怖的龙威浩荡诸天环宇。
<br/>　　不过，即便此龙极为不凡，在苏莫全力催动两种万化神剑元神的一剑之下，也只抗住了两招，便被彻底灭杀。
<br/>　　祖龙塔之外。
<br/>　　龙城和一帮龙族的年轻一代，并未离去，均是伫立在祖龙塔之下等待。
<br/>　　“看，第七层亮了起来！”
<br/>　　“居然这么快！”
<br/>　　“好快的速度，这才十个呼吸左右的时间吧！”
<br/>　　“不知道是谁闯到了第七层，是炎空还是屠霄？”
<br/>　　众人见到祖龙塔第七层的塔壁之上，那雕刻的神龙图案亮了起来，纷纷惊疑不已。
<br/>　　这说明已经有人进入了第七层，而进入祖龙塔之中的人虽然不少，但是，只有炎空和屠霄，有进入第七层的实力。
<br/>　　不过，众人虽然能确定，定然是炎空或者屠霄进入了第七层，但是却无法猜出，具体是谁先进入的第七层。
<br/>　　“应该是炎空，那炎空一身火属性道统，攻伐无比的凌厉，而屠霄虽然厉害，但是，只是防御和梦境厉害，在攻伐方面不如炎空！”龙城面色平静的说道。
<br/>　　虽然他心中很不希望炎空和屠霄，取得很好的成绩，但是，却不得不承认，无论是屠霄还是炎空，都有进入第七层的实力。
<br/>　　“的确有可能是炎空率先进入第七层！”众人点了点头，大都赞成这个猜测。
<br/>　　“屠霄也有一些可能性！”
<br/>　　一位龙族的年轻女子，面现沉吟之色，道：“屠霄天梦的能力，不知道对付龙灵效果如何，而且，屠霄如此天赋，绝对还有其它元神，不可能只有天梦元神。”
<br/>　　“也有道理！”不少人点头，赞成女子的想法，虽然炎空的可能性更大，但是屠霄也有可能。
<br/>　　“他们两人虽然厉害，但是，第七层就是他们的终点！”
<br/>　　龙城轻笑一声，继续说道：“他们是绝对不可能，进入祖龙塔第八……！”
<br/>　　龙城的声音不大，但是却是响彻全场，但是，他话音还未说完，却是突然卡壳了。
<br/>　　因为就在此刻，祖龙塔第八层塔壁之上，那神龙图案陡然亮了起来，虽然亮度不高，但是，在众人的眼中却是极为的璀璨。
<br/>　　一瞬间，所有人的目光，均是瞬间落在了第八层的塔壁之上，个个目瞪口呆，面现不敢置信之色。
<br/>　　“这……！”
<br/>　　龙城的面色，陡然一呆，目光怔怔的望着第八层的塔壁，少倾之后，才低声自语：“这怎么可能？”
<br/>　　他完全不能接受，居然真的闯到了第八层，是屠霄还是炎空，这天赋居然如此逆天，都堪比龙腾了吧！
<br/>　　“天啊，闯到第八层了！”
<br/>　　“第六个，我龙族第六位闯到第八层的人，虽然是外族之人！”
<br/>　　“居然如此逆天，到底是炎空还是屠霄！”
<br/>　　众人炸开了锅，闯入第八层非同小可，堪称是龙族第六人。
<br/>　　这一刻，众人心中纷纷好奇到了极致，到底是谁闯到了第八层，到底是炎空还是屠霄？
<br/>　　随即，立刻有一些人，给族中长辈或者是龙族高层人物传讯，通知此事。
<br/>　　此事在龙族可谓是大事，必须第一时间通知族中长辈高层，上次龙腾闯到第八层，龙族中的高层人物，除了龙帝没有亲自到场，其它的差不多都立刻亲自来到了祖龙塔，便可见此事的分量。
<br/>　　嗖嗖嗖！
<br/>　　而后，几乎是转瞬之间，便见到龙都城各方，有急速的破空声传来，很多龙族的强者，纷纷低空飞掠而来。
<br/>　　龙都城虽然不小，但是，对于龙族的那些八九品的强者，亦或者是神王级别的强者而言，依旧是转瞬极致。
<br/>　　唰唰唰！
<br/>　　几乎是两三个呼吸之间，便有数十名龙族执事和长老，来到了祖龙塔之前。
<br/>　　“谁闯到了第八层？”
<br/>　　“此子何人？”
<br/>　　“到底是何人？”
<br/>　　一众龙族强者，纷纷望着祖龙塔第八层，惊疑不定的询问。
<br/>　　“大长老，是炎空！”
<br/>　　“我觉得也可能是屠霄。”
<br/>　　“炎空或者屠霄。”
<br/>　　“应该是炎空。”
<br/>　　众人七嘴八舌的开口，言语各不相同，没有人能确定是谁。
<br/>　　一众龙族强者，闻言皱眉，随即，一名身材高大，长着金色胡须的金袍老者，看向面色阴沉的龙城，道：“龙城，你来说，是谁闯到了第八层？”
<br/>　　“禀大长老，炎空和屠霄同时进入了祖龙塔，不确定是谁？但肯定是他们之一！”龙城沉声说道，这位金袍老者乃是龙族大长老。
<br/>　　“嗯！”大长老闻言，立刻点了点头，心中知晓了具体的情况。
<br/>　　其它龙族的强者，也都心中恍然，原来是前段时间风头大盛的屠霄和炎空，没想到这两人居然如此逆天。
<br/>　　只是，这两人之中，闯到第八层的人，到底是哪一个呢？
<br/>　　“不会两人都闯到了第八层吧！”
<br/>　　不知道是何人，突然惊疑的开口说道，此言一出，众人纷纷心中一震。
