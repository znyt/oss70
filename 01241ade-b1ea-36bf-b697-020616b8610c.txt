　　苍穹世界各大势力，对于苍穹神宫和帝玄宫、天涯海阁结盟的反应，暂且不谈。
<br/>　　另一边，苏莫和夕儿离开帝玄宫之后，一边飞行，一边闲聊，逍遥自在。
<br/>　　对于结盟事宜，苏莫没有多管，他都已经和帝释商议妥当，天涯海阁那边，也由帝释派人联系。
<br/>　　所以，他只要静等大婚之日的到来便可以了。
<br/>　　“苏莫哥哥，你还记得你当初留下的字迹吗？”飞过帝灵城之时，夕儿瞥了一眼下方帝灵城高大的城墙，微笑着说道。
<br/>　　“当然！”苏莫同样瞥了一眼下方，他当初正是在帝灵城的城墙上，给上官昊下的战书，约定比斗的日期。
<br/>　　“苏莫哥哥，你当初对上上官昊，有几分把握啊！”夕儿好奇的问道。
<br/>　　“不确定！当时我并不清楚上官昊的实力！”苏莫摇头说道。
<br/>　　“当时真的好险！”夕儿想到当初的战况，心有余悸的说道。
<br/>　　两人闲聊之间，飞过了帝灵城上空，向苍穹神宫的方向而去。
<br/>　　帝灵城，一家客栈的顶层房间内，站立着几名黑袍身影。
<br/>　　几人站在房间的窗口处，目视苏莫和夕儿离去的背影，眸中精光熠熠。
<br/>　　“苏莫离开帝玄宫了！”
<br/>　　“大长老所料果然不差，苏莫果然来了帝玄宫！”
<br/>　　“立刻通知大长老，要在苏莫回到苍穹神宫之前进行截杀！”
<br/>　　“这次会有七大元老出手，苏莫插翅难逃！”
<br/>　　低沉的声音在房间中响起，声音中蕴含浓郁至极的杀机。
<br/>　　不错！黄泉魔宗早已在帝灵城安排了人手，时刻注意着苏莫的踪迹。
<br/>　　不仅如此，天涯海阁所在的天海城之内，黄泉魔宗同样派了人手。
<br/>　　上次二魔老和大长老阎泰，以及四大殿主齐齐出手，没有杀死苏莫，他们自然不会轻易放弃。
<br/>　　所以，黄泉魔宗料定，苏莫必然会来这两大势力，一直在守株待兔。
<br/>　　即便苏莫大肆诛灭黄泉魔宗的据点，黄泉魔宗都没有行动，一直在等待。
<br/>　　少顷之后，几名黑袍人匆匆的离开了客栈。
<br/>　　……
<br/>　　苏莫和夕儿飞行的不快，两人即将大婚，这一次旅途，就权当是外出游玩了。
<br/>　　两人一边游山玩水，一边悠然的赶路。
<br/>　　一路上，欢声笑语不断，夕儿的俏脸上时刻洋溢着醉人的笑容。
<br/>　　“苏莫哥哥，你看那处湖泊，此湖名为泪水湖，有一个非常凄美的爱情故事！”
<br/>　　飞行之中，夕儿玉手指着远处一片方圆近万里的湖泊说道。
<br/>　　“哦？什么爱情故事？”苏莫好奇的问道。
<br/>　　“传闻中，数千年前，此湖只有方圆千里大小，但其内却是出了一头女妖皇，名为花珊儿，其本体为花珊鱼，容颜倾城，国色天香！”
<br/>　　“这条花珊鱼和一名凡人男子相爱了，这名凡人没有武魂，无法修炼武道，自然也不能与她天长地久！”
<br/>　　“百年之后，那名凡人男子死了，花珊儿抱着他的遗体，在湖上痛哭了七天七夜，然后便自自缢而亡！”
<br/>　　“而花珊儿的泪水，却是让得泪水湖的水位暴涨，从一个方圆千里的小湖，变成了一个方圆近万里的大湖！”
<br/>　　夕儿面上带着惋惜之色，幽幽的说道，看其面色，这种凄美的爱情故事，仿佛对她心灵的冲击很大。
<br/>　　苏莫闻言瞥了夕儿一眼，心中莞尔。
<br/>　　看来无论是在这个武道世界，还是前世那种文明社会，夕儿这种年纪的姑娘，对于浪漫的爱情，都是心生向往！
<br/>　　接下来，夕儿的心情也受到了感染，不再嬉笑颜开，这让苏莫即好笑又无奈。
<br/>　　随后，两人便没有再过多闲聊，稍稍加快了速度赶路。
<br/>　　又过了两日，苏莫和夕儿飞到了一处层峦叠嶂的丘陵地带。
<br/>　　正在飞行之中的他们，突然看到前方数千里之外，一道光芒自大地上冲天而起，形成一道粗大的光柱，直刺苍穹。
<br/>　　光柱只是停留了一个呼吸的时间，便消失不见。
<br/>　　“那是什么？”苏莫心中疑惑。
<br/>　　“好像是某种异宝发出的异像！”夕儿道。
<br/>　　“异宝？”
<br/>　　苏莫闻言双眉一挑，道：“走！我们过去看看！”
<br/>　　随即，苏莫拉着夕儿的玉手，加快速度向前方飞去。
<br/>　　不多时，苏莫的视线中出现了一座方圆数十里的小镇，之前的光芒便是从小镇上发出。
<br/>　　不过，让得苏莫奇怪的是，这个小镇上死一般的寂静，不见半个人影。
<br/>　　“怎么回事？”苏莫心中疑惑，不过他艺高人胆大，没有丝毫担心，带着夕儿降临在了小镇的街道上。
<br/>　　苏莫本不想在此浪费时间，但刚才的光柱中，传出的劲力波动无比强大，显然不是普通的宝物。
<br/>　　所以，他也被勾起了好奇之心，想要查探清楚。
<br/>　　“苏莫哥哥，这个小镇好奇怪，好像弥漫着淡淡的血腥气息！”夕儿黛眉微撇，凝重的说道。
<br/>　　“嗯！我们小心点！”苏莫点了点头，他灵识释放而出，在整个小镇上扫荡了一圈。
<br/>　　随后，他便拉着夕儿，向前方的一座府邸走去。
<br/>　　在苏莫的灵识查看之下，前方的府邸中传出淡淡的灵气波动，显然异宝便在那座府邸中。
<br/>　　不过，那府邸好像极为不简单，居然能隔绝他的灵识查探。
<br/>　　少顷之后，苏莫走到了府邸的大门前。
<br/>　　这府邸不大，和当初林阳城苏家的府邸规模相仿，府门紧闭，不知其内有何物！
<br/>　　“苏莫哥哥，这里好诡异！”夕儿低声说道。
<br/>　　“的确诡异！”苏莫面色凝重，他自然发现了此地的诡异之处。
<br/>　　如此大的一个小镇，不仅没有半个人影，居然还有异宝出现。
<br/>　　而且，这异宝所在的府邸，居然隔绝他的灵识查探。
<br/>　　唰！
<br/>　　苏莫身形拔高，飞到了数十丈的高空中，目光从上方扫视整座府邸。
<br/>　　然而，府邸中一片迷蒙般的光景，空间仿佛扭曲了一般，根本看不清其内的状况。
<br/>　　就算苏莫催动幽冥魔瞳武魂，也是没有什么效果。
<br/>　　“奇怪！”苏莫皱眉望着眼前的府邸，随即身形再次落下，落在了夕儿的身边。
<br/>　　“苏莫哥哥，我感觉此地有莫名的危机，要不我们走吧！”
<br/>　　夕儿俏脸满含担忧之色，对于她来说，什么宝物都没有苏莫重要，苏莫的安危才是最主要的。
