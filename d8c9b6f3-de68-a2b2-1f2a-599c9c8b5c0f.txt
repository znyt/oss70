　　高空之上，巫殇的脸色难看不已，本以为斩杀苏莫不成问题，没想到苏莫居然如此的诡异。
<br/>　　他的死灵术很强大，其中还有几种变化，能展现出恐怖的威力，但现在连苏莫的踪迹都发现不了，死灵术再强大貌似也是鸡肋了。
<br/>　　时间流逝，时间一息一息的流逝。
<br/>　　骷髅人墙之外，虚无之中，苏莫淡淡的望着满脸戒备，防御的如同铁桶一般的巫殇。
<br/>　　对方上下左右，全部都被骷髅战士包围了，而且警惕性也很高，他没有偷袭的机会。
<br/>　　看来，只能先强势击破对方的骷髅墙防御，再伺机攻击了。
<br/>　　唰！
<br/>　　苏莫陡然出手了，手中剑光暴闪，瞬间连斩九剑，九道浩大无比的三色剑气，劈头盖脸的击向骷髅人墙。
<br/>　　轰隆隆！！
<br/>　　强大的剑气威不可挡，骷髅战士还未反应过来，便瞬间被斩灭一大片。
<br/>　　而后，破灭了一大片骷髅战士的剑气，余威依旧强大，直接向巫殇身上招呼而去。
<br/>　　“给我碎！”
<br/>　　剑气乍一出现，巫殇便发现了，他丝毫无惧，大手连连拍击，掌印翻飞，如同一座座小山一般，击向迎面而立的剑气。
<br/>　　轰轰轰！！
<br/>　　掌印和剑气碰撞，纷纷爆炸湮灭，化为汹涌的气浪，向八方涌动。
<br/>　　然后，击碎了苏莫的攻击之后，巫殇扫视四方，哪里还有苏莫的影子？
<br/>　　苏莫再次不见了！
<br/>　　“可恶！”巫殇心中怒急，这样的战斗，让他感到憋屈不已。
<br/>　　他的实力足够强大，但找不着苏莫，根本无处施展。
<br/>　　随即，他手中尊魂幡连连挥动，被斩碎的骷髅战士，立刻加快复原。
<br/>　　就在此时，巫殇心中一震，因为他感觉到了一股强大的吞噬力量，将他和一众骷髅战士全部笼罩。
<br/>　　在这股顿时力量之下，那些破碎的骷髅，重新复原的速度立刻一滞，而他体内本来急速运转的玄力，同样是猛然一滞。
<br/>　　唰！
<br/>　　蓦地，他头顶上方空间一阵波动，一道犀利无比的寒光，以无与伦比的速度洞穿了玄力气罩，直击他的天灵盖。
<br/>　　“不好！”巫殇心中一惊，来不及多想，他体内神丹猛然一震，立刻便压制住了吞噬之力的影响，手掌一探，直接抓向头顶的寒光。
<br/>　　他的手掌上带着灰色的宝器手套，所以无惧兵刃的攻击，敢用手直接抓取。
<br/>　　下一息，寒光和手掌碰触到了一起。
<br/>　　铛！
<br/>　　一声震耳的金铁交鸣之声响起，刺目的火光四处飞溅，猛烈的玄力瞬间炸裂。
<br/>　　寒光是一柄宝剑，被巫殇一把抓在了手中。
<br/>　　“哈哈！苏莫，这次看你还往哪里躲？”
<br/>　　巫殇大笑出声，虽然他依旧看不见苏莫，但能通过抓住的长剑，感知到苏莫正手握长剑，在他的头顶上方。
<br/>　　然后，巫殇话音一落，陡然全身汗毛炸起，一股极度危险的感觉瞬间侵袭全身。
<br/>　　“什么？”巫殇面色一变，来不及多想是什么危险，他急忙松开了手中抓住的长剑，身形急速暴退。
<br/>　　嗖！
<br/>　　巫殇身形急退，但还是晚了，三柄犀利无比的寒光，瞬间洞穿了他的身体，鲜血飚射长空。
<br/>　　啊！
<br/>　　巫殇口中发出一声惨叫，身形瞬间暴退千里，才停了下来。
<br/>　　“灵剑！”
<br/>　　抬头看向前方，巫殇脸色略显苍白，只见苏莫的身形显露而出，周身有三柄灵剑飞舞。
<br/>　　“巫殇，拿命来吧！”苏莫脸色冷漠，一步一步的向对方走去。
<br/>　　“可恶！”
<br/>　　巫殇脸色变换，此时，他的胸膛上正有三个血洞，鲜血潺潺而流。
<br/>　　这点伤势倒是不严重，但此刻，有大量的锋锐剑气进入了他的体内，在破坏他的肉身。
<br/>　　唰！
<br/>　　手中尊魂幡一挥，那些未被破碎的骷髅战士，顿时向苏莫飞扑而去，而那些破碎的骷髅战士也加速重组。
<br/>　　巫殇则趁着这个时间，立刻将冲进体内的剑气纷纷压制，湮灭。
<br/>　　但此时，苏莫的身影已经再次消失不见了，众多骷髅战士再次的失去了目标。
<br/>　　“这……！”巫殇简直要气的吐血了，这尼玛还怎么打？他完全没想到苏莫居然如此的难以对付。
<br/>　　一众观战的天才均是暗暗心惊，现在的战局，已经完全向苏莫倾斜了。
<br/>　　巫殇不仅无法奈何苏莫，居然还受伤了，强大的巫族禁忌巫术，没有半点作用。
<br/>　　真圣界外，早已一片哗然，议论纷纷。
<br/>　　“巫殇受伤了！”
<br/>　　“苏莫能隐藏身形，已经立于不败之地了！”
<br/>　　“看来巫殇要败了！”
<br/>　　“难怪那几大商会将苏莫排在真圣榜第一，看来也是有原因的，苏莫这个隐身的能力，就是杀手锏啊！”
<br/>　　“……”
<br/>　　嘈杂的声音此起彼伏，但这些话语听在巫族之人的耳中，却是格外的刺耳。
<br/>　　大祭司和一众巫族的天才，个个脸色难看，巫殇若是败了，则会让他们巫族颜面尽失。
<br/>　　但是他们丝毫没有办法，只能静观战局，希望巫殇能有办法对方隐身的苏莫。
<br/>　　大祭司暗叹一声，看来最终还是需要他出手了，巫殇怕是也很难击杀苏莫。
<br/>　　对于苏莫的隐身，大祭司因为身在真圣界外，观看的只是影像，也无法看穿苏莫的踪迹。
<br/>　　“本命剑客？”逍遥剑圣面带淡笑，刚才苏莫的灵剑，他一眼就能看出是本命灵剑，因为他也是一位本命剑客。
<br/>　　只不过，三柄本命灵剑对于武帝境的武者来说，实在是太少了。
<br/>　　真圣界内。
<br/>　　此时，巫殇伫立与高空上，面色急速变换，不想办法应对苏莫的偷袭，他基本上败多胜少。
<br/>　　心念电转，很快，他便有了办法。
<br/>　　“起！”
<br/>　　巫殇陡然大喝一声，全身玄力澎湃，疯狂涌动，他的体外再次的撑起了一个防御气罩。
<br/>　　这个防御气罩急速膨胀，几乎是一瞬间，便暴涨到了方圆百里的大小。
<br/>　　巨大的防御气罩，如同是一个透明的气球一般，悬浮于高空之上。
<br/>　　“苏莫，我看你还怎么偷袭？赶快现身吧！”
<br/>　　撑起了防御气罩，巫殇立刻朗声大喝，他这个办法虽然笨，但却是非常的有效。
<br/>　　有方圆百里的防御气罩守护，苏莫再攻击他的话，他就能有足够的时间来应对了。
