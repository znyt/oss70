        “小溪，要不算了吧？”张蓉蓉不禁安慰起朱小溪来。
<br/>
<br/>        朱小溪一脸不甘心地撅起了嘴巴，回去的路上，跟好友张蓉蓉抱怨起来：“你说她连星儿到底什么来头啊？她只不过是个转校生！一来就抢走了我的全年级第一，这次，不管是老师还是校领导，竟然全都保荐她去美国的HF大学！这些也就算了，学校里的那个大姐大不是很牛逼吗？不是很会整人吗？怎么找了她连星儿一次麻烦后就再也不敢去找第二次了？更可恶的是……”
<br/>
<br/>        说到这儿，朱小溪不禁蹲下身去，捂着脸哭了起来。
<br/>
<br/>        她高中暗恋了三年的班长，高考后好不容易鼓起勇气去表个白，结果人家喜欢的居然是连星儿！！
<br/>
<br/>        朱小溪气不过，于是这次买了那种药，想在今天毁了连星儿，哪知道，最后是这种结果。
<br/>
<br/>        张蓉蓉随之蹲下身去，一边轻轻地拍抚着朱小溪的背脊，一边安慰道：“反正我们都已经毕业了，以后各奔东西了，就不会再见她连星儿了。小溪，你就别伤心了。”
<br/>
<br/>        朱小溪吸了吸鼻子，抹去眼泪站了起来，碎碎念了一句：“也对！反正她连星儿是要去美国念大学了。以后，我们就跟她划清界限。”
<br/>
<br/>        而当初，是她主动要跟连星儿成为朋友的。
<br/>
<br/>        张蓉蓉苦笑着点了点头。
<br/>
<br/>        网咖里，连星儿戴上耳麦后，发现师父已经上线了，立即跟师父在线“吐槽”了一番。
<br/>
<br/>        师父，我今天真衰！遇上一个超没素质的面瘫男！——Star大小姐。
<br/>
<br/>        “Star大小姐”是连星儿的游戏名，也是她的网名。
<br/>
<br/>        好巧，我今天也遇上了一个超蠢乎的傻瓜女。——师父。
<br/>
<br/>        哎，没办法，这世上的极品太多了！——Star大小姐。
<br/>
<br/>        不闲聊了，要开赛了。——师父。
<br/>
<br/>        好，我已经准备好了！——Star大小姐。
<br/>
<br/>        连星儿在连打了五轮战队赛，终于累趴在电脑桌前，就这样枕着臂弯睡着了。
<br/>
<br/>        她一个人在外，其实一点都不害怕会遇上什么不安全的事情。
<br/>
<br/>        因为，她早就知道，爹地派了十个“高手”在暗中轮流保护她。
<br/>
<br/>        好在，那些高手只是负责她的人身安全，不打她私生活上的小报告，不然，她这偷偷上网打通宵游戏的事情要是被爹地知道了，只怕这家网咖都会被她家爹地拆了去。
<br/>
<br/>        没过一会儿，那张“闲人免入”的门开了。
<br/>
<br/>        男生从里面走出来后，突然在连星儿身后驻足了一会儿。
<br/>
<br/>        连星儿的电脑荧幕上游戏还未退出，聊天框也还开着。
<br/>
<br/>        不经意间，男生的嘴角微微上扬，然后收回目光往收银台那边走去。
<br/>
<br/>        “老板，这就回去了吗？”年轻的网管男看到男生走过来，立即起身问候。
<br/>
<br/>        男生双手抄着裤袋，淡定地说道：“三十二号机的那个女生，你关照一下。并吩咐下去，谁都不许打她的主意。”
<br/>
<br/>        “难道，她是我们的老板娘？！”网管男不禁贼贼地打趣道。
<br/>
<br/>        男生扯了扯嘴角：“不，她只是我的一个傻乎乎的徒弟而已。”
<br/>
<br/>        “好嘞！我懂了！”网管男咧嘴一笑。
<br/>
<br/>        男生随之再次瞥了那边的连星儿一眼，意味深长地笑了笑后，只身离开了网咖。
<br/>
<br/>        连星儿不知道自己睡了多久，只知道自己是被“母亲大人”的专属铃声给吵醒的。
<br/>
<br/>        “喂，妈咪——”连星儿拿起放置在电脑桌上的手机，按下接听键后，懒洋洋地唤了声。
<br/>
<br/>        此时，她的双手已经麻得像是有千万只蚂蚁在撕咬。所以，她不得不用肩膀和耳朵夹着手机接听电话。
<br/>
<br/>        “臭丫头，现在是凌晨六点，你若是不赶在七点前回家吃早饭。我想，你爹地大概或许会亲自去‘抓’你回来。”童筱颖故作严肃地说道。
<br/>
<br/>        连星儿顿时睡意全无：“妈咪，你没跟爹地说我有同学毕业聚会活动吗？”
<br/>
<br/>        “说了啊！但是你爹地还是不放心你。”童筱颖笑道。
<br/>
<br/>        连星儿立即坐直了身子：“那我马上回来！”
<br/>
<br/>        “早餐想吃什么？”童筱颖温和地问道。
<br/>
<br/>        连星儿想都没想地脱口而出：“什锦蔬菜沙拉！还有法棍面包和奶油蘑菇浓汤！”
<br/>
<br/>        “好，回家路上注意安全。”童筱颖叮嘱道。
<br/>
<br/>        连星儿和妈咪通完电话后，收拾了一下自己的背包，起身往收银台走去。
<br/>
<br/>        网管男这才仔仔细细地打量起连星儿来，毕竟，这个女孩，不仅是他们老板唯一的徒弟，而且还是他们老板第一次开口说要关照的第一个异性。
<br/>
<br/>        这小姑娘倒是长得挺漂亮的，瓜子脸，大眼睛，樱桃嘴，扎着清爽的高马尾，看上去活泼又靓丽，怪不得能入他们老板的眼。
