　　这刺客说完这一句话后，他的身躯便是猛然破碎开来，当场粉碎，在楚风眠的灵力压迫之下，这刺客居然都足以自毁。
<br/>　　这绝对是一种极为高深的秘术，可以在任何时候，都足以毁灭自身。
<br/>　　楚风眠甚至都没有来得及搜查这刺客的魂魄，他便是急忙自毁了。
<br/>　　“可惜了。”
<br/>　　楚风眠摇了摇头。
<br/>　　他也没有想到，这刺客落入到了他的手中，居然是还有着足以自毁的能力，不过这刺客既然能够从楚风眠的手中自毁，那他随时都可以做到。
<br/>　　他如今说出这一段话，也是故意告诉楚风眠等人的。
<br/>　　似乎是在宣告者，六道魔门的出现。
<br/>　　“六道魔门？这六道魔门，居然是再度出现了。”
<br/>　　现在楚风眠的心思，却是全部开始思考着这刺客所说的话。
<br/>　　听到这个名字的一刻。
<br/>　　楚风眠的眉头都是不由一皱，眼神中都是露出了几分忌惮。
<br/>　　六道魔门，这绝对是在数万年前的九域，可以说每个人听到这里名字，都会无比的忌惮，人人听到这个名字，几乎都要胆战心惊。
<br/>　　大约在三四万年前，这六道魔门，便是当初九域之中最为庞大的宗门，可以说是比起现在七大宗门加起来，还要恐怖的庞然大物。
<br/>　　当时的六道魔门，可以说是笼罩着整个九域，那个时代，魔头无数，人人修魔，整个九域都是魔道昌盛，一直是到后来，六道魔门的毁灭，才让九域也恢复成了如今的样子来。
<br/>　　如今这个时代，虽然关于六道魔门的记载已经很少了，可是这个名字却是无数次的从古籍之中记载，被楚风眠所看见过的，可以说若是真正六道魔门归来的话，从新夺取整个九域，都不是不可能的一件事。
<br/>　　可是六道魔门，却是已经在九域之中消失了数万年了，六道魔门的传承，虽然一直以来可能还传承了下去，可是却早就已经是跟曾经繁盛时期，无法相提并论了。
<br/>　　之前楚风眠在虚空海之中见到过的洛汐修炼的功法，正是六道魔门之一魅魔宗的功法。
<br/>　　所以这六道魔门，虽然是在九域之中已经消失了，但是他们的道统，的确是已经传承下来了不错。
<br/>　　百足之虫死而不僵，六道魔门的道统传承了下来，倒是没有多少令楚风眠吃惊的地方，像是云羽天宫，比起六道魔门还要古老的宗门，如今也一样是有着道统可以传承下来。
<br/>　　不过如今这六道魔门，居然是突然从九域之中大肆崛起出来，倒是让楚风眠的意料之外，而且看起来，这九魔城的分裂，也只是跟六道魔门有关。
<br/>　　本来楚风眠便是想到了，九魔城的分裂，绝对不可能是没有任何理由的，可是没想到现在，居然是跟六道魔门扯上了关系，这倒是令楚风眠便是更加忌惮了。
<br/>　　如今这个时代的武者，也许不知道当初六道魔门的强大，可是楚风眠却是清楚无比，楚风眠的师尊，剑道之主，便是就在那个时代之中出生的，剑道之主经历了六道魔门的强大，也给楚风眠讲述过。
<br/>　　在那个时代，天下九域，便是以六道魔门为尊，无人可以反抗，后来六道魔门的毁灭，也是因为他自身的原因。
<br/>　　六道魔门，其实并非是单单一个整体，而是其中，足足有着六个宗门。
<br/>　　魅魔宗，战魔宗，独魔宗，阎魔宗，天魔宗，人魔宗，这六大宗门，一起组成了六道魔门，这任何一个宗门拿出来，都要比起现在的九魔城还要强大的多。
<br/>　　当时正是因为这六大魔门，产生分歧，才让人寻找到了机会，毁灭了六道魔门。
<br/>　　若是没有这个冲突的话，可能如今这个时代，还是由六道魔门为尊。
<br/>　　只要是任何一件事，跟六道魔门牵扯到了关系，就绝非是一件小事。
<br/>　　“六道魔门？！”
<br/>　　古家家主的眼神，都是目光闪烁，以他的阅历，也听说过这数万年前，传说中的宗门，眼神中露出深深的忌惮。
<br/>　　不管是从任何一本古籍之中的记载，都将当时六道魔门的强大，淋漓尽致的展现了出来，只要是听说过这个名字的武者，都会知道六道魔门的强大。
<br/>　　“什么六道魔门？”
<br/>　　古锦还有那些年轻人之中，倒是对着名字，没有什么敏感的，一个个都是疑惑起来。
<br/>　　“这件事，谁都不许说出去，都散了吧。”
<br/>　　古家家主目光闪烁，便是大手一挥，让在场围观众人，全部散去了。
<br/>　　他的身形一闪，突然之间便是从原地消失了。
<br/>　　“我们也走吧。”
<br/>　　楚风眠看着古锦，开口道。
<br/>　　古锦现在的脸色，还有些慌张，毕竟刚刚经历过了这种刺杀，若是没有楚风眠出手的话，可能她刚刚就已经彻底陨落了。
<br/>　　“是。”
<br/>　　古锦点了点头，也没有在问什么，既然楚风眠跟古家家主，都没有主动说明，那她现在去问，也不过是在自讨苦吃罢了。
<br/>　　古锦将楚风眠，带到了一处宫殿之中，便是缓缓的退了出去。
<br/>　　“楚前辈可以在此宫殿之中休息，若是有事，可以吩咐下人，小女即刻到来。”
<br/>　　看到楚风眠点了点头，古锦便是身躯一动，离开了宫殿之中。
<br/>　　六道魔门，不知道这次六道魔门再度出现，到底是福是祸啊。
<br/>　　楚风眠心中暗自想到，这倒是在他计划之外的事情了，看起来这九域，也不会如此平静了。
<br/>　　在这九域平静的背后，已经是暗潮涌动起来。
<br/>　　与此同时，在宁州古城之外的虚空之中，一处昏暗的宫殿，便是似乎从这虚空之中，被硬生生的开辟了出来。
<br/>　　在这宫殿之中，站着不少的武者，每个人都是一身黑衣，坐在其中最高处的，却是一名身材妙曼的年轻女子。
<br/>　　虽然看不清容貌，可是单单这身影，令人看见，都是会为之着迷。
<br/>　　“什么？王炎那个废物，居然是刺杀古家二小姐的计划，都失败了。”
<br/>　　宫殿之中，一名武者，突然开口道。
<br/>　　“刺杀一个区区生死境八重的小家伙，居然都失败了。”
