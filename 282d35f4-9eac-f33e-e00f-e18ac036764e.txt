　　轰隆隆！！
<br/>　　玄力大手和紫色指芒纷纷爆炸，化为汹涌的狂涛，席卷八方，逆冲九天。
<br/>　　下方的死亡之海，遭到气浪狂涛的冲击，大量的海水蒸发成了虚无，但更多的是向四方狂涌，形成万丈之高的滔天巨浪，排山倒海一般涌动。
<br/>　　这顿时让整个死亡之海，都暴动了起来。
<br/>　　啊！
<br/>　　一声惨叫声响起，巫森虽然极力躲避，但还是被气浪波及，顿时口喷鲜血。
<br/>　　不过，巫森怎么说也是武帝强者，还不至于被余波杀死。
<br/>　　瞬息之间，他便逃到了巫天御等人的身边。
<br/>　　苏莫见此，顿时眉头一皱，不过，他也没有再贸然出手，那阻拦他的巫天御，让他感觉到了一丝压力。
<br/>　　此人极强，实力深不可测！
<br/>　　“你没事吧？”巫天御打量着巫森，皱眉问道。
<br/>　　“没事！”巫森抹了一把嘴角的血迹，微微的摇了摇头，他脸色苍白，心中骇然不已，一丝余波就让他身受重伤。
<br/>　　面前的这名巫族青年，和苏莫的实力，居然都如此的强大。
<br/>　　面前的这名巫族青年强大，巫森不意外，毕竟外面的大世界，强大的武学、宝物不计其数，而能来试炼的人，都是一等一的巫族天才，根本不是苍穹世界可比。
<br/>　　但苏莫的实力，着实让他震惊了，几年时间，达到此等地步，即便他有心里准备，也不免震撼。
<br/>　　这几年，巫森由于肉身毁灭，重新夺舍，实力不仅没有半点增长，甚至还有所倒退。
<br/>　　他和苏莫的差距，已经拉的非常大了。
<br/>　　“你就是祖爷爷的后人？”巫娴仔细打量着巫森，面带惊奇之色。
<br/>　　在孱弱的苍穹世界，能修炼道武帝之境，倒是让人颇感惊奇。
<br/>　　“祖父巫明寂！”巫森沉声说道。
<br/>　　巫森的祖父，乃是千年前的试炼者中，天赋一等一的天才，在苍穹世界试炼之时留下了子嗣。
<br/>　　只不过，当时为了击杀几名苍穹世界的巅峰强者，而耽搁了大量的时间，最后来不及带走其祖母。
<br/>　　其祖母生下了其父，只不过其父中年陨落。
<br/>　　“看来不会错了！”巫娴点了点头，有巫族的族印，有知道大祭司的名讳，那就绝对不会错了！
<br/>　　按照辈分而言，巫森还是巫娴的长辈。
<br/>　　“苍穹世界，只有你一位族人？”巫娴再次问道。
<br/>　　“我还有一子，以及一些下属，目前都在空间宝器之中！”巫森不敢隐瞒，老老实实的说道。
<br/>　　巫娴等人点头，既然没有其他族人了，他们也就省心了，接下来只剩下大开杀戒了！
<br/>　　“你退后吧！待我们扫荡了苍穹世界，自会带你离开！”巫天御摆了摆手道。
<br/>　　“是！”巫森点了点头，便退到了巫天御的身后，他现在心中大定，一切都无忧了，现在只要坐看苏莫如何死就行了！
<br/>　　此时，一直在静观其变的苏莫，眸中却是精芒闪烁，他的目光落在了巫娴的身上。
<br/>　　“祖爷爷？”苏莫心中呢喃自语，看来这名女子身份应该不低。
<br/>　　其祖爷爷这种辈分，很可能是巫族的武尊境强者，若是擒拿了此女，应该能让通道外守候的巫族尊者投鼠忌器。
<br/>　　苏莫心中暗暗打定了注意，将巫娴作为了首要目标。
<br/>　　当然，若是有机会，他也会擒拿巫森，毕竟宏青璇和李峰还在对方的手中。
<br/>　　“土著，你已经激怒我了，我要亲手将你诛杀！”
<br/>　　这时，巫天御遥望苏莫，眸中杀机凌厉，朗声大喝。
<br/>　　唰！
<br/>　　下一刻，一杆紫色的长枪，出现了在巫天御的手中。
<br/>　　这杆长枪长约八尺，通体紫光流转，气息滔天，极为恐怖。
<br/>　　即便巫天御没有注入玄力，枪尖上吞吐的枪芒，都洞穿了层层虚空，将下发的无尽海水洞穿。
<br/>　　“尊……尊器？”海明江、究楚北等老一辈强者，见到巫天御手中出现的长枪，顿时面色大变，惊呼出声。
<br/>　　因为这杆枪，给他们的感觉太强大了，那弥漫的气息，那凌厉的锋芒，远超帝级神兵。
<br/>　　那就只有一个可能了，这是尊器，尊者之器！
<br/>　　苏莫见此，亦是眉头一皱，心中有些惊讶。
<br/>　　此枪比他的帝皇剑气息要强大的多，比他的十亿魔剑也强不少，估计真是尊器！
<br/>　　“一群井底之蛙！”巫天御见此，顿时嗤笑一声，一件尊器而已，居然就让这群土著震惊不已，也是够奇葩的！
<br/>　　“哈哈，一件尊器而已，你看把他们吓的！”
<br/>　　“土著就是土著，没见过世面！”
<br/>　　“天御族兄一人，便可轻易屠杀他们了，我等都无需出手了！”
<br/>　　一众巫族弟子哈哈大笑，满脸的傲然得意之色。
<br/>　　尊器，虽然也算珍贵，但在他们巫族，只要是天才子弟，修为达到高阶武皇之后，都能随意领取。
<br/>　　所以，尊器对于他们来说，根本不值一提。
<br/>　　面对苏莫等人，无论是巫天御还是其他巫族天才，无一例外，每一个人心中都充满了优越感。
<br/>　　这种优越感与生俱来，充斥在他们的心间。
<br/>　　他们是堂堂巫族的天才，而苏莫等人，不过是他们巫族圈养的小世界中的土著，是他们猎杀的猎物。
<br/>　　这种差距，就仿佛是高高在上的帝王，在面对卑微的奴隶一般。
<br/>　　海明江、司空炎等人脸色早已铁青，这些巫族天才张口闭口就是‘土著’，那傲慢的姿态、那满满的优越感，早已让他们怒火中烧。
<br/>　　简直是欺人太甚！
<br/>　　若不是苏莫一直没动手，他们都忍不住要出手了。
<br/>　　此时，巫森亦是眉头皱了起来，这些巫族的天才，虽然个个实力强大，但都是毛头小子，居然如此的轻敌！
<br/>　　他可是知道苏莫的恐怖，武皇境四重修为之时，便能打败他，如今提升到了武皇境八重的巅峰，实力早已逆天了。
<br/>　　唰！
<br/>　　身形一闪，巫森便又再次来到了巫天御的身边，沉声告诫道：“小友切莫轻敌，此子名为苏莫，战力极为逆天！”
<br/>　　“哦？有多逆天？”巫天御闻言，嘴角勾起一丝笑意，面上露出了一丝感兴趣之色。
<br/>　　他也不急着出手，反正有测魂盘在手，没有人能逃得掉。
