　　“苏莫，两年了，紫霄神王回来了！”
<br/>　　绫裳的声音，突然在苏莫的耳畔响起，轻灵而柔和。
<br/>　　“回来了？”苏莫闻言，顿时心中一喜，等了两年，紫霄神王终于回来了。
<br/>　　随即，他立刻结束了修炼，离开了神剑山的内部，再次来到了绫裳的塔楼之前。
<br/>　　当苏莫来到塔楼之时，绫裳也走出了塔楼。
<br/>　　“走吧，我带你去见紫霄神王！”绫裳招呼苏莫一声，便向绫裳剑宫之外走去，苏莫急忙跟着对方身后。
<br/>　　“见过宫主！”
<br/>　　“见过宫主！”
<br/>　　一路上，一些绫裳剑宫的弟子，见到绫裳纷纷行礼。
<br/>　　有些弟子，甚至修为远高于绫裳，但依旧恭敬无比。
<br/>　　要知道，绫裳剑宫的存在，就是因为绫裳，其在绫裳剑宫拥有绝对的威望。
<br/>　　很多修为强大的弟子，甚至是绫裳剑宫的执事，都是绫裳当年教导而出。
<br/>　　所以，即便是九品真神，也对绫裳恭敬无比，不敢有丝毫不敬。
<br/>　　而不会因为绫裳修为未复，有所轻视。
<br/>　　当然，最主要的是，绫裳本是终极神王殿殿主的弟子，有这层关系在，无人敢轻视。
<br/>　　很快，绫裳便带着苏莫，走出了绫裳剑宫，来到了紫霄剑宫。
<br/>　　紫霄剑宫，明显比绫裳剑宫繁华的多，弟子也要更多。
<br/>　　一路无人阻拦，苏莫和绫裳来到了一座恢弘的宫殿门前。
<br/>　　宫殿大气，金碧辉煌，显得尊贵无比。
<br/>　　“紫霄！”
<br/>　　绫裳伫立在宫殿门前，轻轻的呼喊了一声。
<br/>　　少倾，宫殿的大门打开了，一名紫衣男子伫立在门内。
<br/>　　这名男子，看起来二十八九岁许，身材高大而挺拔，五官英挺、浓眉大眼，面目显得英俊而刚毅。
<br/>　　此人穿着一身紫色长袍，只是站在那里，自由一种独特的气质。
<br/>　　“绫裳，你怎么来了？”紫袍男子面带微笑，目光直视着绫裳。
<br/>　　“紫霄，我找你有事！”绫裳沉声说道。
<br/>　　“呵呵，请进！”紫霄轻笑一声，随即招呼绫裳进入殿中。
<br/>　　紫霄至始至终，都没有关注苏莫，一直望着绫裳。
<br/>　　绫裳点了点头，便走进了大殿，苏莫也紧跟着对方走了进去。
<br/>　　走进大殿，殿中是一个正厅，绫裳也没有落座，便开门见山的说出了来意。
<br/>　　“紫霄，我需要你帮一个忙！”绫裳沉声道。
<br/>　　“什么忙值得你亲自过来？”
<br/>　　紫霄面带微笑，走到了绫裳的身前，眸中露出好奇之色。
<br/>　　“我需要你帮我救一个人！”绫裳轻声道，言罢，她向苏莫微微示意。
<br/>　　苏莫了然，意念一动，芊寻月的棺椁便出现在了殿中。
<br/>　　“紫霄，请你帮我救活她！”绫裳望着棺椁中的芊寻月，声音低沉的说道。
<br/>　　紫霄闻言，来到了棺椁之前，他并未开棺，仔细打量了棺椁内的芊寻月一番。
<br/>　　芊寻月的容颜，让紫霄稍稍惊讶，不过，他还是摇了摇头。
<br/>　　“绫裳，我救不活她！”紫霄神王道。
<br/>　　闻听对方之言，苏莫和绫裳均是脸色一沉，救不活吗？
<br/>　　“为何？”绫裳沉声问道。
<br/>　　“救活她代价太大！”紫霄神王摇了摇头。
<br/>　　绫裳闻言沉默，对方的意思，是有能力救活芊寻月，但是，无法承受巨大的代价。
<br/>　　“紫霄宫主，晚辈请求你出手，只要能救活她，大恩大德晚辈必会报答！”苏莫立刻向紫霄神王抱拳行礼。
<br/>　　既然对方有能力，他一定要求得对方出手，因为，芊寻月不能在等下去了。
<br/>　　“一个藏头露尾之人，有何资格请求本宫主？”紫霄神王脸色一沉，凌厉的目光第一次注视在了苏莫的脸上。
<br/>　　虽然他没有正眼看苏莫一次，但是，凭他的修为，也知晓苏莫隐藏了真实的面容。
<br/>　　只是，因为苏莫是和绫裳一起到来，他不知道苏莫和绫裳的关系，所以才没有理会。
<br/>　　苏莫闻言，立刻抱拳，道：“晚辈秦陨，见过紫霄宫主。”
<br/>　　言语之间，苏莫停止了催动天幻神丹，他的真实容貌，立刻显露而出。
<br/>　　紫霄闻言，沉默不语，少倾之后，才问道：“秦陨，这棺椁内的女子，是你的道侣？”
<br/>　　“是！”苏莫重重的点了点头，他和芊寻月有过夫妻之实，称为道侣也不为过。
<br/>　　绫裳闻听苏莫之言，眸中闪过一丝复杂之色，他虽是绫裳，但也拥有宏青璇的记忆，听闻苏莫的话，难免心中有些触动。
<br/>　　紫霄神王点了点头，继而有摇了摇头，道：“秦陨，你拯救道侣之心，本宫主理解，但是以你的弱小，却是无法报答。”
<br/>　　“这……！”苏莫面色一沉，他暂时的确是无法报答对方，但是，并不代表以后不能报答。
<br/>　　“紫霄，算我求你了，你出手相救，我欠你一个人情！”绫裳开口道，她心中清楚，苏莫是不可能说服紫霄的，唯有她开口请求，才有希望。
<br/>　　听闻绫裳之言，紫霄神王沉默不语，他背负着双手，来回踱步。
<br/>　　少倾，紫霄在主座之上，端坐了下来，目光凝视着绫裳。
<br/>　　“绫裳，此女升级完全断绝，想要救活她，无异于逆天改命，我会找到天地规则的反噬！”紫霄神王面色郑重的说道。
<br/>　　“如何补偿你？我会尽力做到！”绫裳沉声问道，她也希望能救活芊寻月，这也算是了却了一桩心愿。
<br/>　　“补偿，倒也的确是可以！”
<br/>　　紫霄神王点了点头，随即沉声道：“我在中级神王之境，已经停留了五万载，你若是能助我突破境界，此女我必然将她救活！”
<br/>　　“我如何助你？”绫裳闻言皱眉，对方的这个要求，未免太过夸张了，她哪里会有能力助对方突破。
<br/>　　比说现如今，就算是前世巅峰之时，也没有这个能力。
<br/>　　“不，你可以，你如今拥有纯阴之体，若是愿意助我，能让我增加两成的希望！”紫霄神王目光灼灼的望着绫裳，面上露出一丝期待之色。
<br/>　　“你……！”绫裳闻言，顿时心中一震，她明白了，对方是想要和她双修，从而借此冲破境界瓶颈。
<br/>　　绫裳怔住了，呆呆的望着紫霄神王，她和对方认识了数十万年，一同拜在殿主的门下，一起在终极神王殿成长。
<br/>　　数十万年的同门之宜，虽然两人之间，以前也互有好感，但是从未有过半分逾越，只是单纯的兄妹之情。
<br/>　　而现如今，这个她曾经有些崇拜的男人，居然要以此事胁迫她？
<br/>　　苏莫的脸色，亦是瞬间阴沉了下来，这个紫霄神王，居然打的这种注意。
<br/>　　要以此事，胁迫绫裳和其双修。
