        &lt;h3&gt;第三卷第二百五十二章  易倾秋赴宴&lt;/h3&gt;
<br/>
<br/>        易倾秋接到了呼延临庆的请客电话。请易倾秋在周五的晚上一起参加他们几个人的聚餐，地点就定在他们周围的国食天香。
<br/>
<br/>        易倾秋毫不犹豫的答应了。
<br/>
<br/>        易倾秋给陈天天打电话：“天天，周五晚上有没有课？”
<br/>
<br/>        陈天天在西城的建大上学，晚上直接回家，都不用住校的。
<br/>
<br/>        “你要干嘛？”陈天天脆生生的半童音欢快的直击易倾秋的耳膜。
<br/>
<br/>        “带你去参加鸿门宴！”易倾秋笑着说，陈天天那种天不怕地不怕的性格她很喜欢。
<br/>
<br/>        “哇哦，易姐姐，你这是要搞事情的节奏？我爸知道吗？”陈天天一听说“鸿门宴”就来了兴趣。
<br/>
<br/>        “我去你爸爸是知道，你去嘛，你要是想让你爸爸知道你就给他说，不想让他知道我就替你保密。”
<br/>
<br/>        “嘿嘿，易姐姐，你好坏哦，想带坏小孩子，告诉我，你想参加的是一个什么样的鸿门宴需要我给你壮胆。”
<br/>
<br/>        “一帮大灰狼加一只狐狸精，我只是觉得我们俩个人去一定是绝配。”易倾秋诱惑着陈天天。
<br/>
<br/>        “既然如此配，那我就陪你一起去。”陈天天曾经参加过他爸爸公司的年会，对公司里的人比较熟悉，再加上她性格活泼，不拘小节，一张口就啪啦啪啦放鞭炮一样，几次年会上她都是引人注目的对象，对于应酬上的事情她更是见怪不怪了。
<br/>
<br/>        易倾秋没有给陈子霖说她要带着陈天天，她给江俞轩汇报了。
<br/>
<br/>        “江总，呼延临庆邀请我去参加他的聚会，说是夏总也会过去，我想带着陈天天。”
<br/>
<br/>        “嗯，俩个人有个照顾，天天答应了吗？”
<br/>
<br/>        “那是个唯恐天下不乱的性格，我一说鸿门宴她就来了兴趣。”
<br/>
<br/>        “呵，你和她一去就不知道是谁的鸿门宴了。”
<br/>
<br/>        “江总对天天了解的如此透彻啊，怕是关注已久了吧。”易倾秋开了一个玩笑。
<br/>
<br/>        “嗯，陈子霖的女儿啊，我们也是经常见面的，那家伙见面就和我斗嘴，天天天天的唧唧歪歪。话说，有时间不见还停想念她的。”
<br/>
<br/>        “嘶。”易倾秋心里说：“好可怕，春心萌动在一个小孩子的身上，妥妥的渣。”
<br/>
<br/>        “你那是什么眼神？”江俞轩发现易倾秋看他的眼神有些怪。
<br/>
<br/>        “没什么，我就是觉得，人这一生，什么人都会碰到，适合自己的才是最好的，唉，有些人都不知道自己想要什么。”易倾秋叹了口气。
<br/>
<br/>        “你知道你自己想要什么？”江俞轩来了兴趣，前倾着身子问易倾秋。
<br/>
<br/>        “我当然知道我想要什么。首先，我不缺钱，我来这里工作是积累经验，适应社会，再有，我想要找一个能与我匹配，最起码是能说得来，感情纯粹的人相处。人啊，如果不知道自己想要什么就悲催了。”
<br/>
<br/>        “你什么时候变成了灵魂导师？开始指导人生了？”
<br/>
<br/>        “我一直都是这样，不过是你不知道而已。”易倾秋那精致的小脸上露出一丝微笑，带着一丝丝调皮。
<br/>
<br/>        易倾秋到公司两年了，两年来她一直低调的完成自己分内的事情，对公司里的事情也是看在眼里装在心里。
<br/>
<br/>        “呵，富婆，灵魂导师，我们公司还真是藏龙卧虎啊。”
<br/>
<br/>        “那可不，最厉害的还是陈总和江总啊，你们领导的好。”易倾秋刚刚给江俞轩上了一堂课，立马又上了一个洋条。
<br/>
<br/>        “你这张嘴，大概率是死人能被你说活，活人也能被你气死吧？”
<br/>
<br/>        “对人而言，对人而言，江总，你放心我绝对不会把你气死。”易倾秋不想和江俞轩聊下去了，她怕聊下去真的会把江俞轩气死，好歹对方是她的领导。
<br/>
<br/>        “你们去吃饭记得不要多喝酒，注意安全。”江俞轩提醒易倾秋。
<br/>
<br/>        “好，那我走了。”
<br/>
<br/>        易倾秋走后，江俞轩靠在沙发上陷入沉思。
<br/>
<br/>        易倾秋将长发扎成活泼的马尾，穿着短款白色羽绒服，黑色牛仔裤，运动鞋，虽然是冬天也挡不住她的细腿纤腰，显得青春靓丽，少了一丝冷情。
<br/>
<br/>        她和陈天天约好了，在建大门口等陈天天，国食天香离得不远，陈天天穿着淡紫色的羽绒服，淡紫色的套头帽，高领羊绒毛衣，黑色牛仔裤，运动鞋，跑跑跳跳的跑到易倾秋的面前。
<br/>
<br/>        “上车。”易倾秋冲陈天天摇了一下头，示意陈天天上车。
<br/>
<br/>        “易姐姐，我们的裤子是一样的颜色，这叫心有灵犀一点通啊。”陈天天上车就看见了易倾秋的穿着。
<br/>
<br/>        “你还是和你江叔叔心有灵犀一点通吧。”易倾秋脱口而出，说完她就后悔了。
<br/>
<br/>        “关我江叔叔什么事情？咦，你车上放了几个柠檬？有没有切开？切成了几瓣？”陈天天霹雳四连轰让易倾秋都有点招架不住。
<br/>
<br/>        “我才知道你江叔叔为什么那么喜欢和你斗嘴。”
<br/>
<br/>        “为什么啊？”
<br/>
<br/>        “因为喜欢，就像我喜欢你一样的喜欢啊。”
<br/>
<br/>        “啊哈哈哈，是因为你们都斗不过我，但是又很想斗过我。”陈天天开心的大笑。
<br/>
<br/>        “你要是能斗过今天的人，以后我就绝对不会和你斗嘴。”易倾秋专注的开着车。
<br/>
<br/>        “你今天吃饭的都是什么人？”陈天天突然就严肃起来了。
<br/>
<br/>        “以后可能会和我们公司正面冲突，生意场上的事情谁也不知道是什么样子，大家如果都遵循规则就好办事，碰到不遵守规则的就会将市场持续打乱。”
<br/>
<br/>        “明白了。”陈天天沉默不语。
<br/>
<br/>        易倾秋停好车和陈天天手拉着手走进酒店。
<br/>
<br/>        易倾秋推开门看见宋长生和顾恩赐、许弋澄都到了，三个人正在聊天。顾恩赐见到易倾秋微微一愣，他没有想到易倾秋会来，这个小妮子三两句话就将他和呼延临庆的距离拉开了。
<br/>
<br/>        “哇，三位老总这么早，我都迟到了。”
<br/>
<br/>        “美女应该姗姗来迟。来快坐吧，正主没到，你们都不算迟到。”许弋澄笑着招呼易倾秋和陈天天。
<br/>
<br/>        “谢谢许总。”
<br/>
<br/>        宋长生和顾恩赐对着易倾秋点点头算是打了招呼。
<br/>
<br/>        夏颜和呼延临庆姗姗来迟。
<br/>
<br/>        大家见夏颜来了都站起来迎接。
<br/>
<br/>        “抱歉，让大家久等了，下午开会，商讨最近的工作计划，所以来迟了，大家快坐吧。”夏颜笑着和大家一一打招呼。
<br/>
<br/>        “这两个小美女是？”夏颜不认识易倾秋和陈天天。
<br/>
<br/>        “我是陈子昂公司的易倾秋，这个是天天，我们现在是公关部的人。”易倾秋微笑着自我介绍。
<br/>
<br/>        “啊，哦，子昂公司的人啊，来来来，和我坐一起，我还正想了解一下子昂最近的情况呢。”夏颜热情的拉着易倾秋和陈天天坐到了她的旁边。
<br/>
<br/>        “夏总想了解什么，我和易姐姐知无不言言无不尽。”陈天天脆生生的童音响起来，大家都吃了一惊，如果光听声音还以为是个孩子。
<br/>
<br/>        “好好，那等一会饭后留下电话，我们以后好好聊一聊。”夏颜很开心，呼延临庆竟然又挖了陈子昂公司里的俩个人。
<br/>
<br/>        不过这俩个女孩看着心思单纯，估计工作不久，她倒是希望她们一直在陈子昂的公司里能为她所用。
