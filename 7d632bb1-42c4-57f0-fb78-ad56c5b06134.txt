　　要知道那可是武器啊！
<br/>　　冷兵器的锻造不要时间？
<br/>　　每一把兵器制造而出哪怕有着流水线，但在这个末世纪元一切从简的状况中，到底是怎么造出高达五千数量的？
<br/>　　而五百热武器就更不用说了，这会儿段氏基地的人才搞明白，前些天他们送到南署基地，那些看起来古怪无比被放进城墙内部的玩意儿，竟然就是科技部研发的新武器。
<br/>　　无论从哪个方面来看，这都太不科学！
<br/>　　何峰倒是不意外，思索道：“五千新武器只能先分配给狼牙老兵，其余八万人用那些基地提供的武器，我记得南都和川域基地承诺尽可能提供武器的吧？”
<br/>　　鲁初雪从震惊中回过神，点头道：“没错，在南署基地的那五万新兵已经人手一把武器分配好了，你们的三万人武器都放在了分配物资之中，不用担心。”
<br/>　　“那就好。”何峰欣赏的看了鲁初雪一眼，此女能当上中将，其能力果然值得信任，这样一来他们狼牙就能将很多杂事交于段氏基地来处理，专心负责对抗尸潮就好。
<br/>　　紧接着何峰又道：“警戒队的人还没有自己的武器，五千冷兵器给他们，三战队的武器不换。”
<br/>　　三战队的武器要么是一开始就由卢宏盛亲自打造而成的，要么就是楚涵从阴阳谷的制裁之地带出来的，虽然武器各式各样，但三战队的风格多样化，万分巧合的正是这批武器的最佳人选。
<br/>　　尤其是杀羽战队和神隐战队的成员，他们手中的武器很大一部分都来自阴阳谷，除了何峰没有人认出这批武器的来历，毕竟与徐峰用的那柄金枪还是有着一定差距，但也比当下其余的冷兵器要好上很多。
<br/>　　神秘家族用的东西，当然不凡！
<br/>　　将物资的分配都定好之后，何峰也不管还在震惊中的那些人，有些慎重的望向了研究部的人：“那东西研究出来了吗？”
<br/>　　江左严肃的摇头：“还没有，袁熙叶部·长没有传消息过来，就是还在进行中。”
<br/>　　何峰面容严峻起来：“时间可来不及的了啊！其余三种药剂呢？”
<br/>　　江左立即回答：“数量充足，哪怕开打了这里也不会停止调配，只要原材料充足，我们就能力保供应不断。”
<br/>　　“好。”何峰拍了拍江左的肩膀。
<br/>　　与狼牙几个部门的短暂交谈就此结束，何峰等人也将一路上三万多人参与考核后，从考核石壁内取得的东西尽数交给了狼牙各部门，而后便拖着疲惫的身躯在段氏基地休息了一夜，第二天一大早，三万五千人就整装待发，他们穿着干净的新衣服，背着装满了食物的包裹，手中拿着刚刚分配到的武器，就这么义无反顾的踏出段氏基地大门，前往了第一战场南署基地！
<br/>　　留在基地负责后勤的那批人，则是站在了高处目送这批队伍离开，不少人心中都有些沉重。
<br/>　　无论是上一批离开的五万五千人，还是今日离开的三万五千人，这批两支已经被冠上了狼牙标志的军队，都离开的如此静悄悄，一点都不大张旗鼓，甚至当他们的身影消失不见时，还有孩童在熟睡不醒。
<br/>　　他们的步伐踩踏在一起只发出极轻的声响，他们目不斜视的看着前方，面上不带一丝懦弱和悔意，哪怕他们知道此行不一定还能回得来。
<br/>　　一名医护人员轻声感叹：“昨天我帮一个士兵包扎了伤口，本以为只有手上有伤，哪知身上到处都是。”
<br/>　　“我这里的也是。”另一名医护人员道：“我问他这是怎么伤到的，他说过河时总有大石块撞击。”
<br/>　　“还有下山时的滑坡，据说都是直接侧身滚下来的，因为可以节省时间……”
<br/>　　不少人都沉默下来，他们难以想象这些人都经历了什么，如何能将导致浑身是伤的过程说的如此轻松淡然？
<br/>　　——————
<br/>　　三万五千人从段氏基地出发前往南署基地，休息了一夜的他们速度更快，横穿两个地界到达南署基地只用了短短三天。
<br/>　　此时距离两个月的期限，也只剩下了最后三天！
<br/>　　南署基地两大军队首次会面，两批队伍只是相互点了点头，便一声不吭的朝着制定的待命区而去，所有新加入的士兵都有狼牙老兵带领，警戒队的五千人在见到从前线回归的五千人后，相互间有着一种默契，让两批队伍中的新兵都不禁隐隐期待着什么。
<br/>　　整整九万人终于在第一战区集会，这些人让这座改造过后的南署基地变得拥挤起来，但却出乎意料的依然维持着军队肃静。
<br/>　　当参战者全员到达之后，苏行的工程队终于停了下来，不再没日没夜的继续改造，他们站在了高处最后看了眼这座已经完全变样的基地，便集体乘坐直升机离开。
<br/>　　所有狼牙战团的高级军官们，也终于聚集一堂，在最后三天的时间里进行了最彻底的战略会议。
<br/>　　彼此熟悉的人没有什么开场白，于是落座后第一个出声的竟然是话不多的陈少爷：“老大还没来。”
<br/>　　他说的是陈述句，语气和心情一听便知。
<br/>　　“不用太担心，这和分界线战役前的情况差不多。”何峰安慰道。
<br/>　　其余人都点点头，而后·进入了高度认真的状态，包括行程中斗了一路那三个战队队长，也在当下收起了争锋相对。
<br/>　　这时候一人将改造过后的南署基地地图铺开，顿时所有人都目光都忘了过去，要开打首先当然要熟悉战区地理形势，而楚涵大胆的将战区设定在了这座基地，他们当然要利用已知，将一切优势扩到最大。
<br/>　　“这份地图我好像见过啊！”忽的路冰泽惊讶出声，指着地图长大了嘴巴，更是激动道：“对！我见过！但不是这么详细的，是老大手绘的那个版本，而且还是在南署基地原本地图上加以修改的版本。”
<br/>　　听着路冰泽的大叫声，何峰并不意外，解释道：“你看到的是楚涵的初步设计图，他直接在原本地图上加以修改，我们现在看到的是经过苏行改良达成合理化的详细地图，其上每一处地方甚至拐角，都与我们眼下所在的战区完全一致！就是一个缩小版的全面图！”
