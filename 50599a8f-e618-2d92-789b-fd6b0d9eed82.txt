　　　　涂小安跟青衣男子站到了比武场上，随着时间的推移，现场不单单只有天才班的学员，围观的普通班学员也越来越多，可谓是渐渐的人山人海起来。
<br/>　　每一个挑战天才班的人，都会在比武场比试，每一次比赛都是学院的大新闻，大事件。
<br/>　　没人会不来观看这场决斗。
<br/>　　天才班出来的，每个人都是前途无量，不是国家未来的人才，就是各方势力的传人。
<br/>　　况且这次挑战的主角可是公认的最强新生，拥有一头货真价实的超等战宠，众人眼中还浮现清晨最强新生站在仙灵孔雀身上翱翔空中的雄姿。
<br/>　　学员越来越多，现场的嘈杂声也此起彼伏起来。
<br/>　　有意思的是，不单单的地面汇聚了很多学员，即便在半空中，也能看到好几道身影凌空而立，举手投足间流露出高手的姿态。
<br/>　　他们身形平稳，在空中如履平地，好像失去了地吸引力的安排。
<br/>　　“哇...你们看，是天才班前五的大神学长们，好霸气呀”
<br/>　　“东位的是白君然大神，白家的少主，好有男人味，听说这次是他放话给最强新生，说你有本事进超等天才班”
<br/>　　“北位的是扬子杰大神，扬家的继承人，二年前，一进古学院，直接进入超等天才班，因为要跟他比试的那个人，连跟他战斗的勇气都没有，直接投降”
<br/>　　“你们看，南位的是肖何大神，一头的红发，迎风而立，帅到爆裂”
<br/>　　“还有西位的万东大神，也是一表人才，听说他是天才班资历最老的学员，足足待了四年了”
<br/>　　很多女学员在窃窃私语，崇拜的看着上空的学长们。
<br/>　　忽然，男学员也骚动起来，因为站在上空中间位置的是一位风华绝代的大美女，肌肤娇嫩，神态悠闲，美目流盼间眸中冷艳无双。
<br/>　　“大小姐邪七七，唯一一位天才班前五的女生，但听说她的实力已经步入了三阶生灵，什么人都要让她三分”
<br/>　　底下议论纷纷，痴迷的看着那位风姿卓越的女子。
<br/>　　这四男一女都是古学院一等一的高手，他们不屑跟普通学员一样站在地面观看，已东南西北中五个方位凌空而望。
<br/>　　身处半空，俯视而下，才能彰显他们的身份。
<br/>　　没错，他们就是那么的牛逼，那么的与众不同。
<br/>　　“你们说是最强新生胜出，还是天才班学长守门成功”
<br/>　　往年，新生挑战天才班学长成功的几率只有不到三成，但今年不同，这个新生太强，一入学就成为了传奇人物，还有一头超等仙灵孔雀，被视为一个强大隐藏家族的传人，身份非常的神秘。
<br/>　　此时，副院长上去交代了几声，然后下场了，众人知道，比试要开始了，现场的议论声也渐渐的消失了，全神贯注这场比试。
<br/>　　只见最强新生跟天才班学长青衣男子遥遥对立而望。
<br/>　　“在下凌岩航”青衣男子先自我介绍，然后脸上带着一股冷然的看着涂小安，已长辈的姿态道：“你就是这一届的最强新生是吧，大名鼎鼎呀，希望等一下比武开始，你不要让我失望”
<br/>　　涂小安古怪的一笑：“不让你失望，就会让你绝望的”
<br/>　　呃...
<br/>　　凌岩航闻言一楞，这是挑衅啊，旋即冷笑，：“你可知道，我已经打败了好几届的最强新生，他们都来挑战我，想踩着我的肩膀进入天才班，可惜呀，最后都灰溜溜的回到上等班，小师弟，你觉得自己是个例外吗”
<br/>　　小师弟这三个字，咬字咬的特别重。
<br/>　　新生就是新生，在厉害，能厉害到哪里去，唯一让凌岩航忌惮的是，这位新生能跟白君然对拼一招而不落下风。
<br/>　　但那一幕他也没亲眼看到。
<br/>　　“不是例外，是必然”涂小安其实都觉得自己是在欺负他。
<br/>　　可从对方脸上的表情看来，非常的自信，也不知道谁给他的自信。
<br/>　　凌岩航脸上阴郁了下来，好大的口气呀，他虽然在天才班排名最末，但也能轻松的蹂躏上等班的学员。
<br/>　　“小学弟，我到想看看你有多少底气撑起你说这些话”
<br/>　　“废话少说，跟你不熟，开始吧，别耽误我功夫”
<br/>　　对于涂小安来说，这就是走一个过场而已，对方不过一个二十六级的修武者，他一眼就看透了。
<br/>　　清晨之前，涂小安还刚收服了一头三十二级的三阶妖灵，现在面对这些学员，简直提不起什么兴趣。
<br/>　　什么天才班不天才班，忽然，涂小安下巴微抬，看到上空凌空而立的四男一女，一个个看上去不可一世，高人一等似的。
<br/>　　“滚下来，要看下去看，杵在我头上干嘛”
<br/>　　涂小安微微的蹙眉，上空几个人的姿态让他有点反感，其中白君然就在其中。
<br/>　　“小师弟，你还没进入天才班，就这样狂，小心日后没好果子吃”
<br/>　　站在西位空中的一位器宇轩昂的男子冷冷的开口。
<br/>　　“是万东大神，天才班排名第五的高手”
<br/>　　人群中有人开口。
<br/>　　“我们就站在你头顶看，你乃我们何”
<br/>　　白君然也开口了，高高在上的藐视涂小安，敌意好不隐藏。
<br/>　　“小学弟，师姐看好你，加油哦”
<br/>　　涂小安正中间上空，一位风华绝代的美女笑呵呵的朝他开口，露出了猫抓老鼠般的神情。
<br/>　　“你长的不错”
<br/>　　涂小安回了一句过去。
<br/>　　“将你的仙灵孔雀送于我，本小姐答应给你约会一次如何”
<br/>　　“不好意思，我对你没兴趣”
<br/>　　涂小安冷淡无波的一笑，上面的女子长的是非常的漂亮，但是他又不花痴。
<br/>　　美女都吞吃过一个，还是妖精管理局的组长呢。
<br/>　　“什么，最强新生好大的口气，敢对古学院第一大美女说出这样违心的话”
<br/>　　底下的男学员骚动了起来，很多邪七七的粉丝义愤填膺起来。
<br/>　　“你们确定不下来？”
<br/>　　上空五个人凌空不动，就是一副高高在上的样子。
<br/>　　涂小安笑了，扭头看了下方的副院长一眼，传音过去：“老头，你不管管，你要不管，我可就不客气了”
<br/>　　苍白的声音传来：“这几个人，都是各方势力的传人，心高气傲，目空一切，你这条蛇精要是有本事，教训教训他们，老夫没意见”
<br/>　　“好，是你说的”
<br/>　　涂小安懂了，这老头既然都这样说了，那么他就不用有丝毫的忌惮。
<br/>　　如今的他，不像之前，做什么事情，都要唯唯若若，思前顾后。
<br/>　　说句装逼的话，他现在光三阶妖灵都有二只，要是召唤过来，能将古学院闹的天翻地覆。
