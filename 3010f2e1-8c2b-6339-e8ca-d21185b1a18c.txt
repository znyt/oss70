　　时间飞快，眨眼间，距离紫霄宫鸿钧第二次讲道不过百年时间。
<br/>　　在这一天，玉虚宫门大开，元始从中走了出来。经过近三千年时间的炼化，再加上元始本身修为就比太一要高得多，故而，混沌钟已经被他炼化道三十五道先天禁制了，已经可以堪比极品先天灵宝了。
<br/>　　但是，越往后，后面的禁制就越难炼化，因为不仅后面的先天禁制更加复杂深奥，而且炼化需要的修为也必须更高。所以，想要炼化后面的，必须花费更多时间，而他如今最差的就是时间。
<br/>　　是故，炼化灵宝这种事，还是等他修为更高了再来炼化，就如今而言，它也足够元始用的了。
<br/>　　在元始走出玉虚宫的同时，老子与通天同样走出太清宫与上清宫。元始见到两人，便向两人问道：
<br/>　　“如何？”
<br/>　　“大兄，吾之境界没有提升，但道行却是提高了许多，法力更加精纯了。贫道还在此基础上创出《太清道德经》。至于灵宝，并没有炼化，只是简单祭炼了一番，但威力也绝对不俗。”
<br/>　　“吾亦一样，吾创出《上清灵宝经》，更是在炼化昆吾剑之时，领悟了几招剑诀，有时间还要找大兄，我们切磋一番。”通天还是一如既往的好战。
<br/>　　“好，你们都有所得就好，但你们也不要为此骄傲，须知人外有人。紫霄宫再启，道祖鸿钧第二次讲道在即，我等也应该出发了。”
<br/>　　元始对着两人说道。
<br/>　　“善！”
<br/>　　“大善！”
<br/>　　两人口中同时称善，之后，三人不再犹豫，出了昆仑，向着三十三天外的紫霄宫而去。
<br/>　　正所谓一回生，二回熟，第一次境紫霄宫还需要道混沌之中慢慢寻找，但此次进紫霄宫，刚入混沌不久，脚下便出现一道金桥，三人毫不犹豫地踏上金桥。
<br/>　　一下金桥，便到了紫霄宫外，门外却站着两个童子，正是昊天与瑶池。
<br/>　　两童子见元始三清走来，顿时向着三人走来，齐齐拱手作揖，道：“见过三位真人。”
<br/>　　三人看着昊天与瑶池，略微点头示意，之后便迈步走进紫霄宫。
<br/>　　紫霄宫中早已有人，但大多数人都在闭目不语，好似在修炼道法，不敢大声喧哗。元始抬头看了一眼，其中大多数人都是第一次来听讲之人，但也有一些新来的。而且，第一次来听讲道之人，亦有一些没有来。元始不用想也知道，这些人恐怕已经没命了。
<br/>　　洪荒之中，争斗是不息的主题，洪荒每日都有争斗。这些人连听讲道这样的大事都不来，怕是已经没命来了。
<br/>　　而且，元始还注意到，此次后土与玄冥两位祖巫也没有来，他也没有多管，两女显然是不会就此陨落的，她们现在不来，想来是因为听不懂道祖所讲，故而没来浪费时间吧。
<br/>　　见元始三人的到来，众人睁开眼睛，看了三人一眼，不知道的人在向周边人询问，知道了稍微点了点头，向三人示好。
<br/>　　其中，帝俊、太一等人，看见三人，顿时眼中闪过一丝愤怒以及仇恨，但在这紫霄宫中，他们还是隐忍了起来，没人敢在此地动手。
<br/>　　元始面不改色，一脸贵气，不急不缓地走到最前方的云台之下，第一个蒲团之前，坐了下来，看都不看周围人一眼，便闭目不语。
<br/>　　老子面无表情，与元始一样，坐到第二个蒲团上，闭目沉思。
<br/>　　通天一脸桀骜，眼神坚毅，两条眉毛翘起，宛若宝剑出鞘，让人感到一股凌厉之气扑面而来。他来到第三个蒲团面前，看了众人一眼，特别是在妖族众人面前多留意了几眼，之后盘腿坐下，闭目不语。
<br/>　　“哼！”
<br/>　　女娲见此，冷哼一声，表示自己的不满，上一次被三人围攻击败，她心中一直耿耿于怀。此次见到三清，特别是三清对她的无视，更是令她火气直冒，快有些控制不住自己的情绪了，还是他后面的伏羲及时拉着了她，她才不至于站起来干架。
<br/>　　元始听到她的冷哼声，睁开眼睛，淡漠无情的眼光在她身上扫了一遍，之后才收回目光，闭目不语，静等道祖的前来。
<br/>　　女娲被这冷漠的眼光吓一大跳，之后便更加气恼了，这是什么意思，威胁吗，还是看不起自己？她就想要发作，但想到之前元始的战力，顿时不再言语，也一样闭上双眼，沉默不语，只是她的脸色更加难看了。
<br/>　　随着镇元子、红云、鲲鹏等一众修士前来，坐在三清身后，最后，西方的接引、准提两位大神来临，坐到最后两个蒲团之上。
<br/>　　不多时，两个童子童女昊天与瑶池关闭宫门，一同走到云台前，对着众修士大声道：
<br/>　　“老师第二次讲道即将开始，众人休得喧哗！”
<br/>　　有些正在交谈的众人，听闻昊天与瑶池的声音，纷纷安静下来，一时间紫霄宫中，四下一片寂静，之前正在闭目深思的人也纷纷睁开双眼，看向前方云台之处。
<br/>　　就在此时，一声钟鸣，众人却见一道苍老的道者身影，如梦幻虚无之中走出，缓缓浮现在道台之上。
<br/>　　“拜见圣人！”
<br/>　　如梦似幻，鸿钧道祖身影浮现，恢宏圣人之威，宛若天道凌驾，众生臣服，一种修士纷纷恭声开口，齐齐行礼参拜。
<br/>　　元始瞳孔微缩，心里更是惊骇，因为他发现这鸿钧的修为至少已经圣人三重天巅峰了，就快要突破圣人中期了。这鸿钧的修炼速度真快，从成圣道此时，不过一万三千多年，就已经快要圣人中期了，简直令人难以置信。按理说到了圣人境界，修为提升应该更慢了才是，这鸿钧明显是不合理。
<br/>　　但，元始知道这是真的，而且，他发现此时的鸿钧变得比之前更加高深莫测了，他的眼神比之前更加淡漠无情，他身上的天道气息更加浓郁了。
<br/>　　元始猜测，他能做到如此，想必是借助天道之力了。他的造化玉蝶有沟通天道之效，他借助天道之力修炼，也不是不可能。
<br/>　　“尔等坐好，听贫道讲道。”鸿钧见众人都来了，便毫不废话，直接开口讲道：
<br/>　　“夫万物以人为主，人以心为宗，无主则法不生，无心则身不立。心法多门，妙用非一，有无二体。……”
<br/>　　“但天下之道，疏通而同归，百虑而一致，从厅入妙，权实则二阶，总迩符真，是非同一乎……”
<br/>　　……
<br/>　　鸿钧讲道，一如既往地天地同庆。金花乱坠，地涌金莲，星辰耀天，龙飞凤舞，仙音袅袅，道门显现……各种异象，尽皆显现。
<br/>　　大道法则交织，道则奥妙显现，玄之又玄。众修士很快就沉浸在大道的美妙之中，不可自拔。
<br/>　　……
<br/>(本章完)
